# 初级 48 课语法索引

> 来源：标日初级上册+下册 OCR 电子稿，仅提取语法点名称和一句话描述。
> 用途：作为大纲编写和词汇分配的基础，后续逐条经 Bunpro 校验。

## 第1課：李さんは 中国人です

1. 名は 名です — 肯定判断（～是～）
2. 名は 名では ありません — 否定判断（～不是～）
3. 名は 名ですか — 疑问句及应答（はい/いいえ）
4. 名の 名 — 所属/从属关系（～的～）

## 第2課：これは 本です

1. これ/それ/あれ は 名です — 指示事物的近/中/远称代词
2. だれですか / 何ですか — 疑问词问人、问事物
3. 名の 名 — 助词の表示所属（扩展用法）
4. この/その/あの + 名 — 指示连体词修饰名词
5. どれ/どの + 名 — 三者以上的不确定疑问词

## 第3課：ここは デパートです

1. ここ/そこ/あそこ は 名です — 指示场所的近/中/远称代词
2. 名は 名[场所]です — 某事物存在于某场所
3. 名は どこですか — 询问存在的场所
4. 名も 名です — 助词も表示"也"
5. 名は 名ですか、名ですか — 选择疑问句

## 第4課：部屋に 机と いすが あります

1. あります / います — 无意志事物用あります，有意志的用います
2. 场所に 名が あります/います — 存在句（～有～）
3. 名は 场所に あります/います — 位置句（～在～）
4. 名と 名 — 助词と并列两个名词（和）
5. 上/下/前/後ろ/隣/中/外 — 方位词
6. ね — 句尾助词，征求同意
7. 疑问词 + も + 否定 — 全面否定（谁也没有/什么也没有）

## 第5課：森さんは 7時に 起きます

1. 今～時～分です — 表示时间
2. ～ます/～ません/～ました/～ませんでした — 动词礼貌体四种形式
3. 名[时间]に + 动 — 助词に表示动作发生的时间点
4. 名[时间]から 名[时间]まで + 动 — 时间范围（从～到～）
5. いつ～ますか — 用いつ询问时间
6. は 的对比用法 — は暗示对比（其他情况不同）

## 第6課：吉田さんは 来月 中国へ 行きます

1. 名[场所]へ + 动 — 助词へ表示目的地（读作え）
2. 名[场所]から + 动 — 助词から表示移动起点
3. 名[人]と + 动 — 助词と表示共同做某事的对象
4. 名[交通工具]で + 动 — 助词で表示交通手段
5. 名[场所]から 名[场所]まで + 动 — 移动范围的起点和终点
6. に/で/へ/から/まで/と + は — 复合助词，表示对比

## 第7課：李さんは 毎日 コーヒーを 飲みます

1. 名を + 动 — 助词を表示动作对象（读作お）
2. 名[场所]で + 动 — 助词で表示动作发生的场所
3. 名か 名 — 助词か连接名词，表示"或者"
4. 名を ください — 买东西/点菜/索取某物

## 第8課：李さんは 日本語で 手紙を 書きます

1. 名[工具]で + 动 — 助词で表示手段及原材料
2. 名1は 名2に 名3を あげます — 给予（に标记接受者）
3. 名1は 名2に/から 名3を もらいます — 接受（に/から标记赠送者）
4. 名[人]に 会います — 见某人（所见对象用に）
5. よ — 句尾助词，提醒对方注意
6. もう — 表示"已经"

## 第9課：四川料理は 辛いです

1. 名は 一类形です — 一类形容词做谓语
2. 一类形做谓语的否定 — い→くないです/くありません
3. 一类形做谓语的过去 — い→かったです；否定：くなかったです
4. 一类形 + 名 — 一类形容词直接修饰名词
5. を → は — 话题化或对比时，を替换为は
6. あまり + 否定 — 程度不高（不太～）
7. 程度副词① — とても/少し/あまり/全然
8. 常用一类形容词一览

## 第10課：京都の 紅葉は 有名です

1. 名は 二类形です/でした — 二类形容词做谓语
2. 二类形 では ありません/では ありませんでした — 二类形否定及过去否定
3. 二类形な + 名 — 二类形容词修饰名词时加な
4. 名 でした/では ありません/では ありませんでした — 名词谓语的过去和否定
5. どんな + 名 — 询问人或事物的性质
6. どうですか/どうでしたか — 询问意见或感想
7. でも / そして — 转折连词 / 并列连词
8. 常用二类形容词一览

## 第11課：小野さんは 歌が 好きです

1. 名1は 名2が 一类形/二类形です — が标记情感对象，表达好恶
2. 名1は 名2が 分かります/できます — が标记能力对象
3. 名や 名 — 助词や列举部分代表性项目（～啦～什么的）
4. ～から / だから — から接原因小句；だから是原因连词
5. 频率副词 — いつも/よく/時々/たまに/あまり/全然
6. どうしてですか — 询问原因，回答用～から

## 第12課：李さんは 森さんより 若いです

1. 名1は 名2より 一类形/二类形です — 比较句（名1比名2更～）
2. 名1より 名2のほうが 一类形/二类形です — 比较句（名2比名1更～）
3. 名1は 名2ほど 一类形くないです — 否定比较（名1不如名2那么～）
4. 名の中で 名が いちばん 一类形/二类形です — 最高级
5. 名1と 名2と どちらが 一类形/二类形ですか — 两者比较疑问
6. どの名/いつ/どれ/だれ/何が いちばん… — 三者以上的最高级疑问

## 第13課：机の上に 本が 3冊 あります

1. 名[数量] + 动 — 数量词放在动词前面
2. 名[时间] + 动 — 时间数量词说明持续时间（不加に）
3. 名[时间]に 名[次数] + 动 — 一定时间内进行若干次
4. 动词的ます形 — ます形及去掉ます后的形式
5. 名[场所]へ 动に 行きます/来ます — 移动行为的目的
6. 名[数量] + で — 计价方式（几个多少钱）

## 第14課：昨日 デパートへ 行って、買い物しました

1. 动词的て形 — 三类动词变换为て形的规则
2. 动て 动 — て形连接，表示动作按时间顺序发生
3. 动てから 动 — 做完前一动作后再做后一动作
4. 动て ください — 请求某人做某事
5. 名[场所]を 动 — を表示经过或离开的场所

## 第15課：小野さんは 今 新聞を 読んで います

1. 动ています① — 动作正在进行
2. 动てもいいです — 表示许可
3. 动てはいけません — 表示禁止
4. 名に 动（附着点） — に表示人或物体的附着点
5. 名に 动（目的地） — 移动目的地可用へ也可用に

## 第16課：ホテルの 部屋は 広くて 明るいです

1. 一类形くて 一类形 — 一类形容词并列（い→くて）
2. 二类形で 二类形 — 二类形容词并列
3. 名で 名 — 名词并列
4. 动ています② — 动作结束后留下的结果状态
5. 小句が、小句 — が连接两个小句，表示转折
6. まだ + 动（否定） — 还没有～

## 第17課：わたしは 新しい 洋服が 欲しいです

1. 名が 欲しいです — 想要某物
2. 名を 动たいです — 想做某事
3. 动ませんか — 否定疑问提议（～好吗？）
4. 动ましょう — 提议（～吧）
5. 疑问词 + でも — 任何情况都相同（什么都～/谁都～）
6. ね②（缓和语气） — 舒缓地向对方传达想法

## 第18課：携帯電話は とても 小さく なりました

1. 一类形く なります — 性质状态的变化（变得～）
2. 一类形く します — 意志性动作引起变化（把～弄成～）
3. 二类形/名 に なります — 二类形/名词的状态变化
4. 二类形/名 に します — 二类形/名词的意志性变化
5. 一类形/二类形 ほうが いいです — 表示选择或建议
6. 自動詞と他動詞① — 自动词与他动词的区分

## 第19課：部屋の かぎを 忘れないで ください

1. 动词的ない形 — 动词否定变形规则
2. 动 ないで ください — 请不要～
3. 动 なければ なりません — 必须～
4. 动 なくても いいです — 不～也行
5. 名1が 名2です — 新信息做主语时用が（强调主语）

## 第20課：スミスさんは ピアノを 弾く ことが できます

1. 动词的基本形 — 辞书形的变换规则
2. 名は 动（基本形）ことが できます — 表示能力（会～/能～）
3. 名は 动（基本形）ことです — 谓语内容为动作行为
4. 动（基本形）前に、～ — ～之前
5. 疑问词 + か — 疑问词加か表示不确定（某时/某地/某人）
6. ～よね — 征求对方同意

## 第21課：わたしは すき焼きを 食べた ことが あります

1. 动词的た形 — 过去式变形规则（て→た）
2. 动（た形）ことが あります — 过去的经历（曾经～过）
3. 动（た形）後で、～ — ～之后
4. 动（た形）ほうが いいです — 建议（还是～比较好）
5. 动ましょうか — 提议为对方做某事（我来～好吗？）

## 第22課：森さんは 毎晩 テレビを 見る

1. 敬体形和简体形 — 根据场合选用ます/です（敬体）或简体
2. 动词的简体形 — 基本形/ない形/た形/なかった形
3. 一类形容词谓语的简体形 — 四种简体形式
4. 二类形容词谓语的简体形 — 四种简体形式
5. 名词谓语的简体形 — 四种简体形式
6. 小句 けど、小句 — けど表示转折或铺垫（口语）

## 第23課：休みの 日、散歩したり 買い物に 行ったり します

1. 动たり 动たり します — 列举若干种代表性动作（有时～有时～）
2. 一类形かったり/二类形だったり/名だったりです — 状态的多种可能
3. 小句 + か — 不确定内容（～かどうか / 疑问词小句+か）

## 第24課：李さんは もうすぐ 来ると 思います

1. 小句（简体形）と 思います — 说话人的想法（我觉得～）
2. 名[人]は 小句（简体形）と 言いました — 转述他人的话（某人说～）
3. ～のです/んです — 说明状况或解释原因
4. 小句が、小句（铺垫） — が用于引出接下来的内容

# 初级下册（第25課～第48課）

## 第25課：これは明日会議で使う資料です

1. 小句（动词简体形）+ 名词 — 动词小句修饰名词（定语从句）
2. 小句（动词简体形）+ 名词 + は + 名词/形容词 + です — 修饰名词短语做主语
3. 小句（动词简体形）+ 名词 + を/に/から + 动词 — 修饰名词短语做宾语等
4. 小句（一类形/二类形/名）+ 名词 — 形容词和名词修饰名词的接续方式

## 第26課：自転車に2人で乗るのは危ないです

1. 小句（动词简体形）+ のは + 形容词です — 动词小句名词化做主语
2. 小句（动词简体形）+ のを + 动词 — 动词小句名词化做宾语
3. 小句（简体形）+ でしょう — 不确定事项的推测
4. 小句（简体形）+ かもしれません — 约50%可能性的推量

## 第27課：子供の時,大きな地震がありました

1. 小句（简体形）+ 時 — 时间定语从句
2. 动词ます形 + ながら — 同一主体同时进行两个动作
3. 小句（简体形）+ でしょう? — 向对方确认信息
4. 动词て形 + います③ — 反复性或习惯性动作
5. 名词 + で — で表示原因、理由
6. 名词 + と + 动词 — と表示互动对方（结婚、吵架等）

## 第28課：馬さんはわたしに地図をくれました

1. 名1は 名2に 名3を くれます — 别人给说话人一方某物
2. 动词て形 + あげます — 为别人做某事
3. 动词て形 + もらいます — 请别人为自己做某事
4. 动词て形 + くれます — 别人为说话人做某事

## 第29課：電気を消せ

1. 动词的命令形 — 命令表达的变形规则
2. ～なさい — 比命令形稍客气的要求
3. 动词基本形 + な — 禁止对方做某事
4. ～て / ～ないで — 日常亲密关系中的祈使
5. 名1 + という + 名2 — 提供新名称信息
6. 名1は～ / 小句 + という + 名2です — 给名词下定义

## 第30課：もう11時だから寝よう

1. 动词的意志形 — 当场表示意志或提议的变形规则
2. 动词意志形 + と思います — 温和表示自己要做某事
3. 动词意志形 + と思っています — 持续一段时间的意志或打算
4. 小句1 + ので + 小句2 — 原因理由（比から更礼貌）

## 第31課：このボタンを押すと,電源が入ります

1. 小句1（基本形/ない形）と, 小句2 — 条件表达（恒常/真理/习惯/发现）
2. 动词基本形/ない形 + ことがあります — 偶尔发生某事
3. 形容词的副词性用法 — 一类形い→く、二类形+に 修饰动词
4. 小句（简体形）+ でしょうか — 礼貌程度较高的疑问

## 第32課：今度の日曜日に遊園地へ行くつもりです

1. 动词基本形/ない形 + つもりです — 事先形成的意志、打算
2. 动词基本形/ない形 + ことにします — 说话人自己决定实施某行为
3. 动词基本形/ない形 + ことになりました — 因外在原因形成的决定
4. 小句（简体形）+ そうです — 传闻表达

## 第33課：電車が急に止まりました

1. 自动词和他动词② — 词形对应的自他动词
2. 自动词 + ています④ — 自动词变化后的结果状态
3. 动词て形 + てしまいます — 结果令人不快或动作完成
4. ～そうです②（样态/推测） — 根据外观推测事态发展

## 第34課：壁にカレンダーが掛けてあります

1. 他动词 + てあります — 有意进行的动作结果的存续
2. 动词て形 + ておきます — 为准备而有意保留动作结果
3. 动词て形 + てみます — 尝试做某事
4. 小句（基本形）ために / 名詞のために — 表示目的

## 第35課：明日雨が降ったら,マラソン大会は中止です

1. 小句1 たら, 小句2 — 假定条件（如果…就…）
2. 小句1 ても, 小句2 — 即使…也…
3. 名詞 + だけ — 限定（只有…）
4. 名詞 + しか + 否定 — 带主观语气的限定（只…）
5. 名詞 + でも（极端例） — 提示极端例子（连…也…）

## 第36課：遅くなって,すみません

1. 小句1 て/で, 小句2（原因理由） — て形连接表示原因
2. 名詞 + に（用途/基准） — に表示用途或评价基准
3. 动词基本形 + のに（用途/基准） — 动词+のに表示用途或基准
4. 名詞 ばかり / 动词て形 ばかりいます — 总是做同一件事（光是…）
5. 小句（动词简体形）のが + 見えます/聞こえます — 看到/听到的对象

## 第37課：優勝すれば,オリンピックに出場することができます

1. ば形 — 动词/一类形的假定条件变形规则
2. 小句1 ば, 小句2 — ば形假定条件（如果…就…）
3. 小句1（简体形）なら, 小句2 — 根据对方言行提出建议
4. 名詞 + でも — 举出代表性选项进行提议
5. 名1 とか 名2 とか — 列举同类性质的若干例子

## 第38課：戴さんは英語が話せます

1. 可能形式 — 表示能够进行某动作的动词变形
2. 小句（基本形/ない形）ように, 小句 — 为了使某种状态成立
3. 小句（基本形/ない形）ようになります — 能力/状况/习惯变成新状态
4. 小句（基本形/ない形）ようにします — 努力使某行为成为习惯

## 第39課：眼鏡をかけて本を読みます

1. 小句1 て / ないで + 小句2 — て形/ないで表示伴随状态
2. 小句1（简体形）ために, 小句2（原因） — ために表示非意志性原因
3. 动词て形 + いきます/きます — 移动方向/相继动作/伴随状态

## 第40課：これから友達と食事に行くところです

1. 动词 + ところです — 动作即将/正在/刚刚的阶段
2. 动词た形 + ばかりです — 动作刚刚结束
3. 动词 + 始めます/出します — 动作的开始
4. 动词 + 続けます — 动作的持续
5. 动词 + 終わります — 动作的结束

## 第41課：李さんは部長にほめられました

1. 被动形式 — 动词被动变形规则
2. ～は（～に）～（ら）れます — 基本被动句
3. ～は～に～を～（ら）れます — 物主被动句（拥有者遭受损失）
4. ～は～に～（ら）れます — 间接受害被动句
5. ～が/は～（ら）れます — 事物做主语的被动句
6. ～は～によって～（ら）れます — によって提示特定动作主体

## 第42課：テレビをつけたまま,出かけてしまいました

1. 小句（た形/ない形）まま, 小句2 — 保持理应改变的状态做另一动作
2. 小句1（简体形）のに, 小句2 — 转折（实际不符合常理）
3. 小句（简体形）はずです — 有把握的推断
4. 小句（简体形）はずがありません — 否定性推断

## 第43課：陳さんは,息子をアメリカに留学させます

1. 使役形式 — 动词使役变形规则
2. ～は～を（自动）（さ）せます — 自动词使役句
3. ～は～に～を（他动）（さ）せます — 他动词使役句
4. ～（さ）せてください — 请求允许自己做某事
5. ～やすいです/にくいです — 易于/难于做某事

## 第44課：玄関のところにだれかいるようです

1. 小句（简体形）ようです — 根据感知推测
2. 小句（简体形）みたいです — 推测（口语化）
3. 小句（简体形）らしいです — 观察推测或婉转传闻
4. ～過ぎます — 超过正常程度（太…了）
5. 一类形 + さ — い→さ 构成表示状态的名词

## 第45課：少子化が進んで,日本の人口はだんだん減っていくでしょう

1. ～ていきます/きました — 从基准时间观察的持续或变化
2. ～てきました — 某种状态的开始或出现
3. 小句1し, 小句2し, 小句3 — 并列叙述多个原因
4. 动/一类形 ば…ほど — 越…越…
5. で/へ/から/まで/と + の — 这些助词可后续の修饰名词

## 第46課：これは柔らかくて,まるで本物の毛皮のようです

1. 名+の / 动（简体形）ようです/みたいです②（比喻） — 用于比喻
2. 名1 + らしい + 名2（典型性） — 具有典型特征
3. 名（时间）までに — 截止期限
4. 名+の / 动（简体形）+ 間/間に — 持续时间段/事态发生的时间范围
5. ～のような味/においがします — 感觉到某种味道、气味、声音

## 第47課：周先生は明日日本へ行かれます

1. 敬语概述 — 尊他语、自谦语、礼貌语三大分类
2. 尊他语 — 通过抬高对方表示敬意
3. ～（ら）れます — 被动形式兼用作尊他语
4. お + 动词 + になります — 尊他语（较深敬意）
5. お + 动词 + ください / ご + 三类动 + ください — 尊他语劝说
6. 尊他语特殊形式 — 召し上がる/いらっしゃる/おっしゃる等

## 第48課：お荷物は私がお持ちします

1. 自谦语概述 — 通过压低自己表示敬意
2. お + 动词 + します / ご + 三类动 + します — 常用自谦语形式
3. 自谦语特殊形式 — 拝見する/いただく/参る/伺う/いたす等
4. ～ていただけますか — 礼貌请求对方做某事
5. ～（さ）せていただきます — 高度自谦地表示做某事
6. ございます / ～でございます — あります/です的更礼貌说法
