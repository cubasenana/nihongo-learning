## 语法点

### 1. 动词的「た形」规则

- **含义**：动词过去/完成的基础变形
- **接续**：把「て形」的「て」→「た」、「で」→「だ」
- **要点**：
  - 变形规律与て形完全对应：書いて→書いた、読んで→読んだ
  - 二类动词：食べて→食べた、見て→見た
  - 三类动词：して→した、来て（きて）→来た（きた）
- **例句**：
  - 書きます→書いた、飲みます→飲んだ、食べます→食べた
  - 昨日、映画を見た。（昨天看了电影。）
- **Bunpro 校验**：待验证

### 2. 動（た形）ことがあります

- **含义**：曾经做过……（过去的经历）
- **接续**：动词た形 + ことがあります
- **要点**：
  - 表示曾经有过某经历，通常不用于刚发生且时间明确的普通过去事实
  - 否定：～たことがありません
  - 强调从未：一度も＋～たことがありません
- **例句**：
  - 北海道へ行ったことがあります。（去过北海道。）
  - 日本料理を作ったことがありますか。——いいえ、一度もありません。（做过日本料理吗？——不，一次也没有。）
- **Bunpro 校验**：待验证

### 3. 動（た形）後で、～

- **含义**：做完……之后（时间先后）
- **接续**：动词た形 + 後で / 名词 + の + 後で
- **要点**：
  - 「後で」前面的动词必须是た形
  - 名词接续时加「の」：仕事の後で
  - 对比「～前に」：前に用基本形，後で用た形
- **例句**：
  - ご飯を食べた後で、散歩します。（吃完饭后去散步。）
  - 授業の後で、図書館へ行きます。（下课后去图书馆。）
- **Bunpro 校验**：待验证

### 4. 動（た形）ほうがいいです

- **含义**：还是做……比较好（建议）
- **接续**：动词た形 + ほうがいいです / 动词ない形 + ほうがいいです
- **要点**：
  - 建议做某事：た形＋ほうがいい
  - 建议不做某事：ない形＋ほうがいい
  - 常加「よ」表示亲切提醒或加强建议语气
- **例句**：
  - 風邪ですから、今日は早く寝たほうがいいですよ。（感冒了，今天还是早点睡比较好。）
  - あまりお酒を飲まないほうがいいですよ。（还是别喝太多酒好。）
- **Bunpro 校验**：待验证

### 5. 動ましょうか（提议）

- **含义**：我来……好吗？/……好吗？（主动提议为对方做事）
- **接续**：动词ます形去「ます」+ ましょうか
- **要点**：
  - 可提议为对方做事，也可提议共同做事
  - 接受：はい、お願いします / はい、そうしましょう
  - 拒绝：いいえ、大丈夫です / いいえ、結構です
- **例句**：
  - 荷物を持ちましょうか。——はい、お願いします。（我来帮你拿行李吧？——好的，麻烦你了。）
  - 窓を開けましょうか。——いいえ、大丈夫です。（开窗好吗？——不用了，没关系。）
- **Bunpro 校验**：待验证

## 课文

> Arc 6「告别东京」第1话 — 歌舞伎之夜

**登場人物**：陳明、田中 — 帰りの電車

> 使用语法：≤L21（+た形、動たことがあります、動た後で、動たほうがいいです、動ましょうか）

---

田中：陳さん、歌舞伎を 見た ことが ありますか。

陳：いいえ、一度も ありません。

田中：じゃあ、今度 一緒に 見に 行きましょうか。

陳：いいですね。チケットは もう 買いましたか。

田中：はい、買いました。公演は 土曜日です。

陳：見た 後で、どこかで 食事を しませんか。

田中：いいですね。駅の 近くに いい レストランが ありますよ。

陳：何を 着て 行った ほうが いいですか。

田中：普通の 服で 大丈夫ですよ。

---

**语法覆盖**：✅ た形 ✅ たことがあります ✅ た後で ✅ たほうがいいです ✅ ましょうか

## 表达要点

| 要点 | 说明 |
|------|------|
| そのほうがいいです | 口语中用「その」指代对方提议的内容表示赞同 |
| 何度か | 「疑問詞+量詞+か」表示不确定的数量："去过几次" |
| ～過ぎ | 接时间或年龄表示"过了……"：9時過ぎ、30過ぎ |
| 連休 | 连续的休假，最长的是四月末至五月初的ゴールデンウィーク |
