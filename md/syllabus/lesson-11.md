## 语法点

### 1. 名1は 名2が 好きです / 嫌いです

- **含义**：表达喜好或厌恶的对象
- **接续**：名1（主体）は + 名2（对象）が + 好きです / 嫌いです / 怖いです
- **要点**：
  - 名1 为感情主体，名2 为情感对象
  - 对象用「が」标记（N5/N4 阶段固定用 が；を 在某些口语场景中也可见，后续再学）
  - 同类表达：怖いです（害怕）
- **例句**：
  - 私は音楽が好きです。（我喜欢音乐。）
  - 弟は野菜が嫌いです。（弟弟讨厌蔬菜。）
- **Bunpro 校验**：待验证

### 2. 名1は 名2が 分かります / できます

- **含义**：表达能力或理解的对象
- **接续**：名1（主体）は + 名2（对象）が + 分かります / できます / 上手です / 下手です
- **要点**：
  - 能力对象用「が」标记
  - 分かります＝理解、懂得
  - できます＝会、能做到
  - 上手です/下手です＝擅长/不擅长
- **例句**：
  - 姉はフランス語が分かります。（姐姐懂法语。）
  - 私は料理ができません。（我不会做饭。）
- **Bunpro 校验**：待验证

### 3. 名 や 名（列举）

- **含义**：列举代表性的事物，"……啦……什么的"
- **接续**：名词A + や + 名词B（+ など）
- **要点**：
  - 「や」只列举部分代表性事物，暗示还有其他
  - 「と」则是全部列举
  - 常搭配「など」使用，构成「～や～など」
- **例句**：
  - 週末は映画や買い物などをします。（周末看电影、购物什么的。）
  - 冷蔵庫に牛乳やジュースがあります。（冰箱里有牛奶和果汁等。）
- **Bunpro 校验**：待验证

### 4. ～から / だから（原因、理由）

- **含义**：说明原因理由
- **接续**：小句（原因）+ から、小句（结果）/ だから + 小句（结果）
- **要点**：
  - 「から」接在原因小句后面，构成原因从句
  - 可以先说原因再说结果，也可以先说结果再补原因
  - 「だから/ですから」是连词，独立放在结果句前面
  - **名词/二类形做谓语时，から 前面必须加 だ（简体）或 です（敬体）**：× 試験から → ○ 試験だから / 試験ですから
- **例句**：
  - 明日は試験ですから、今晩勉強します。（明天有考试，所以今晚学习。）
  - お腹が痛いです。だから、何も食べません。（肚子疼。所以什么也不吃。）
- **Bunpro 校验**：待验证

### 5. 频率副词

- **含义**：表示动作发生频率的副词体系
- **接续**：频率副词 + 动词
- **要点**：
  - 从高到低：いつも ＞ よく ＞ 時々 ＞ たまに ＞ あまり〜ない ＞ 全然〜ない
  - 「あまり」「全然」后接否定形式
  - 可与时间词搭配使用
- **例句**：
  - 父はいつも朝6時に起きます。（父亲总是早上6点起床。）
  - 私はたまに美術館へ行きます。（我偶尔去美术馆。）
- **Bunpro 校验**：待验证

### 6. どうしてですか

- **含义**：询问原因理由，相当于"为什么"
- **接续**：どうしてですか / どうして + 小句 + か
- **要点**：
  - 回答时在谓语后面加「から」或用「～からです」结尾
  - 「だから」是连接结果句的连词，不用于直接回答 どうして（× だから、好きです → ○ 好きだからです）
  - 语气比「なぜ」柔和，日常对话常用
- **例句**：
  - どうして日本語を勉強しますか。——日本の文化が好きですから。（为什么学日语？——因为喜欢日本文化。）
  - どうして昨日休みましたか。——熱がありましたから。（为什么昨天请假了？——因为发烧了。）
- **Bunpro 校验**：待验证

## 课文

> Arc 3「社交展开」第3话 — 爱好交流

**登場人物**：陳明、田中、佐藤 — 金曜日の昼休み

> 使用语法：≤L11（+Nが好き/嫌い、Nが分かります/できます、NやN、から/だから、頻率副詞、どうして）

---

田中：陳さんは 音楽が 好きですか。

陳：はい、とても 好きです。ギターや ピアノが 好きです。

佐藤：陳さんは ギターが できますか。

陳：はい、少し できます。佐藤さんは？

佐藤：私は 音楽が あまり 分かりません。スポーツが 好きです。魚が 嫌いです。

田中：どうしてですか。

佐藤：匂いが 好きではありませんから。

陳：私は 時々 映画を 見ます。田中さんは よく 何を しますか。

田中：私は いつも 本を 読みます。料理も 好きですから、週末は よく 作ります。

陳：そうですか。

---

**语法覆盖**：✅ Nが好き/嫌い（音楽が好き/魚が嫌い） ✅ Nができます/分かります（ギターができます/音楽が分かりません） ✅ NやN（ギターやピアノ） ✅ から（匂いが好きではありませんから） ✅ 頻率副詞（時々/よく/いつも） ✅ どうして

## 表达要点

| 要点 | 说明 |
|------|------|
| ぼく | 男性自称代词，对关系亲近的人使用，比「私」随意 |
| 場所 + で + 事件 + があります | 表示某处发生某事（区别于「に」表存在） |
| いかがですか | 「どうですか」的礼貌形式，用于建议或推荐 |
| 気に入りました | 第一次见到某事物后表示中意，不用于表达持续性爱好 |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：兴趣爱好大调查**

- **场景**：公司新来了一位同事，午休时大家围坐在一起聊各自的兴趣爱好和日常习惯。你参与对话，介绍自己的爱好，并询问他人。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：

    - ✅ は〜が分かります / できます
    - ✅ や（列举）
    - ✅ から / だから（原因理由）
    - ✅ 频率副词（至少使用2个不同的）
    - ✅ どうして
- **词汇限制**：仅使用 lesson ≤ 11 的词汇
- **AI 验收标准**：

  2. 覆盖检查：是否全部语法点都用到了
