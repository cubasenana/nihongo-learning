## 语法点

### 1. 他动词て形＋あります

- **含义**：表示某人有意进行的动作之后，其结果状态仍然存续
- **接续**：他动词て形 + あります
- **要点**：
  - 强调"有意为之"的结果存续，不把动作主体作为焦点，而是突出人为动作后的结果状态
  - 助词变化：动作句用「を」，结果状态句用「が」
  - 对比：「窓を開けました」（打开了窗户）→「窓が開けてあります」（窗户开着）
  - 与「自动词＋ています」的区别：てあります暗示是人为做的
- **例句**：
  - 冷蔵庫にビールが入れてあります。（冰箱里放了啤酒。）
  - 会議室の机の上に資料が並べてあります。（会议室桌上摆好了资料。）
  - パスポートはバッグの中にしまってあります。（护照收在包里了。）
- **Bunpro 校验**：待验证

### 2. 動て形＋おきます

- **含义**：事先做好某事，或做了某事并保持其结果状态
- **接续**：动词て形 + おきます
- **要点**：
  - 核心语感：做了某事并让结果保留（为后续做准备）
  - 否定形：～ておかなくてもいいです（不需要事先做）
  - 常见搭配：「～前に、～ておきます」
- **例句**：
  - お客さんが来る前に、部屋を片づけておきます。（客人来之前先把房间收拾好。）
  - 明日のプレゼンの資料をコピーしておいてください。（请事先复印好明天演示用的资料。）
  - 帰りの切符を買っておきましょうか。（要不要提前买好回程的票？）
- **Bunpro 校验**：待验证

### 3. 動て形＋みます

- **含义**：尝试做某事，看看结果如何
- **接续**：动词て形 + みます
- **要点**：
  - 表示"试着做"，带有不确定结果的期待
  - 常用于建议对方尝试：「～てみてください」
  - 常与「～そうですね」搭配：看着不错→试试看
- **例句**：
  - この靴、履いてみてもいいですか。（这双鞋可以试穿一下吗？）
  - 新しいレストランができたので、食べてみました。（新开了一家餐厅，就去尝了尝。）
  - おかしいですね。もう一度やってみてください。（真奇怪。请你再试一次。）
- **Bunpro 校验**：待验证

### 4. 小句（基本形）ために / 名詞＋の＋ために（目的）

- **含义**：为了某个目的而做某事
- **接续**：
  - 动词基本形 + ために
  - 名词 + の + ために
- **要点**：
  - 目的通常属于主句行为者（前后主语通常一致）
  - 前接意志性动词的基本形（非可能形式）
  - 与「～ように」区分：ために接意志动词，ように接非意志/可能形式
- **例句**：
  - 日本語を上達させるために、毎日ニュースを聞いています。（为了提高日语水平，每天都听新闻。）
  - 健康のために、週に3回ジョギングをしています。（为了健康，每周慢跑三次。）
  - 新しい家を買うために、貯金しています。（为了买新房子，正在攒钱。）
- **Bunpro 校验**：待验证

## 课文

> Arc 9「接待朋友·上」第2话 — 招待准备

**登場人物**：陳明、田中 — 陳明のアパート

> 使用语法：≤L34（+てあります、ておきます、てみます、ために目的）

---

陳：部屋に 入って ください。ビールが 冷蔵庫に 入れて ありますよ。

田中：ありがとうございます。テーブルの 上に 料理が 並べて ありますね。

陳：田中さんの ために、中華料理を 作りました。

田中：すごいですね。この料理は 初めてです。食べて みても いいですか。

陳：どうぞ。辛いかもしれませんから、少し 食べて みて ください。

田中：おいしいです！ 明日 北京を 案内するために、レストランも 予約して おきました。

陳：ありがとうございます。明日の 切符も 買って おきましたか。

田中：はい、もう 買って おきました。

---

**语法覆盖**：✅ てあります（入れてあります/並べてあります） ✅ ておきます（予約しておきました/買っておきました） ✅ てみます（食べてみて/食べてみても） ✅ ために目的（田中さんのために＝Nのために / 案内するために＝動ために）
