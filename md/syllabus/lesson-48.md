## 语法点

### 1. 自谦语概述

- **含义**：通过压低说话人自身或己方来表示对对方的敬意
- **接续**：无固定接续，是一套表达体系
- **要点**：
  - 自谦语的核心逻辑：降低自己 → 相对抬高对方
  - 谈及自己家人或自己公司上级时也用自谦语（对外时）
  - 与尊他语形成对照：对方用尊他语，自己用自谦语
  - 自称用「わたくし」「わたくしども」，公司用「社」「弊社」
- **例句**：
  - 私、李秀麗と申します。（我叫李秀丽。）
  - こちらは私どもの新しい製品でございます。（这是我们的新产品。）
- **Bunpro 校验**：待验证

### 2. お＋动词＋します / ご＋三类动汉字部分＋します

- **含义**：我来为您做……（自谦语中最常用的形式）
- **接续**：
  - 一类动/二类动：お + ます形去ます + します
  - 三类动词：ご + 汉字部分 + します
- **要点**：
  - ます形去ます后只有一个音节的动词不能用（× お見します）
  - 来る/する本身不能用此形式；サ変复合词多用ご～します/ご～いたします（ご連絡します等）
  - 部分三类动词用「お」而非「ご」：お電話します、お返事します
  - 与尊他语「お～になります」形式对照记忆
- **例句**：
  - お荷物は私がお持ちします。（您的行李我来拿。）
  - 昨日メールをお送りしたんですが。（昨天我给您发了邮件。）
  - 明日の午後、ご連絡します。（明天下午我跟您联系。）
- **Bunpro 校验**：待验证

### 3. 自谦语的特殊形式

- **含义**：部分高频动词有专用的自谦语形式
- **接续**：固定对应关系（需要记忆）
- **要点**：
  - 見る → 拝見します
  - 食べる/飲む/もらう → いただきます
  - 行く/来る → 参ります（泛用，己方人员来/去）/ 伺います（多用于去对方处拜访）
  - 聞く（问） → 伺います
  - する → いたします
  - 言う → 申します
  - あげる → 差し上げます
  - 知っている → 存じています（否定：存じません）
  - います → おります（～ています → ～ております）
- **例句**：
  - 明日、私がそちらへ伺います。（明天我去您那儿拜访。）
  - 中国の北京から参りました。（我从北京来。）
  - 案内状、拝見いたしました。（开业通知，我们已经拜读了。）
- **Bunpro 校验**：待验证

### 4. ～ていただけますか

- **含义**：能请您为我做……吗？（非常礼貌的请求表达）
- **接续**：动词て形 + いただけますか / いただけませんか
- **要点**：
  - 「いただけます」是「いただきます」的可能形式
  - 「～ていただけませんか」比「～ていただけますか」更礼貌
  - 是自谦语体系中的请求形式
  - 比「～てくれますか」「～てもらえますか」敬意程度更高
- **例句**：
  - この服、ちょっと小さいので、取り替えていただけますか。（这件衣服小了点，能给我换一下吗？）
  - 教えていただけませんか。（您能教我一下吗？）
  - 一度弊社までお越しいただけますか。（能请您来一趟公司吗？）
- **Bunpro 校验**：待验证

### 5. ～（さ）せていただきます

- **含义**：请允许我做……（获得对方允许/恩惠后让我做某事，非普通自谦的高级替换）
- **接续**：动词使役形式て形 + いただきます
- **要点**：
  - 使役形式 + いただきます = "获得对方允许我做某事"
  - 自谦程度非常高，商务场景频繁使用
  - 请求允许时用「～（さ）せていただけますか/いただけませんか」
  - 常用于自我介绍、开始/结束发言等正式场合
- **例句**：
  - 早速、資料を届けさせていただきます。（我这就把资料给您送过去。）
  - 明日、もう一度連絡させていただきます。（请允许我明天再跟您联络一次。）
  - 頭が痛いんですが、帰らせていただけますか。（我头疼，能允许我回去吗？）
- **Bunpro 校验**：待验证

### 6. ございます / ～でございます

- **含义**：「あります」「です」的更礼貌说法
- **接续**：
  - ございます（= あります 的礼貌形）
  - 名词 + でございます（= ～です 的礼貌形）
- **要点**：
  - 商务电话、接待客人时的标准用语
  - 不是自谦语，而是更高级的礼貌语
  - 常见用法：「～でございます」（电话应答）、「おつりでございます」（找零）
  - 「ございます」的否定是「ございません」
- **例句**：
  - はい、JC企画上海事務所でございます。（您好，这里是JC策划公司上海事务所。）
  - 社長、1つお伺いしたいことがございます。（经理，我有一个问题向您请教。）
  - あのう、お客様、おつりでございます。（这是找您的零钱。）
- **Bunpro 校验**：待验证

## 课文

> Arc 12「開業」第4话 — 开业后的首通商务电话（自謙語）

**登場人物**：陳明、取引先の山本さん — 電話

> 使用语法：≤L48（+自謙語、お/ご～します、特殊自謙、ていただけますか、させていただきます、ございます）

---

陳：はい、JC企画 上海事務所で ございます。

山本：お世話に なって おります。ABC商事の 山本と 申します。

陳：山本様、いつも お世話に なって おります。陳と 申します。

山本：開業の ご案内を 拝見いたしました。

陳：ありがとうございます。

山本：一度、そちらに 伺いたいのですが。

陳：ありがとうございます。それでは、ご都合を 教えて いただけますか。

山本：来週の 火曜日は いかがでしょうか。

陳：来週の 火曜日ですね。日程を お知らせします。

山本：火曜日ですか。午後なら 参れます。

陳：では、午後2時に お待ちして おります。資料は 事前に お送りさせて いただきます。

山本：よろしく お願いいたします。

陳：こちらこそ、よろしく お願いいたします。失礼いたします。

---

**语法覆盖**：✅ ございます（JC企画でございます） ✅ おります（お世話になっております/お待ちしております） ✅ 申します（陳と申します） ✅ お～します（お知らせします/お待ちして） ✅ ていただけますか（教えていただけますか） ✅ 拝見いたしました（特殊自謙） ✅ 伺いたい（特殊自謙） ✅ 参れます（特殊自謙） ✅ させていただきます（お送りさせていただきます）

## 表达要点

| 要点 | 说明 |
|------|------|
| 右も左も分かりません | 表示初来乍到什么都不懂，常用于新人入职自我介绍 |
| いつもお世話になっております | 商务电话中的标准寒暄，表示"承蒙多方关照" |
| 何のお構いもしませんで | 客人告辞时主人说的话，表示"招待不周" |
| おります / ～ております | 「います/～ています」的更谦和说法 |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。确认自谦语特殊形式的对应关系，区分尊他语和自谦语的使用对象。

### Level 2：组装（30%）

给词造句，3-5 题。将普通句子改写为自谦语形式。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：新事务所开业后的首通商务电话**

- **场景**：你是新开设的上海事务所副所长。收到一家日资企业的电话，对方看到了你们的开业通知。你接电话自报家门（用でございます），做自我介绍（用申します），感谢对方一直以来的关照（用ております），表示会尽快送去资料（用させていただきます），请求对方允许你明天前去拜访（用伺います/いただけますか）。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ お～します / ご～します
    - ✅ 自谦特殊形式（至少使用4种不同的特殊形式）
    - ✅ ～ていただけますか
    - ✅ ～（さ）せていただきます
    - ✅ ～でございます
- **词汇限制**：仅使用 lesson ≤ 48 的词汇
- **AI 验收标准**：
  1. 语法检查：自谦语形式是否正确，尊他/自谦是否分清对象
  2. 覆盖检查：是否全部语法点都用到了，特殊形式是否达到4种
  3. 自然度评分：商务电话的寒暄和用语是否自然得体
