## 语法点

### 1. 小句（动词た形/ない形）まま，小句

- **含义**：保持着某种状态不变去做另一件事
- **接续**：
  - 肯定：动词た形 + まま
  - 否定：动词ない形 + まま
  - 名词：名词 + の + まま
- **要点**：
  - 表示某状态保持不变；常用于"本应改变但未改变"的场景
  - 「まま」前不能用敬体形（× つけましたまま）
  - 名词接续用「の」：空腹のまま
  - 常与「～てしまいました」搭配，表示无意中的遗漏
- **例句**：
  - テレビをつけたまま、出かけてしまいました。（开着电视就出门了。）
  - 李さんはコートを着たまま、ベッドで寝てしまいました。（小李穿着外套就躺在床上睡了。）
  - 馬さんは行き先を告げないまま、家を出てしまいました。（小马没有说去哪里就离开了家。）
- **Bunpro 校验**：待验证

### 2. 小句（简体形）のに，小句

- **含义**：尽管……却……（结果不符合常识常理，含意外、不满语气）
- **接续**：
  - 动词/一类形容词简体形 + のに
  - 二类形容词/名词：～な + のに
- **要点**：
  - 后项出现了与前项不匹配的结果，说话人感到意外或不满
  - 「のに」前不能用敬体形（× 入れましたのに）
  - 「のに」的后项不能是命令、请求句
  - 与「が/けど」的区别：「のに」前后独立性弱、不满语气更强
- **例句**：
  - 目覚ましをかけておいたのに、今朝は起きられませんでした。（尽管上了闹钟，今天早上还是没起来。）
  - せっかくアメリカに留学したのに、英語が上手になりませんでした。（煞费苦心去了美国留学，英语却没能学好。）
  - 田中さんは作家なのに、読書が嫌いです。（田中先生尽管是作家却讨厌读书。）
- **Bunpro 校验**：待验证

### 3. 小句（简体形）はずです

- **含义**：应该……/理应……（根据某种理由做出有把握的推断）
- **接续**：
  - 动词/一类形容词简体形 + はずです
  - 二类形容词：～な + はずです
  - 名词：～の + はずです
- **要点**：
  - 说话人有根据地认为理应如此
  - 可用于自己记忆不太确定时：「たしか、～はずです」
  - 也可用于实际情况与预期不一致时
  - 与「～でしょう」相比，「はずです」更强调有理由/依据
- **例句**：
  - 会議は5時までですから、もうすぐ終わるはずです。（会议5点结束，应该快完了。）
  - たしか、書類は机の上に置いたはずです。（我记得文件应该是放在桌子上了。）
  - キムさんはこの大学の学生のはずです。（金女士应该是这所大学的学生。）
- **Bunpro 校验**：待验证

### 4. 小句（简体形）はずがありません

- **含义**：不可能……/不应该……（根据某种理由做出否定推断）
- **接续**：
  - 动词/一类形容词简体形 + はずがありません
  - 二类形容词：～な + はずがありません
  - 名词：～の + はずがありません
- **要点**：
  - 对某事的可能性做出强烈否定（"不可能/没道理"）
  - 也可以说成「はずはないです」「はずはありません」
  - 常用于反驳不合理的传言或说法
  - 语气比「～ないはずです」更强烈
- **例句**：
  - 張さんは入院中ですから、旅行に行くはずがありません。（小张在住院，不可能去旅行。）
  - 7月のオーストラリアは冬ですから、暑いはずがありません。（7月的澳大利亚是冬季，不可能热。）
  - 森さんはスキーがあんなに上手ですから、初心者のはずがありません。（森先生滑雪那么好，不可能是初学者。）
- **Bunpro 校验**：待验证

## 课文

> Arc 11「項目冲刺」第2话 — 文件追踪风波

**登場人物**：陳明、王 — オフィス

> 使用语法：≤L42（+まま、のに、はずです、はずがありません）

---

陳：あの 書類、王さんに 渡した はずですが……。

王：ええ、確かに 受け取りました。机の 上に 置いた はずです。

陳：でも、そこに ないんです。

王：おかしいですね。なくなる はずが ありません。

陳：鍵を かけないまま、部屋を 出ましたか。

王：ああ、そうです。急いで いたので、鍵を かけないまま 出て しまいました。

陳：せっかく 準備したのに、書類が 見つかりません。

王：あ、ちょっと 待って ください。……ありました！ かばんの 中に 入れたまま でした。

陳：よかったです。今度は 気をつけて くださいね。

---

**语法覆盖**：✅ まま（かけないまま/入れたまま） ✅ のに（準備したのに見つからない） ✅ はずです（渡したはず/置いたはず） ✅ はずがありません（なくなるはずがない）

## 表达要点

| 要点 | 说明 |
|------|------|
| ～ですもの/だもん | 表示原因理由（多为女性用语），带有申明正当性的语气 |
| せっかく～のに | 对付出没有得到回报表示惋惜、遗憾 |
| こちらこそ | 回应对方的道歉或感谢时使用："哪里，我才应该……" |
| 早速 | 积极主动地迅速开始行动，也用于书信开头的习惯用法 |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。确认「まま」「のに」「はずです」「はずがありません」的接续和含义。

### Level 2：组装（30%）

给词造句，3-5 题。用给定情境分别造句。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：项目文件追踪风波**

- **场景**：你负责一个重要项目，把完成的策划方案交给了同事转交客户。几天后客户说没收到。你和同事确认情况、推理可能的原因（用はず推断）、表达对结果的意外（用のに），最终发现文件被同事保管着忘了转交（用まま）。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ ～たまま / ～ないまま
    - ✅ ～のに
    - ✅ ～はずです
    - ✅ ～はずがありません
- **词汇限制**：仅使用 lesson ≤ 42 的词汇
- **AI 验收标准**：
  1. 语法检查：各语法点的接续和敬体/简体使用是否正确
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：推断和转折的语境是否自然合理
