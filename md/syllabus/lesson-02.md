## 语法点

### 1. これ / それ / あれ は 名 です

- **含义**：指示事物的代词，根据距离分为三种
- **接续**：これ / それ / あれ + は + 名词 + です
- **要点**：
  - これ：距说话人较近的事物
  - それ：距听话人较近的事物
  - あれ：距双方都较远的事物
  - 说话人和听话人在同一位置时，これ＝近、それ＝远、あれ＝更远
- **例句**：
  - これは ノートです。（这是笔记本。）
  - それは 雑誌ですか。（那是杂志吗？）
  - あれは 学校です。（那是学校。）
- **Bunpro 校验**：待验证

### 2. だれですか / 何ですか

- **含义**：对不确定的人或物进行提问
- **接续**：名 + は + だれ / 何 + ですか
- **要点**：
  - だれ：不知道是什么人时使用
  - 何（なん）：不知道是什么事物时使用
  - 即使句中有疑问词，句尾也必须加 か
  - だれ 的礼貌说法为 どなた
- **例句**：
  - あの 人は だれですか。（那个人是谁？）
  - これは 何ですか。（这是什么？）
  - あの 方は どなたですか。（那位是哪位？）
- **Bunpro 校验**：待验证

### 3. 名1 の 名2

- **含义**：用助词 の 连接两个名词，表示所属关系
- **接续**：名词1 + の + 名词2
- **要点**：
  - 表示所属（"〜的"）
  - 前面的名词修饰后面的名词
- **例句**：
  - わたしの 本です。（是我的书。）
  - 日本語の 雑誌を 読みます。（读日语杂志。）
- **Bunpro 校验**：待验证

### 4. この / その / あの + 名

- **含义**：修饰名词时，これ/それ/あれ 要变为 この/その/あの
- **接续**：この / その / あの + 名词
- **要点**：
  - この/その/あの 后面必须接名词，不能单独使用；单独指代时用 これ/それ/あれ
  - 距离指代规则与 これ/それ/あれ 相同
  - 「田中さんのです」中的 の 代替了前文已提到的名词（如 傘），相当于「田中さんの傘です」
- **例句**：
  - この 傘は 田中さんのです。（这把伞是田中先生的。）
  - その 辞書は だれのですか。（那本词典是谁的？）
  - あの 建物は 図書館です。（那栋建筑是图书馆。）
- **Bunpro 校验**：待验证

### 5. どれ / どの + 名

- **含义**：在三个以上的事物中进行选择提问
- **接续**：どれ + ですか / どの + 名词 + ですか
- **要点**：
  - 单独使用时用 どれ
  - 修饰名词时用 どの
  - 用于三者以上的选择
- **例句**：
  - あなたの かばんは どれですか。（你的包是哪个？）
  - 田中さんの 車は どの 車ですか。（田中先生的车是哪辆？）
- **Bunpro 校验**：待验证

## 课文

> Arc 1「到达·安顿」第2话 — 办公室里的东西

**登場人物**：陳明（ちん めい）、田中（たなか）、佐藤（さとう）— 同僚

> 使用语法：≤L02（L01全部 + これ/それ/あれ、だれ/何ですか、この/その/あの+名、どれ/どの、の準体）

---

田中：陳さん、これは陳さんの机です。

陳：ありがとうございます。あのう、それは何ですか。

田中：これは会社のノートパソコンです。陳さんのです。

陳：そうですか。では、あれは何ですか。

田中：あれはコピー機です。

陳：この辞書はだれのですか。

田中：それは佐藤さんのです。

陳：佐藤さん？ あの方はだれですか。

田中：あの方は佐藤さんです。営業の佐藤さんです。

陳：あのう、私のかばんはどれですか。

田中：そのかばんです。

陳：ああ、これですか。ありがとうございます。陳さんの机はどの机ですか。

田中：この机です。

---

**语法覆盖**：
- ✅ これ/それ/あれ は 名 です（これは会社のノートパソコンです / あれはコピー機です）
- ✅ だれですか / 何ですか（それは何ですか / あの方はだれですか）
- ✅ この/その/あの + 名（この辞書 / そのかばん / あの方 / この机）
- ✅ どれ / どの+名（かばんはどれですか / どの机ですか）
- ✅ の（準体）（陳さんのです / 佐藤さんのです）
- ✅ A の B（会社のノートパソコン / 陳さんの机 / 営業の佐藤さん）

## 表达要点

| 要点 | 说明 |
|------|------|
| 方（かた） | これ/それ/あれ の 人 → この/その/あの 方，是指人的礼貌说法 |
| どうぞ | 给对方东西或请对方收下时使用 |
| えっ | 对对方所说的话感到意外时使用；也可用于没听清时的反问（读升调） |
| はい 和 ええ | ええ 是 はい 比较随便的说法，但只用于肯定回答；被叫到名字时只能用 はい |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：失物招领处**

- **场景**：你在学校的失物招领处，有很多物品摆在桌上。你需要和工作人员确认哪个是你的东西，同时帮朋友问一下他丢的东西是否在这里。
- **要求**：
  - 5-7 句
  - 必须覆盖以下全部语法点：
    - ✅ これ/それ/あれ
    - ✅ だれ/何ですか
    - ✅ 名の名
    - ✅ この/その/あの
    - ✅ どれ/どの
- **词汇限制**：仅使用 lesson ≤ 2 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
