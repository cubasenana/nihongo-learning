# 初级下册 24 课语法索引

## 第25課：これは明日会議で使う資料です

1. 动词小句（简体形）+ 名词 — 动词简体形修饰名词（定语从句）
2. 小句（动词简体形）+ 名词 + は ～です — 动词小句修饰的名词短语做主语
3. 小句（动词简体形）+ 名词 + を/に/から ～ます — 动词小句修饰的名词短语做宾语等成分
4. 小句（一类形/二类形/名）+ 名词 — 形容词小句和名词小句修饰名词

## 第26課：自転車に2人で乗るのは危ないです

1. 小句（动词简体形）+ のは ～です — 动词小句加「の」名词化做主语
2. 小句（动词简体形）+ のを ～ — 动词小句加「の」名词化做宾语
3. 小句（简体形）+ でしょう — 对不确定事项的推测
4. 小句（简体形）+ かもしれません — 表示可能性（约50%）的推测

## 第27課：子供の時、大きな地震がありました

1. 小句（简体形）+ 時 — 表示时间的名词短语（～的时候）
2. 动词ます形 + ながら — 同一主体同时进行两个动作
3. 小句（简体形）+ でしょう？ — 向对方确认（升调确认/降调叮问）
4. ～ています③ — 表示反复或习惯性的动作
5. 名词 + で — 助词「で」表示原因、理由
6. ～と + 动词 — 助词「と」表示互动行为的对方

## 第28課：馬さんはわたしに地図をくれました

1. 名1は 名2に 名3を くれます — 别人给说话人或说话人一方的人某物
2. ～てあげます — 说话人或说话人一方为别人做某事
3. ～てもらいます — 请别人做某事/承受别人动作带来的恩惠
4. ～てくれます — 别人为说话人或说话人一方做某事

## 第29課：電気を消せ

1. 动词的命令形 — 对听话人下命令的表达形式
2. 动词ます形 + なさい — 比命令形稍客气的要求（师长对晚辈）
3. 动词基本形 + な — 禁止听话人做某事
4. ～て/～ないで — 日常祈使的说法（亲密关系间）
5. 名1 + という + 名2 — 提供对方不知道的新信息（花名、地名等）
6. 名1は ～という 名2です — 给名词下定义或做解释

## 第30課：もう11時だから寝よう

1. 动词的意志形 — 当场表示自己决心、意志或提议
2. 动词（意志形）+ と思います — 向听话人表示自己要做某事的意志
3. 动词（意志形）+ と思っています — 表示已持续一段时间的意志
4. 小句1 + ので、小句2 — 表示原因、理由（比「から」更礼貌郑重）

## 第31課：このボタンを押すと、電源が入ります

1. 小句1（动词基本形/ない形）+ と、小句2 — 表示恒常性条件（真理、习惯、发现新情况）
2. 动词（基本形/ない形）+ ことがあります — 表示有时会发生某种事态
3. 形容词的副词性用法 — 一类形容词「い→く」、二类形容词加「に」修饰动词
4. 小句（简体形）+ でしょうか — 礼貌程度较高的疑问表达

## 第32課：今度の日曜日に遊園地へ行くつもりです

1. 动词（基本形/ない形）+ つもりです — 表示说话之前已形成的意志、打算
2. 动词（基本形/ない形）+ ことにします/ことにしました — 说话人自己决定实施某行为
3. 动词（基本形/ない形）+ ことになりました — 由于外在原因形成了某种决定
4. 小句（简体形）+ そうです① — 传闻（表示信息来源于他人）

## 第33課：電車が急に止まりました

1. 自动词和他动词② — 词形上有对应关系的自他动词
2. 自动词 + ています④ — 表示变化结果的存续状态
3. ～てしまいます — 表示令人不愉快的结果或动作的完成
4. 动词ます形/一类形/二类形 + そうです② — 描述样态的观感或根据情形推测事态发展

## 第34課：壁にカレンダーが掛けてあります

1. 他动词 + てあります — 表示有意进行的动作结果的存续状态
2. ～ておきます — 为某种准备而有意识地进行动作（留存结果）
3. ～てみます — 尝试做某事
4. 小句（基本形）+ ために / 名词のために — 表示目的

## 第35課：明日雨が降ったら、マラソン大会は中止です

1. 小句1 + たら、小句2 — 表示假定条件
2. 小句1 + ても、小句2 — 表示即使前项成立后项结果也不变（让步条件）
3. ～だけ — 表示限定（这就是全部）
4. ～しか + 否定形式 — 表示排他限定（带主观感情色彩）
5. 名词 + でも — 提示极端例子（即便如此结果也一样）

## 第36課：遅くなって、すみません

1. 小句1て/で、小句2 — 「て」「で」形表示原因、理由
2. 名词 + に — 助词「に」表示用途或基准
3. 动词基本形 + のに — 「のに」表示用途或基准
4. 名词 + ばかり / 动词て形 + ばかりいます — 表示全部相同或总是进行同样动作
5. 小句（动词简体形）のが + 見えます/聞こえます — 表示感觉到看见或听见某事

## 第37課：優勝すれば、オリンピックに出場することができます

1. 动词/一类形容词的ば形 — 假定条件变形规则
2. 小句1ば、小句2 — 表示「小句2」成立须以「小句1」事态产生为条件
3. 小句（简体形）なら、小句2 — 根据对方言谈或现场情况陈述意见、提出请求或忠告
4. 名词 + でも — 从几个选项中举出有代表性的一项进行提议
5. 名1とか名2とか / 小句1とか小句2とか — 列举同样性质的几个例子

## 第38課：戴さんは英語が話せます

1. 可能形式 — 表示能够进行某动作的动词变形规则
2. 小句（基本形/ない形）+ ように、小句 — 表示「为了使某种状态成立」的目的
3. 小句（基本形/ない形）+ ようになります — 表示能力、状况、习惯等变成了某种新状态
4. 小句（基本形/ない形）+ ようにします — 表示努力使某种行为或状况变成现实

## 第39課：眼鏡をかけて本を読みます

1. 小句1て + 小句2 / 小句1ないで + 小句2 — 表示进行动作时主体处于某种状态
2. 小句1（简体形）ために、小句2 — 表示非意志性事态作为原因、理由
3. 动词ていきます/きます — 表示移动方向、独立动作、动作主体状况

## 第40課：これか��友達と食事に行くところです

1. 动词 + ところです — 表示动作处于即将进行/持续中/刚结束某阶段
2. 动词た形 + ばかりです — 表示动作或事情刚刚结束
3. 动词ます形 + 始めます/出します — 表示动作或变化的开始
4. 动词ます形 + 続けます — 表示动作或状态的不间断持续
5. 动词ます形 + 終わります — 表示动作或行为的结束

## 第41課：李さんは部長にほめられました

1. 被动形式 — 表示主语承受某种动作或影响的动词变形规则
2. 名は（名に）动词（ら）れます — 动作对象做主语的基本被动句
3. 名は 名に 名を 动词（ら）れます — 物主做主语的被动句（遭受麻烦或损失）
4. 名は 名に 动词（ら）れます — 纯粹表示受害的被动句（含自动词被动）
5. 名が/は 动词（ら）れます — 事物做主语的被动句（动作主体不确定）
6. 名は 名によって 动词（ら）れます — 事物做主语、动作主体特定时用「によって」

## 第42課：テレビをつけたまま、出かけてしまいました

1. 小句1（动词た形/ない形）まま、小句2 — 将理应改变的状态保持着进行另一动作
2. 小句1（简体形）のに、小句2 — 表示后项不符合前项的常识常理（转折）
3. 小句（简体形）+ はずです — 根据某种理由做出有把握的推断
4. 小句（简体形）+ はずがありません — 根据某种理由做出否定性推断

## 第43課��陳さんは、息子をアメリカに留学させます

1. 使役形式 — 使役主体强制、指示或许可动作主体进行动作的变形规则
2. 名は 名を 自动词（さ）せます — 自动词使役句（动作主体用「を」）
3. 名は 名に 名を 他动词（さ）せます — 他动词使役句（动作主体用「に」）
4. 动词（さ）せてください — 请求别人允许自己做某事
5. 动词ます形 + やすいです/にくいです — 表示事物易于或难于做某事的性质

## 第44課：玄関のところにだれかいるようです

1. 小句（简体形）+ ようです / 名のようです — 根据感知的情况进行推测
2. 小句（简体形）+ みたいです — 与「ようです」相同的推测（较随便的说法）
3. 小句（简体形）+ らしいです — 根据观察进行推测或婉转叙述传闻
4. 动词/一类形/二类形 + 過ぎます — 表示动作或性质超过正常的量或程度
5. 一类形容词 + さ — 构成表示某种状态的名词

## 第45課：少子化が進んで、日本の人口はだんだん減っていくでしょう

1. 动词ていきます/きました — 从基准时间观察的动作持续或性质状态的变化
2. 动词てきました — 表示某种状态的开始或出现
3. 小句1し、小句2し、小句3 — 叙述并列的几个事项（原因、理由的并列）
4. 动词ば形 + 动词基本形 + ほど — 表示正比例关系（越～越～）
5. で/へ/から/まで/と + の — 助词后续「の」修饰名词的用法

## 第46課：これは柔らかくて、まるで本物の毛皮のようです

1. 名のようです / 小句（简体形）ようです / みたいです — 表示比喻（好像～一样）
2. 名1 + らしい + 名2 — 表示「名2」具有「名1」的典型特征
3. 名（时间）+ までに — 表示某种事态发生或完成的期限终点
4. 名の / 动词（简体形）+ 間/間に — 表示持续时间段或事态发生的时间范围
5. ～のような 味/におい が します — 表示感觉到某种味道、气味、声音等

## 第47課：周先生は明日日本へ行かれます

1. 敬语概述 — 尊他语、自谦语、礼貌语三种分类
2. 动词（ら）れます — 动词被动形式作为尊他语使用
3. お + 动词ます形 + になります — 尊他语的表达形式
4. お/ご + 动词 + ください — 尊他语的劝说表达
5. 尊他语的特殊形式 — 召し上がる・ご覧になる・いらっしゃる・おっしゃる・なさる・くださる等

## 第48課：お荷物は私がお持ちします

1. 自谦语概述 — 通过压低说话人自身表示敬意的表达方式
2. お + 动词ます形 + します / ご + 三类动汉字部分 + します — 自谦语最常用的表达形式
3. 自谦语的特殊形式 — 拝見する・いただく・参る・伺う・いたす・申す・差し上げる・存じている等
4. 动词ていただけますか — 请求对方为自己做某事的礼貌请求形式
5. 动词（さ）せていただきます — 自谦程度高于「お/ご～します」的使役自谦表达
6. ございます / 名 + でございます — 「あります」「です」的更礼貌说法
