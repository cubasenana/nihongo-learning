# L25-L28 课文与生词表审核报告

审核对象：`docs/syllabus/lesson-25.md` 到 `docs/syllabus/lesson-28.md` 的新增 `## 课文` 与 `## 生词表`。

审核基准：

- 语法解锁：`docs/syllabus/grammar-unlock.md`
- 只检查 `lesson <= X` 的语法是否越界
- 对照每课 `## 语法点` 下的 `###` 标题检查覆盖
- 同时检查日语自然度、叙事连续性、生词表 `📖` 标记和 Markdown 表格列数

## 总结

整体可用，但 L26-L28 仍有硬问题：

- L26 有人物设定冲突：陈明是中国人，王作为上海同事夸他“中国語を話すのが上手”不自然。
- L26 `漢字を読むのはできます` 不自然，且不符合本课 `動のは + 形容词です` 覆盖目标。
- L27 `会えませんでした` 使用可能形，严格按 `grammar-unlock.md` 属于 L38 越界。
- L28 `手伝ってもらうと助かります` 使用条件 `と`，属于 L31 越界。
- L28 `くれます` 覆盖清单把 `てくれます` 当成裸 `くれます`，实际缺少 `Nをくれます` 用例。
- L25-L27 的 `新動詞` 表头缺少 `📖` 列，但数据行有该列，Markdown 表格列数不一致。

## L25

### ❌ 语法覆盖缺失：`動小句 + N + を/に/から + 動`

位置：`lesson-25.md:89`, `lesson-25.md:95`

原文：

```text
王：ここが 陳さんが 使う 部屋です。
```

覆盖清单写：

```text
修飾N做賓語（陳さんが使う部屋）
```

问题：

`陳さんが使う部屋` 在原句中是名词谓语，不是 `小句 + 名词 + を/に/から + 动词`。本课第 3 个语法点要求被修饰名词后面带 `を/に/から` 再接动词，目前课文没有真正覆盖。

建议：

```text
王：ここから 陳さんが 使う 部屋に 行きます。
```

或：

```text
陳：窓から 見える 景色を 写真に 撮っても いいですか。
```

### ⚠ 覆盖清单引用不准确：`修飾N做主語`

位置：`lesson-25.md:87`, `lesson-25.md:95`

覆盖清单把 `入り口が広い建物` 标成“修饰 N 做主语”，但这句主要覆盖的是 `形容词小句 + 名词`，不是 `動小句 + N`。

更合适的覆盖说明应引用：

```text
空港から来た道は混んでいましたか
あそこに見える高いビルが私たちのオフィスです
```

### ⚠ 生词表格式错误：`新動詞` 表列数不一致

位置：`lesson-25.md:117-124`

表头是 4 列：

```markdown
| 词条 | 读音 | 中文 | 类型 |
```

但数据行是 5 列，额外多了 `📖`。应把表头改成：

```markdown
| 词条 | 读音 | 中文 | 类型 | 📖 |
```

### 💡 可选：若要求本课生词表覆盖课文全部新词，可补充

课文中出现但本课表没有单独列出的词包括：

- `去年`
- `窓`
- `部屋`
- `オフィス`
- `空港`

其中部分已在前课出现，是否重复收录取决于生词表设计原则。

## L26

### ❌ 叙事连续性/人物设定冲突

位置：`lesson-26.md:75`, `lesson-26.md:79`

原文：

```text
王：陳さん、上海で 生活するのは 大変ですか。
王：中国語を 話すのが 上手ですね。
```

问题：

陈明在 L01 已设定为中国人，L25 已调到上海。上海同事夸他“中国语说得好”，叙事上不自然，也会让学习者误解人物设定。

建议改成日语或英语相关表达，例如：

```text
王：日本語を 話すのが 上手ですね。
陳：いいえ、まだまだです。日本語を 聞くのは 難しいです。
```

### ❌ 语法/自然度错误：`読むのはできます`

位置：`lesson-26.md:81`, `lesson-26.md:93`

原文：

```text
漢字を 読むのは できますが、聞くのは 難しいです。
```

问题：

`読むのはできます` 很不自然；而且覆盖清单把它当作 `動のは + 形`，但 `できます` 是动词，不是形容词。

建议：

```text
漢字を 読むのは 簡単ですが、聞くのは 難しいです。
```

或如果想表达“能读”，可用已学过的 L20：

```text
漢字を 読むことは できますが、聞くのは 難しいです。
```

但第二种不能用来覆盖本课 `動のは + 形`。

### ⚠ 生词表格式错误：`新動詞` 表列数不一致

位置：`lesson-26.md:129-135`

表头缺 `📖` 列，但数据行有 `📖`。应统一为：

```markdown
| 词条 | 读音 | 中文 | 类型 | 📖 |
```

### ⚠ 生词表遗漏

课文中出现但 L26 生词表未列出的重点词：

- `中国語`
- `漢字`
- `午後`

其中 `中国語/漢字` 与本课语言学习场景直接相关，建议收进 L26 生词表。

## L27

### ❌ 语法越界：`会えませんでした`

位置：`lesson-27.md:120`, `lesson-27.md:159`

原文：

```text
忙しくて、会えませんでした。
```

问题：

`会えませんでした` 是 `会います` 的可能形。根据 `grammar-unlock.md`，可能形式在 L38 才正式解锁。即使生词表把 `会えます` 当成新动词列出，对 0 基础学习者来说仍然会提前引入“可能形”规则。

建议：

```text
忙しくて、会いませんでした。
```

如果确实要表达“没能见到”，可用 L20 已学：

```text
忙しくて、会うことができませんでした。
```

### ⚠ 指代略显不清

位置：`lesson-27.md:118-120`

原文：

```text
王：田中さん、先週 上海に 来たでしょう？
陳：はい、仕事で 来ました。忙しくて、会えませんでした。
```

问题：

从上下文看，王想说“田中上周来上海了吧”，但下一句省略主语后，`仕事で来ました` 容易被读成“陈明自己因工作来的”。建议补 `田中さんは` 或重写回答。

建议：

```text
王：田中さんは 先週 上海に 来たでしょう？
陳：はい、仕事で 来ました。でも、忙しくて、会いませんでした。
```

### ⚠ 生词表格式错误：`新動詞` 表列数不一致

位置：`lesson-27.md:156-162`

表头缺 `📖` 列，但数据行有 `📖`。应统一为：

```markdown
| 词条 | 读音 | 中文 | 类型 | 📖 |
```

### ⚠ 生词表遗漏

位置：`lesson-27.md:104`

`子供` 出现在课文中，但 L27 生词表未列出。它不是专有名词，也不是语法功能词，建议收进本课生词表。

## L28

### ❌ 语法越界：条件 `と`

位置：`lesson-28.md:92`

原文：

```text
やっぱり、だれかに 手伝って もらうと 助かりますね。
```

问题：

这里的 `もらうと` 是条件 `と`，根据 `grammar-unlock.md` 属于 L31。L28 尚未解锁。

建议：

```text
やっぱり、だれかに 手伝って もらうのは 助かりますね。
```

或：

```text
手伝って もらって、助かりました。
```

### ❌ 语法覆盖缺失：裸 `くれます`

位置：`lesson-28.md:8`, `lesson-28.md:80`, `lesson-28.md:84`, `lesson-28.md:96`

本课第 1 个语法点是：

```text
名1[人]は + 名2[人]に + 名3[物]を + くれます
```

但课文中的 `くれました` 都是：

```text
荷物を 運んで くれました
レストランも 教えて くれました
```

这两句都是 `動てくれます`，不是 `Nをくれます`。覆盖清单把 `てくれます` 误算成了裸 `くれます`。

建议补一句：

```text
李さんは 私に お茶を くれました。
```

或：

```text
李さんが 日本の お菓子を くれました。
```

### ⚠ 自然度建议：`何かしましたか`

位置：`lesson-28.md:86`

原文：

```text
陳さんは 李さんに 何か しましたか。
```

这句话语法没错，但有点泛，像是在问“你对李做了什么事”。结合上下文更自然的是：

```text
陳さんは 李さんに 何か お礼を しましたか。
```

### ⚠ 生词表 `📖` 标记可能不一致

位置：`lesson-28.md:70`, `lesson-28.md:109`

`隣人` 在登场人物说明中出现为 `李さん（隣人）`，但对话正文中实际使用的是 `隣の李さん`，不是 `隣人`。

如果 `📖` 只统计对话正文，应去掉 `隣人` 的 `📖`，或把正文改成：

```text
隣人の 李さんが 荷物を 運んで くれました。
```

### ⚠ 生词表词性/形态建议

位置：`lesson-28.md:76`, `lesson-28.md:107`

课文使用：

```text
引っ越しました
```

生词表只写：

```text
引っ越し | 搬家
```

建议标成 `引っ越し / 引っ越します`，或注明 `名・自動3`，避免学生不知道它在课文里为什么能变成动词。

## 建议修复顺序

1. 先修 L28 `もらうと` 越界和裸 `くれます` 覆盖缺失。
2. 再修 L26 人物设定冲突与 `読むのはできます`。
3. 再修 L27 `会えませんでした` 可能形越界。
4. 再同步 L25 覆盖清单，补真正的 `小句 + N + を/に/から + 動`。
5. 最后统一 L25-L27 `新動詞` 表格列数，并按规则整理生词表 `📖`。
