# L29-L32 课文与生词表审核报告

审核对象：`docs/syllabus/lesson-29.md` 到 `docs/syllabus/lesson-32.md` 的新增 `## 课文` 与 `## 生词表`。

审核基准：

- 语法解锁：`docs/syllabus/grammar-unlock.md`
- 只检查 `lesson <= X` 的语法是否越界
- 对照每课 `## 语法点` 下的 `###` 标题检查覆盖
- 同时检查日语自然度、叙事连续性、生词表 `📖` 标记、表格列数、词源一致性

## 总结

本批整体结构清楚，但硬错误比 L25-L28 略多，集中在“为了自然表达而提前用了后面课的语法”：

- L29 `書いてあります` 提前使用 L34 `てあります`。
- L29 `入ってみましょう` 提前使用 L34 `てみます`。
- L30 `雨が降ったら` 提前使用 L35 `たら`。
- L31 `使えるでしょうか` 提前使用 L38 可能形。
- L31 `9時まででしたら` 提前使用条件表达。
- L32 `楽しんできてください` / `買ってくることにします` 提前使用 L39 `てくる`。
- L32 `黄金周 | ゴールデンウィーク` 写法和读音不匹配，容易误导 0 基础学生。

Markdown 表格列数检查通过，L29-L32 没有发现表格列数不一致。

## L29

### ❌ 语法越界：`書いてあります`

位置：`lesson-29.md:105`

原文：

```text
陳：あの 看板に 「止まれ」と 書いて あります。どういう 意味ですか。
```

问题：

`書いてあります` 是 `てあります`，根据 `grammar-unlock.md` 属于 L34。L29 尚未解锁。

建议：

```text
陳：あの 看板は 「止まれ」です。どういう 意味ですか。
```

或：

```text
陳：あの 看板の 「止まれ」は どういう 意味ですか。
```

### ❌ 语法越界：`入ってみましょう`

位置：`lesson-29.md:121`

原文：

```text
陳：なるほど。ラーメン屋という 店が ありますね。入って みましょう。
```

问题：

`入ってみましょう` 使用 `てみます`，根据 `grammar-unlock.md` 属于 L34。L29 尚未解锁。

建议：

```text
陳：なるほど。ラーメン屋が ありますね。入りましょう。
```

### ⚠ 语法覆盖不严：`ないで` 祈使没有真正独立出现

位置：`lesson-29.md:115`, `lesson-29.md:125`

原文：

```text
王：普段は 使わないで ください。「～て」の ほうが 自然ですよ。
```

覆盖清单写：

```text
て/ないで（使わないで/静かにして）
```

问题：

课文实际是 `使わないでください`，这是 L19 已学的 `ないでください`，不是 L29 这里要教的亲密/省略式 `～ないで`。`静かにして` 覆盖了 `て`，但 `ないで` 独立祈使未真正覆盖。

建议补一句：

```text
王：「使わないで」は 「使わないでください」と 同じです。
```

或更自然地：

```text
王：友達には 「心配しないで」と 言います。
```

### ⚠ 自然度：`ラーメン屋という店` 冗余

位置：`lesson-29.md:121`

问题：

`ラーメン屋という店` 字面是“叫作拉面店的店”，比较冗余。`ラーメン屋` 本身就是店的种类。

建议：

```text
「一番」という ラーメン屋が ありますね。
```

如果不想引入新店名：

```text
ラーメン屋が ありますね。入りましょう。
```

## L30

### ❌ 语法越界：`雨が降ったら`

位置：`lesson-30.md:93`

原文：

```text
王：雨が 降ったら、どうしますか。
```

问题：

`たら` 条件在 `grammar-unlock.md` 中是 L35。L30 尚未解锁。

建议：

```text
王：雨が 降るかもしれません。どうしますか。
```

或使用 L27 已学的 `時`：

```text
王：雨の 時は、どうしますか。
```

### ⚠ 覆盖清单可更清楚：`と思っています`

位置：`lesson-30.md:89`, `lesson-30.md:99`

原文：

```text
実は、前から 陳さんの 料理を 食べようと 思って いたんです。
```

问题：

这句语法上可以成立，`前から` 也能说明持续意志；但课文实际用了过去形式 `思っていたんです`，而本课标题是 `動意志形 + と思っています`。对 0 基础学生来说，覆盖清单写 `と思っています`，正文却出现 `と思っていたんです`，会增加理解成本。

建议：

```text
実は、前から 陳さんの 料理を 食べようと 思っています。
```

## L31

### ❌ 语法越界：可能形 `使える`

位置：`lesson-31.md:81`, `lesson-31.md:97`

原文：

```text
陳：分かりました。プールは 何時まで 使えるでしょうか。
```

问题：

`使える` 是 `使います` 的可能形，根据 `grammar-unlock.md`，可能形式在 L38 才解锁。L31 尚未解锁。

建议：

```text
陳：分かりました。プールは 何時まででしょうか。
```

如果要保留“能使用”的意思，可用 L20 已学：

```text
陳：プールを 何時まで 使うことが できるでしょうか。
```

但对 0 基础学生来说，第一种更干净。

### ❌ 语法越界：`でしたら`

位置：`lesson-31.md:87`

原文：

```text
スタッフ：はい、9時まででしたら 大丈夫です。
```

问题：

`でしたら` 是条件表达，和 `たら/なら` 系列相关；根据 `grammar-unlock.md`，`たら` 是 L35，`なら` 是 L37。L31 尚未解锁这种用法。

建议：

```text
スタッフ：はい、9時までです。大丈夫です。
```

或：

```text
スタッフ：はい、夜 9時まで 大丈夫です。
```

### ⚠ 自然度/用法边界：`ゆっくり泳ぐといいですよ`

位置：`lesson-31.md:91`

原文：

```text
大きく 息を 吸って、ゆっくり 泳ぐと いいですよ。
```

这句日语本身自然，也没有明显硬越界。但它带有建议语气，和本课 `と条件` 的“恒常/真理/发现新情况”主线略有偏离。若想更贴近本课核心，可以改成更客观的条件结果：

```text
ゆっくり 泳ぐと、長く 泳ぐことが できますよ。
```

不过这会再次引入 `ことができます`，是否使用取决于你是否愿意复用 L20。

### ✅ 语法覆盖

除 `使える` 越界导致 `でしょうか` 例句需要替换外，四个语法点都有覆盖：

- `と条件`：`押すとドアが開きます` / `まっすぐ行くと右にプールがあります`
- `ことがあります`：`遅く来ることがあります`
- 形容词副词用法：`静かに` / `自由に` / `大きく` / `ゆっくり`
- `でしょうか`：当前用例需替换为无可能形版本

## L32

### ❌ 语法越界：`楽しんできてください`

位置：`lesson-32.md:92`

原文：

```text
王：きれいでしょうね。楽しんで きてください。
```

问题：

`楽しんでくる` 使用 `てくる`，根据 `grammar-unlock.md`，`てきます/てくる` 在 L39 才解锁。L32 尚未解锁。

建议：

```text
王：きれいでしょうね。楽しんで ください。
```

### ❌ 语法越界：`買ってくることにします`

位置：`lesson-32.md:94`

原文：

```text
陳：ありがとうございます。お土産を 買って くる ことに します。
```

问题：

同样使用了 L39 的 `てくる`。本句虽然覆盖了 `ことにします`，但同时引入了未解锁语法。

建议：

```text
陳：ありがとうございます。お土産を 買う ことに します。
```

### ❌ 生词/自然度硬问题：`黄金周 | ゴールデンウィーク`

位置：`lesson-32.md:78`, `lesson-32.md:108`

原文：

```text
陳：黄金周は 何を する つもりですか。
```

生词表：

```markdown
| 黄金周 | ゴールデンウィーク | 黄金周 | ★★★ | 📖 |
```

问题：

日语里最自然的是 `ゴールデンウィーク`。`黄金周` 更像中文写法；如果用汉字词，常见是 `黄金週間`，不是把 `黄金周` 直接读作 `ゴールデンウィーク`。这对 0 基础学生很容易造成“汉字写作黄金周、读作ゴールデンウィーク”的误解。

建议：

```text
陳：ゴールデンウィークは 何を する つもりですか。
```

生词表改为：

```markdown
| ゴールデンウィーク | | 黄金周；日本长假 | ★★★ Golden Week | 📖 |
```

如果想表达中国语境的黄金周，建议另写成 `連休` 或 `国慶節の連休`，不要混用 `黄金周 | ゴールデンウィーク`。

### ✅ 语法覆盖

L32 四个语法点均有覆盖：

- `つもりです`：`帰るつもりです` / `何をするつもりですか`
- `ことにしました`：`北京に旅行することにしました`
- `ことになりました`：`田中さんが来ることになりました`
- `そうです` 传闻：`いい天気だそうです` / `桜が咲いたそうです`

修掉 `買ってくる` 后，`お土産を買うことにします` 仍可保留 `ことにします` 覆盖。

### ✅ 叙事连续性

L32 讨论黄金周/长假去北京，且田中从日本来，承接 L24 以后田中仍在日本、陈明在上海的叙事线，没有发现硬矛盾。

### ⚠ 生词表/词源一致性

若严格执行 CSV 词源，以下词条未能在 CSV 中精确匹配到词条：

- `黄金周`
- `実家`
- `お土産`（CSV 中有 `土産`，不是精确词条）
- `咲きます`（CSV 中能看到 `咲く/咲いた` 的例句或 `桜` 词条，但没有精确的 `咲きます` 词条）

另外，`桜` 在 CSV 中存在，但如果你把 CSV 的课号字段也当作词汇解锁顺序，它标在后面课次；是否允许 L32 提前使用需要另行确认。

## 建议修复顺序

1. 先修 L29/L30/L31/L32 的硬越界：`てあります`、`てみます`、`たら`、可能形、`でしたら`、`てくる`。
2. 再修 L32 `黄金周 | ゴールデンウィーク` 的词表和课文写法。
3. 再补 L31 场景核心词：`プール`、`泳ぎます`、`押します`、`開きます`、`息`。
4. 再调整 L29 `ないで` 祈使覆盖，让它真正出现无 `ください` 的形式。
5. 最后按是否严格执行 CSV 词源，统一处理未精确匹配 CSV 的词条。
