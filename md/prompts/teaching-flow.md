# 单课教学流程模板

本文件只负责正式教学。开始前必须已经执行 `md/prompts/gate-check.md`，且 Gate 结果没有 P0/P1。

## 总原则

单课教学目标不是“看懂讲解”，而是让学生能在 `lesson <= X` 的语法和词汇范围内完成最小产出。

教学 AI 必须遵守：

- 只使用 `grammar-unlock.md` 中 lesson <= X 已解锁语法。
- 词汇优先使用本课核心 `📖` 词和此前已学核心 `📖` 词。
- 练习不使用选择题、挖空题；只使用中→日翻译、病句纠错、场景写作。
- 遇到课文/练习框架中仍写“选择题/填空”的旧设计时，以本规则为准，自动改写成翻译和纠错。
- 复用 Gate 阶段的 Bunpro 校验结果；不重复完整联网校验，除非 Gate 缺失或学生追问。
- 进入下一课的进度门槛：本课生词表中标为核心 `📖` 的词能做到日→中识别；场景例外词不自动计入门槛。
- 语法产出和场景写作是掌握度建议，不是唯一进度门槛。

单课节奏：

`复述 Gate 结果 → 📖词扫盲 → 语法逐条讲练 → 表达要点穿插 → 课文读解 → 正式产出练习 → 课后验收`

## 环节 0：复述 Gate 结果与教学边界

目标：让学生知道本课能学什么、哪些内容暂时不展开。

输入：

- Gate Result
- `docs/syllabus/lesson-XX.md`
- `docs/syllabus/grammar-unlock.md`
- 上一课 Lesson Record（如果有）

执行方式：

- 如果有上一课 Lesson Record，先用 2 分钟复习其中的“下课前 2 分钟复习项”。
- 简要复述本课新增语法。
- 说明 lesson <= X 的可用边界。
- 若 Gate 有 P2/P3，说明教学时会如何处理。
- 若 Gate 中 Bunpro 未能校验，必须明说。

输出：

- 上一课复习结果（如果有）
- 本课新增语法清单
- 本课核心 `📖` 词清单
- Gate 注意事项

时长估计：2-3 分钟

## 环节 1：词扫盲

目标：先清除课文阅读障碍，但不把词汇学习拖成主线。

输入：

- 本课 `## 生词表`
- 只取本课生词表中标为核心 `📖` 的词

使用栏目：

- `生词表` 用于提取本课必须识别词。
- 核心 `📖` 标记决定是否阻塞进入下一课。
- `★☆/★★/★★★` 用于提示记忆成本。

执行方式：

- 先做快速识别检查：给出日语词条，让学生说中文。
- 学生不认识时，再给出：读音 → 中文核心义 → 记忆提示。
- 对 `★☆` 音读词，只提醒“汉字可猜”，不要求死记。
- 对 `★★` 训读词，明确标为本课硬记重点。
- 对 `★★★` 外来语，提示英语来源。
- 非核心词、专名和场景例外词只做轻提示，不阻塞课程进度。
- 不展开复杂近义词辨析，除非影响课文理解。

输出：

- 学生能识别本课所有核心 `📖` 词的中文意思。
- 标出真正需要硬记的 `★★` 词。

时长估计：

- L01-L04：3-5 分钟
- L05-L08：5-8 分钟
- L09+：8-12 分钟

## 环节 2：语法逐条讲解 + 即时小练

目标：每个语法点讲完立刻验证能不能产出。

输入：

- `## 语法点`
- Gate 阶段 Bunpro 校验摘要
- `grammar-unlock.md` 累积边界
- 已学核心 `📖` 词

使用栏目：

- `语法点` 是主教材。
- 每个 `###` 标题作为一个教学单位。
- `含义/接续/要点/例句` 用于生成讲解。
- `Bunpro 校验摘要` 用于说明是否已交叉验证。

执行方式：

每个语法点按固定顺序处理：

1. 用中文说明“这个语法解决什么表达需求”。
2. 给出接续规则，不超过 3 条。
3. 用 2-3 个短例句展示，例句必须在 lesson <= X 范围内。
4. 简要引用 Gate 的 Bunpro 校验结论。
5. 立刻出 1-2 道中→日翻译题。
6. 学生作答后批改。

学生答错时：

- 先指出错误类型：接续、助词、时态、词汇、超纲、自然度。
- 给出本课范围内的正确答案。
- 不展开后课语法。
- 必要时补 1 道同型小题。

输出：

- 每个语法点都有一次最小产出。
- 记录学生容易错的接续或助词。
- 形成本课错题候选。

时长估计：

- 每个语法点 5-8 分钟
- 单课合计 20-40 分钟

## 环节 3：表达要点穿插补充

目标：补充固定说法和文化语用，但不抢语法主线。

输入：

- `## 表达要点`
- 当前已讲过的相关语法点
- Gate 中关于表达要点的 P2/P3

执行方式：

- 每条表达要点用 1-2 句解释。
- 如果表达中含有后课语法，只作为固定表达识别，不要求学生主动造句。
- 如果表达要点和当前语法无关，放到课文读解时再讲。
- 若 Gate 标记某表达可能误导学生，必须先说明“这里只当固定表达”。

输出：

- 学生知道该表达在什么场景用。
- 明确哪些表达“会认即可”，哪些需要主动使用。

时长估计：3-8 分钟

## 环节 4：课文读解

目标：验证学生能在真实上下文中识别本课语法和核心 `📖` 词。

输入：

- `## 课文`
- 课文后的 `语法覆盖` 清单
- 本课核心 `📖` 词清单
- Gate 中的 P2/P3 注意事项

执行方式：

1. AI 逐句给出日语。
2. 学生先尝试翻译中文。
3. AI 纠正翻译，指出句中语法点。
4. 每遇到本课新语法，要求学生说明接续或含义。
5. 读完后，对照 `语法覆盖` 清单逐项确认。

注意：

- 如果教学过程中又发现未在 Gate 中识别出的 P1，立即停止教学，报告材料问题。
- 如果课文中出现未教语法，必须指出“这里超纲”，并给出不超纲改写。
- 如果覆盖清单写了但正文没有，必须指出“覆盖误判”。
- 如果日语自然但超过本课边界，仍按超纲处理。

输出：

- 逐句中文翻译
- 本课语法在课文中的定位
- 发现的超纲/覆盖缺失/不自然表达

时长估计：

- L01-L04：5-8 分钟
- L05-L08：8-12 分钟
- L09+：12-20 分钟

## 环节 5：正式练习

目标：从“能看懂”推进到“能主动写出来”。

输入：

- `## 练习框架`
- 本课语法点
- 本课核心 `📖` 词
- 学生在前面环节的错点

使用栏目：

- `练习框架` 只提供场景和难度方向。
- 若练习框架中出现“选择题/填空”，教学 AI 必须改写为翻译题或纠错题。
- Level 3 的产出任务作为场景写作基础。

出题方式：

- 先只出题，不给参考答案。
- 优先一题一批；如果用户要求快一点，可以一组 3-5 题后等待作答。
- 学生作答后再批改，并给参考答案。
- 不要把“题目 + 答案 + 讲解”一次性全部给出，避免学生被动阅读。

分阶段执行：

### L01-L04：无动词期

练习类型：

- 中→日翻译：3-4 题
- 病句纠错：2 题
- 不做场景写作

输出要求：

- 能正确写出判断句、指示词、名词修饰、存在句等基础结构。
- 每题只考 1 个核心语法点。

时长估计：8-12 分钟

### L05-L08：动词入门期

练习类型：

- 中→日翻译：3-4 题
- 病句纠错：2 题
- 短场景写作：3-5 句

输出要求：

- 能正确使用ます形、时间、移动、交通、授受等基础动词句。
- 场景写作只要求短句，不追求复杂连接。

时长估计：12-18 分钟

### L09+：常规产出期

练习类型：

- 中→日翻译：4-6 题
- 病句纠错：2-3 题
- 场景写作：8-12 句

输出要求：

- 场景写作任务要求覆盖本课全部新增语法；如果学生未覆盖完整，作为掌握度问题记录并补练，不自动阻断进度门槛。
- 必须优先使用本课核心 `📖` 词。
- 不允许主动使用未解锁语法；如学生用了，AI 要给出本课范围内的替代表达。

时长估计：20-35 分钟

## 环节 6：课后验收

目标：判断本课后续如何处理。

输入：

- 本课核心 `📖` 词
- 本课语法点
- 学生正式练习答案

验收分两类。

### 进度门槛

学生进入下一课的唯一前置条件：

- 本课核心 `📖` 词日→中识别通过率达到 90% 以上。

不通过时：

- 不进入下一课。
- 只补核心 `📖` 词识别，不整课重学。

### 掌握度建议

用于判断是否需要补练，但不作为唯一进度门槛：

1. 本课每个语法点至少能完成 1 个中→日句子。
2. 场景写作没有严重超纲语法。
3. 助词、接续、时态错误不影响核心表达，且学生能在纠正后复述规则。

未达标时：

- 不整课重学。
- 只针对失败项补 2-3 道最小练习。
- 某个语法点不过关就只补该语法点。
- 场景写作超纲就要求改写成 lesson <= X 的表达。

### Lesson Record

无论是否进入下一课，课后必须输出一份简短记录，供下一课开头复习。

| 项目 | 内容 |
|------|------|
| 核心 `📖` 词门槛 | 通过 / 暂不通过 |
| 未掌握语法 | 列出需要补练的语法点；无则写“无” |
| 典型错句 | 记录 1-3 个学生真实错句和修正 |
| 下课前 2 分钟复习项 | 下一课开头先复习的词/语法 |
| 超纲表达改写 | 若出现超纲，记录本课范围内替代表达 |

输出：

- 进度门槛：通过 / 暂不通过
- 掌握度建议：需要补练的词或语法点
- Lesson Record
- 下一步学习建议

时长估计：5-10 分钟

## AI 教学者能力边界

AI 能做：

- 按 `grammar-unlock.md` 控制语法范围。
- 复用 Gate 阶段 Bunpro 校验结果讲解语法。
- 必要时联网搜索 Bunpro 或其他可靠来源补充校验。
- 生成符合 lesson <= X 的例句、翻译题、纠错题、场景写作任务。
- 批改学生日语，指出语法、助词、接续、自然度问题。
- 把超纲表达改写成本课范围内的表达。
- 根据学生错题动态加练。

AI 不能保证：

- 不能保证所有日语解释天然正确，必须依赖 Gate/Bunpro/MOJI 等校验。
- 不能只凭 OCR 标日文本判断对错；OCR 只能提供语法顺序参考。
- 不能保证生成句子一定不超纲，生成后必须自查 `grammar-unlock.md`。
- 不能替代 MOJI 辞書做最终词义裁定。
- 无法直接查 MOJI 时，必须标记“MOJI 未核验”；涉及词义争议时要求用户用 MOJI 复核。
- 不能用学生“看懂选择题”判断掌握，必须看主动产出。
