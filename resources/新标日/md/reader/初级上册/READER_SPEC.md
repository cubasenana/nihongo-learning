# 新标日 Reader 规范 v1

## 目标

这套 `reader` 稿的目标不是纸书级复刻，而是：

- 保留原课的学习价值
- 保留页码锚点和课内结构
- 优先让人类在网页里顺畅阅读
- 允许为了网页阅读做有限度的重组
- 明确区分“已重建”与“暂缓处理”，不伪装完整度

## 分层

### `structured`

- 机器清洗稿
- 允许存在 OCR 残留
- 作为脚手架和检索底稿

### `reader`

- 人类可读稿
- 对原 PDF 页图逐页核对后重建
- `VitePress` 导出时优先于 `structured`

## 模块模型

| 模块 | 名称 | 目标 | 处理原则 |
| --- | --- | --- | --- |
| `M0` | 课头与页码 | 锁定课次和原书锚点 | 课名、页码范围、页标记必须保留 |
| `M1` | 基本课文 | 保住本课主干 | `基本句 + 会话 A/B/C/D` 尽量全文忠实 |
| `M2` | 语法解释 | 让规则可学可查 | 每个语法点单独成节，例句尽量保留 |
| `M3` | 表达及词语讲解 | 吃透语感与文化点 | 允许条目化重写，但不要丢信息点 |
| `M4` | 应用课文 | 保证能连续阅读 | 按场景重组，对话顺序必须正确 |
| `M5` | 练习 | 保持教材训练价值 | 能全文则全文；依赖录音/图表时至少保留题型、题干和训练点 |
| `M6` | 生词表 | 方便记忆和检索 | 统一成 `词条 / 专有名词 / 表达 / 专栏` |
| `M7` | 补充页 | 处理图示、文化页、专题页 | 可重排、可拆专题页，但不能混成乱码正文 |

## 完成等级

| 等级 | 含义 | 验收标准 |
| --- | --- | --- |
| `S1` | 主干可读 | `M1/M2/M4/M6` 已完成，能顺着学完本课核心内容 |
| `S2` | 教材可用 | `M3/M5` 基本完成，图示页和补充页不再明显拖后腿 |
| `S3` | 高保真学习版 | `M0-M7` 全部有明确归宿；练习和补充页完成重排；难点允许补 `ruby` 或说明；剩余缺口必须显式标注 |

> 说明：`S3` 仍不是出版社电子版。它是“高保真学习版”，重点是完整学习价值和网页可读性，不追求逐像素复刻。

## 处理流程

1. 先看原 PDF 页图，确定各页属于哪个模块。
2. 用 `structured` 稿做脚手架，不直接把脏稿视作成品。
3. 在 `reader` 里按 `M0-M7` 重建。
4. 对无法自动恢复的内容，优先改成专题块或学习版速查，不强行保留坏版式。
5. 导出到 `docs/`，再进 `VitePress` 预览。
6. 检查本课完成等级，不满足标准就不标成更高等级。

## 页面类型规则

### 普通正文页

- 保持原书顺序
- 日文原句在前，必要中文说明在后

### 对话页

- 用 `会话 A/B/C/D` 或 `场景 1/2/3`
- 不允许角色顺序塌掉

### 语法页

- 一个语法点一个小节
- 必须有规则说明
- 必须有例句

### 图示页

- 不强行复刻原版布局
- 改成“速查表 / 分类清单 / 专题块”
- 如果图本身是学习重点，再考虑补图

### 练习页

- 能保留原题就尽量保留
- 依赖录音 / 路线图 / 日历 / 插图时，至少保留：
  - 题号
  - 例题
  - 小题范围
  - 训练重点

### 补充页 / 文化页

- 可以拆成专题块
- 必须说明它和本课主内容的关系

## 图像处理规范

### A 类：文字主导页

- 课文、语法、生词表这类页面，信息主体是文字
- 默认不保留原页图
- 直接做成纯文字重排版

### B 类：图辅助页

- 原页有插图，但主要信息可以转成文字清单或专题块
- 默认做“文字主版 + 可选参考图”
- 如果图本身不影响理解，可以不放图

### C 类：图本体信息页

- 路线图、流程图、看图答题、图示词汇等页面
- 必须至少保留一张参考图
- 同时提供文字版说明、题干或转写，避免只有图片

### D 类：装饰性图片页

- 不承载核心学习信息
- 直接删去，不进入站点正文

## 图像落地规则

- 参考图只放关键页，不追求每课都塞图
- 没有彩色原图不是问题，黑白页图可接受
- 图下方必须配文字说明，不能只丢图片
- 如果图页已经被完整文字化，图片可以降级为“参考图”
- 如果图就是题目本体，图片必须保留

## 文本规范

- 默认保留 `> Page xxx`
- 不伪造“全文完成”
- 不确定的内容宁可标注“待核”，也不要硬猜
- 中文说明以“帮助理解”为目标，不把整课改写成中文讲义

### 注音（`ruby`）规范

- 当前 `reader` 默认采用“初学者模式”，允许在**日语正文层**使用重度注音
- “日语正文层”包括：
  - 基本课文
  - 会话 / 应用课文
  - 语法页例句
  - 练习题面
  - 图示页中的日语条目
- 中文说明段、中文导语、纯中文专栏正文默认**不做整段注音**
- 词汇表里已经通过 `读法（汉字）` 明示读音的代码行，不再额外套 `ruby`
- 优先给这些内容补注音：
  - 人名 / 地名 / 专有名词
  - 动词变形后的正文出现形
  - 初级阶段容易误读的时间、日期、数量说法
  - 图示页和专题页里脱离上下文也要独立阅读的词条
- `ruby` 的目标是“帮助入门者直接读”，不是“把整页变成注音噪音”
- 如果某一页已经因为 `ruby` 过密而明显影响排版，可降级处理：
  - 正文句子保留重度注音
  - 中文说明和专栏段落只保留关键术语注音

## 当前试点：第 5-8 课

| 课次 | 本轮重点 | 目标 |
| --- | --- | --- |
| `第5课` | `A` 类文字页为主 | 稳定为 `S3` 基线样板 |
| `第6课` | `C` 类路线图参考图 + 练习完整度 | 从“摘要版”推进到“高保真学习版” |
| `第7课` | `B/C` 类动作图示页 + 练习完整度 | 让图示页不再只是简化速查 |
| `第8课` | `B/C` 类词汇图示页与标识页 | 解决缺页，完成补充专题重排 |

## Lesson 审稿清单

- `M1` 是否存在空对话、缺句、错序
- `M2` 是否每个语法点都有规则和例句
- `M4` 是否能连续阅读，不依赖猜测
- `M5` 是否明确区分“全文题面”和“保留题型”
- `M7` 是否仍有挂起页；如有，必须显式写出页码和原因

## 结论

后续每一课都按这份规范推进，不再接受“这一课像教材、下一课像读书笔记”的状态。
