## 语法解释

### 1. 一类形容词并列：`〜くて`

两个以上 `い` 形容词并列时，把前面的 `い` 变成 `くて`：

- ホテルの <ruby>部屋<rt>へや</rt></ruby>は <ruby>広くて<rt>ひろくて</rt></ruby> <ruby>明るい<rt>あかるい</rt></ruby>です。  
- この <ruby>本<rt>ほん</rt></ruby>は <ruby>安くて<rt>やすくて</rt></ruby> おもしろいです。  
- この りんごは <ruby>大きくて<rt>おおきくて</rt></ruby>、<ruby>甘くて<rt>あまくて</rt></ruby>、おいしいです。  

修饰名词时也一样：

- <ruby>軽くて<rt>かるくて</rt></ruby> <ruby>暖かい<rt>あたたかい</rt></ruby> コート  

### 2. 二类形容词并列：`〜で`

二类形容词并列时，用：

`二类形容词 + で`

- この コンピュータの <ruby>操作<rt>そうさ</rt></ruby>は <ruby>簡単<rt>かんたん</rt></ruby>で <ruby>便利<rt>べんり</rt></ruby>です。  
- <ruby>森<rt>もり</rt></ruby>さんは <ruby>親切<rt>しんせつ</rt></ruby>で <ruby>優しい<rt>やさしい</rt></ruby>です。  
- この <ruby>公園<rt>こうえん</rt></ruby>は <ruby>静か<rt>しずか</rt></ruby>で <ruby>広い<rt>ひろい</rt></ruby>です。  

### 3. 名词并列：`A で B です`

表示一个人或东西具有两层身份、属性时：

- スミスさんは <ruby>旅行会社<rt>りょこうがいしゃ</rt></ruby>の <ruby>社員<rt>しゃいん</rt></ruby>で、<ruby>営業部<rt>えいぎょうぶ</rt></ruby>の <ruby>部長<rt>ぶちょう</rt></ruby>です。  
- この <ruby>機械<rt>きかい</rt></ruby>は <ruby>日中電気<rt>にっちゅうでんき</rt></ruby>の <ruby>製品<rt>せいひん</rt></ruby>で、<ruby>中国製<rt>ちゅうごくせい</rt></ruby>です。  

> Page 209

### 4. `〜ています` 还可以表示结果状态

除了“正在进行”，`〜ています` 还可以表示“动作结束后留下的状态”：

- <ruby>森<rt>もり</rt></ruby>さんは <ruby>車<rt>くるま</rt></ruby>を <ruby>持って<rt>もって</rt></ruby>います。  
- わたしは <ruby>北京<rt>ペキン</rt></ruby>に <ruby>住んで<rt>すんで</rt></ruby>います。  
- <ruby>田中<rt>たなか</rt></ruby>さんを <ruby>知って<rt>しって</rt></ruby>いますか。  

否定时：

- いいえ、<ruby>持って<rt>もって</rt></ruby>いません。  
- いいえ、<ruby>知りません<rt>しりません</rt></ruby>。  

这里 `知っています` 的否定不能说 `知っていません`，要说 `知りません`。

### 5. `が` 表示转折

连接前后两个分句，表示“但是 / 不过”：

- 3か<ruby>月<rt>げつ</rt></ruby> パソコンを <ruby>勉強しました<rt>べんきょうしました</rt></ruby>が、まだ あまり できません。  
- この レストランは おいしいですが、ちょっと <ruby>高い<rt>たかい</rt></ruby>です。  

### 6. `まだ`

`まだ` 表示“还没到那个程度”。常和否定搭配：

- 2<ruby>年<rt>ねん</rt></ruby> <ruby>勉強しました<rt>べんきょうしました</rt></ruby>。でも、まだ あまり できません。  
- <ruby>小野<rt>おの</rt></ruby>さんは まだ <ruby>結婚して<rt>けっこんして</rt></ruby>いません。

## 表达及词语讲解

### 1. 描写外貌的常见搭配

本课常见的身体部位 + 形容词搭配：

- <ruby>背<rt>せ</rt></ruby>が <ruby>高い<rt>たかい</rt></ruby> / <ruby>低い<rt>ひくい</rt></ruby>
- <ruby>脚<rt>あし</rt></ruby>が <ruby>長い<rt>ながい</rt></ruby> / <ruby>短い<rt>みじかい</rt></ruby>
- <ruby>目<rt>め</rt></ruby>が <ruby>大きい<rt>おおきい</rt></ruby> / <ruby>細い<rt>ほそい</rt></ruby>
- <ruby>鼻<rt>はな</rt></ruby>が <ruby>高い<rt>たかい</rt></ruby> / <ruby>低い<rt>ひくい</rt></ruby>
- <ruby>顔<rt>かお</rt></ruby>が <ruby>小さい<rt>ちいさい</rt></ruby> / <ruby>大きい<rt>おおきい</rt></ruby>

### 2. `〜費 / 〜料 / 〜代`

汉语里常说“……费”，日语根据对象不同有不同后缀：

- `〜費`：<ruby>交通費<rt>こうつうひ</rt></ruby>、<ruby>食費<rt>しょくひ</rt></ruby>
- `〜料`：<ruby>入場料<rt>にゅうじょうりょう</rt></ruby>、<ruby>使用料<rt>しようりょう</rt></ruby>、<ruby>送料<rt>そうりょう</rt></ruby>
- `〜代`：<ruby>電気代<rt>でんきだい</rt></ruby>、<ruby>水道代<rt>すいどうだい</rt></ruby>、タクシー<ruby>代<rt>だい</rt></ruby>

### 3. `IDカード`

`IDカード` 指身份证件、员工证之类的卡。  
课文里是进入展览会场时要出示的证件。

### 4. `ちゃんと`

表示“好好地、规规矩矩地、确实地”：

- ちゃんと <ruby>持って<rt>もって</rt></ruby>います。  
- ちゃんと <ruby>片づけて<rt>かたづけて</rt></ruby> ください。  

> Page 211

### 5. `ほら`

提醒对方看某样东西时说：

- ほら、これです。  
- ほら、<ruby>森<rt>もり</rt></ruby>さん。この <ruby>書類<rt>しょるい</rt></ruby>、<ruby>間違い<rt>まちがい</rt></ruby>が ありますよ。  

### 6. `ずいぶん`

表示程度很高，常带有“比想象中更……”的感觉：

- <ruby>李<rt>り</rt></ruby>さん、<ruby>日本語<rt>にほんご</rt></ruby>が ずいぶん <ruby>上手<rt>じょうず</rt></ruby>ですね。  
- その かばん、ずいぶん <ruby>大きい<rt>おおきい</rt></ruby>ですね。  

### 7. `派手` 和 `地味`

- `派手`：耀眼、花哨
- `地味`：朴素、不显眼

日本人传统审美里，太夸张、太亮眼的东西常会被形容成 `派手`。

### 8. `嫌いじゃありません`

字面是否定，但意思不是“讨厌”，而是：

- 不算讨厌
- 也还可以
- 我并不排斥

语气比直接说 `好きです` 要委婉。

- `おもしろい / にぎやか`

- 例：<ruby>旅行<rt>りょこう</rt></ruby> / <ruby>天気<rt>てんき</rt></ruby>が いい / <ruby>楽しい<rt>たのしい</rt></ruby>
