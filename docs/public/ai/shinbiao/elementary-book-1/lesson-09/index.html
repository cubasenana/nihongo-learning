## 语法解释

### 1. `い` 形容词做谓语

本课第一次系统学习 `い` 形容词。它们一般以 `い` 结尾，直接接在名词后面描述性质。

- <ruby>四川料理<rt>しせんりょうり</rt></ruby>は <ruby>辛い<rt>からい</rt></ruby>です。  
  四川菜很辣。
- この スープは <ruby>熱い<rt>あつい</rt></ruby>です。  
  这个汤很烫。
- この <ruby>国<rt>くに</rt></ruby>は <ruby>広い<rt>ひろい</rt></ruby>です。  
  这个国家很辽阔。

### 2. `い` 形容词的否定与过去

`い` 形容词的变化方式很重要：

| 形式 | 规则 | 例子 |
| --- | --- | --- |
| 肯定 | `〜いです` | <ruby>熱い<rt>あつい</rt></ruby>です |
| 否定 | `〜くないです / 〜くありません` | <ruby>熱くない<rt>あつくない</rt></ruby>です |
| 过去 | `〜かったです` | <ruby>楽しかった<rt>たのしかった</rt></ruby>です |
| 过去否定 | `〜くなかったです / 〜くありませんでした` | <ruby>寒くなかった<rt>さむくなかった</rt></ruby>です |

例如：

- この スープは <ruby>熱くない<rt>あつくない</rt></ruby>です。  
  这个汤不烫。
- <ruby>昨日<rt>きのう</rt></ruby>は <ruby>寒くなかった<rt>さむくなかった</rt></ruby>です。  
  昨天不冷。
- <ruby>旅行<rt>りょこう</rt></ruby>は とても <ruby>楽しかった<rt>たのしかった</rt></ruby>です。  
  旅行非常愉快。

`いい` 比较特殊：

- 否定：`よくないです / よくありません`
- 过去：`よかったです`
- 过去否定：`よくなかったです / よくありませんでした`

### 3. `い` 形容词直接修饰名词

`い` 形容词修饰名词时，直接放在名词前面，**中间不能加 `の`**。

- <ruby>広い<rt>ひろい</rt></ruby> <ruby>国<rt>くに</rt></ruby>  
  幅员辽阔的国家
- <ruby>青い<rt>あおい</rt></ruby> <ruby>海<rt>うみ</rt></ruby>  
  蓝色的大海
- おいしい <ruby>料理<rt>りょうり</rt></ruby>  
  好吃的菜

### 4. `を` 改成 `は`

已经学过的宾语句里，如果想把宾语拿出来做主题或对比，就把 `を` 改成 `は`。

- わたしは コーヒーを <ruby>飲みません<rt>のみません</rt></ruby>。  
- わたしは コーヒーは <ruby>飲みません<rt>のみません</rt></ruby>。  

第二句的意思更像“咖啡我不喝”。

### 5. `あまり 〜 ません / 〜 ないです`

`あまり` 要和否定搭配，表示“不太……”。

- この スープは あまり <ruby>熱くない<rt>あつくない</rt></ruby>です。  
  这个汤不太热。
- <ruby>試験<rt>しけん</rt></ruby>は あまり <ruby>難しくありませんでした<rt>むずかしくありませんでした</rt></ruby>。  
  考试不太难。

常见程度副词的大致轻重：

- `とても / たいへん`：很，非常
- `少し / ちょっと`：有一点
- `あまり 〜 ません`：不太……
- `全然 〜 ません`：完全不……

### 6. 本课常见 `い` 形容词

- <ruby>新しい<rt>あたらしい</rt></ruby>：新
- <ruby>古い<rt>ふるい</rt></ruby>：旧
- <ruby>熱い<rt>あつい</rt></ruby>：热，烫
- <ruby>冷たい<rt>つめたい</rt></ruby>：凉
- <ruby>高い<rt>たかい</rt></ruby>：高，贵
- <ruby>安い<rt>やすい</rt></ruby>：便宜
- <ruby>多い<rt>おおい</rt></ruby>：多
- <ruby>少ない<rt>すくない</rt></ruby>：少
- <ruby>難しい<rt>むずかしい</rt></ruby>：难
- <ruby>やさしい<rt>やさしい</rt></ruby>：容易

## 表达及词语讲解

### 1. `多い / 少ない`

`多い` 和 `少ない` 虽然是 `い` 形容词，但一般**不单独直接修饰名词**。

- <ruby>公園<rt>こうえん</rt></ruby>に たくさんの <ruby>人<rt>ひと</rt></ruby>が います。  
  公园里有很多人。

不说：

- `× 多い本`

### 2. `〜用`

`〜用` 表示“……用的”。

- <ruby>子供用<rt>こどもよう</rt></ruby>：孩子用的
- <ruby>女性用<rt>じょせいよう</rt></ruby>：女性用的
- ワイン<ruby>用<rt>よう</rt></ruby>の グラス：喝葡萄酒的杯子

### 3. `あら`

`あら` 多用于女性，表示惊讶或突然注意到什么。

- あら、<ruby>李さん<rt>りさん</rt></ruby>が いませんね。  
  咦，小李不在呀。

### 4. `熱くないですか`

这是“确认对方是不是和自己有同样看法”的问法。  
回答时和中文习惯不完全一样：

- <ruby>熱くない<rt>あつくない</rt></ruby>ですか。  
  不烫吗？
- はい、<ruby>熱くない<rt>あつくない</rt></ruby>です。  
  是的，不烫。
- いいえ、ちょうど いいです。  
  不，正合适。

### 5. `お湯` 和 `水`

在日语里：

- `お湯`：热水 / 开水
- `水`：凉水

所以可以说：

- <ruby>熱い<rt>あつい</rt></ruby> お<ruby>湯<rt>ゆ</rt></ruby>
- <ruby>冷たい<rt>つめたい</rt></ruby> <ruby>水<rt>みず</rt></ruby>

但一般不说 `熱い水`。

### 6. 常见味觉词

- <ruby>甘い<rt>あまい</rt></ruby>：甜
- <ruby>辛い<rt>からい</rt></ruby>：辣
- <ruby>塩辛い<rt>しおからい</rt></ruby> / しょっぱい：咸
- <ruby>酸っぱい<rt>すっぱい</rt></ruby>：酸
- <ruby>苦い<rt>にがい</rt></ruby>：苦
- おいしい：好吃 / 好喝
- まずい：不好吃 / 不好喝

#### 场景 3：温泉水不好喝

### 练习 I

#### 1. 把 `い` 形容词改成否定式

- 例：かばんは <ruby>大きい<rt>おおきい</rt></ruby>です → この かばんは <ruby>大きくない<rt>おおきくない</rt></ruby>です
- 练习词：
  - この お<ruby>茶<rt>ちゃ</rt></ruby>は <ruby>熱い<rt>あつい</rt></ruby>
  - パーティーは <ruby>楽しい<rt>たのしい</rt></ruby>
  - この <ruby>料理<rt>りょうり</rt></ruby>は <ruby>辛い<rt>からい</rt></ruby>
  - <ruby>日本<rt>にほん</rt></ruby>の <ruby>食べ物<rt>たべもの</rt></ruby>は おいしい

#### 2. 把过去式改成过去否定式

- 例：<ruby>公園<rt>こうえん</rt></ruby>は <ruby>広かった<rt>ひろかった</rt></ruby>です → <ruby>広くなかった<rt>ひろくなかった</rt></ruby>です
- 练习词：
  - <ruby>駅<rt>えき</rt></ruby>は <ruby>遠かった<rt>とおかった</rt></ruby>
  - その <ruby>学校<rt>がっこう</rt></ruby>は <ruby>近かった<rt>ちかかった</rt></ruby>
  - <ruby>天気<rt>てんき</rt></ruby>は <ruby>悪かった<rt>わるかった</rt></ruby>

#### 3. 用 `はい / いいえ` 回答

- その <ruby>靴<rt>くつ</rt></ruby>は <ruby>新しい<rt>あたらしい</rt></ruby>ですか。  
- <ruby>昨日<rt>きのう</rt></ruby>は <ruby>暑かった<rt>あつかった</rt></ruby>ですか。  
- <ruby>旅行<rt>りょこう</rt></ruby>は <ruby>楽しかった<rt>たのしかった</rt></ruby>ですか。  

#### 4. 练 `形容词 + 名词`

- <ruby>安い<rt>やすい</rt></ruby> かばん
- おいしい ケーキ
- かわいい <ruby>花<rt>はな</rt></ruby>
- <ruby>熱い<rt>あつい</rt></ruby> お<ruby>茶<rt>ちゃ</rt></ruby>

### 练习 II

#### 1. 写反义词

- <ruby>熱い<rt>あつい</rt></ruby> ↔ <ruby>冷たい<rt>つめたい</rt></ruby>
- <ruby>新しい<rt>あたらしい</rt></ruby> ↔ <ruby>古い<rt>ふるい</rt></ruby>
- <ruby>高い<rt>たかい</rt></ruby> ↔ <ruby>安い<rt>やすい</rt></ruby>
- <ruby>広い<rt>ひろい</rt></ruby> ↔ <ruby>狭い<rt>せまい</rt></ruby>

#### 2. 在句子中填入适当词语

- <ruby>日本語<rt>にほんご</rt></ruby>は <ruby>難しい<rt>むずかしい</rt></ruby>ですか。  
- いいえ、（　　　　）です。  
- あなたの うちは <ruby>大きい<rt>おおきい</rt></ruby>ですか。  
- いいえ、（　　　　）です。  

#### 3. 本课翻译题

- 四川菜很辣。  
- 天安门是很雄伟的建筑。  
- 这个汤不太热。
