# 第3单元 小李在箱根 概览

> Pages: 130-130

## 单元导语

11 月初，小李和小野一起去箱根采访。秋天的箱根正迎来红叶的季节；采访的前一天，小李和小野住进了箱根的旅馆。

## 课次与应用课文

### 第9課 四川料理は辛いです

应用课文：`温泉`

### 第10課 京都の 紅葉は 有名です

应用课文：`美術館`

### 第11課 小野さんは 歌が 好きです

应用课文：`お土産`

### 第12課 李さんは 森さんより 若いです

应用课文：`お酒とお茶`
