## 语法解释

### 1. 动词的 `ない形`

本课学习动词否定的基础形式 `ない形`。  
变换规律：

- 一类动词：把最后一个音改成 `あ段 + ない`
- 二类动词：去掉 `ます`，直接加 `ない`
- 三类动词：`来ます → 来ない`，`します → しない`

例如：

- <ruby>行きます<rt>いきます</rt></ruby> → <ruby>行かない<rt>いかない</rt></ruby>
- <ruby>買います<rt>かいます</rt></ruby> → <ruby>買わない<rt>かわない</rt></ruby>
- <ruby>食べます<rt>たべます</rt></ruby> → <ruby>食べない<rt>たべない</rt></ruby>
- <ruby>見ます<rt>みます</rt></ruby> → <ruby>見ない<rt>みない</rt></ruby>

### 2. `〜ないで ください`

表示“请不要……”：

- <ruby>部屋<rt>へや</rt></ruby>の かぎを <ruby>忘れないで<rt>わすれないで</rt></ruby> ください。  
- この <ruby>部屋<rt>へや</rt></ruby>に <ruby>入らないで<rt>はいらないで</rt></ruby> ください。  
- <ruby>無理<rt>むり</rt></ruby>を しないで ください。  

> Page 245

### 3. `〜なければ なりません / 〜ないと いけません`

表示“必须……”：

- <ruby>今日<rt>きょう</rt></ruby> <ruby>早く<rt>はやく</rt></ruby> <ruby>帰らなければ<rt>かえらなければ</rt></ruby> なりません。  
- この <ruby>薬<rt>くすり</rt></ruby>は <ruby>毎日<rt>まいにち</rt></ruby> <ruby>飲まなければ<rt>のまなければ</rt></ruby> なりません。  
- すぐに <ruby>書類<rt>しょるい</rt></ruby>を <ruby>送らないと<rt>おくらないと</rt></ruby> いけません。  

`〜なければ なりません` 更正式；口语中 `〜ないと いけません`、甚至省略成 `〜ないと` 也很常见。

### 4. `〜なくても いいです`

表示“不做某事也可以”：

- <ruby>明日<rt>あした</rt></ruby>は <ruby>残業しなくても<rt>ざんぎょうしなくても</rt></ruby> いいです。  
- <ruby>靴<rt>くつ</rt></ruby>を <ruby>脱がなくても<rt>ぬがなくても</rt></ruby> いいですか。  

### 5. `名词2 は 名词1 です / 名词1 が 名词2 です`

表示“某个已知对象是什么”时，可以两种说法：

- <ruby>山田<rt>やまだ</rt></ruby><ruby>先生<rt>せんせい</rt></ruby>は あの <ruby>方<rt>かた</rt></ruby>です。  
- あの <ruby>方<rt>かた</rt></ruby>が <ruby>山田<rt>やまだ</rt></ruby><ruby>先生<rt>せんせい</rt></ruby>です。  

### 6. 疑问词作主语时用 `が`

- だれが いますか。  
- どこが <ruby>痛い<rt>いたい</rt></ruby>ですか。  
- <ruby>何<rt>なに</rt></ruby>が <ruby>好き<rt>すき</rt></ruby>ですか。

## 表达及词语讲解

### 1. `〜なければ なりません` 和 `〜ないと いけません`

两者都表示义务，但：

- `〜なければ なりません`：更正式、更书面
- `〜ないと いけません`：更口语

### 2. `上級者`

表示技能等级时，日语常说：

- <ruby>初心者<rt>しょしんしゃ</rt></ruby>
- <ruby>中級者<rt>ちゅうきゅうしゃ</rt></ruby>
- <ruby>上級者<rt>じょうきゅうしゃ</rt></ruby>

### 3. `何 + 量词 + も`

表示数量很多：

- <ruby>何度<rt>なんど</rt></ruby>も <ruby>転びました<rt>ころびました</rt></ruby>。  
- <ruby>水<rt>みず</rt></ruby>を <ruby>何杯<rt>なんばい</rt></ruby>も <ruby>飲みました<rt>のみました</rt></ruby>。  

### 4. `だいぶ`

表示“相当、很大程度上”：

- <ruby>風邪<rt>かぜ</rt></ruby>は だいぶ よく なりました。  
- ずいぶん と近い语感，但 `だいぶ` 更常用于“变化后好转很多”的场景。  

> Page 247

### 5. `大丈夫`

这里不是“男子汉”的意思，而是：

- 没关系
- 不要紧
- 我能行
