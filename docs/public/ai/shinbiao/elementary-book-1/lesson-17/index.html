## 语法解释

### 1. `名词が 欲しいです`

表示“想要某样东西”时，用：

`名词1 は 名词2 が 欲しいです`

- わたしは <ruby>新しい<rt>あたらしい</rt></ruby> <ruby>洋服<rt>ようふく</rt></ruby>が <ruby>欲しい<rt>ほしい</rt></ruby>です。  
- <ruby>何<rt>なに</rt></ruby>が <ruby>欲しい<rt>ほしい</rt></ruby>ですか。  
- <ruby>新しい<rt>あたらしい</rt></ruby> パソコンが <ruby>欲しい<rt>ほしい</rt></ruby>です。  

### 2. `动词ます形去ます + たいです`

表示“想做某事”时，用：

`动词ます形去ます + たいです`

- <ruby>映画<rt>えいが</rt></ruby>を <ruby>見たい<rt>みたい</rt></ruby>です。  
- <ruby>今日<rt>きょう</rt></ruby>は お<ruby>酒<rt>さけ</rt></ruby>を <ruby>飲みたくない<rt>のみたくない</rt></ruby>です。  
- <ruby>何<rt>なに</rt></ruby>を したいですか。  

`欲しい` 的对象只能用 `が`，但 `たい` 的对象有时可用 `を`，有时也会说成 `が`：

- <ruby>水<rt>みず</rt></ruby>を <ruby>飲みたい<rt>のみたい</rt></ruby>です。  
- <ruby>水<rt>みず</rt></ruby>が <ruby>飲みたい<rt>のみたい</rt></ruby>です。  

### 3. 全面否定：`何も / だれにも / どこへも`

表示“什么都不…… / 谁也不…… / 哪儿也不……”时：

- <ruby>何<rt>なに</rt></ruby>も したくありません。  
- だれにも <ruby>会いたくない<rt>あいたくない</rt></ruby>です。  
- どこへも <ruby>行きたくない<rt>いきたくない</rt></ruby>です。  

> Page 225

### 4. `〜ませんか`

`〜ませんか` 语法形式上是疑问句，功能上常用来邀请、建议：

- いっしょに お<ruby>茶<rt>ちゃ</rt></ruby>を <ruby>飲みません<rt>のみません</rt></ruby>か。  
- <ruby>少し<rt>すこし</rt></ruby> <ruby>休みません<rt>やすみません</rt></ruby>か。  
- <ruby>明日<rt>あした</rt></ruby> <ruby>故宮<rt>こきゅう</rt></ruby>へ <ruby>行きません<rt>いきません</rt></ruby>か。  

### 5. `〜ましょう`

把 `ます` 换成 `ましょう`，可以表示“咱们……吧”：

- ちょっと <ruby>休みましょう<rt>やすみましょう</rt></ruby>。  
- そろそろ <ruby>行きましょう<rt>いきましょう</rt></ruby>。  
- いっしょに お<ruby>茶<rt>ちゃ</rt></ruby>を <ruby>飲みましょう<rt>のみましょう</rt></ruby>。  

它也可以作为对别人邀请的回应：

- コンサートへ <ruby>行きません<rt>いきません</rt></ruby>か。  
- いいですね。<ruby>行きましょう<rt>いきましょう</rt></ruby>。  

### 6. `何でも / いつでも / だれでも`

在肯定句里，`疑问词 + でも` 表示“无论什么 / 无论何时 / 无论谁”：

- <ruby>何でも<rt>なんでも</rt></ruby> いいです。  
- いつでも <ruby>電話<rt>でんわ</rt></ruby>して ください。  
- だれでも <ruby>分かります<rt>わかります</rt></ruby>。

## 表达及词语讲解

### 1. 对长辈别直接问 `何が欲しいですか`

`欲しいですか`、`〜たいですか` 直接问对方愿望，对熟人、晚辈、朋友没问题；  
但对长辈、上级会显得有点直接。更稳的说法通常是：

- どうですか
- 〜ませんか

### 2. `〜中に`

表示“在……结束之前”：

- <ruby>今年中<rt>ことしじゅう</rt></ruby>に <ruby>結婚したい<rt>けっこんしたい</rt></ruby>です。  
- <ruby>今日中<rt>きょうじゅう</rt></ruby>に この <ruby>仕事<rt>しごと</rt></ruby>を <ruby>終わります<rt>おわります</rt></ruby>。  
- <ruby>来週中<rt>らいしゅうちゅう</rt></ruby>に もう <ruby>一度<rt>いちど</rt></ruby> <ruby>連絡します<rt>れんらくします</rt></ruby>。  

### 3. `ぜひ`

表示强烈希望、积极邀请时常用：

- ぜひ <ruby>行きたい<rt>いきたい</rt></ruby>です。  
- ぜひ <ruby>食べたい<rt>たべたい</rt></ruby>です。  
- ぜひ <ruby>遊び<rt>あそび</rt></ruby>に <ruby>来て<rt>きて</rt></ruby> ください。  

### 4. 句尾的 `ね`

除了确认对方是否同意，`ね` 还可以用来舒缓语气、把自己的感觉轻轻抛给对方：

- まず <ruby>恋人<rt>こいびと</rt></ruby>が <ruby>欲しい<rt>ほしい</rt></ruby>ですね。  
- まじめで <ruby>優しい<rt>やさしい</rt></ruby> <ruby>人<rt>ひと</rt></ruby>が いいですね。  

> Page 227

### 5. `初詣`

`初詣（はつもうで）` 指新年第一次去神社或寺庙参拜。  
人们常为这些事祈愿：

- <ruby>健康<rt>けんこう</rt></ruby>
- <ruby>恋愛<rt>れんあい</rt></ruby>
- <ruby>結婚<rt>けっこん</rt></ruby>
- <ruby>仕事<rt>しごと</rt></ruby>
- <ruby>試験<rt>しけん</rt></ruby>

### 专栏：新年的首次参拜

日本很多人会在元旦后几天去 `初詣`。  
从学习角度看，这一页最值钱的不是文化知识本身，而是把“愿望表达”和“邀请表达”放进了很真实的生活场景里。
