## 语法解释

### 1. 数量词放在动词前

本课开始系统学习数量词。日语里数量词常常放在宾语后、动词前面：

- <ruby>卵<rt>たまご</rt></ruby>を 1<ruby>個<rt>こ</rt></ruby> <ruby>食べます<rt>たべます</rt></ruby>。  
- <ruby>本<rt>ほん</rt></ruby>を 2<ruby>冊<rt>さつ</rt></ruby> <ruby>買いました<rt>かいました</rt></ruby>。  
- <ruby>教室<rt>きょうしつ</rt></ruby>に <ruby>学生<rt>がくせい</rt></ruby>が 4<ruby>人<rt>にん</rt></ruby> います。  

常见问法：

- 何<ruby>冊<rt>さつ</rt></ruby> <ruby>買いました<rt>かいました</rt></ruby>か。  
- 何<ruby>枚<rt>まい</rt></ruby> ありますか。  

### 2. 表示持续时间时，后面不加 `に`

表示动作持续了多久时，时间数量词后面**不能加 `に`**：

- <ruby>李<rt>り</rt></ruby>さんは <ruby>毎日<rt>まいにち</rt></ruby> 7<ruby>時間<rt>じかん</rt></ruby> <ruby>働きます<rt>はたらきます</rt></ruby>。  
- <ruby>昼<rt>ひる</rt></ruby> 1<ruby>時間<rt>じかん</rt></ruby> <ruby>休みます<rt>やすみます</rt></ruby>。  
- <ruby>森<rt>もり</rt></ruby>さんは 9<ruby>時間<rt>じかん</rt></ruby> <ruby>寝ます<rt>ねます</rt></ruby>。  

### 3. `〜期間に 〜回`

在一定时间里做几次某个动作时，用：

`期间 + に + 次数`

- <ruby>李<rt>り</rt></ruby>さんは 1<ruby>週間<rt>しゅうかん</rt></ruby>に 2<ruby>回<rt>かい</rt></ruby> プールへ <ruby>行きます<rt>いきます</rt></ruby>。  
- この <ruby>花<rt>はな</rt></ruby>は 2<ruby>年<rt>ねん</rt></ruby>に 1<ruby>度<rt>ど</rt></ruby> <ruby>咲きます<rt>さきます</rt></ruby>。  

### 4. 动词的 `ます形`

把 `働きます / 休みます / 起きます / 寝ます` 这类形式统称为动词的 `ます形`。  
以后很多句型都要用到“去掉 `ます` 之后的部分”：

- <ruby>働きます<rt>はたらきます</rt></ruby> → <ruby>働き<rt>はたらき</rt></ruby>  
- <ruby>休みます<rt>やすみます</rt></ruby> → <ruby>休み<rt>やすみ</rt></ruby>  
- <ruby>泳ぎます<rt>およぎます</rt></ruby> → <ruby>泳ぎ<rt>およぎ</rt></ruby>  

### 5. `场所へ 动词ます形去ます + に 行きます / 来ます`

表示“去某处做某事”，用这个句型：

- <ruby>郵便局<rt>ゆうびんきょく</rt></ruby>へ <ruby>荷物<rt>にもつ</rt></ruby>を <ruby>出し<rt>だし</rt></ruby>に <ruby>行きます<rt>いきます</rt></ruby>。  
- プールへ <ruby>泳ぎ<rt>およぎ</rt></ruby>に <ruby>行きます<rt>いきます</rt></ruby>。  
- <ruby>友達<rt>ともだち</rt></ruby>の <ruby>家<rt>いえ</rt></ruby>へ <ruby>遊び<rt>あそび</rt></ruby>に <ruby>来ました<rt>きました</rt></ruby>。  

这里的 `に` 不是时间或地点的 `に`，而是表示“目的”。

### 6. `数量 + で + 金额`

按“几个一共多少钱”售卖时，用：

`数量 + で + 金额`

- この ケーキは 3<ruby>個<rt>こ</rt></ruby>で 500<ruby>円<rt>えん</rt></ruby>です。  
- はがきは 5<ruby>枚<rt>まい</rt></ruby>で 250<ruby>円<rt>えん</rt></ruby>です。  

但数量是 1 个时，通常不用 `で`：

- この ケーキは 1<ruby>個<rt>こ</rt></ruby> 200<ruby>円<rt>えん</rt></ruby>です。

## 表达及词语讲解

### 1. 数词和量词的读法会变化

很多量词和数字组合时，读音会发生变化，尤其是：

- `〜個`
- `〜本`
- `〜杯`
- `〜匹`

例如：

- 1<ruby>個<rt>いっこ</rt></ruby>
- 3<ruby>本<rt>さんぼん</rt></ruby>
- 1<ruby>杯<rt>いっぱい</rt></ruby>
- 6<ruby>匹<rt>ろっぴき</rt></ruby>

初学阶段不需要一次性全背完，但 `1 / 3 / 6 / 8 / 10` 这些变化要尽早熟悉。

### 2. `週に 2回`

`1週間に 2回` 常常简化成 `週に 2回`。  
同理还有：

- `日に 〜回`
- `月に 〜回`
- `年に 〜回`

例如：

- <ruby>年<rt>ねん</rt></ruby>に 4<ruby>回<rt>かい</rt></ruby>ぐらい <ruby>北京<rt>ペキン</rt></ruby>へ <ruby>行きます<rt>いきます</rt></ruby>。  
- <ruby>週<rt>しゅう</rt></ruby>に 1<ruby>回<rt>かい</rt></ruby> ボーリングを します。  

### 3. `くらい / ぐらい`

表示大概数量、时间、距离时常用：

- ここから <ruby>駅<rt>えき</rt></ruby>まで 1キロメートルぐらいです。  
- <ruby>電車<rt>でんしゃ</rt></ruby>で 1<ruby>時間<rt>じかん</rt></ruby>ぐらい かかります。  
- だいたい 20<ruby>万円<rt>まんえん</rt></ruby>ぐらいです。  

### 4. `どのぐらい かかりますか`

`かかります` 可以表示“花时间”或“花钱”：

- <ruby>家<rt>いえ</rt></ruby>から <ruby>会社<rt>かいしゃ</rt></ruby>まで どのぐらい かかりますか。  
- <ruby>修理<rt>しゅうり</rt></ruby>は どのぐらい かかりますか。  

### 5. `飲みに行きます`

如果不特别说明喝什么，`飲みに行きます` 往往默认指“去喝酒”：

- これから <ruby>森<rt>もり</rt></ruby>さんと <ruby>飲み<rt>のみ</rt></ruby>に <ruby>行きます<rt>いきます</rt></ruby>。  
- <ruby>最近<rt>さいきん</rt></ruby> あまり <ruby>飲み<rt>のみ</rt></ruby>に <ruby>行きません<rt>いきません</rt></ruby>。  

> Page 181

### 6. `とりあえず`

`とりあえず` 表示“先来…… / 暂时先……”。在居酒屋点单时非常常见：

- とりあえず <ruby>生<rt>なま</rt></ruby>ビールを 3つ お<ruby>願い<rt>ねがい</rt></ruby>します。  

意思是：先把最容易决定的先点了，别的等会儿再说。

### 7. `生ビールを 3つ`

现实口语里，点餐时常常省略量词本身，直接说：

- ビールを 2つ
- コーヒーを 1つ

严格说应该是 `2杯 / 1杯`，但口语省略很常见。

### 专栏：居酒屋里的“先点这个”

在居酒屋里，`とりあえず 生ビール` 是非常高频的说法。  
它不是“最后只喝啤酒”，而是“先把啤酒上了，别的等会儿再点”。  
这类口语场景题，重点不是逐字翻译，而是把日本人点单时的语气和节奏先听熟。
