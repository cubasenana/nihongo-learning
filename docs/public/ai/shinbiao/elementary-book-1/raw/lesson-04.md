## 语法解释

### 1. `あります` 和 `います`

表示存在时最常用的两个动词：

- `あります`：用于没有意志的事物
- `います`：用于有意志的人、动物

例如：

- <ruby>部屋<rt>へや</rt></ruby>に <ruby>机<rt>つくえ</rt></ruby>が あります。  
- <ruby>部屋<rt>へや</rt></ruby>に <ruby>猫<rt>ねこ</rt></ruby>が います。  

### 2. 存在句：`场所 に 名 が あります / います`

表示“某处有某物 / 某人”。

- <ruby>部屋<rt>へや</rt></ruby>に <ruby>机<rt>つくえ</rt></ruby>が あります。  
  房间里有桌子。
- <ruby>公園<rt>こうえん</rt></ruby>に <ruby>子供<rt>こども</rt></ruby>が います。  
  公园里有孩子。
- あそこに だれが いますか。  
  那里有谁？

### 3. 位置句：`名 は 场所 に あります / います`

表示“某物 / 某人在某处”。

- いすは <ruby>部屋<rt>へや</rt></ruby>に あります。  
  椅子在房间里。
- <ruby>吉田<rt>よしだ</rt></ruby>さんは <ruby>庭<rt>にわ</rt></ruby>に います。  
  吉田先生在院子里。
- <ruby>犬<rt>いぬ</rt></ruby>は どこに いますか。  
  狗在哪儿？

> Page 71

### 4. `名1 と 名2`

助词 `と` 可以连接两个名词，表示并列，相当于“和”。

- <ruby>時計<rt>とけい</rt></ruby>と <ruby>眼鏡<rt>めがね</rt></ruby>  
- ビールと ウイスキー  
- <ruby>居間<rt>いま</rt></ruby>に テレビと ビデオが あります。  

### 5. `上 / 下 / 前 / 後ろ / 隣 / 中 / 外`

表示具体位置时，常用：

- `名词 + の + 上`
- `名词 + の + 下`
- `名词 + の + 前`
- `名词 + の + 後ろ`
- `名词 + の + 隣`
- `名词 + の + 中`
- `名词 + の + 外`

例如：

- <ruby>机<rt>つくえ</rt></ruby>の <ruby>上<rt>うえ</rt></ruby>に <ruby>猫<rt>ねこ</rt></ruby>が います。  
- <ruby>箱<rt>はこ</rt></ruby>の <ruby>中<rt>なか</rt></ruby>に <ruby>眼鏡<rt>めがね</rt></ruby>が あります。  
- <ruby>駅<rt>えき</rt></ruby>の <ruby>外<rt>そと</rt></ruby>に <ruby>売店<rt>ばいてん</rt></ruby>が あります。  

### 6. `ね`

当说话人希望对方认同、确认时，句尾常用 `ね`。

- あそこに <ruby>犬<rt>いぬ</rt></ruby>が いますね。  
- <ruby>駅<rt>えき</rt></ruby>の <ruby>前<rt>まえ</rt></ruby>に <ruby>銀行<rt>ぎんこう</rt></ruby>が ありますね。  

### 7. `疑问词 + も + 否定`

表示全面否定。

- <ruby>教室<rt>きょうしつ</rt></ruby>に だれも いません。  
  教室里谁也没有。
- <ruby>冷蔵庫<rt>れいぞうこ</rt></ruby>に <ruby>何<rt>なん</rt></ruby>も ありません。  
  冰箱里什么也没有。

## 表达及词语讲解

### 1. `上` 的范围

日语里的 `上` 通常只表示“正上方”。  
像“墙上有开关”这种说法，日语通常说：

- <ruby>壁<rt>かべ</rt></ruby>に スイッチが あります。  

而不是：

- <ruby>壁<rt>かべ</rt></ruby>の <ruby>上<rt>うえ</rt></ruby>に …

### 2. `ええと`

回答前思考、找地图、整理信息时常说：

- ええと

有点像“嗯……我想想”。

### 3. `ご家族 / ご両親 / ご兄弟`

这些是说到对方家人时更礼貌的表达：

- ご<ruby>家族<rt>かぞく</rt></ruby>：您的家人
- ご<ruby>両親<rt>りょうしん</rt></ruby>：您的父母
- ご<ruby>兄弟<rt>きょうだい</rt></ruby>：您的兄弟姐妹

### 4. `兄弟`

`きょうだい` 不只指兄弟，也可以泛指兄弟姐妹。

> Page 73

### 5. `JR` 和 `地下鉄`

- `JR`：日本主要铁路系统之一
- `地下鉄`：地铁

在日本都市通勤语境里，这两个词非常常见。

### 6. `一人暮らし`

表示“一个人住、单身生活”。

- わたしは <ruby>一人暮らし<rt>ひとりぐらし</rt></ruby>です。  
  我一个人住。
