## 语法解释

### 1. `名1 は 名2 です`

相当于汉语的“`A 是 B`”。  
助词 `は` 提示主题，读作 `わ`。

- <ruby>李<rt>り</rt></ruby>さんは <ruby>中国人<rt>ちゅうごくじん</rt></ruby>です。  
  小李是中国人。
- わたしは <ruby>日本人<rt>にほんじん</rt></ruby>です。  
  我是日本人。
- わたしは <ruby>王<rt>おう</rt></ruby>です。  
  我姓王。

### 2. `名1 は 名2 では ありません`

相当于汉语的“`A 不是 B`”。  
口语里 `では` 也常说成 `じゃ`。

- <ruby>森<rt>もり</rt></ruby>さんは <ruby>学生<rt>がくせい</rt></ruby>では ありません。  
  森先生不是学生。
- わたしは <ruby>日本人<rt>にほんじん</rt></ruby>では ありません。  
  我不是日本人。
- わたしは <ruby>田中<rt>たなか</rt></ruby>じゃ ありません。  
  我不是田中。

### 3. `名1 は 名2 ですか`

句尾加 `か` 表示疑问，日语问句句尾一般不用 `?`。

- あなたは <ruby>小野<rt>おの</rt></ruby>さんですか。  
  您是小野女士吗？
- キムさんは <ruby>中国人<rt>ちゅうごくじん</rt></ruby>ですか。  
  金女士是中国人吗？

回答时常用：

- はい、そうです。  
- いいえ、ちがいます。  
- 分かりません。  

### 4. `名1 の 名2`

助词 `の` 连接名词和名词，表示所属、从属机构、国家或属性。

- <ruby>李<rt>り</rt></ruby>さんは <ruby>JC企画<rt>ジェーシーきかく</rt></ruby>の <ruby>社員<rt>しゃいん</rt></ruby>です。  
  小李是 JC 策划公司的职员。
- <ruby>北京旅行社<rt>ペキンりょこうしゃ</rt></ruby>は <ruby>中国<rt>ちゅうごく</rt></ruby>の <ruby>企業<rt>きぎょう</rt></ruby>です。  
  北京旅行社是中国的企业。
- デュポンさんは <ruby>大学<rt>だいがく</rt></ruby>の <ruby>先生<rt>せんせい</rt></ruby>です。  
  迪蓬先生是大学老师。

## 表达及词语讲解

### 1. 人称

- `わたし`：我
- `あなた`：你 / 您  
  但在实际会话里，直接用 `あなた` 叫对方有时会显得生硬。
- `あの人`：那个人

更自然的做法通常是：

- 用对方的姓 + `さん`
- 或者直接用职务称呼对方

### 2. `〜さん / 〜ちゃん / 〜くん`

- `〜さん`：最常见、最稳妥的称呼后缀
- `〜ちゃん`：多用于小孩或关系亲近者
- `〜くん`：多用于年轻男性、后辈或同辈男性

注意：

- 不能给自己名字后面加 `さん`
- `先生` 不能简单当作 `さん` 的替代品

### 3. 省略

日语会大量省略上下文已经明确的第一、第二人称。

- （あなたは）<ruby>吉田<rt>よしだ</rt></ruby>さんですか。  
- いいえ、（わたしは）<ruby>吉田<rt>よしだ</rt></ruby>じゃ ありません。<ruby>森<rt>もり</rt></ruby>です。  

### 4. `あっ`

表示突然注意到某件事、吃惊或碰撞时的自然反应。

- あっ、<ruby>森<rt>もり</rt></ruby>さん！  
- あっ、すみません。  

> Page 43

### 5. `どうぞ よろしく お願いします`

自我介绍结束时常说，意思接近“请多关照”。

- はじめまして、<ruby>李<rt>り</rt></ruby>です。どうぞ よろしく <ruby>お願いします<rt>おねがいします</rt></ruby>。  
- こちらこそ、どうぞ よろしく <ruby>お願いします<rt>おねがいします</rt></ruby>。  

更礼貌的说法是：

- どうぞ よろしく <ruby>お願い<rt>おねがい</rt></ruby>いたします

### 6. `会社員` 和 `社員`

- `会社員`：一种职业，意思是“公司职员”
- `社員`：某个公司的成员，通常要带所属单位

例如：

- わたしは <ruby>会社員<rt>かいしゃいん</rt></ruby>です。  
- わたしは <ruby>JC企画<rt>ジェーシーきかく</rt></ruby>の <ruby>社員<rt>しゃいん</rt></ruby>です。

### 练习 II

#### 1. 在括号中填入适当的词语

- 例：`あなたは（ 日本人 ）ですか。 → はい、日本人です。`
- `(1) 李さんは（　　　）ですか。 → はい、会社員です。`
- `(2) あなたは（　　　）ですか。 → いいえ、小野では ありません。`
- `(3) 森さんは（　　　）の 社員ですか。 → はい、JC企画の 社員です。`
- `(4) キムさんは 学生ですか。 → （　　　）、学生では ありません。`
- `(5) スミスさんは アメリカ人ですか。 → はい、（　　　）です。`

#### 2. 在正确答案上画 `○`

- 例：`李さんは 中国人ですか。 → （はい・いいえ）`
- `(1) キムさんは 中国人ですか。 → （はい・いいえ）`
- `(2) あなたは 研修生ですか。 → 研修生（です・では ありません）`
- `(3) あなたは 小野さんですか。 → （小野・小野さん）です。`
- `(4) 張さんは（会社員・学生）ですか。`
- `(5) デュポンさんは アメリカ人ですか。 → （はい・いいえ）`

#### 3. 听录音，根据自己的实际情况回答提问

- 例：`あなたは 中国人ですか。`
- 网页版保留任务结构：
  - `(1)〜(4)` 听后按自身情况回答
  - 重点练 `はい、〜です / いいえ、〜では ありません`

#### 4. 用花括号中的词语造句

- 例：`{森さん / です / は / 日本人} → 森さんは 日本人です。`
- `(1) {小野さん / 課長 / ありません / では / は}`
- `(2) {デュポンさん / 東京大学 / 教授 / の / は / です}`
- `(3) {森さん / JC企画 / 課長 / ありません / は / では / の}`
- `(4) {スミスさん / JC企画 / の / は / か / 社員 / です}`
  - 追问：`いいえ、〜では ありません。〜の 社員です。`

#### 5. 将下面的句子译成日语

- `(1) 小李是中国人。`
- `(2) 森先生不是学生。`
- `(3) 小李是 JC 策划公司的职员。`

### 专栏：成田机场

成田机场是日本的重要国际机场，位于千叶县成田市，距离东京市中心约 `60` 公里。  
原书补充页还提到：

- 机场有两个候机楼
- 与中国往来的航班主要使用第二候机楼
