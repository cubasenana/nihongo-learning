## 语法解释

### 1. `A は B より 〜です`

表示 A 比 B 更具有某种性质：

- <ruby>李<rt>り</rt></ruby>さんは <ruby>森<rt>もり</rt></ruby>さんより <ruby>若い<rt>わかい</rt></ruby>です。  
- <ruby>中国<rt>ちゅうごく</rt></ruby>は <ruby>日本<rt>にほん</rt></ruby>より <ruby>広い<rt>ひろい</rt></ruby>です。  

如果差距很大，可以加 `ずっと`：

- <ruby>中国<rt>ちゅうごく</rt></ruby>は <ruby>日本<rt>にほん</rt></ruby>より ずっと <ruby>広い<rt>ひろい</rt></ruby>です。  

### 2. `A より B の ほうが 〜です`

如果想强调“B 更……”，就用这个句型：

- <ruby>日本<rt>にほん</rt></ruby>より <ruby>中国<rt>ちゅうごく</rt></ruby>の ほうが <ruby>広い<rt>ひろい</rt></ruby>です。  
- <ruby>北京<rt>ペキン</rt></ruby>より <ruby>東京<rt>とうきょう</rt></ruby>の ほうが <ruby>暖かい<rt>あたたかい</rt></ruby>です。  

### 3. `A は B ほど 〜くない / 〜では ありません`

表示“A 没有 B 那么……”：

- <ruby>東京<rt>とうきょう</rt></ruby>の <ruby>冬<rt>ふゆ</rt></ruby>は <ruby>北京<rt>ペキン</rt></ruby>の <ruby>冬<rt>ふゆ</rt></ruby>ほど <ruby>寒くない<rt>さむくない</rt></ruby>です。  
- <ruby>神戸<rt>こうべ</rt></ruby>は <ruby>大阪<rt>おおさか</rt></ruby>ほど にぎやかでは ありません。  

注意句尾必须是否定形式。

### 4. `A の 中で B が いちばん 〜です`

在一个范围内说“最……”：

- スポーツの <ruby>中<rt>なか</rt></ruby>で サッカーが いちばん おもしろいです。  
- <ruby>日本料理<rt>にほんりょうり</rt></ruby>の <ruby>中<rt>なか</rt></ruby>で <ruby>寿司<rt>すし</rt></ruby>が いちばん おいしいです。  

表示场所或时间范围时，常用：

- <ruby>日本<rt>にほん</rt></ruby>で いちばん <ruby>高い<rt>たかい</rt></ruby> <ruby>山<rt>やま</rt></ruby>
- <ruby>一年<rt>いちねん</rt></ruby>で いちばん <ruby>寒い<rt>さむい</rt></ruby> <ruby>月<rt>つき</rt></ruby>

### 5. `A と B と どちらが 〜ですか`

两者比较时用 `どちら`。

- <ruby>日本語<rt>にほんご</rt></ruby>と <ruby>英語<rt>えいご</rt></ruby>と どちらが <ruby>難しい<rt>むずかしい</rt></ruby>ですか。  
- <ruby>紅茶<rt>こうちゃ</rt></ruby>と コーヒーと どちらが <ruby>好き<rt>すき</rt></ruby>ですか。  

回答：

- <ruby>日本語<rt>にほんご</rt></ruby>の ほうが <ruby>難しい<rt>むずかしい</rt></ruby>です。  
- どちらも <ruby>好き<rt>すき</rt></ruby>です。  

### 6. 三个以上时的疑问词

当比较对象超过两个时，要换疑问词：

- `だれが`：谁最……
- `どれが / どの 〜 が / 何が`：哪个 / 什么最……
- `いつが`：什么时候最……

例如：

- この <ruby>中<rt>なか</rt></ruby>で どの <ruby>料理<rt>りょうり</rt></ruby>が いちばん おいしいですか。  
- <ruby>季節<rt>きせつ</rt></ruby>の <ruby>中<rt>なか</rt></ruby>で いつが いちばん <ruby>好き<rt>すき</rt></ruby>ですか。

## 表达及词语讲解

### 1. `最近`

表示从较近的过去到现在的一段时间：

- <ruby>最近<rt>さいきん</rt></ruby> <ruby>少し<rt>すこし</rt></ruby> <ruby>太りました<rt>ふとりました</rt></ruby>。  

它一般不表示将来的“最近”。

### 2. `〜が いいです`

根据上下文不同，可以表示“选这个比较好”或“我更喜欢这个”。

- <ruby>箱根<rt>はこね</rt></ruby>の お<ruby>土産<rt>みやげ</rt></ruby>は <ruby>何<rt>なに</rt></ruby>が いいですか。  
- わたしは <ruby>肉<rt>にく</rt></ruby>より <ruby>魚<rt>さかな</rt></ruby>の ほうが いいです。  

### 3. `やっぱり`

口语里比 `やはり` 更常见，表示“果然”“还是”。

- やっぱり <ruby>雨<rt>あめ</rt></ruby>が <ruby>降りました<rt>ふりました</rt></ruby>ね。  
- やっぱり ウーロン<ruby>茶<rt>ちゃ</rt></ruby>です。  

### 4. 句尾的 `が`

日本人常在句尾留一个 `が`，把后半句省掉，让语气更柔和、更含蓄。

- わたしは ジャスミン<ruby>茶<rt>ちゃ</rt></ruby>の ほうが <ruby>好き<rt>すき</rt></ruby>ですが……。
