## 基本课文

### 基本句

1. これは <ruby>本<rt>ほん</rt></ruby>です。  
2. それは <ruby>何<rt>なん</rt></ruby>ですか。  
3. あれは だれの <ruby>傘<rt>かさ</rt></ruby>ですか。  
4. この カメラは スミスさんのです。  

### 会话 A

甲：これは テレビですか。  
乙：いいえ、それは テレビでは ありません。パソコンです。  

### 会话 B

甲：それは <ruby>何<rt>なん</rt></ruby>ですか。  
乙：これは <ruby>日本語<rt>にほんご</rt></ruby>の <ruby>本<rt>ほん</rt></ruby>です。  

### 会话 C

甲：<ruby>森<rt>もり</rt></ruby>さんの かばんは どれですか。  
乙：あの かばんです。  

### 会话 D

甲：その ノートは だれのですか。  
乙：わたしのです。

## 语法解释

### 1. `これ / それ / あれ は 名 です`

这三个词都用于指事物：

- `これ`：离说话人近的
- `それ`：离听话人近的
- `あれ`：离双方都远的

例如：

- これは <ruby>本<rt>ほん</rt></ruby>です。  
  这是书。
- それは パソコンです。  
  那是电脑。
- あれは テレビです。  
  那是电视。

### 2. `だれですか / 何ですか`

- 不知道“人”是谁时，用 `だれ`
- 不知道“东西”是什么时，用 `何`

例如：

- それは <ruby>何<rt>なん</rt></ruby>ですか。  
  那是什么？
- あの <ruby>人<rt>ひと</rt></ruby>は だれですか。  
  那个人是谁？

更礼貌的“哪位”是：

- どなた

### 3. `名1 の 名2`

`の` 可以表示所属。

- わたしの かぎ  
  我的钥匙
- <ruby>田中<rt>たなか</rt></ruby>さんの <ruby>車<rt>くるま</rt></ruby>  
  田中先生的车

> Page 51

### 4. `この / その / あの + 名`

修饰名词时，要把：

- `これ / それ / あれ`

改成：

- `この / その / あの`

例如：

- この カメラは スミスさんのです。  
  这个照相机是史密斯先生的。
- その <ruby>自転車<rt>じてんしゃ</rt></ruby>は <ruby>森<rt>もり</rt></ruby>さんのです。  
  那辆自行车是森先生的。
- あの ノートは だれのですか。  
  那个笔记本是谁的？

### 5. `どれ / どの + 名`

在三个以上的事物中问“哪一个”时：

- 单独用：`どれ`
- 修饰名词用：`どの`

例如：

- <ruby>森<rt>もり</rt></ruby>さんの かばんは どれですか。  
  森先生的包是哪一个？
- <ruby>小野<rt>おの</rt></ruby>さんの <ruby>机<rt>つくえ</rt></ruby>は どの <ruby>机<rt>つくえ</rt></ruby>ですか。  
  小野女士的桌子是哪一张？

## 表达及词语讲解

### 1. `方`

`方（かた）` 是 `人` 的礼貌说法。

- あの<ruby>方<rt>かた</rt></ruby>は <ruby>田中<rt>たなか</rt></ruby>さんです。  
  那位是田中先生。

所以：

- <ruby>日本人<rt>にほんじん</rt></ruby>ですか  

直接这样问有时会显得太直。更礼貌的说法是：

- <ruby>日本<rt>にほん</rt></ruby>の <ruby>方<rt>かた</rt></ruby>ですか

### 2. 问年龄

- <ruby>何歳<rt>なんさい</rt></ruby>ですか  
- おいくつですか

后者更礼貌。

### 3. `どうぞ`

给对方东西、请对方收下时常说：

- これは <ruby>中国<rt>ちゅうごく</rt></ruby>の <ruby>名産品<rt>めいさんひん</rt></ruby>です。どうぞ。  

### 4. 叹词：`えっ` / `わあ`

- `えっ`：吃惊、没听清时的反应
- `わあ`：感动、惊喜时的反应

> Page 53

### 5. `はい` 和 `ええ`

`ええ` 是 `はい` 较随意的说法。  
但被别人点名时，一般用 `はい`，不用 `ええ`。

### 6. 外来词

日语里从外语借来的词，一般用片假名写。

例如：

- シルク：silk
- ハンカチ：handkerchief
- スミス：Smith
- ロンドン：London

### 7. `どうも ありがとうございます`

`どうも` 可以加强谢意。  
略表示感谢时，也可以只说：

- どうも

### 专栏：家庭的形态

原书补充页强调了日本家庭结构的变化：

- 20 世纪 60 年代平均每个家庭约 `4.14` 人
- 到 2000 年已经减少到约 `2.67` 人
- 双亲加两个孩子的家庭在减少
- 双亲加一个孩子、或者没有孩子的家庭在增加
- 晚婚和单身生活方式都在影响家庭形态
