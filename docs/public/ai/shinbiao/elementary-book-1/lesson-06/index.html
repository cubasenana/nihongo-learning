## 语法解释

### 1. `名[场所]へ + 动`

表示移动行为的目的地，用助词 `へ`。  
这里的 `へ` 读作 `え`。

- <ruby>吉田さん<rt>よしださん</rt></ruby>は <ruby>中国<rt>ちゅうごく</rt></ruby>へ <ruby>行きます<rt>いきます</rt></ruby>。  
  吉田先生去中国。
- <ruby>森さん<rt>もりさん</rt></ruby>は <ruby>日本<rt>にほん</rt></ruby>へ <ruby>帰ります<rt>かえります</rt></ruby>。  
  森先生回日本。
- <ruby>李さん<rt>りさん</rt></ruby>は どこへ <ruby>行きました<rt>いきました</rt></ruby>か。  
  小李去哪儿了？

### 2. `名[场所]から + 动`

表示移动的起点，用助词 `から`。

- <ruby>李さん<rt>りさん</rt></ruby>は <ruby>先月<rt>せんげつ</rt></ruby> <ruby>北京<rt>ペキン</rt></ruby>から <ruby>来ました<rt>きました</rt></ruby>。  
  小李上个月从北京来。
- あの <ruby>方<rt>かた</rt></ruby>は どこから <ruby>来ました<rt>きました</rt></ruby>か。  
  那个人从哪儿来？

### 3. `名[人]と + 动`

共同做某事的对象，用助词 `と` 表示。

- <ruby>小野さん<rt>おのさん</rt></ruby>は <ruby>友達<rt>ともだち</rt></ruby>と <ruby>帰りました<rt>かえりました</rt></ruby>。  
  小野女士和朋友一起回去了。
- <ruby>李さん<rt>りさん</rt></ruby>は だれと <ruby>日本<rt>にほん</rt></ruby>へ <ruby>来ました<rt>きました</rt></ruby>か。  
  小李和谁一起来日本的？

### 4. `名[交通工具]で + 动`

交通手段用助词 `で` 表示。  
如果不是乘坐交通工具，而是步行，一般说 `歩いて`。

- <ruby>上海<rt>シャンハイ</rt></ruby>まで <ruby>飛行機<rt>ひこうき</rt></ruby>で <ruby>行きます<rt>いきます</rt></ruby>。  
  坐飞机去上海。
- わたしは バスで <ruby>帰ります<rt>かえります</rt></ruby>。  
  我坐公共汽车回去。
- <ruby>李さん<rt>りさん</rt></ruby>は アパートへ <ruby>歩いて<rt>あるいて</rt></ruby> <ruby>帰りました<rt>かえりました</rt></ruby>。  
  小李步行回了公寓。

### 5. `名[场所]から 名[场所]まで + 动`

表示移动范围时，起点用 `から`，终点用 `まで`。

- <ruby>森さん<rt>もりさん</rt></ruby>は <ruby>東京<rt>とうきょう</rt></ruby>から <ruby>広島<rt>ひろしま</rt></ruby>まで <ruby>新幹線<rt>しんかんせん</rt></ruby>で <ruby>行きます<rt>いきます</rt></ruby>。  
  森先生从东京坐新干线去广岛。
- <ruby>李さん<rt>りさん</rt></ruby>は <ruby>駅<rt>えき</rt></ruby>から アパートまで <ruby>歩いて<rt>あるいて</rt></ruby> <ruby>帰りました<rt>かえりました</rt></ruby>。  
  小李从车站走回公寓。

### 6. `に / で / へ / から / まで / と + は`

助词后面加 `は`，可以形成带对比意味的说法。

- わたしの <ruby>部屋<rt>へや</rt></ruby>には <ruby>電話<rt>でんわ</rt></ruby>が ありません。  
  我的房间里没有电话。
- <ruby>韓国<rt>かんこく</rt></ruby>へは <ruby>行きました<rt>いきました</rt></ruby>。<ruby>中国<rt>ちゅうごく</rt></ruby>へは <ruby>行きませんでした<rt>いきませんでした</rt></ruby>。  
  我去了韩国，没去中国。

## 表达及词语讲解

### 1. `たしか`

`たしか〜` 表示“凭记忆大概是……”，说话人并不是百分之百确定。

- あの <ruby>人<rt>ひと</rt></ruby>は だれですか。  
  那个人是谁？
- たしか <ruby>李さん<rt>りさん</rt></ruby>の <ruby>会社<rt>かいしゃ</rt></ruby>の <ruby>人<rt>ひと</rt></ruby>です。  
  我记得好像是小李公司的人。

### 2. `〜です` 的用法 ②

在上下文已经明确时，可以用 `〜です` 简洁代替前面的完整句子。

- <ruby>何で<rt>なんで</rt></ruby> <ruby>帰りました<rt>かえりました</rt></ruby>か。タクシーですか。  
  怎么回去的？坐出租车吗？
- <ruby>電車<rt>でんしゃ</rt></ruby>です。  
  坐电车。

这里省略的是：

- <ruby>電車<rt>でんしゃ</rt></ruby>で <ruby>帰りました<rt>かえりました</rt></ruby>。

### 3. `家` 和 `うち`

两者都可以表示“家”，但侧重点不同：

- `家`：更偏向房子、住处这个场所
- `うち`：更偏向自己的家、家人这一侧

- わたしの <ruby>家<rt>いえ</rt></ruby>は ここです。  
  我家在这儿。
- うちには <ruby>子供<rt>こども</rt></ruby>が いません。  
  我家没有孩子。

### 4. `まっすぐ 帰りました`

`まっすぐ` 本义是“笔直”，和 `帰ります` 连用时表示“不顺路去别处，直接回去”。

- <ruby>今日<rt>きょう</rt></ruby>は まっすぐ <ruby>帰ります<rt>かえります</rt></ruby>。  
  今天直接回家。

### 5. `それ` 的用法

这里的 `それ` 不一定指眼前具体物品，也可以指前面说到的整件事。

- <ruby>毎日<rt>まいにち</rt></ruby> <ruby>夜<rt>よる</rt></ruby> <ruby>11時<rt>じゅういちじ</rt></ruby>まで <ruby>働きます<rt>はたらきます</rt></ruby>。  
  我每天工作到晚上 11 点。
- それは <ruby>大変ですね<rt>たいへんですね</rt></ruby>。  
  那可真不容易。

> Page 99

### 6. `お疲れ様でした`

这是常见的寒暄语，可以表示：

- 对对方辛苦工作的体谅
- 下班或分别时的礼貌问候

例如：

- <ruby>お先<rt>おさき</rt></ruby>に <ruby>失礼します<rt>しつれいします</rt></ruby>。  
  我先走了。
- <ruby>お疲れ様でした<rt>おつかれさまでした</rt></ruby>。  
  辛苦了。

### 7. 自転車

在日本都市通勤场景里，很多人会先骑自行车到车站，再换乘电车去公司。

### 8. 终电

日语里末班电车常说：

- 最終電車
- 終電

如果错过末班车，往往只能打车回家。

### 专栏：新干线的爱称

从 1964 年东海道新干线问世到 2004 年的 40 年间，相继开通了山阳、东北、上越、北陆等新干线。各新干线的列车依等级、目的地不同被冠以不同的爱称。

东海道、山阳新干线，依等级不同爱称不尽相同。特快列车有“のぞみ（希望号）”和“ひかり（光速号）”。其中速度最快的是“のぞみ”，东京到新大阪（大阪府）之间，单程用两个半小时，东京到博多（福冈县）只需 4 小时 49 分。每站都停的慢车叫“こだま（回声号）”。

东北新干线以东京为起点，各条线路都有自己的爱称。去仙台（宫城县）和盛冈（岩手县）的叫“やまびこ（回音号）”，去八户（青森县）的叫“はやて（疾风号）”。去秋田（秋田县）和山形（山形县）的分别叫做“こまち（美女号）”和“つばさ（羽翼号）”。

此外，连接东京和新潟（新潟县）的上越新干线叫“とき（朱鹮号）”，东京和长野（长野县）的北陆新干线叫“あさま（浅间号）”。连接鹿儿岛中央（鹿儿岛县）和新八代（熊本县）的九州新干线叫做“つばめ（燕子号）”。
