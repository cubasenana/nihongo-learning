## 语法解释

### 1. 敬体和简体

到这一课为止，我们已经学过了很多 `〜ます / 〜です` 的句子。这些属于 `敬体`，适合和不太熟的人、长辈、正式场合使用。

而和朋友、家人、同辈聊天时，经常改用 `简体`：

- <ruby>見ます<rt>みます</rt></ruby> → <ruby>見る<rt>みる</rt></ruby>
- <ruby>見ました<rt>みました</rt></ruby> → <ruby>見た<rt>みた</rt></ruby>
- <ruby>見ません<rt>みません</rt></ruby> → <ruby>見ない<rt>みない</rt></ruby>
- <ruby>見ませんでした<rt>みませんでした</rt></ruby> → <ruby>見なかった<rt>みなかった</rt></ruby>

### 2. 动词的简体形

动词的简体形其实就是我们已经学过的这些形式：

- `基本形`
- `ない形`
- `た形`
- `なかった形`

例如：

- コーヒー、<ruby>飲む<rt>のむ</rt></ruby>。  
- ううん、<ruby>飲まない<rt>のまない</rt></ruby>。  
- <ruby>昨日<rt>きのう</rt></ruby> テレビ、<ruby>見た<rt>みた</rt></ruby>。  
- ううん、<ruby>見なかった<rt>みなかった</rt></ruby>。  

### 3. 一类形容词的简体形

- <ruby>忙しい<rt>いそがしい</rt></ruby>です → <ruby>忙しい<rt>いそがしい</rt></ruby>
- <ruby>忙しくない<rt>いそがしくない</rt></ruby>です → <ruby>忙しくない<rt>いそがしくない</rt></ruby>
- <ruby>忙しかった<rt>いそがしかった</rt></ruby>です → <ruby>忙しかった<rt>いそがしかった</rt></ruby>
- <ruby>忙しくなかった<rt>いそがしくなかった</rt></ruby>です → <ruby>忙しくなかった<rt>いそがしくなかった</rt></ruby>

要特别注意：一类形容词的简体后面**不能随便再加 `だ`**。

### 4. 二类形容词和名词的简体形

二类形容词和名词在简体里常常会出现 `だ / ではない / だった / ではなかった`：

- <ruby>簡単<rt>かんたん</rt></ruby>です → <ruby>簡単<rt>かんたん</rt></ruby>だ
- <ruby>簡単<rt>かんたん</rt></ruby>では ありません → <ruby>簡単<rt>かんたん</rt></ruby>では ない
- <ruby>晴れ<rt>はれ</rt></ruby>です → <ruby>晴れ<rt>はれ</rt></ruby>だ
- <ruby>晴れ<rt>はれ</rt></ruby>でした → <ruby>晴れ<rt>はれ</rt></ruby>だった

例如：

- この ネクタイ、<ruby>派手<rt>はで</rt></ruby>。  
- ううん、<ruby>全然<rt>ぜんぜん</rt></ruby> <ruby>派手<rt>はで</rt></ruby>じゃないわ。  
- <ruby>明日<rt>あした</rt></ruby>は <ruby>水曜日<rt>すいようび</rt></ruby>だね。  

### 5. `けど`

`けど` 在口语里非常常见，有两种典型用法：

- 表示转折：有点难，**不过**大体做出来了
- 表示铺垫：下周要办欢送会，**你方便吗**

例如：

- ちょっと <ruby>難しかった<rt>むずかしかった</rt></ruby>けど、まあまあ できたよ。  
- <ruby>送別会<rt>そうべつかい</rt></ruby>を するけど、<ruby>都合<rt>つごう</rt></ruby>は どうかな。

## 表达及词语讲解

### 1. 简体会话更像真实口语

到了这一课，课本开始让你接触“朋友之间怎么说话”。  
重点不是一口气全学会，而是先建立感觉：

- 简体更短
- 更口语
- 助词常常会省掉

### 2. 简体疑问

简体句提问时，经常直接升调，不一定加 `か`：

- これ、<ruby>日本製<rt>にほんせい</rt></ruby>。  
- <ruby>昨日<rt>きのう</rt></ruby>の <ruby>試験<rt>しけん</rt></ruby>、どうだった。  

### 3. `うん / ううん`

- `はい` → `うん`
- `いいえ` → `ううん`

这是最常见的口语化变化之一。

### 4. `かな`、`の`、`〜って`

- `かな`：自言自语或语气较轻的猜测  
  例：<ruby>今日<rt>きょう</rt></ruby> <ruby>李<rt>り</rt></ruby>さんは <ruby>来る<rt>くる</rt></ruby>かな。  
- `の`：询问原因、确认情况  
  例：どうしたの。  
- `〜って`：口语里的“听说……”  
  例：<ruby>太田<rt>おおた</rt></ruby>、<ruby>今度<rt>こんど</rt></ruby> <ruby>中国<rt>ちゅうごく</rt></ruby>へ <ruby>転勤<rt>てんきん</rt></ruby>だって。

## 单元末专题：亲昵的用语

### 1. 句尾的 `わ / よ`

课文里的 `全然 派手じゃないわ` 这种说法，传统上更偏女性口语。  
但现在真实口语里，男女说法差异已经没有以前那么绝对了。

### 2. 电话号码的读法

电话号码一般是一位一位地读：

- 3493-3945  
  → さん よん きゅう さん の さん きゅう よん ご  

### 3. `〜方`

把动词的 `ます形` 去掉 `ます`，再加 `方`，就变成“做法 / 读法 / 用法”：

- <ruby>話します<rt>はなします</rt></ruby> → <ruby>話し方<rt>はなしかた</rt></ruby>
- <ruby>読みます<rt>よみます</rt></ruby> → <ruby>読み方<rt>よみかた</rt></ruby>
- <ruby>使います<rt>つかいます</rt></ruby> → <ruby>使い方<rt>つかいかた</rt></ruby>

### 4. `タメ口`

`タメ口` 指的是同辈之间那种很随意、很亲近的说话方式。  
这页文化点真正想让你注意的是：简体并不是“更简单的语法”，而是**和人际距离绑定**的表达方式。
