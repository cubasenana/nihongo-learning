## 语法解释

### 1. 动词的 `て形`

本课核心是 `て形`。之后很多句型都依赖它，所以这一课要反复熟悉。

常见变化规律：

| 动词类型 | `ます形` 去掉 `ます` 后 | `て形` |
| --- | --- | --- |
| `書きます` | `書き` | `書いて` |
| `急ぎます` | `急ぎ` | `急いで` |
| `待ちます / 売ります / 買います` | `待ち / 売り / 買い` | `待って / 売って / 買って` |
| `読みます / 飲みます / 死にます` | `読み / 飲み / 死に` | `読んで / 飲んで / 死んで` |
| `話します` | `話し` | `話して` |
| 二类动词 `食べます / 見ます` | `食べ / 見` | `食べて / 見て` |
| 三类动词 `来ます / します` | `来 / し` | `来て / して` |

注意例外：

- <ruby>行きます<rt>いきます</rt></ruby> → <ruby>行って<rt>いって</rt></ruby>

### 2. `て形 + 动词`

按时间顺序连续做几个动作时，可以用 `て形` 连接：

- デパートへ <ruby>行って<rt>いって</rt></ruby>、<ruby>買い物<rt>かいもの</rt></ruby>しました。  
- <ruby>家<rt>いえ</rt></ruby>へ <ruby>帰って<rt>かえって</rt></ruby>、<ruby>宿題<rt>しゅくだい</rt></ruby>を します。  
- <ruby>図書館<rt>としょかん</rt></ruby>へ <ruby>行って<rt>いって</rt></ruby>、<ruby>本<rt>ほん</rt></ruby>を <ruby>借りて<rt>かりて</rt></ruby>、<ruby>家<rt>いえ</rt></ruby>へ <ruby>帰ります<rt>かえります</rt></ruby>。  

### 3. `てから`

想强调“做完前一个动作之后，再做后一个动作”，就用：

`动词て形 + から + 动词`

- ラジオを <ruby>聞いて<rt>きいて</rt></ruby>から <ruby>寝ます<rt>ねます</rt></ruby>。  
- <ruby>昼ご飯<rt>ひるごはん</rt></ruby>を <ruby>食べて<rt>たべて</rt></ruby>から <ruby>出かけます<rt>でかけます</rt></ruby>。  

它和普通 `て` 的区别在于：`てから` 更明确表示“前项完成后再后项”，而且一个句子里一般不会连续重复太多次。

### 4. `〜て ください`

请求别人做某事时，用：

`动词て形 + ください`

- ここに <ruby>住所<rt>じゅうしょ</rt></ruby>と <ruby>名前<rt>なまえ</rt></ruby>を <ruby>書いて<rt>かいて</rt></ruby> ください。  
- この <ruby>荷物<rt>にもつ</rt></ruby>を <ruby>航空便<rt>こうくうびん</rt></ruby>で <ruby>送って<rt>おくって</rt></ruby> ください。  

### 5. 经过或离开的场所用 `を`

`を` 除了表示宾语，还能表示“经过的地方”或“离开的地方”：

- この バスは <ruby>駅前<rt>えきまえ</rt></ruby>を <ruby>通ります<rt>とおります</rt></ruby>。  
- この <ruby>道<rt>みち</rt></ruby>を まっすぐ <ruby>行って<rt>いって</rt></ruby>、<ruby>橋<rt>はし</rt></ruby>を <ruby>渡って<rt>わたって</rt></ruby> ください。  
- <ruby>李<rt>り</rt></ruby>さんは <ruby>毎朝<rt>まいあさ</rt></ruby> 7<ruby>時<rt>じ</rt></ruby>に <ruby>家<rt>いえ</rt></ruby>を <ruby>出ます<rt>でます</rt></ruby>。  
- <ruby>来年<rt>らいねん</rt></ruby> <ruby>大学<rt>だいがく</rt></ruby>を <ruby>卒業します<rt>そつぎょうします</rt></ruby>。  

### 6. `〜ます` 前接汉字词可做三类动词

很多汉字词加上 `します` 就能变成动词：

- <ruby>勉強<rt>べんきょう</rt></ruby>します
- <ruby>旅行<rt>りょこう</rt></ruby>します
- <ruby>掃除<rt>そうじ</rt></ruby>します
- <ruby>整理<rt>せいり</rt></ruby>します
- <ruby>出張<rt>しゅっちょう</rt></ruby>します

## 表达及词语讲解

### 1. `それから`

`て形` 最多连接两三个动作就够了，再多会累。要继续接动作时，经常改用：

- それから

例如：

- <ruby>図書館<rt>としょかん</rt></ruby>へ <ruby>行って<rt>いって</rt></ruby>、<ruby>勉強<rt>べんきょう</rt></ruby>を します。それから <ruby>家<rt>いえ</rt></ruby>へ <ruby>帰って<rt>かえって</rt></ruby>、<ruby>手紙<rt>てがみ</rt></ruby>を <ruby>書きます<rt>かきます</rt></ruby>。  

### 2. `駅前`

`駅前` 不一定只指“车站正前方”，而常指车站周围一带。  
如果想精确说“车站前面”，更常用：

- <ruby>駅<rt>えき</rt></ruby>の <ruby>前<rt>まえ</rt></ruby>

### 3. `なかなか`

`なかなか` 后面接肯定表达时，表示“相当……、很不错”，含有超出预期的感觉：

- <ruby>李<rt>り</rt></ruby>さんの <ruby>日本語<rt>にほんご</rt></ruby>は なかなか <ruby>上手<rt>じょうず</rt></ruby>ですね。  
- <ruby>小野<rt>おの</rt></ruby>さんの <ruby>料理<rt>りょうり</rt></ruby>は なかなか おいしいです。  

### 4. `くださいませんか`

比 `〜て ください` 更客气的请求方式：

- <ruby>後で<rt>あとで</rt></ruby> <ruby>写真<rt>しゃしん</rt></ruby>も <ruby>見て<rt>みて</rt></ruby> くださいませんか。  
- スケジュール<ruby>表<rt>ひょう</rt></ruby>を ファックスで <ruby>送って<rt>おくって</rt></ruby> くださいませんか。  

### 5. `もう`

本课里的 `もう` 常常是“这就 / 马上”的意思，不是以前学过的“已经”：

- もう <ruby>会社<rt>かいしゃ</rt></ruby>を <ruby>出ます<rt>でます</rt></ruby>か。  

> Page 191

### 6. `すみませんが`

要麻烦别人时，在请求前先说：

- すみませんが、〜

比直接开口更礼貌，也更自然。

### 7. `そうしてください`

这里的 `そう` 不是指物品，而是承接前面说过的“做法 / 状态”：

- <ruby>速達<rt>そくたつ</rt></ruby>で <ruby>出します<rt>だします</rt></ruby>か。  
- ええ、そうして ください。  

### 8. `お金を 下ろします`

`下ろします` 原意是“从上面拿下来”，`お金を 下ろします` 则是“取钱”：

- <ruby>銀行<rt>ぎんこう</rt></ruby>で お<ruby>金<rt>かね</rt></ruby>を <ruby>下ろします<rt>おろします</rt></ruby>。

### 专栏：日本的邮政编码

日本的邮政编码常写成 `〒123-4567` 这种形式。  
看到 `〒` 这个符号，基本就能判断后面跟的是邮编。  
教材里写地址时，经常把 `〒`、都道府县、市区町村、门牌号连在一起出现，初学时先认层级，不必一次性全部背熟。
