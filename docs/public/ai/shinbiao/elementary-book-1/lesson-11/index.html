## 语法解释

### 1. `名词1 は 名词2 が 好きです / 嫌いです / 怖いです`

表示喜好、厌恶、害怕等感情时，经常用：

- `A は B が 好きです`
- `A は B が 嫌いです`
- `A は B が 怖いです`

这里：

- `A` 是有这种感情的人
- `B` 是感情指向的对象

例如：

- <ruby>小野<rt>おの</rt></ruby>さんは <ruby>歌<rt>うた</rt></ruby>が <ruby>好き<rt>すき</rt></ruby>です。  
- <ruby>王<rt>おう</rt></ruby>さんは <ruby>肉<rt>にく</rt></ruby>が <ruby>嫌い<rt>きらい</rt></ruby>です。  
- <ruby>林<rt>はやし</rt></ruby>さんは <ruby>犬<rt>いぬ</rt></ruby>が <ruby>怖い<rt>こわい</rt></ruby>です。  

### 2. `分かります / できます / 上手です / 下手です`

这些词也常用 `が` 来提示对象：

- スミスさんは <ruby>韓国語<rt>かんこくご</rt></ruby>が <ruby>分かります<rt>わかります</rt></ruby>。  
- わたしは パソコンが できません。  
- <ruby>小野<rt>おの</rt></ruby>さんは <ruby>料理<rt>りょうり</rt></ruby>が <ruby>上手<rt>じょうず</rt></ruby>です。  
- わたしは <ruby>歌<rt>うた</rt></ruby>が <ruby>下手<rt>へた</rt></ruby>です。  

### 3. `や` 和 `など`

`と` 是把项目全部列出来，`や` 则更像“……呀……之类的”。常与 `など` 呼应。

- <ruby>吉田<rt>よしだ</rt></ruby>さんは <ruby>時々<rt>ときどき</rt></ruby> <ruby>中国<rt>ちゅうごく</rt></ruby>や <ruby>韓国<rt>かんこく</rt></ruby>へ <ruby>行きます<rt>いきます</rt></ruby>。  
- <ruby>森<rt>もり</rt></ruby>さんは サッカーや <ruby>野球<rt>やきゅう</rt></ruby>が <ruby>好き<rt>すき</rt></ruby>です。  
- わたしは お<ruby>店<rt>みせ</rt></ruby>で <ruby>野菜<rt>やさい</rt></ruby>や <ruby>果物<rt>くだもの</rt></ruby>などを <ruby>買いました<rt>かいました</rt></ruby>。  

### 4. `〜から`

说明原因、理由时，用 `から` 接在原因句后面。

- <ruby>森<rt>もり</rt></ruby>さんは お<ruby>酒<rt>さけ</rt></ruby>が <ruby>好き<rt>すき</rt></ruby>ですから、<ruby>毎日<rt>まいにち</rt></ruby> <ruby>飲みます<rt>のみます</rt></ruby>。  
- <ruby>寒い<rt>さむい</rt></ruby>ですから、<ruby>窓<rt>まど</rt></ruby>を <ruby>閉めます<rt>しめます</rt></ruby>。  

### 5. `だから / ですから`

这是“所以、因此”的连词形式：

- キムさんは <ruby>野菜<rt>やさい</rt></ruby>が <ruby>嫌い<rt>きらい</rt></ruby>です。だから、あまり <ruby>食べません<rt>たべません</rt></ruby>。  
- <ruby>明日<rt>あした</rt></ruby>は <ruby>休み<rt>やすみ</rt></ruby>です。ですから、<ruby>子供<rt>こども</rt></ruby>と <ruby>動物園<rt>どうぶつえん</rt></ruby>へ <ruby>行きます<rt>いきます</rt></ruby>。

### 6. 频率副词

本课要开始熟悉表示频率的副词：

- いつも：总是
- よく：经常
- <ruby>時々<rt>ときどき</rt></ruby>：有时
- たまに：偶尔
- あまり：不太（后接否定）
- <ruby>全然<rt>ぜんぜん</rt></ruby>：完全不（后接否定）

例如：

- わたしは たまに コンサートへ <ruby>行きます<rt>いきます</rt></ruby>。  
- <ruby>李<rt>り</rt></ruby>さんは <ruby>時々<rt>ときどき</rt></ruby> <ruby>映画<rt>えいが</rt></ruby>を <ruby>見ます<rt>みます</rt></ruby>。  
- <ruby>田中<rt>たなか</rt></ruby>さんは あまり <ruby>横浜<rt>よこはま</rt></ruby>へは <ruby>行きません<rt>いきません</rt></ruby>。  

### 7. `どうしてですか`

问原因时可以说：

- どうしてですか。  

回答时常接 `〜から`：

- どうしてですか。  
- ここで <ruby>友達<rt>ともだち</rt></ruby>の <ruby>結婚式<rt>けっこんしき</rt></ruby>が ありますから。

## 表达及词语讲解

### 1. `ぼく`

`ぼく` 是男性在比较随便的场合自称“我”的说法。女性一般不用。

### 2. `写真展があります`

第 4 课学过 `あります` 表示“有东西”。到了这里，`あります` 也能表示“有活动 / 有事情发生”。

- <ruby>来週<rt>らいしゅう</rt></ruby>、ホテルで パーティーが あります。  
- この <ruby>近く<rt>ちかく</rt></ruby>の ホテルで <ruby>写真展<rt>しゃしんてん</rt></ruby>が あります。  

### 3. `いかがですか`

这是比 `どうですか` 更礼貌的说法。用于推荐、建议时很常见。

- <ruby>寄木細工<rt>よせざいく</rt></ruby>は いかがですか。  

### 4. `そうですね`

除了表示“是啊”，还常用来给自己一点思考时间。

- うーん、そうですね……。  

### 5. `迷います`

表示“很难马上决定”。常用于面对很多选项时。

- うーん、<ruby>迷います<rt>まよいます</rt></ruby>ね。  

### 6. `気に入ります`

表示“喜欢上、看中”。这里不是长期爱好，而是“看到某样东西后觉得中意”。

- <ruby>気に入りました<rt>きにいりました</rt></ruby>か。  
  喜欢吗？
- はい、とても。  
  嗯，很喜欢。

### 专栏：<ruby>寄木細工<rt>よせざいく</rt></ruby>

“<ruby>寄木細工<rt>よせざいく</rt></ruby>”是箱根地区很有代表性的传统工艺品。它的特点是利用不同木材天然颜色的差异，拼接出几何纹样。

对初学者来说，本课里最有用的不是工艺史，而是把它当成礼品店场景里的核心词：

- <ruby>箱根<rt>はこね</rt></ruby>の お<ruby>土産<rt>みやげ</rt></ruby>は <ruby>何<rt>なに</rt></ruby>が いいですか。  
- <ruby>寄木細工<rt>よせざいく</rt></ruby>は いかがですか。
