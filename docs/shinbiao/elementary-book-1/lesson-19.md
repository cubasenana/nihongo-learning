## 语法解释

### 1. 动词的"ない形"

第 14 课我们学习了动词的"て形"（☞第 14 课"语法解释 1"），本课学习动词的"ない形"。"ない形"的变换方式如下：

- 一类动词：把"ます形"去掉"ます"后的最后一个音变成相应的"あ段"音，后加"ない"。如果最后音为"い"时，把"い"变成"わ"后加"ない"。（注意：此时不能将"い"变为"あ"。）
- 二类动词：把"ます形"的"ます"去掉后加"ない"。
- 三类动词：把"来ます"变成"来ない"，把"します"变成"しない"。

| 类别 | ます形 | 去掉ます | → | ない形 |
|------|--------|----------|---|--------|
| 一类动词 | <ruby>書<rt>か</rt></ruby>きます | かき | → | かかない |
| | <ruby>急<rt>いそ</rt></ruby>ぎます | いそぎ | → | いそがない |
| | <ruby>飛<rt>と</rt></ruby>びます | とび | → | とばない |
| | <ruby>読<rt>よ</rt></ruby>みます | よみ | → | よまない |
| | <ruby>死<rt>し</rt></ruby>にます | しに | → | しなない |
| | <ruby>待<rt>ま</rt></ruby>ちます | まち | → | またない |
| | <ruby>売<rt>う</rt></ruby>ります | うり | → | うらない |
| | <ruby>買<rt>か</rt></ruby>います | かい | → | かわない |
| | <ruby>話<rt>はな</rt></ruby>します | はなし | → | はなさない |
| 二类动词 | <ruby>食<rt>た</rt></ruby>べます | たべ | → | たべない |
| | <ruby>見<rt>み</rt></ruby>ます | み | → | みない |
| 三类动词 | <ruby>来<rt>き</rt></ruby>ます | き | → | こない |
| | します | し | → | しない |

### 2. 动 ないで ください

表示否定的命令时，使用"～て ください"的否定形式。其构成方式为"ない形 + で + ください"。

- <ruby>部屋<rt>へや</rt></ruby>の かぎを <ruby>忘<rt>わす</rt></ruby>れないで ください。（请别忘了房间钥匙。）
- この <ruby>部屋<rt>へや</rt></ruby>には <ruby>入<rt>はい</rt></ruby>らないで ください。（请不要进入这个房间。）
- <ruby>無理<rt>むり</rt></ruby>を しないで ください。（请不要勉强。）

### 3. 动 なければ なりません

表示必须。接续方法是将动词"ない形"中的"ない"换成"なければ なりません"。也可以用"ないと いけません"的形式代替"なければ なりません"。

- <ruby>李<rt>り</rt></ruby>さんは <ruby>今日<rt>きょう</rt></ruby> <ruby>早<rt>はや</rt></ruby>く <ruby>帰<rt>かえ</rt></ruby>らなければ なりません。（小李今天必须早点回家。）
- この <ruby>薬<rt>くすり</rt></ruby>は <ruby>毎日<rt>まいにち</rt></ruby> <ruby>飲<rt>の</rt></ruby>まなければ なりません。（这个药必须每天吃。）
- すぐに <ruby>書類<rt>しょるい</rt></ruby>を <ruby>送<rt>おく</rt></ruby>らないと いけません。（得马上把文件寄出去。）

### 4. 动 なくても いいです

表示不做某事也可以。意思相当于"～なければ なりません"的否定。接续方法是将动词"ない形"中的"ない"换成"なくても いいです"。

- <ruby>明日<rt>あした</rt></ruby>は <ruby>残業<rt>ざんぎょう</rt></ruby>しなくても いいです。（明天不加班也行。）
- <ruby>慌<rt>あわ</rt></ruby>てなくても いいですよ。（不要慌慌张张的！）
- <ruby>靴<rt>くつ</rt></ruby>を <ruby>脱<rt>ぬ</rt></ruby>がなくても いいですか。（不脱鞋也可以吗？）

### 5. 名1 が 名2 です

表示作为话题的"名词 2"是什么时，用"名词 2 は 名词 1 です"的形式。这里"名词 2"是旧信息，"名词 1"是新信息。同样的内容也可以用"名词 1 が 名词 2 です"的形式来表示。在这个句型里"名词 1"仍然是新信息，"名词 2"仍然是旧信息。新信息做主语时用"が"来表示。

- <ruby>山田<rt>やまだ</rt></ruby><ruby>先生<rt>せんせい</rt></ruby>は どの <ruby>方<rt>かた</rt></ruby>ですか。（山田老师是哪位？）  
  ——<ruby>山田<rt>やまだ</rt></ruby><ruby>先生<rt>せんせい</rt></ruby>は あの <ruby>方<rt>かた</rt></ruby>です。（山田老师是那位。）  
  ——あの <ruby>方<rt>かた</rt></ruby>が <ruby>山田<rt>やまだ</rt></ruby><ruby>先生<rt>せんせい</rt></ruby>です。（那位是山田老师。）
- <ruby>吉田<rt>よしだ</rt></ruby><ruby>課長<rt>かちょう</rt></ruby>は いますか。（吉田科长在吗？）  
  ——わたしが <ruby>吉田<rt>よしだ</rt></ruby>ですが…。（我就是吉田……）
- あなたの かばんは どれですか。（你的包是哪一个？）  
  ——これが わたしの かばんです。（这个是我的包。）

> **参考** 叙述直接观察到的某事物，而不是作为话题在谈论这个事物时，使用"名词 が 动词"的形式。因为这里也同样是新信息在做主语。
>
> - <ruby>子供<rt>こども</rt></ruby>が <ruby>公園<rt>こうえん</rt></ruby>で <ruby>遊<rt>あそ</rt></ruby>んで います。（孩子在公园里玩儿。）
> - バスが <ruby>来<rt>き</rt></ruby>ました。（公共汽车来了。）

> **注意** "何""だれ""どこ""いつ"等表示疑问的词做主语时不能用"は"，只能用"が"。因为这些词不能表示旧信息。
>
> - だれが いますか。（有谁呀？） × だれは いますか。
> - どこが <ruby>痛<rt>いた</rt></ruby>いですか。（哪儿疼啊？） × どこは <ruby>痛<rt>いた</rt></ruby>いですか。
> - <ruby>何<rt>なに</rt></ruby>が <ruby>好<rt>す</rt></ruby>きですか。（喜欢什么？） × <ruby>何<rt>なに</rt></ruby>は <ruby>好<rt>す</rt></ruby>きですか。

## 表达及词语讲解

### 1. "～なければ なりません"和"～ないと いけません"

两者都是表示义务、必要性的说法，但"～なければ なりません"比"～ないと いけません"正式些。"～なければ なりません"多用于书面语，"～ないと いけません"主要用于口语。另外，后者的省略形式"～ないと"在口语中也用得很多。

- <ruby>子供<rt>こども</rt></ruby>は 6<ruby>歳<rt>さい</rt></ruby>から <ruby>学校<rt>がっこう</rt></ruby>へ <ruby>行<rt>い</rt></ruby>かなければ なりません。（从 6 岁起，孩子必须上学。）
- あっ、8<ruby>時<rt>じ</rt></ruby>ですね。もう <ruby>駅<rt>えき</rt></ruby>へ <ruby>行<rt>い</rt></ruby>かないと。（啊，8 点了，我得去车站了。）

### 2. ＊"<ruby>初心者<rt>しょしんしゃ</rt></ruby>""<ruby>上級者<rt>じょうきゅうしゃ</rt></ruby>"

汉语中，表示技能的等级时，说"初级/中级/高级"，但日语中不说"高級"而说"上級"。尽管日语中有"高級"这个词，但它不用于描述人的技能、技术等，只限于表示高质量的物品。

- <ruby>森<rt>もり</rt></ruby>さん、<ruby>英会話<rt>えいかいわ</rt></ruby>の <ruby>勉強<rt>べんきょう</rt></ruby>は どうですか。  
  ——やっと <ruby>先月<rt>せんげつ</rt></ruby> <ruby>上級<rt>じょうきゅう</rt></ruby>クラスに <ruby>入<rt>はい</rt></ruby>りました。  
  （森先生，英语会话学得怎么样了？——上个月，好不容易进了高级班。）

### 3. ＊<ruby>何度<rt>なんど</rt></ruby>も

本课的"何 + 量词 + も + 肯定形式"是表示数量多的说法。（☞第 4 课"语法解释 5"，第 13 课"语法解释 1"）

- <ruby>何度<rt>なんど</rt></ruby>も <ruby>転<rt>ころ</rt></ruby>びましたが、とても おもしろいです。（我摔了好几次，不过挺有意思的。）
- のどが <ruby>渇<rt>かわ</rt></ruby>きましたから、<ruby>水<rt>みず</rt></ruby>を <ruby>何杯<rt>なんばい</rt></ruby>も <ruby>飲<rt>の</rt></ruby>みました。（口渴了，喝了好几杯水。）

### 4. ＊だいぶ ［表示程度的副词④］

表示程度相当高。用于"～なりました"等表示变化的句型中时，突出强调变化的程度。

- <ruby>昨日<rt>きのう</rt></ruby>は だいぶ お<ruby>酒<rt>さけ</rt></ruby>を <ruby>飲<rt>の</rt></ruby>みました。（昨天喝了不少酒。）
- <ruby>風邪<rt>かぜ</rt></ruby>は よく なりましたか。（感冒好了吗？）  
  ——だいぶ よく なりました。（好多了。）

### 5. ＊<ruby>大丈夫<rt>だいじょうぶ</rt></ruby>です

和汉语中的"大丈夫"汉字相同，但意思完全不同。是"没问题""可以"的意思。依据上下文关系可以有各种不同的解释，应用课文中使用的"大丈夫です"是"我保证（不进去）"的意思。

- <ruby>上級者<rt>じょうきゅうしゃ</rt></ruby>の コースには <ruby>入<rt>はい</rt></ruby>らないで ください。  
  ——<ruby>大丈夫<rt>だいじょうぶ</rt></ruby>です。<ruby>心配<rt>しんぱい</rt></ruby>しないで ください。  
  （注意别滑进熟练者滑道里去。——我保证不进去，请别担心。）

- <ruby>田中<rt>たなか</rt></ruby>さん、<ruby>病気<rt>びょうき</rt></ruby>は もう <ruby>治<rt>なお</rt></ruby>りましたか。（田中先生，病好了吗？）  
  ——はい、もう <ruby>大丈夫<rt>だいじょうぶ</rt></ruby>です。（嗯，已经不要紧了。）

### 6. <ruby>残業<rt>ざんぎょう</rt></ruby>

公司职员在工作时间以外工作，称为"残業（加班）"。以前一般以为日本的工薪人员无一例外地都要加班，但近年来一部分年轻人为了提高自己的生活质量，不仅不主动、积极地加班，甚至即使上司要求加班也予以拒绝。不过，与此同时，由于工作量大，上班时间内无法完成，所以不拿报酬在超出规定的加班时间后仍继续加班的人也为数不少。

### 7. ＊スキー

日本地形南北狭长，全国多山，因此许多地方冬天积雪，具备享受滑雪、单板滑雪等冬季体育运动的条件。这样，城市里的人们也开车或乘坐滑雪旅游专车、火车等涌向滑雪场。应用课文中的"新潟县苗场滑雪场"从东京开车两个小时左右即可到达。

滑雪场配备不同的滑道，滑雪技术水平不一的人可以尽情享受。山坡平缓的部分是面向初学者的滑道，山坡陡峭的部分是面向熟练者的滑道。有时，会发生初学者在不知情的情况下滑进熟练者的滑道，因而受伤的事故，因此应用课文中小野提醒小李注意。

## 专栏：冬季的体育运动

提起冬季的体育运动，有名的是滑雪和溜冰。在日本，"スノーボード Snowboard（单板滑雪）"在年轻人中间正掀起一股热潮。

"单板滑雪"是侧身站在滑雪板上滑雪的比赛项目，据说 20 世纪 60 年代起源于美国，70 年代传入日本，当时参加比赛的人很少，因此鲜为人知。进入 80 年代之后，举办大型的比赛活动，参加这项比赛活动的人才渐渐多了起来。

1998 年长野举办的冬季奥运会将其规定为正式项目后，国际上都认可了这项运动。

"单板滑雪"刚进入日本的时候，常常可以看到极少数人在滑雪场的角落里默默地滑着，但最近，滑雪场为"单板滑雪"设立了专用滑道，"スノボー"这一略称也固定了下来。冬季滑雪场因有很多以滑雪、"单板滑雪"为乐趣的人们，显得热闹异常。
