## 语法解释

### 1. 敬体形和简体形

在口语里，日语的谓语形式根据说话人和听话人之间的上下关系、亲疏关系的不同而不同。迄今为止学习的"～ます""～です"是对长辈或关系不太密切的人使用的形式，这种形式我们称做"敬体形"。与此相反，如对方是自己的同辈或晚辈，或关系较密切时则使用"简体形"。"简体形"是不使用"ます""です"的形式。

动词本身即具有"敬体形"和"简体形"，如以前学过的"～ます""～ません""～ました""～ませんでした"等四种礼貌的表达形式即是"敬体形"，而以前所学的动词的"基本形""ない形""た形"都属于简体形。但一类形容词、二类形容词和名词本身没有"敬体形"和"简体形"之分，只有在做谓语时，其谓语形式才具有"敬体形"和"简体形"。以下"语法解释 2～5"里我们将分别予以说明。

### 2. 动词的简体形

| | 敬体形 | | 简体形 | |
|---|---|---|---|---|
| 现在形式 | 肯定 | <ruby>買<rt>か</rt></ruby>います（买） | <ruby>買<rt>か</rt></ruby>う（基本形） | ある |
| | 否定 | <ruby>買<rt>か</rt></ruby>いません | <ruby>買<rt>か</rt></ruby>わない（ない形） | ない |
| 过去形式 | 肯定 | <ruby>買<rt>か</rt></ruby>いました | <ruby>買<rt>か</rt></ruby>った（た形） | あった |
| | 否定 | <ruby>買<rt>か</rt></ruby>いませんでした | <ruby>買<rt>か</rt></ruby>わなかった（なかった形） | なかった |

- <ruby>森<rt>もり</rt></ruby>さんは <ruby>毎晩<rt>まいばん</rt></ruby> テレビを **<ruby>見<rt>み</rt></ruby>る**。（森先生每天晚上看电视。）
- コーヒー、**<ruby>飲<rt>の</rt></ruby>む**？（喝咖啡吗？）  
  ——ううん、**<ruby>飲<rt>の</rt></ruby>まない**。（不，不喝。）
- <ruby>昨日<rt>きのう</rt></ruby> テレビ、**<ruby>見<rt>み</rt></ruby>た**？（昨天看电视了吗？）  
  ——ううん、**<ruby>見<rt>み</rt></ruby>なかった**。（不，没看。）

> 简体的会话中常常省略助词。另外，"ありません"的简体形是"ない"。

### 6. `小句` けど、`小句`【转折】【铺垫】

"けど"连接两个小句，表示转折。前后两个小句子的时态由各自谓语的形式决定。用于口语。

- <ruby>昨日<rt>きのう</rt></ruby>の <ruby>試験<rt>しけん</rt></ruby>、どうだった？（昨天的考试怎么样啊？）  
  ——ちょっと <ruby>難<rt>むずか</rt></ruby>しかった**けど**、まあまあ できたよ。（有点难，不过大概都做出来了。）

另外，"けど"还有不表示转折而表示铺垫的用法。

- <ruby>来週<rt>らいしゅう</rt></ruby>、<ruby>送別会<rt>そうべつかい</rt></ruby>を する**けど**、<ruby>都合<rt>つごう</rt></ruby>は どうかな？  
  （下星期要开一个欢送会，你时间方便吗？）

## 表达及词语讲解

### 1. 简体会话

#### (1) 敬体和简体

使用敬体形的文体为敬体，使用简体形的文体为简体。在正式场合谈话时使用敬体，书信一般也用敬体。但是，与家人、朋友交谈时则用简体，有关法律的文章、报纸、商业文书等也都用简体。以下分别是简体会话及简体文章中的句子。

- お<ruby>母<rt>かあ</rt></ruby>さん、あの おもちゃが <ruby>欲<rt>ほ</rt></ruby>しい。（妈妈，我想要那个玩具。）（孩子和母亲的对话）
- <ruby>昨日<rt>きのう</rt></ruby> <ruby>駅前<rt>えきまえ</rt></ruby>で <ruby>大<rt>おお</rt></ruby>きい <ruby>火事<rt>かじ</rt></ruby>が あった。（昨天车站附近发生了重大的火灾。）（日记文章）

#### (2) 简体疑问

用简体表示疑问的时候，名词、形容词省略敬体形中含有的"～ですか"，句尾升调。动词要把谓语敬体形变成简体形，去掉表示疑问的助词"か"，句尾升调。另外，为了表示疑问须加疑问符号"？"。

- これ、<ruby>日本製<rt>にほんせい</rt></ruby>？（这个，是日本产的吗？）

#### (3) "うん"和"ううん"

简体会话中，"はい"说成"うん"，"いいえ"说成"ううん"。"うん"和"ううん"的语调比较特殊，请听录音仿读。

#### (4) 简体助词

"かな"和"の"这两个助词只能接在简体形后面。

##### ① かな

助词"かな"本来只用于自言自语的场合。如有听话人在场时则表示通过让对方听到自己的自问自答而向对方提供一种不太确实的信息。

- <ruby>今日<rt>きょう</rt></ruby>、<ruby>李<rt>り</rt></ruby>さんは <ruby>来<rt>く</rt></ruby>る**かな**。（小李今天会来吗？）
- <ruby>期間<rt>きかん</rt></ruby>は 4<ruby>年<rt>ねん</rt></ruby>か 5<ruby>年<rt>ねん</rt></ruby>**かな**。（时间好像是 4 年或者 5 年。）

##### ② の

助词"の"用于要求说明或确认某事。

- ああ、<ruby>清水<rt>しみず</rt></ruby><ruby>君<rt>くん</rt></ruby>、どうした**の**？（清水，你好。有什么事啊？）
- <ruby>薬<rt>くすり</rt></ruby>、<ruby>飲<rt>の</rt></ruby>んだ**の**？（药吃了吗？）

#### (5) ～って ①【传闻】

"～って"是在简体形"転勤だ"的后面加"って"构成的。"～って"表示传闻。

- <ruby>太田<rt>おおた</rt></ruby>、<ruby>今度<rt>こんど</rt></ruby> <ruby>中国<rt>ちゅうごく</rt></ruby>へ <ruby>転勤<rt>てんきん</rt></ruby>だ**って**。（听说太田要调到中国去工作了。）

#### (6) 省略 ④【简体的省略】

在口语里，简体句中的一些音往往不发，如会话中"<ruby>森<rt>もり</rt></ruby>さんの <ruby>電話番号<rt>でんわばんごう</rt></ruby>、<ruby>知<rt>し</rt></ruby>ってる？"省略了"ている"的"い"。另外"<ruby>電話番号<rt>でんわばんごう</rt></ruby>"中的助词"を"也省略了。

### 2. "わ"和"よ"

基本课文 C 中的"<ruby>全然<rt>ぜんぜん</rt></ruby> <ruby>派手<rt>はで</rt></ruby>じゃないわ（一点儿也不花哨）"的"わ"主要是女性使用。而在这种场合男性一般使用"よ"。但是，最近男女之间的语体差异逐渐缩小，一些女性也开始使用"<ruby>派手<rt>はで</rt></ruby>だよ""できたよ"这样的表达方式了。另外，"二类形容词＋よ""名词＋よ"的形式主要是女性使用。

- この <ruby>携帯<rt>けいたい</rt></ruby>、とても **<ruby>便利<rt>べんり</rt></ruby>**よ。（这个手机，很方便哟。）
- あそこが わたしの **うち**よ。（那儿就是我家。）

### 3. 电话号码的读法

读电话号码的时候，数字要一个一个地读，以便对方听明白。另外，也有在局号和个人的号码之间加"の"的读法。（☞第 1 册第 100 页下方的数字）

- <ruby>電話番号<rt>でんわばんごう</rt></ruby>は "3493-3945" です。（电话号码 3493-3945。）

### 4. ～<ruby>方<rt>かた</rt></ruby>

动词"ます形"去掉"ます"的形式加上"<ruby>方<rt>かた</rt></ruby>"就成了"～の做法"的意思，用来表示动作的方法。但并不是所有的动词都适用这种形式。

| ます形 | 去掉ます | ～<ruby>方<rt>かた</rt></ruby> | 做法 |

| <ruby>話<rt>はな</rt></ruby>します | <ruby>話<rt>はな</rt></ruby>し | <ruby>話<rt>はな</rt></ruby>し<ruby>方<rt>かた</rt></ruby> | 说法 |
