# 精修规范（S3+ 标准）v2

> 基于第 1-5 课实际精修过程中踩过的全部坑固化而成。
> 后续第 6-24 课按本规范一次通过，不再需要多轮回校。

## 目标

不求与原书 100% 像素级还原，只求：

1. **AI 可直接抓取当教材**——结构清晰、内容完整，无需再 OCR
2. **人类可阅读学习**——排版舒适、注音到位、翻译准确

## 输入源

| 源 | 用途 |
|----|------|
| `resources/新标日/pages/page-XXX.png` | 原书 400dpi 渲染页，逐页对照的**唯一真相** |
| `docs/shinbiao/elementary-book-1/lesson-XX.md` | 现有 S3 版本，在此基础上修正 |

## 逐页校对流程

```
对每一课（已知页码范围 Pages: XX-YY）：
  1. 读取 docs/ 中现有 markdown 全文
  2. 逐页读取对应 PNG（page-XXX.png ~ page-YYY.png）
  3. 按下方检查清单【每条都过】逐项比对
  4. 一次性修正全部问题 → 更新完成度标记为 S3+
```

## 检查清单

### A. 基本课文核验（优先级最高）

> **教训：第 4 课基本句第 2 句完全写错（部屋に猫→机の上に猫），这是最致命的错误。**

- [ ] **基本句 1-4 逐字核对 PDF**——每个助词、每个词都要对
- [ ] **会话 A-D 逐句核对**——注意甲/乙、句尾标点
- [ ] 页底生词速查行不需要单独收录（已在生词表中）

### K. Ruby 标注准确性（v2.1 新增，基于第 6-7 课教训）

> **教训：第 7 课「昼ご飯」标注为 ひるごめし（应为ひるごはん），中文「要走的人说」标注了胡乱日语 ruby。第 6 课「こどもの日」标注为 こどものにち（应为こどものひ）。**

- [ ] **ruby 读音逐个核实**——特别注意：ご飯=ごはん（非ごめし）、こどもの日=こどものひ（非にち）等易混读音
- [ ] **ruby 只标注汉字**——平假名/片假名不需要 ruby（`銀座です` 的 `です` 不标 `ぎんざです`）
- [ ] **中文文本绝对不加日语 ruby**——舞台说明、分类标签等中文内容不得有 `<rt>` 标注
- [ ] **节日/专有名词特殊读音**——こどもの日(ひ)、七夕(たなばた) 等需要查证，不能套用常规读音规则

### B. 语法解释核验

> **教训：第 5 课「星期后一般加に」说反了原书——事实性错误最危险。**

- [ ] **说明文字与原书逐句比对**——不改措辞、不意译、不加自己的概括
- [ ] **例句数量与原书一致**——原书有 6 个例句就写 6 个，不缩减（第 4 课语法 2/3 被缩到 3 个）
- [ ] **例句翻译全部保留**——每个 `►` 例句后的括号翻译都要有（第 4 课「と」3 例全缺翻译）
- [ ] **应答部分不丢**——`——はい/いいえ` 的回答要跟上（第 1 课语法 3 被砍掉）
- [ ] **重要注意事项/反例不漏**——原书的 `注意` `参考` 标注、`×` 反例、中日对比说明（第 4 课缺「の不能省略」，第 5 课缺「汉语只用有无但日语必须ました」）
- [ ] **活用表/变形表完整**——原书列了几个动词就写几个（第 5 课活用表从 5 个动词缩到 1 个）
- [ ] **交叉引用保留**——「☞第 X 课」的交叉引用要保留

### C. 表达及词语讲解核验

> **教训：这是历次校验中问题最集中的部分。第 1 课~さん、第 4 课全部 6 节、第 5 课人称②都是摘要。**

- [ ] **绝对禁止摘要化**——原书写了一段话就保留一段话，不得用 bullet 列表替代
- [ ] **原书例句和对话全部保留**——包括 `甲/乙` 对话、场景说明括号
- [ ] **标签保留**——原书的 `【礼貌语言 ①】` `【叹词 ③】` `【寒暄语 ①】` 等标签要写上
- [ ] **翻译保留**——每个例句的中文翻译不能省
- [ ] **说明文字忠于原文**——不用自己的话概括（第 4 课 JR 历史、一人暮らし 背景全被一句话替代）

### D. 补充表格/参考表核验

> **教训：第 1 课国家表 17→5，第 4 课城市设施 40+→11，第 5 课时间表严重缩水。**

- [ ] **表格条目数与原书一致**——数一下行数
- [ ] **特殊读音加粗/标记**——原书用阴影或加粗标记的音变要保留（如 300=さんびゃく）
- [ ] **表格标题和分类与原书一致**

### E. 应用课文核验

- [ ] **场景说明用原书原文**——不改写、不概括
- [ ] **舞台说明用括号**——`（小李朝牌子的方向走去）`，不改成 `#### 场景 1：` 等编辑性标题（第 5 课犯过）
- [ ] **中文舞台说明不加日语 ruby**——`翻了一下` 的「下」不标 `した`（第 2、4 课犯过）
- [ ] **对话逐句与 PDF 核对**

### F. 练习核验

- [ ] **例句对话完整展开**——多轮对话不能只写第一句（第 1 课练习 4 缺 4 句示范对话）
- [ ] **选择题带完整应答上下文**——`（はい・いいえ）` 后面的 `、そうです` 不能丢（第 1 课练习 II-2）
- [ ] **小题号和替换项与原书一致**

### G. 生词表核验

- [ ] **条目与原书核对**——不多不少（第 4 课把 ええと 错写成 ええ）
- [ ] **词性标签一致**——`[名]` `[动1]` `[动2]` `[动3]` `[代]` `[疑]` `[叹]` `[副]` `[专]`
- [ ] **翻译不自行添加**——原书写「学生」就不加「（大）」（第 1 课犯过）
- [ ] **表达列表完整**

### H. 专栏核验

> **教训：第 1-3 课专栏用 meta 语言（"原书补充页还提到"），第 4 课专栏整段缺失。**

- [ ] **专栏不能缺失**——每课的专栏/コラム 必须存在
- [ ] **用原书叙述**——不用 bullet 摘要，不加 "原书补充页还提到" 等 meta 语言
- [ ] **不加编辑性总结**——如「本课的出迎え场景就发生在…」这种联系性语句不加

### I. 翻译准确性

> **教训：第 3 课 これ 翻译成「那」，かばん 翻译成「书包」。**

- [ ] **これ/それ/あれ 对应 这/那**——不能混
- [ ] **专有名词翻译忠于原书**——不自创译名
- [ ] **语法术语用原书措辞**——原书说「疑问句及应答」不改成「名1は名2ですか」

### J. 格式一致性

- [ ] 完成度行：`> 当前完成度：\`S3+（精修版）\`。...`
- [ ] 各 Page 分隔用 `<div class="page-marker">`
- [ ] 章节顺序：基本课文 → 语法解释 → （补充表格）→ 表达及词语讲解 → 应用课文 → 练习 → 生词表 → 专栏
- [ ] 不留 meta 语言：「这一页原书给了…」「网页版先收…」「网页版保留…」一律删除

## 不做的事

| 不做 | 原因 |
|------|------|
| 不转写录音题音频内容 | 无音频源，保留题型和作答方式即可 |
| 不还原原书排版/分栏布局 | 目标是可读性，不是像素还原 |
| 不添加原书没有的内容 | 精修 ≠ 扩写（不加「本课学习检查」、不加场景标题） |
| 不修改章节编排顺序 | 保持与原书一致 |
| 不给生词表加 ruby 标签 | 生词表本身已用「かな（漢字）」格式 |
| 不给中文舞台说明加日语 ruby | 「翻了一下」的「下」是中文 |
