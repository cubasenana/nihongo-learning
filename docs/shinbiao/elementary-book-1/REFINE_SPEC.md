# 精修规范（S3+ 标准）

> 本文档记录从 S3 → S3+（精修版）的逐页校对标准。
> 基于第 1 课实际执行过程固化而成。

## 目标

不求与原书 100% 像素级还原，只求：

1. **AI 可直接抓取当教材**——结构清晰、内容完整，无需再 OCR
2. **人类可阅读学习**——排版舒适、注音到位、翻译准确

## 输入源

| 源 | 用途 |
|----|------|
| `resources/新标日/pages/page-XXX.png` | 原书 400dpi 渲染页，逐页对照的**唯一真相** |
| `docs/shinbiao/elementary-book-1/lesson-XX.md` | 现有 S3 版本，在此基础上修正 |
| `resources/新标日/md/raw/初级上册/lesson-XX.md` | OCR 原始提取，辅助交叉验证 |

## 逐页校对流程

```
对每一课（已知页码范围 Pages: XX-YY）：
  1. 读取 docs/ 中现有 markdown 全文
  2. 逐页读取对应 PNG（page-XXX.png ~ page-YYY.png）
  3. 按下方检查清单逐项比对
  4. 修正 → 更新完成度标记为 S3+
```

## 检查清单

### A. 内容完整性（最重要）

- [ ] **语法例句带完整应答**：原书的 `►` 例句如果有 `——` 应答部分，必须一并收录
- [ ] **表格/列表不缩水**：如国家表、动词变形表等，必须还原原书完整条目数
- [ ] **讲解段落不省略**：「表达及词语讲解」每个小节的说明文字和例句都要在
- [ ] **练习示范对话完整**：练习题的「例」如果是多轮对话，要完整展开
- [ ] **补充说明不丢失**：原书在语法点末尾的附加说明（如「×わたし 父」这类反例提示）

### B. 注音（ruby）

- [ ] 基本课文、会话：所有汉字词标注 `<ruby>漢字<rt>かな</rt></ruby>`
- [ ] 语法例句：保持注音
- [ ] 生词表：以 `かな（漢字）` 格式呈现，不需要 ruby 标签
- [ ] 练习题面：关键词注音，替换项可简写
- [ ] 纯假名词（如 わたし、あなた）：不加 ruby

### C. 结构一致性

- [ ] 每课顶部：`# 第X課` + 课名（带 ruby）
- [ ] `<div class="page-range">Pages: XX-YY</div>` 页码范围
- [ ] `> 当前完成度：S3+（精修版）...` 完成度说明
- [ ] 各 Page 分隔用 `<div class="page-marker" data-page="XX">Page XX</div>`
- [ ] 章节顺序固定为：基本课文 → 语法解释 → （补充表格）→ 表达及词语讲解 → 应用课文 → 练习 → 生词表 → 专栏

### D. 翻译与准确性

- [ ] 日语原文与 PDF 逐字核对，修正 OCR 残留错字（如「日禾人→日本人」「吉亩→吉田」）
- [ ] 中文翻译忠于原书，不自行发挥
- [ ] 语法术语使用原书用语（如原书写「疑问句及应答」就不改成别的说法）

### E. 不做的事

| 不做 | 原因 |
|------|------|
| 不转写录音题音频内容 | 无音频源，保留题型和作答方式即可 |
| 不还原原书排版/分栏布局 | 目标是可读性，不是像素还原 |
| 不添加原书没有的内容 | 精修 ≠ 扩写 |
| 不修改章节编排顺序 | 保持与原书一致 |
| 不给生词表加 ruby 标签 | 生词表本身已用「かな（漢字）」格式 |

## 完成度标记

修完后更新第一行 blockquote：

```markdown
> 当前完成度：`S3+（精修版）`。已逐页对照原书 PDF 校对；[具体改动摘要]。
```
