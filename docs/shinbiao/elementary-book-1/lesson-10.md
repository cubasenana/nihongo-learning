4. <ruby>昨日<rt>きのう</rt></ruby>は <ruby>日曜日<rt>にちようび</rt></ruby>でした。

## 语法解释

### 1. `名は 二类形 です / でした`

二类形容词是不以「い」结尾的形容词。做谓语时要在后面加「です」。其过去形式是「でした」。

- <ruby>京都<rt>きょうと</rt></ruby>の <ruby>紅葉<rt>もみじ</rt></ruby>は <ruby>有名<rt>ゆうめい</rt></ruby>です。（京都的红叶很有名。）  
- この <ruby>通り<rt>とおり</rt></ruby>は <ruby>静か<rt>しずか</rt></ruby>です。（这条街很安静。）  
- この <ruby>町<rt>まち</rt></ruby>は にぎやかでした。（这个城市从前很热闹。）  

### 2. `二类形 では ありません / では ありませんでした`

二类形容词做谓语时的否定形式是"二类形容词 + `では ありません`"。其过去形式的否定形式是"二类形容词 + `では ありませんでした`"。在口语里「では」一般说成「じゃ」。

- この <ruby>通り<rt>とおり</rt></ruby>は にぎやかでは ありません。（这条街不热闹。）  
- <ruby>昨日<rt>きのう</rt></ruby>は <ruby>暇<rt>ひま</rt></ruby>じゃ ありませんでした。（昨天不空闲。）  

### 3. `二类形な + 名`

二类形容词修饰名词时，用"二类形容词 + `な` + 名词"的形式。

- <ruby>奈良<rt>なら</rt></ruby>は <ruby>静か<rt>しずか</rt></ruby>な <ruby>町<rt>まち</rt></ruby>です。（奈良是座安静的城市。）  
- <ruby>奈良<rt>なら</rt></ruby>は にぎやかな <ruby>町<rt>まち</rt></ruby>では ありません。（奈良不是热闹的城市。）  

### 4. `名 でした / では ありません / では ありませんでした`

名词做谓语时的过去形式是"名词 + `でした`"。其否定形式是"名词 + `では ありません`"，过去的否定形式是"名词 + `では ありませんでした`"。

- <ruby>昨日<rt>きのう</rt></ruby>は <ruby>日曜日<rt>にちようび</rt></ruby>でした。（昨天是星期日。）  
- おとといは <ruby>休み<rt>やすみ</rt></ruby>でした。（前天是休息日。）  
- <ruby>昨日<rt>きのう</rt></ruby>は <ruby>雨<rt>あめ</rt></ruby>では ありませんでした。（昨天没下雨。）  
- おとといは <ruby>曇り<rt>くもり</rt></ruby>じゃ ありませんでした。（前天没有阴天。）  

### 5. `どんな + 名`

询问人或事物的性质时用「どんな + 名词」。「どんな」相当于汉语的"什么样的"。

- <ruby>横浜<rt>よこはま</rt></ruby>は どんな <ruby>町<rt>まち</rt></ruby>ですか。（横滨是个什么样的城市？）  
  ——とても にぎやかな <ruby>町<rt>まち</rt></ruby>です。（是个非常热闹的城市。）  
- <ruby>張<rt>ちょう</rt></ruby>さんは どんな <ruby>人<rt>ひと</rt></ruby>ですか。（小张是什么样的人？）  
  ——<ruby>親切<rt>しんせつ</rt></ruby>な <ruby>人<rt>ひと</rt></ruby>です。（是个很热情的人。）  

> 参考：「の」有询问内容和材料两种用法。如：「<ruby>何<rt>なん</rt></ruby>の <ruby>本<rt>ほん</rt></ruby>ですか——<ruby>自動車<rt>じどうしゃ</rt></ruby>の <ruby>本<rt>ほん</rt></ruby>です（什么书？——关于汽车的书。）」「<ruby>何<rt>なん</rt></ruby>の <ruby>料理<rt>りょうり</rt></ruby>ですか——<ruby>魚<rt>さかな</rt></ruby>の <ruby>料理<rt>りょうり</rt></ruby>です（什么菜？——鱼做的菜。）」  
> 「どんな」只用于询问性质。如：「どんな <ruby>本<rt>ほん</rt></ruby>ですか——おもしろい <ruby>本<rt>ほん</rt></ruby>です（什么样的书？——很有趣的书。）」「どんな <ruby>料理<rt>りょうり</rt></ruby>ですか——ちょっと <ruby>甘い<rt>あまい</rt></ruby> <ruby>料理<rt>りょうり</rt></ruby>です（什么样的菜？——有点儿甜的菜。）」

### 6. `どうですか / どうでしたか`

「どうですか」是询问对方对某状态的意见或感想时的表达方式。如果询问的是现在或未来的事情时用「どうですか」，询问的是过去的事情时用「どうでしたか」。另外「どうですか」有时还用于劝请对方进行某动作。

- この <ruby>料理<rt>りょうり</rt></ruby>は どうですか。（这道菜怎么样？）  
- <ruby>京都<rt>きょうと</rt></ruby>は どうでしたか。（京都怎么样？）  
- お<ruby>茶<rt>ちゃ</rt></ruby>は どうですか。（您喝杯茶吧。）  
  ——ありがとう ございます。（谢谢。）

### 7. `でも` 和 `そして`

「でも」是表示转折关系的连词，一般用于口语，而不用于正式的书面语。

- わたしの <ruby>部屋<rt>へや</rt></ruby>は <ruby>狭い<rt>せまい</rt></ruby>です。でも、きれいです。（我的房间虽小，但是很干净。）  

「そして」是表示并列关系的连词。

- わたしの <ruby>部屋<rt>へや</rt></ruby>は <ruby>狭い<rt>せまい</rt></ruby>です。そして、<ruby>暗い<rt>くらい</rt></ruby>です。（我的房间很小，而且很暗。）

## 表达及词语讲解

### 1. きれいです

由于一类形容词都以「い」结尾，因此经常有人误把二类形容词「きれい」当作一类形容词。一类形容词如「<ruby>新しい<rt>あたらしい</rt></ruby>」「<ruby>大きい<rt>おおきい</rt></ruby>」等，即使是用汉字书写也要把词尾「い」的部分用假名标示出来，而「きれい」是汉字写作「<ruby>綺麗<rt>きれい</rt></ruby>」，其中的「い」为二类形容词词尾汉字读音的一部分。

- きれいな <ruby>服<rt>ふく</rt></ruby>（漂亮的衣服）　× きれい <ruby>服<rt>ふく</rt></ruby>

### 2. おとといは <ruby>雨<rt>あめ</rt></ruby>でした

「<ruby>下雨<rt></rt></ruby>」「<ruby>下雪<rt></rt></ruby>」等表示自然现象的词句，在日语里常常使用「<ruby>雨<rt>あめ</rt></ruby>です」「<ruby>雪<rt>ゆき</rt></ruby>です」等名词句的形式。

- <ruby>晴れ<rt>はれ</rt></ruby>です（晴天）
- <ruby>雨<rt>あめ</rt></ruby>です（雨天）
- <ruby>曇り<rt>くもり</rt></ruby>です（阴天）
- <ruby>雪<rt>ゆき</rt></ruby>です（下雪）

### 3. あれえ 【叹词⑥】

感叹吃惊或意外而情不自禁发出的声音，由「あれ」的发音变化而来。发音时读开口的长音「えー」。

应用课文中小李对相当著名的美术馆竟然没有人来参观一事感到吃惊。

- あれえ、だれも いませんね。（咦，怎么一个人也没有？）  

### 4. ところで

转换话题时使用。

- ところで、この <ruby>美術館<rt>びじゅつかん</rt></ruby>には どんな <ruby>作品<rt>さくひん</rt></ruby>が ありますか。  
  （嗯，是吗。哦，请问这个美术馆里都有什么作品？）  

### 5. ～<ruby>中<rt>じゅう/ちゅう</rt></ruby>

「<ruby>場所<rt>ばしょ</rt></ruby>」+「<ruby>中<rt>じゅう</rt></ruby>」表示某场所范围内的全部。「<ruby>時間<rt>じかん</rt></ruby>/<ruby>期間<rt>きかん</rt></ruby>」+「<ruby>中<rt>じゅう</rt></ruby>」表示"在某个期间一直"。

| 场所 + 中 | | 时间/期间 + 中 | |
|---|---|---|---|
| <ruby>世界中<rt>せかいじゅう</rt></ruby> | 全世界 | <ruby>一日中<rt>いちにちじゅう</rt></ruby> | 一整天 |
| <ruby>日本中<rt>にほんじゅう</rt></ruby> | 全日本 | <ruby>一晩中<rt>ひとばんじゅう</rt></ruby> | 一整晚 |
| <ruby>学校中<rt>がっこうじゅう</rt></ruby> | 全校 | <ruby>一年中<rt>いちねんじゅう</rt></ruby> | 全年 |

### 6. <ruby>形容詞<rt>けいようし</rt></ruby> + の

应用课文中长岛说到「おもしろいのが あります」（还有很有意思的），这里的「の」代替了前文已经提到的名词。关于那雕刻的话题前面已经交代过了，因此不需要重复使用同样的词语，才能造出好的句子。

- おもしろいのが あります。（有有意思的〔东西〕。）

### 7. <ruby>紅葉<rt>もみじ</rt></ruby>：「こうよう」和「もみじ」

「<ruby>紅葉<rt>こうよう</rt></ruby>」有「こうよう」和「もみじ」两种读法。「こうよう」是表示秋天园林中树的落叶变红或叶子全都变成红色。「もみじ」多表示枫叶的红叶。

秋天，日本全国各地都能看到红叶。其中尤其以京都的红叶最为著名，这片红叶每年同样牵动着日本人的心。

### 8. <ruby>修学旅行<rt>しゅうがくりょこう</rt></ruby>

「<ruby>修学旅行<rt>しゅうがくりょこう</rt></ruby>」指的是让中小学生亲身体验日常生活中接触不到的异地风土人文化而由教育管理部门组织的旅行活动。一般安排在最后的学年。根据地区、学校的不同，所去的地方也不尽相同。有的学校还组织到中国或韩国等地去海外旅行。也作为修学旅行的目的地参观，很多学校都去京都和奈良。

### 专栏：<ruby>奈良<rt>なら</rt></ruby>和<ruby>京都<rt>きょうと</rt></ruby>

奈良和京都都作为游览胜地举世有名。两者都与中国有着深厚的渊源。

奈良有唐招提寺，是鉴真和尚于 759 年建造的。鉴真和尚受当时的圣武天皇邀请，历经 10 多年才由中国东渡日本，成为向日本传播佛教文化的高僧。唐招提寺于 1998 年被确定为世界文化遗产。鉴真和尚传奇的文化交流故事被完好地保存在那里。

在京都岚山的龟山公园建有一块「<ruby>周恩来<rt>しゅうおんらい</rt></ruby><ruby>総理<rt>そうり</rt></ruby>」碑。碑中「岚山」是已故的周恩来总理在日本留学时所作诗歌的题目。访问岚山时的诗句「<ruby>雨中岚山<rt></rt></ruby>」碑是 1979 年中日和平友好条约缔结后建造的，现已成为许多中国人访日时的游览地之一。
