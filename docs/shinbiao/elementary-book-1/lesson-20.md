## 语法解释

### 1. 动词基本形

本课开始正式使用 `基本形`。词典里的动词通常都写成这种形式：

- <ruby>書きます<rt>かきます</rt></ruby> → <ruby>書く<rt>かく</rt></ruby>
- <ruby>急ぎます<rt>いそぎます</rt></ruby> → <ruby>急ぐ<rt>いそぐ</rt></ruby>
- <ruby>食べます<rt>たべます</rt></ruby> → <ruby>食べる<rt>たべる</rt></ruby>
- <ruby>見ます<rt>みます</rt></ruby> → <ruby>見る<rt>みる</rt></ruby>
- <ruby>来ます<rt>きます</rt></ruby> → <ruby>来る<rt>くる</rt></ruby>
- します → する

### 2. `基本形 + ことが できます`

表示能力、可能：

- スミスさんは ピアノを <ruby>弾く<rt>ひく</rt></ruby> ことが できます。  
- <ruby>森<rt>もり</rt></ruby>さんは <ruby>中華料理<rt>ちゅうかりょうり</rt></ruby>を <ruby>作る<rt>つくる</rt></ruby> ことが できますか。  

简答时可以只说：

- はい、できます。  
- いいえ、できません。  

### 3. `〜ことです`

把动作名词化以后，可以用来说明“兴趣、特长、梦想”等：

- わたしの <ruby>趣味<rt>しゅみ</rt></ruby>は <ruby>切手<rt>きって</rt></ruby>を <ruby>集める<rt>あつめる</rt></ruby> ことです。  
- <ruby>王<rt>おう</rt></ruby>さんの <ruby>夢<rt>ゆめ</rt></ruby>は <ruby>外国<rt>がいこく</rt></ruby>で <ruby>働く<rt>はたらく</rt></ruby> ことです。  

<div class="page-marker" data-page="255">Page 255</div>

### 4. `基本形 + 前に`

表示“在……之前”：

- こちらへ <ruby>来る<rt>くる</rt></ruby> <ruby>前<rt>まえ</rt></ruby>に <ruby>電話<rt>でんわ</rt></ruby>を かけて ください。  
- <ruby>毎日<rt>まいにち</rt></ruby> <ruby>寝る<rt>ねる</rt></ruby> <ruby>前<rt>まえ</rt></ruby>に シャワーを <ruby>浴びます<rt>あびます</rt></ruby>。  

名词后面时则说：

- <ruby>会議<rt>かいぎ</rt></ruby>の <ruby>前<rt>まえ</rt></ruby>に、<ruby>資料<rt>しりょう</rt></ruby>を コピーします。  

### 5. `だれか / どこか / 何か`

`疑问词 + か` 可以表示“某人 / 某地 / 某物”：

- <ruby>部屋<rt>へや</rt></ruby>に だれか いますか。  
- <ruby>冬休み<rt>ふゆやすみ</rt></ruby>、どこかへ <ruby>行きたい<rt>いきたい</rt></ruby>です。  
- おなかが すきましたね。<ruby>何か<rt>なにか</rt></ruby> <ruby>食べましょう<rt>たべましょう</rt></ruby>。  

### 6. `よね`

表示说话人提出自己的判断，同时期待对方同意：

- <ruby>手作り<rt>てづくり</rt></ruby>の <ruby>餃子<rt>ぎょうざ</rt></ruby>は <ruby>皮<rt>かわ</rt></ruby>が おいしいですよね。  
- この <ruby>傘<rt>かさ</rt></ruby>は <ruby>森<rt>もり</rt></ruby>さんのですよね。

## 表达及词语讲解

### 1. `〜ことが できます` 有时也表示“不允许”

在规则、制度的语境里，它也可以用来表达“可以 / 不可以”：

- ここでは <ruby>写真<rt>しゃしん</rt></ruby>を <ruby>撮る<rt>とる</rt></ruby> ことが できません。  

### 2. `みんなで`

`で` 在这里表示“一起、作为一个整体地”：

- みんなで <ruby>餃子<rt>ぎょうざ</rt></ruby>を <ruby>作ります<rt>つくります</rt></ruby>。  
- クラス <ruby>全員<rt>ぜんいん</rt></ruby>で キャンプへ <ruby>行きました<rt>いきました</rt></ruby>。  

### 3. `趣味` 和 `興味`

- `趣味`：爱好
- `興味`：兴趣

所以：

- わたしの <ruby>趣味<rt>しゅみ</rt></ruby>は <ruby>切手<rt>きって</rt></ruby>を <ruby>集める<rt>あつめる</rt></ruby> ことです。  
- わたしは <ruby>日本<rt>にほん</rt></ruby>の <ruby>書道<rt>しょどう</rt></ruby>に <ruby>興味<rt>きょうみ</rt></ruby>が あります。  

### 4. `ごちそうします`

表示“请客、请别人吃饭”：

- <ruby>今度<rt>こんど</rt></ruby> <ruby>日本料理<rt>にほんりょうり</rt></ruby>を ごちそうします。
