## 语法解释

### 1. 动词的"基本形"

"基本形"是动词的基本形式。词典中的词条都使用这一形式。各类动词"ます形"去掉"ます"的形式和"基本形"的对应关系如下：

- 一类动词："基本形"最后的发音为"う"段，"ます形"去掉"ます"后的发音为"い"段。
- 二类动词："基本形"为"ます形"去掉"ます"后加"る"。
- 三类动词："基本形"分别为"来る"和"する"，"ます形"去掉"ます"后则为"来"和"し"。

| 类别 | ます形 | 去掉ます | → | 基本形 |
|------|--------|----------|---|--------|
| 一类动词 | <ruby>書<rt>か</rt></ruby>きます | かき | → | <ruby>書<rt>か</rt></ruby>く |
| | <ruby>急<rt>いそ</rt></ruby>ぎます | いそぎ | → | <ruby>急<rt>いそ</rt></ruby>ぐ |
| | <ruby>飛<rt>と</rt></ruby>びます | とび | → | <ruby>飛<rt>と</rt></ruby>ぶ |
| | <ruby>読<rt>よ</rt></ruby>みます | よみ | → | <ruby>読<rt>よ</rt></ruby>む |
| | <ruby>死<rt>し</rt></ruby>にます | しに | → | <ruby>死<rt>し</rt></ruby>ぬ |
| | <ruby>待<rt>ま</rt></ruby>ちます | まち | → | <ruby>待<rt>ま</rt></ruby>つ |
| | <ruby>売<rt>う</rt></ruby>ります | うり | → | <ruby>売<rt>う</rt></ruby>る |
| | <ruby>買<rt>か</rt></ruby>います | かい | → | <ruby>買<rt>か</rt></ruby>う |
| | <ruby>話<rt>はな</rt></ruby>します | はなし | → | <ruby>話<rt>はな</rt></ruby>す |
| 二类动词 | <ruby>食<rt>た</rt></ruby>べます | たべ | → | <ruby>食<rt>た</rt></ruby>べる |
| | <ruby>見<rt>み</rt></ruby>ます | み | → | <ruby>見<rt>み</rt></ruby>る |
| 三类动词 | <ruby>来<rt>き</rt></ruby>ます | き | → | <ruby>来<rt>く</rt></ruby>る |
| | します | し | → | する |

### 2. 名［人］は 动（基本形）ことが できます

"动词基本形 + ことが できます"表示能力。在回答别人的询问时，肯定的情况下可以单用"できます"，否定的情况下可以单用"できません"。

- スミスさんは ピアノを <ruby>弾<rt>ひ</rt></ruby>く ことが できます。（史密斯先生会弹钢琴。）
- <ruby>森<rt>もり</rt></ruby>さんは <ruby>中華料理<rt>ちゅうかりょうり</rt></ruby>を <ruby>作<rt>つく</rt></ruby>る ことが できますか。（森先生会做中国菜吗？）  
  ——はい、できます。（是的，会做。）

### 3. 名は 动（基本形）ことです

当谓语部分的内容是一种动作、行为时使用这个句型。

<div class="page-marker" data-page="238">Page 238</div>

- わたしの <ruby>趣味<rt>しゅみ</rt></ruby>は <ruby>切手<rt>きって</rt></ruby>を <ruby>集<rt>あつ</rt></ruby>める ことです。（我的爱好是集邮。）
- <ruby>王<rt>おう</rt></ruby>さんの <ruby>特技<rt>とくぎ</rt></ruby>は <ruby>胡弓<rt>こきゅう</rt></ruby>を <ruby>弾<rt>ひ</rt></ruby>く ことです。（小王的特长是拉胡琴。）
- <ruby>森<rt>もり</rt></ruby>さんの <ruby>夢<rt>ゆめ</rt></ruby>は <ruby>外国<rt>がいこく</rt></ruby>で <ruby>働<rt>はたら</rt></ruby>く ことです。（森先生的理想是在外国工作。）

> **参考** "动词基本形 + こと"可以起到和名词同样的作用。因此，也可以用名词或名词短语来代替。名词短语指下面例子中的"車の運転"这种形式。
>
> - <ruby>小野<rt>おの</rt></ruby>さんは <ruby>車<rt>くるま</rt></ruby>を <ruby>運転<rt>うんてん</rt></ruby>する ことが できません。（小野不会开车。）
> - <ruby>小野<rt>おの</rt></ruby>さんは <ruby>車<rt>くるま</rt></ruby>の <ruby>運転<rt>うんてん</rt></ruby>が できません。（小野不会开车。）

### 4. 动（基本形）<ruby>前<rt>まえ</rt></ruby>に、～

表示一个动作在另一个动作之前发生时，使用这个句型。

- こちらへ <ruby>来<rt>く</rt></ruby>る <ruby>前<rt>まえ</rt></ruby>に、<ruby>電話<rt>でんわ</rt></ruby>を かけて ください。（来之前请打个电话。）
- <ruby>毎日<rt>まいにち</rt></ruby> <ruby>寝<rt>ね</rt></ruby>る <ruby>前<rt>まえ</rt></ruby>に、シャワーを <ruby>浴<rt>あ</rt></ruby>びます。（每天睡觉前淋浴。）

这个句型也可以以"名词 + の + 前に"的形式使用。

- <ruby>会議<rt>かいぎ</rt></ruby>の <ruby>前<rt>まえ</rt></ruby>に、<ruby>資料<rt>しりょう</rt></ruby>を コピーします。（开会之前复印资料。）

### 5. 疑问词 + か

疑问词"いつ""どこ""だれ"等词后续"か"，表示"某时""某地""某人"等意思。

- <ruby>部屋<rt>へや</rt></ruby>に だれか いますか。（房间里有人吗？）
- <ruby>部屋<rt>へや</rt></ruby>に だれが いますか。（房间里有谁？）

"いつか""どこか""だれか""<ruby>何<rt>なに</rt></ruby>か"等本身并不表示疑问的意思，所以可用于陈述句。

- <ruby>冬休<rt>ふゆやす</rt></ruby>み、どこかへ <ruby>行<rt>い</rt></ruby>きたいです。（寒假想去个什么地方。）
- おなかが すきましたね。<ruby>何<rt>なに</rt></ruby>か <ruby>食<rt>た</rt></ruby>べましょう。（肚子饿了吧。吃点东西吧。）

### 6. ～よね

第 4 课和第 8 课分别学习了助词"ね"（☞第 4 课"语法解释 4"）和"よ"（☞第 8 课"语法解释 5"）。这两个助词组成"よね"，表示提出自己的意见，以征求对方的同意。

当说话人确信对方和自己的意见、想法完全相同时，只用"ね"，而说话人对自己的意见、想法没有足够的把握时用"よね"。

- <ruby>手作<rt>てづく</rt></ruby>りの <ruby>餃子<rt>ギョーザ</rt></ruby>の <ruby>皮<rt>かわ</rt></ruby>は おいしいですよね。（自己擀的饺子皮可好吃了。）
- この <ruby>傘<rt>かさ</rt></ruby>は <ruby>森<rt>もり</rt></ruby>さんのですよね。（这把伞是森先生的吧？）
- <ruby>小野<rt>おの</rt></ruby>さんも いっしょに <ruby>行<rt>い</rt></ruby>きますよね。（小野女士也一起去吧？）

## 表达及词语讲解

### 1. <ruby>写真<rt>しゃしん</rt></ruby>を <ruby>撮<rt>と</rt></ruby>る ことが できます ［可能和允许］

"动词基本形 + ことが できます"基本功能是表示可能，但有时根据上下文关系也表示允许。基本课文 B 中的"ここで 写真を 撮っても いいですか（可以在这儿照相吗？）"是一个希望得到允许的疑问句，回答是"いいえ、ここでは 撮る ことが できません（不行，不能在这儿照相）"表示不允许。允许的内容，从国家的规章制度到入住公司宿舍的条件等，有许多种类。

- <ruby>日本<rt>にほん</rt></ruby>では <ruby>何歳<rt>なんさい</rt></ruby>から <ruby>結婚<rt>けっこん</rt></ruby>する ことが できますか。  
  ——<ruby>女<rt>おんな</rt></ruby>の <ruby>人<rt>ひと</rt></ruby>は 16<ruby>歳<rt>さい</rt></ruby>から、<ruby>男<rt>おとこ</rt></ruby>の <ruby>人<rt>ひと</rt></ruby>は 18<ruby>歳<rt>さい</rt></ruby>から できます。  
  （在日本，多大年龄可以结婚？——女的 16 岁，男的 18 岁。）
- <ruby>寮<rt>りょう</rt></ruby>の <ruby>洗濯機<rt>せんたくき</rt></ruby>は <ruby>何時<rt>なんじ</rt></ruby>まで <ruby>使<rt>つか</rt></ruby>う ことが できますか。  
  ——8<ruby>時<rt>じ</rt></ruby>までです。それからは <ruby>使<rt>つか</rt></ruby>う ことが できません。  
  （宿舍的洗衣机可以用到几点？——用到 8 点，那以后就不能用了。）

### 2. ＊みんなで

由"みんな + で"组成。这里的"で"表示动作主体的样态，如"1人で（一个人）""自分で（自己）""全員で（全体人员）"。

- みんなで <ruby>餃子<rt>ギョーザ</rt></ruby>を <ruby>作<rt>つく</rt></ruby>ります。（大家在一起包饺子。）
- <ruby>先月<rt>せんげつ</rt></ruby> クラス <ruby>全員<rt>ぜんいん</rt></ruby>で キャンプへ <ruby>行<rt>い</rt></ruby>きました。（上个月我们班全体同学一起去野营了。）

### 3. "<ruby>趣味<rt>しゅみ</rt></ruby>"和"＊<ruby>興味<rt>きょうみ</rt></ruby>"

因为汉语中也有"趣味"这个词，所以致使不少人对日语的"趣味"屡屡用错。日语的"趣味"是"爱好""喜好"的意思，与"兴趣"无缘。表示"兴趣"的时候，使用"興味"一词。

- わたしの <ruby>趣味<rt>しゅみ</rt></ruby>は <ruby>切手<rt>きって</rt></ruby>を <ruby>集<rt>あつ</rt></ruby>める ことです。（我的爱好是集邮。）
- わたしは <ruby>日本<rt>にほん</rt></ruby>の <ruby>書道<rt>しょどう</rt></ruby>に <ruby>興味<rt>きょうみ</rt></ruby>が あります。（我对日本的书法感兴趣。）

### 4. ＊ごちそうします

"ごちそう"这个词本身的意思是奢华而高级的菜肴，后续动词"します"变成"ごちそうします"，则为请别人吃饭的意思。应用课文中，森对小野说"ごちそうして ください（请我们吃饭吧）"，意思是"希望你让我们品尝一下你做的饭菜"的意思。

- わたしは <ruby>時々<rt>ときどき</rt></ruby> <ruby>家<rt>いえ</rt></ruby>で <ruby>李<rt>り</rt></ruby>さんに <ruby>日本料理<rt>にほんりょうり</rt></ruby>を ごちそうします。（我时不时在家请小李吃日本料理。）
- あの <ruby>先生<rt>せんせい</rt></ruby>は <ruby>授業<rt>じゅぎょう</rt></ruby>が <ruby>終<rt>お</rt></ruby>わってから、よく <ruby>学生<rt>がくせい</rt></ruby>に ごちそうします。（那个老师上完课后常常请学生吃饭。）

## 专栏：中华街

在日本，有很多被称为"中華街"的街道。其中，横滨市、神户市和长崎市的中华街最为有名，被称为"3 大中華街"。

规模最大的当数横滨市的中华街，仅菜馆就有近 200 家，其中一半是广东菜馆。加上食品材料店、杂货店等，店铺总数约有 500 家以上。每年超过 1 800 万的人来此造访，据说是世界上最大的中华街。

神户市的中华街称为"南京町"，菜馆、杂货店等有 100 家以上。菜馆也多为广东菜馆。

长崎市有"新地中華街"，约 40 家菜馆、杂货店鳞次栉比。这里的石板路是长崎市的友好城市福州市协作修成的。

另外，日本和中国之间有许多像长崎市与福州市这样结成合作关系的友好城市。横滨市和上海市、神户市和天津市就是有合作关系的友好城市。

有了阿健以后，小野回家的时间就早了，阿健还不会"坐"。回家后小野训练"阿健"坐起来。阿健喜欢和小野一起玩。虽然小野每天也想早点回家陪它玩，但偶尔必须加班。小野将手机的待机画面设定为阿健的照片，因此她随时都能看到阿健。
