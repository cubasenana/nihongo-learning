## 语法点

### 1. 名1[人]は + 名2[人]に + 名3[物]を + くれます

- **含义**：（别人）给我/我方的人某物
- **接续**：人は + 人に + 物を + くれます
- **要点**：
  - 「くれます」表示别人给说话人或说话人一方的人
  - 与「あげます」「もらいます」都译为"给"，但视角不同
  - 「くれる」从给予者对我方的恩惠看，「もらう」从接受者看，两者视角不同但都常用
  - 也可用于别人给自己家人（＝我方的人）的情况：馬さんが妹（＝我的妹妹）にお土産をくれました
- **例句**：
  - 友達が誕生日にきれいな花をくれました。（朋友在我生日时送了我漂亮的花。）
  - 先輩がおもしろい本をくれました。（前辈给了我一本有趣的书。）
  - 母が手作りのケーキをくれました。（妈妈给了我手工做的蛋糕。）
- **Bunpro 校验**：待验证

### 2. 动词て形 + あげます

- **含义**：为别人做某事
- **接续**：动词て形 + あげます
- **要点**：
  - 表示说话人或说话人一方为别人做事
  - 含有施恩于人的语感，不能直接对尊长面对面使用
  - 主动提议帮忙时用「～ましょうか」；描述已做的事时避开「てあげる」，用中性表达
  - 受益者需要说明时用「に」标记；上下文清楚时可省略
- **例句**：
  - 重い荷物を持ってあげました。（帮忙拿了沉重的行李。）
  - 弟に漢字の読み方を教えてあげました。（教了弟弟汉字的读法。）
  - 道に迷っている人に駅までの行き方を教えてあげました。（给迷路的人指了去车站的路。）
- **Bunpro 校验**：待验证

### 3. 动词て形 + もらいます

- **含义**：请别人为自己做某事 / 承受别人做某事带来的恩惠
- **接续**：动词て形 + もらいます（做事的人用「に」标记）
- **要点**：
  - 两层含义：请别人做事 + 承受恩惠
  - 做事的人用「に」：李さんに案内してもらいました
  - 希望请别人做事时：～てもらいたいのですが
- **例句**：
  - 先生に作文を直してもらいました。（请老师改了作文。）
  - 日本人の友達に発音をチェックしてもらいました。（请日本朋友帮我检查了发音。）
  - すみません、もう少し待ってもらいたいのですが。（不好意思，想请您再稍等一下。）
- **Bunpro 校验**：待验证

### 4. 动词て形 + くれます

- **含义**：（别人）为我/我方的人做某事
- **接续**：动词て形 + くれます
- **要点**：
  - 主语是说话人以外的人，为说话人或说话人一方做事
  - 疑问形式可用于请求：～てくれますか（偏直接，适合关系亲密的人）
  - 否定形式「～てくれませんか」比肯定更客气；对上级更稳的是「ていただけませんか」
- **例句**：
  - 同僚が仕事を手伝ってくれました。（同事帮我做了工作。）
  - 隣の人がごみの出し方を教えてくれました。（邻居告诉了我垃圾的扔法。）
  - すみません、この漢字の読み方を教えてくれませんか。（不好意思，能告诉我这个汉字怎么读吗？）
- **Bunpro 校验**：待验证

## 课文

> Arc 7「新城市」第4话 — 搬家互助

**登場人物**：陳明、王、李さん（隣人）— 陳明の新しいアパート

> 使用语法：≤L28（+くれます、動てあげます、動てもらいます、動てくれます）

---

陳：先週 新しい アパートに 引っ越しました。

王：大変でしたね。だれかに 手伝って もらいましたか。

陳：はい、隣の 李さんが 荷物を 運んで くれました。

王：いい 人ですね。

陳：ええ。李さんは お茶も くれました。それから、近くの おいしい レストランも 教えて くれました。

王：陳さんは 李さんに 何か お礼を しましたか。

陳：はい、李さんが 日本の お菓子を 欲しいと 言いましたから、お菓子を あげました。それから、パソコンの 使い方を 教えて あげました。

王：すごいですね。私も 引っ越しの 時、友達に 手伝って もらいました。

陳：手伝って もらって、とても 助かりました。

---

**语法覆盖**：✅ くれます（お茶もくれました） ✅ てあげます（教えてあげました/お菓子をあげました） ✅ てもらいます（手伝ってもらいました） ✅ てくれます（運んでくれました/教えてくれました）

## 表达要点

| 要点 | 说明 |
|------|------|
| ～てあげます的礼貌问题 | 对尊长不能直接说「～てあげます」，改用「～ましょうか」 |
| それに | 用于前后内容为累加关系："而且……"，多见于口语 |
| ～先 | 表示移动性动作的到达地/归属：引っ越し先（搬迁地址）、出張先（出差地） |
| どの辺 | 询问大致场所时使用，回答大致地名或地标即可 |
