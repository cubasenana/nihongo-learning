## 语法点

### 1. 小句（动词简体形）+ 名词（定语从句）

- **含义**：用动词小句修饰名词，相当于中文的"……的（名词）"
- **接续**：动词普通形 + 名词（名词前不能用ます形，不加「の」）
- **要点**：
  - 将来/习惯性动作用基本形：明日使う資料（明天要用的资料）
  - 已完成动作用た形：倒産した会社（倒闭的公司）
  - 也可用ない形/ている形等：まだ届いていない荷物（还没送到的包裹）
  - 名词前绝对不能用ます形：× 座りました場所
  - 名词前不能加「の」：× 座るの場所
- **例句**：
  - 毎朝飲むコーヒーはブラジル産です。（每天早上喝的咖啡是巴西产的。）
  - 昨日届いた荷物を開けました。（打开了昨天送到的包裹。）
  - 来月発表するレポートの準備をしています。（正在准备下个月要发表的报告。）
- **Bunpro 校验**：待验证

### 2. 小句（动词简体形）+ 名词 + は～です（修饰名词做主语）

- **含义**：由动词小句修饰的名词短语充当句子主语
- **接续**：小句（动词简体形）+ 名词 + は + 名词/一类形/二类形 + です
- **要点**：
  - 定语可以很长，日语允许较长的前置修饰语
  - 主语的「は」标记不变
  - 可将两个短句合并为一个含长定语的句子
- **例句**：
  - 母が作る料理はいつもおいしいです。（妈妈做的菜总是很好吃。）
  - 先週読んだ小説はとてもおもしろかったです。（上周读的小说非常有趣。）
  - 駅前にできた店はラーメン屋です。（车站前面开的店是拉面馆。）
- **Bunpro 校验**：待验证

### 3. 小句（动词简体形）+ 名词 + を/に/から + 动词（修饰名词做宾语等）

- **含义**：由动词小句修饰的名词短语充当宾语或其他成分
- **接续**：小句（动词简体形）+ 名词 + を/に/から + 动词
- **要点**：
  - 修饰名词的动词小句中，主语通常用「が」而非「は」
  - × わたしは知らない人 → ○ わたしが知らない人
  - 被修饰名词可以带任何助词做各种句子成分
- **例句**：
  - 友達が撮った写真をSNSに載せました。（把朋友拍的照片发到了社交媒体上。）
  - 日本に住んでいる友達に手紙を書きます。（给住在日本的朋友写信。）
  - わたしが知らない番号から電話がかかってきました。（一个我不认识的号码打来了电话。）
- **Bunpro 校验**：待验证

### 4. 小句（一类形/二类形/名）+ 名词

- **含义**：用形容词或名词小句修饰名词
- **接续**：一类形容词基本形/二类形容词＋な/名词＋の + 名词
- **要点**：
  - 一类形容词直接用基本形修饰：入り口が広い建物
  - 二类形容词用「な」连接：操作が簡単なパソコン
  - 名词用「の」连接：数学が専門の先生
  - 形容词修饰名词时不能加「の」：× 広いの建物
- **例句**：
  - 値段が安くて品質がいい商品を探しています。（在找价格便宜品质又好的商品。）
  - デザインがきれいな服が欲しいです。（想要设计漂亮的衣服。）
  - 英語が専門の先生に相談しました。（向教英语的老师咨询了。）
- **Bunpro 校验**：待验证

## 课文

> Arc 7「新城市」第1话 — 初到上海

**登場人物**：陳明、王さん（上海の同僚）— 空港からオフィスへ

> 使用语法：≤L25（+動小句+N定語、修飾N做主語/賓語、形/N小句+N）

---

王：陳さん、お疲れ様です。空港から 来た 道は 混んで いましたか。

陳：いいえ、それほど 混んで いませんでした。

王：あそこに 見える 高い ビルが 私たちの オフィスです。

陳：きれいな ビルですね。

王：駅の 前に できた 新しい ビルです。去年 建てた ビルですから、中も きれいですよ。

陳：入り口が 広い 建物ですね。

王：ここが 陳さんが 使う 部屋です。

陳：ありがとうございます。窓から 見える 景色が きれいですね。去年 撮った 写真を 見せて ください。

王：はい、どうぞ。

---

**语法覆盖**：✅ 動小句+N（来た道/見える高いビル/できた新しいビル/使う部屋/見える景色/撮った写真） ✅ 修飾N做主語（空港から来た道は混んでいましたか） ✅ 修飾N做賓語（去年撮った写真を見せてください） ✅ 形/N+N（新しいビル/きれいなビル/入り口が広い建物）

## 表达要点

| 要点 | 说明 |
|------|------|
| ～のところ | 把非场所名词变成场所表达：窓のところ（窗户那儿）、課長のところ（科长那儿） |
| ～でしたね | 用于确认双方都知道但记得不太确切的事情，常与「たしか」呼应 |
| 大きな / 小さな | 与「大きい/小さい」意思相同，但只能修饰名词不能结句 |
| このあたり | 表示某场所的周边一带：このあたりはよく渋滞します |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。确认语法规则是否记住。

### Level 2：组装（30%）

给词造句，3-5 题。主动运用语法。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：初到新城市的见闻**

- **场景**：你刚到一个新的城市出差，同事开车带你从机场去酒店。一路上你看到各种建筑和风景，向同事提问，同事做介绍。描述你看到的事物、认识的人、以及对这座城市的第一印象。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ 动词小句修饰名词（定语从句）
    - ✅ 修饰名词做主语
    - ✅ 修饰名词做宾语
    - ✅ 形容词/名词修饰名词
- **词汇限制**：仅使用 lesson ≤ 25 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
