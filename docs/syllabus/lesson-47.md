## 语法点

### 1. 敬语概述

- **含义**：对会话中涉及的人物或听话人表示敬意的语言体系
- **接续**：无固定接续，是一套分类体系
- **要点**：
  - 尊他语：抬高对方或话题中的人物（本课重点）
  - 自谦语：压低说话人自身或己方人物（第48课重点）
  - 礼貌语：使用「です/ます」等表示对听话人的敬意
  - 对自己的家人不使用尊他语
  - 对外谈及自己公司的上级时也不用尊他语
- **例句**：
  - 社長がそうおっしゃいました。（总经理这样说了。）[尊他语]
  - 私が参ります。（我去。）[自谦语]
- **Bunpro 校验**：待验证

### 2. ～（ら）れます（尊他语）

- **含义**：用动词被动形式作为尊他语的一种（表示较轻微的敬意）
- **接续**：与被动形式完全相同（一类去ない+れる、二类去ない+られる、三类特殊）
- **要点**：
  - 形式与被动句相同，但意义不同——只是单纯表示尊敬
  - 不改变句子成分的位置和助词
  - 三种尊他形式中敬意程度最低
  - 根据上下文区分是被动还是尊他
- **例句**：
  - 周先生は明日日本へ行かれます。（周老师明天去日本。）
  - 今朝何時に起きられましたか。（您今天早上几点起床的？）
  - 部長は毎朝公園を散歩されるそうです。（据说部长每天早上都在公园散步。）
- **Bunpro 校验**：待验证

### 3. お＋动词＋になります

- **含义**：表示尊他的正式表达，敬意程度高于「～（ら）れます」
- **接续**：お + 一类动/二类动的ます形去ます + になります
- **要点**：
  - ます形去ます后只有一个音节的动词不能用此形式（× お見になります）
  - する/来る本身不能用此形式；サ変复合词另按お/ご规则处理（ご利用になります等）
  - 比「～（ら）れます」表示更深的敬意
- **例句**：
  - お客様はもうお帰りになりました。（客人已经回去了。）
  - お食事はもうお済みになりましたか。（您已经用过饭了吗？）
  - 旅行の日程はもうお決めになりましたか。（旅行日程您已经定下来了吗？）
- **Bunpro 校验**：待验证

### 4. お＋动词＋ください / ご＋三类动汉字部分＋ください

- **含义**：请您……（礼貌地劝说对方做某事）
- **接续**：
  - 一类动/二类动：お + ます形去ます + ください
  - 三类动词：ご + 汉字部分 + ください
- **要点**：
  - 用于劝说对方做有益于对方或公共利益的事
  - ます形去ます后只有一个音节的动词不能用（× お見ください）
  - 来る/する本身不能用此形式；サ変复合词可用ご～ください（ご利用ください、ご確認ください等）
  - 比单纯「～てください」更尊敬
- **例句**：
  - どうぞお座りください。（请坐。）
  - エスカレーターをご利用ください。（请利用自动扶梯。）
  - こちらのドアからお入りください。（请从这个门进去。）
- **Bunpro 校验**：待验证

### 5. 尊他语的特殊形式

- **含义**：部分高频动词有专用的尊他语形式，实际使用中优先用这些形式
- **接续**：固定对应关系（需要记忆）
- **要点**：
  - 見る → ご覧になります
  - 食べる/飲む → 召し上がります
  - 行く/来る/いる → いらっしゃいます / おいでになります
  - する → なさいます
  - 言う → おっしゃいます
  - くれる → くださいます
  - ～ている → ～ていらっしゃいます
  - 知っています → ご存じです
  - 有特殊形式的动词优先使用特殊形式
- **例句**：
  - 先生、何を召し上がりますか。（老师，您吃点什么？）
  - もうその資料をご覧になりましたか。（您已经看了那份资料了吗？）
  - 木村部長は何時に戻るっておっしゃいましたか。（木村部长说几点回来？）
- **Bunpro 校验**：待验证

## 课文

> Arc 12「開業」第3话 — 総経理视察（尊他語）

**登場人物**：陳明、社長 — 上海事務所

> 使用语法：≤L47（+敬語概述、～(ら)れます尊他、お～になります、お/ご～ください、特殊尊他形式）

---

陳：社長、お忙しい 中、上海まで お越しくださいまして、ありがとうございます。

社長：いや、新しい 事務所を 見たかったんだ。

陳：まず、こちらの 会議室を ご覧ください。

社長：きれいだね。もう スタッフは 集まったか。

陳：はい、先週 全員 そろいました。社長、こちらの 資料は もう 読まれましたか。

社長：ああ、読んだよ。昼食は もう 召し上がったか。

社長：いや、まだだ。

陳：では、近くの レストランで お食事を なさいませんか。

社長：そうだな。陳くん、この 1年で ずいぶん 成長したね。

陳：ありがとうございます。皆様が いろいろ 教えて くださいました。

社長：田中は 元気に して いるか。

陳：はい、田中さんは 東京で お元気に していらっしゃいます。

---

**语法覆盖**：✅ (ら)れます尊他（資料は読まれましたか） ✅ お～になります（お越しくださいまして） ✅ お/ご～ください（ご覧ください） ✅ 特殊尊他（ご覧になる/召し上がる/なさる/いらっしゃる/くださる）

## 表达要点

| 要点 | 说明 |
|------|------|
| ～じゃないか | 与「ね」语义相近但更强烈，多为男性上级对下级使用，降调 |
| ～ていらっしゃいます | 「～ています」的尊他形式（不是「ていきます/てきます」的尊他形式） |
| ～てくださって | 「～てくれて」的尊他形式，含感谢对方恩惠的语气 |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。确认三种尊他形式的使用规则和特殊形式的对应关系。

### Level 2：组装（30%）

给词造句，3-5 题。将普通句子改写为尊他语。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：总经理视察新办公室**

- **场景**：东京总公司的总经理来视察你们新开设的办公室。你作为接待人员，在机场迎接总经理，询问总经理是否已用餐、是否先去酒店还是直接去办公室，到办公室后介绍设施并请总经理入座，转达同事对总经理的问候，汇报总经理对办公室的评价。全程使用尊他语。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ ～（ら）れます（尊他）
    - ✅ お～になります
    - ✅ お～ください / ご～ください
    - ✅ 尊他特殊形式（至少使用3种不同的特殊形式）
- **词汇限制**：仅使用 lesson ≤ 47 的词汇
- **AI 验收标准**：
  1. 语法检查：尊他语形式是否正确，不可用的动词是否避开了
  2. 覆盖检查：是否全部尊他形式都用到了，特殊形式是否达到3种
  3. 自然度评分：商务接待的语境和敬语使用是否自然得体
