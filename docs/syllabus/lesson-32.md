## 语法点

### 1. 动（基本形/ない形）つもりです

- **含义**：打算做……/打算不做……（表示说话前已形成的意志/计划）
- **接续**：动词基本形/ない形 + つもりです
- **要点**：
  - 与「～ようと思っています」意思相近
  - 肯定：行くつもりです（打算去）
  - 否定：行かないつもりです（打算不去）
  - 不宜对尊长用「～つもりですか」提问，可能听起来像质问"你真打算这么做？"
- **例句**：
  - 来月から毎日運動するつもりです。（打算从下个月起每天运动。）
  - 夏休みに北海道を旅行するつもりです。（打算暑假去北海道旅行。）
  - もうお酒は飲まないつもりです。（打算不再喝酒了。）
- **Bunpro 校验**：待验证

### 2. 动（基本形/ない形）ことにします / ことにしました

- **含义**：决定做……/决定不做……（说话人自己做出的决定）
- **接续**：动词基本形/ない形 + ことにします/ことにしました
- **要点**：
  - 强调是说话人自己主动做出的决定
  - 「ことにします」表示做出决定（不限于当场）
  - 「ことにしました」表示已经决定了
  - 「こと」不能换成「の」
- **例句**：
  - 明日から毎朝走ることにします。（我决定从明天起每天早上跑步。）
  - 飛行機で行くことにしました。（决定坐飞机去了。）
  - 今年の夏は旅行に行かないことにしました。（决定今年夏天不去旅行了。）
- **Bunpro 校验**：待验证

### 3. 动（基本形/ない形）ことになりました

- **含义**：定下来要……（表示决定的结果，不强调是谁做的决定）
- **接续**：动词基本形/ない形 + ことになりました
- **要点**：
  - 与「ことにしました」对比：后者是自己决定，前者是外在因素导致
  - 常用于公司安排、规定变更等场景
  - 也可用于婉转表达自己的决定：結婚することになりました
  - 「こと」不能换成「の」
- **例句**：
  - 来月から上海に転勤することになりました。（定下来下个月调去上海了。）
  - 今週の会議は中止になりました。（这周的会议取消了。）
  - 来年から会社の休みが増えることになりました。（从明年起公司假期增加了。）
- **Bunpro 校验**：待验证

### 4. 小句（简体形）そうです（传闻）

- **含义**：听说……/据说……（转述从别处获得的信息）
- **接续**：小句简体形 + そうです；消息来源用「～によると」
- **要点**：
  - 明确表示信息不是自己直接感知的，而是从别人那里听到的
  - 提示消息来源：天気予報によると（据天气预报说）、ニュースによると（据新闻说）
  - 「そうです」本身不变否定/过去形；被转述的内容可带否定/过去
  - × 暇だそうではありません → ○ 暇ではないそうです / ○ 行ったそうです
- **例句**：
  - 天気予報によると、明日は雪が降るそうです。（据天气预报说，明天会下雪。）
  - 友達の話によると、あのレストランはとてもおいしいそうです。（据朋友说，那家餐馆很好吃。）
  - ニュースによると、来月ガソリンの値段が上がるそうです。（据新闻说，下个月汽油要涨价。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| 今度 | 语义比中文的"这次"广，有时相当于"下次"：今度の日曜日（这个/下个星期天） |
| ずっと（期间） | 表示某期间内动作/状态一直持续：連休中はずっと晴れだそうです |
| ～てほしい | 希望别人去做某事（礼貌程度低，不能对尊长直接使用）：手伝ってほしい |
| お邪魔します | 进入别人家时说「お邪魔します」（打扰了），告辞时说「お邪魔しました」 |
