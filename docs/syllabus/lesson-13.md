## 语法点

### 1. 名【数量】+ 动

- **含义**：在动词前加数量词，表示事物的数量
- **接续**：名词 + を/が + 数量词 + 动词
- **要点**：
  - 数量词直接放在动词前面
  - 量词因所修饰事物种类不同而不同（冊、枚、台、本、杯、個等）
  - 询问数量：「何 + 量词」（何枚、何冊等）
- **例句**：
  - ノートを3冊買いました。（买了3本笔记本。）
  - 部屋に猫が2匹います。（房间里有两只猫。）
- **Bunpro 校验**：待验证

### 2. 名【时间数量】+ 动

- **含义**：时间数量词说明动作或状态的持续时间
- **接续**：时间数量词 + 动词（时间词后面不加「に」）
- **要点**：
  - 表示持续时间时，时间词后不加「に」
  - 常见时间量词：～時間、～日、～週間、～か月、～年
  - 注意：「に」表示时间点，不加「に」表示时间段
- **例句**：
  - 毎日8時間働きます。（每天工作8小时。）
  - 昼1時間休みます。（中午休息1小时。）
- **Bunpro 校验**：待验证

### 3. 名【时间】に 名【次数】+ 动

- **含义**：在一定时间内进行若干次动作
- **接续**：时间 + に + 次数 + 动词
- **要点**：
  - 时间段用「に」标记表示"每……"
  - 常省略「1」和「間」：1週間→週に、1か月→月に、1年→年に
  - 常搭配「ぐらい」表示大概频率
- **例句**：
  - 週に3回ジムへ行きます。（一周去3次健身房。）
  - 月に2回映画を見ます。（一个月看2次电影。）
- **Bunpro 校验**：待验证

### 4. 动词词干（ます形去ます）

- **含义**：动词去掉「ます」后的词干形式，用于构成各种句型
- **接续**：动词ます形 - ます = 词干（「ます形」指「食べます」整体，「食べ」是词干）
- **要点**：
  - 書きます → 書き、食べます → 食べ、来ます → 来（き）
  - 词干形式在后续课程中会大量使用
  - 「します」→「し」
- **例句**：
  - 読みます → 読み（用于后续句型构成）
  - 散歩します → 散歩し（用于后续句型构成）
- **Bunpro 校验**：待验证

### 5. 名【场所】へ 动（ます形去ます）に 行きます / 来ます

- **含义**：表示移动行为的目的，"去/来做某事"
- **接续**：场所 + へ + 动词词干 + に + 行きます / 来ます
- **要点**：
  - 动词用ます形去掉ます的形式
  - 前面可以带宾语：名词 + を + 动词词干 + に + 行きます
  - 也可以省略场所，直接说「动词词干 + に行きます」（飲みに行きます、買いに行きます）
- **例句**：
  - デパートへ服を買いに行きます。（去商场买衣服。）
  - 友達が家へ遊びに来ました。（朋友来我家玩了。）
- **Bunpro 校验**：待验证

### 6. 名【数量】+ で

- **含义**：表示以几个为一组计价
- **接续**：数量词 + で + 金额 + です
- **要点**：
  - 用于按组/打计价的场合
  - 单价通常不用「で」：「1個 200円」
  - 成组/合计计价用「で」：「3個で 500円」
- **例句**：
  - このりんごは5個で800円です。（这苹果5个800日元。）
  - 切手は10枚で820円です。（邮票10张820日元。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| くらい/ぐらい | 接在数量词后表示大概，「どのぐらい」用于询问 |
| どのぐらいかかりますか | 「かかります」用于时间和金钱，意为"花费、需要" |
| 飲みに行きます | 省略宾语时，「飲みに行きます」特指"去喝酒" |
| とりあえず | "暂且先……"，点菜时常用，先点最急需的 |
