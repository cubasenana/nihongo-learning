## 语法点

### 1. ～ようです② / ～みたいです②（比喻）

- **含义**：像……一样（用于比喻，非推测）
- **接续**：
  - 名词 + の + ようです / 名词 + みたいです（みたい前不加の）
  - 动词简体形 + ようです / みたいです
- **要点**：
  - 第44课学的是推测用法，本课是比喻用法
  - 比喻时常与副词「まるで」呼应使用
  - 修饰名词：～ような/みたいな + 名词
  - 修饰动词/形容词：～ように/みたいに + 动词/形容词
- **例句**：
  - これは柔らかくて、まるで本物の毛皮のようです。（这个软软乎乎的，好像真皮一样。）
  - まるでヨーロッパの町並みを見ているようですね。（真像看到了欧洲的街道。）
  - この野菜、まるで果物みたいですね。（这种菜好像水果一样。）
- **Bunpro 校验**：待验证

### 2. 名1＋らしい＋名2（典型性）

- **含义**：具有……的典型特征（像样的……）
- **接续**：名词1 + らしい + 名词2
- **要点**：
  - 与第44课推测用法不同，这里表示「名词2」具有「名词1」的典型特征
  - 常与副词「いかにも」呼应使用
  - 「名词1」=「名词2」时，意思是"真正像样的～"：仕事らしい仕事
- **例句**：
  - この着物はいかにも日本らしい柄ですね。（这件和服的花纹是典型的日本式。）
  - 今日は春らしい天気です。（今天像是春天的天气。）
  - このごろ忙しくて、食事らしい食事をしていません。（最近很忙，没正经吃过一顿饭。）
- **Bunpro 校验**：待验证

### 3. 名（时间）までに

- **含义**：在……之前（表示某事态完成的截止时间）
- **接续**：时间名词 + までに
- **要点**：
  - 「まで」= 持续到某时间点（3時まで勉強する → 学到3点）
  - 「までに」= 在某时间点之前完成（3時までに終わる → 3点之前完成）
  - 两者含义不同，容易混淆
  - 「までには」把截止点拿出来强调/对比，常含"至少到那时为止要完成"的意思
- **例句**：
  - 明日の9時までにこの書類を完成させなければなりません。（明天9点之前必须完成这份文件。）
  - 今月の末までに新しい家を見つけたいです。（月底之前想找到一处新房子。）
  - 来月の初めまでには事務所開設の準備を済ませたいです。（下个月初之前要完成开设事务所的准备工作。）
- **Bunpro 校验**：待验证

### 4. ～間 / ～間に

- **含义**：在……期间（一直）/ 在……期间（发生/完成了某事）
- **接续**：
  - 动词简体形 + 間/間に
  - 名词 + の + 間/間に
- **要点**：
  - 「間」：后接持续性动作，常与「ずっと」搭配（整个期间都在做）
  - 「間に」：后接在期间内发生或完成的事项（不占满整段期间）
  - 区分关键：后项是否贯穿整个期间（是→間，否→間に）
- **例句**：
  - 部屋の中を見ている間、ずっと外で車の音がしていましたよ。（看房间的时候，外面一直有车的噪音。）
  - わたしが留学している間に、家の周りもずいぶん変わりました。（我留学期间，我家那一带发生了很大的变化。）
  - 子供が寝ている間に、部屋を掃除しました。（趁孩子睡觉的时候，打扫了房间。）
- **Bunpro 校验**：待验证

### 5. ～のような味/においがします

- **含义**：有……那样的味道/气味（描述感官体验的比喻表达）
- **接续**：名词 + のような + 味/におい + がします
- **要点**：
  - 「味がします」＝尝到/感觉到味道；「味がありません」＝没有味道/缺乏味道（两者都可用，含义不同）
  - 也可用于声音（声/音がします）和感觉（感じがします）
  - 与比喻的「ようです」组合使用
  - 身体状况也可以：頭痛がします、吐き気がします
- **例句**：
  - この野菜、レモンのような味がしますね。（这菜有股柠檬的味道。）
  - このスープは日本のみそ汁のようなにおいがします。（这汤有股日本酱汤的味儿。）
  - 外で子供の声がします。（外面有孩子们的声音。）
- **Bunpro 校验**：待验证

## 课文

> Arc 12「開業」第2话 — 事务所选址

**登場人物**：陳明、王 — 候補物件を見学

> 使用语法：≤L46（+よう/みたい比喻、らしい典型性、までに、間/間に、がします）

---

王：この オフィスは まるで ホテルの ロビーの ようですね。

陳：きれいですね。いかにも 上海らしい 雰囲気です。

王：月末までに 決めなければ なりませんね。

陳：はい。見学して いる 間に、いくつか 気づいた ことが あります。

王：何ですか。

陳：この 部屋は コーヒーの ような 匂いが しますね。

王：あ、隣に カフェが ありますね。この 部屋は 図書館みたいに 静かですね。

陳：見学して いる 間、ずっと この 匂いが して いました。

王：来月の 初めまでには 契約を 済ませたいですね。

陳：そうですね。今週中に 課長に 報告します。

---

**语法覆盖**：✅ ようです比喻（ホテルのロビーのよう） ✅ みたいです比喻（図書館みたいに静か） ✅ らしい典型性（上海らしい雰囲気） ✅ までに（月末までに/来月の初めまでには） ✅ 間/間に（見学している間/間に） ✅ がします（匂いがします）

## 表达要点

| 要点 | 说明 |
|------|------|
| 済ませます / 済みます | 他动词"结束某事" / 自动词"事情结束"，一对自他动词 |
| ～んじゃないですか | 婉转陈述意见，比「のではないでしょうか」礼貌程度略低 |
| いかにも | 与「らしい」呼应使用时，只能表示典型性，不能表示推测 |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。区分「ようです/みたいです」的推测和比喻用法，区分「間」和「間に」，区分「まで」和「までに」。

### Level 2：组装（30%）

给词造句，3-5 题。用给定情境造比喻句、らしい典型性句、間/間に句。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：上海事务所选址实地考察**

- **场景**：你和同事去上海为公司新事务所选址。你们参观了几个候选地点，讨论各自的感受（某个地方像教室一样、某个街区像欧洲的街道一样），考虑截止期限（月底之前必须决定），描述在看房期间发生的事情（看房的时候外面一直在施工），最后在当地品尝特色料理并描述味道。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ ～ようです / ～みたいです（比喻）
    - ✅ ～らしい（典型性）
    - ✅ ～までに
    - ✅ ～間 / ～間に
    - ✅ ～のような味がします
- **词汇限制**：仅使用 lesson ≤ 46 的词汇
- **AI 验收标准**：
  1. 语法检查：比喻和推测用法是否区分正确，「間」和「間に」是否选择正确
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：实地考察的场景和感官描述是否自然
