## 语法点

### 1. ば形（条件形变形规则）

- **含义**：日语动词和一类形容词的假定条件形式
- **接续**：
  - 一类动词：基本形末音变为「え」段 + ば（書く→書けば）
  - 二类动词：基本形去「る」+ れば（食べる→食べれば）
  - 三类动词：来る（くる）→来れば（くれば）、する→すれば
  - 一类形容词：去「い」+ ければ（楽しい→楽しければ）
- **要点**：
  - 变形口诀：う段→え段＋ば
  - ない→なければ（否定条件）
  - 初级阶段二类形容词/名词的条件形用「なら」；正式场合也有「～であれば」的形式
- **例句**：
  - 毎日練習すれば、上手になりますよ。（每天练习的话就会变好的。）
  - 安ければ、買いたいです。（如果便宜就想买。）
  - 時間がなければ、行かなくてもいいです。（没有时间的话不去也行。）
- **Bunpro 校验**：待验证

### 2. 小句1 ば、小句2（ば条件句）

- **含义**：如果……就……（表示后句成立的必要前提条件）
- **接续**：小句1（ば形） + 小句2
- **要点**：
  - 原则上后句不能用意志/请求/命令表达（～たい、～てください等）
  - 例外1：前句谓语为状态性谓语时可以（安ければ買いたいです）
  - 例外2：前后主语不同时可以（雨が降らなければ、行きましょう）
  - 与「たら」对比：たら更自由，ば有后句限制
  - 不能用于"做了之后发现"的过去事实（× 窓を開ければ、富士山が見えました）
- **例句**：
  - この大会で優勝すれば、オリンピックに出場できます。（在这次大会上获胜的话就能参加奥运会。）
  - 部屋がもう少し広ければいいのですが。（房间要是再大一点就好了。）
  - 李さんに聞けば、すぐ分かりますよ。（问小李的话马上就知道了。）
- **Bunpro 校验**：待验证

### 3. 小句1（简体形）なら、小句2（建议/忠告）

- **含义**：如果是那样的话……（根据对方的情况提出建议或看法）
- **接续**：小句1（简体形） + なら + 小句2；二类形容词/名词去「だ」+ なら
- **要点**：
  - 核心功能：承接已知信息、假设前提或对方话题，提出判断/建议
  - 后句为说话者主观判断/建议/提议
  - 与「たら」「ば」「と」的核心区别：なら最适合"针对对方意图提建议"
  - 名词可直接接：日曜日なら時間があります
- **例句**：
  - 天安門へ行くなら、地下鉄が便利ですよ。（要去天安门的话，坐地铁很方便。）
  - 魚を買うなら、駅前の市場がいいですよ。（要买鱼的话，车站前面的市场好。）
  - 忙しいなら、無理しなくてもいいですよ。（忙的话不用勉强。）
- **Bunpro 校验**：待验证

### 4. 名詞＋でも（从选项中提议）

- **含义**：……之类的怎么样？（举出代表性的一项进行提议）
- **接续**：名词 + でも + 動詞
- **要点**：
  - 不是"限定"而是"举一个代表性的选项"
  - 常用于提议/劝说：コーヒーでも飲みませんか
  - 通常用于提议/建议，面向将来的行为
  - 与第17课全面肯定的「でも」和第35课极端例示的「でも」不同
- **例句**：
  - 疲れたでしょう。お茶でも飲みませんか。（累了吧。喝点茶什么的怎么样？）
  - 運動でもしたらどうですか。（运动运动怎么样？）
  - 週末、映画でも見に行きましょうか。（周末去看看电影什么的吧？）
- **Bunpro 校验**：待验证

### 5. 名1 とか 名2 とか / 小句1 とか 小句2 とか（列举）

- **含义**：……啦……啦（列举同类事物中的几个代表性例子）
- **接续**：名词/小句 + とか + 名词/小句 + とか
- **要点**：
  - 与「～や～など」意思基本相同
  - 区别：とか可连接小句，や只能连接名词
  - とか偏口语/随意，书面中也可见但比「や/など」随便
  - 暗示还有其他没列举的同类事物
- **例句**：
  - 休みの日は映画を見るとか、本を読むとか、いろいろしています。（休息天看看电影啦、读读书啦，做各种事。）
  - コートとかマフラーとか、冬の物をたくさん買いました。（外套啦围巾啦，买了很多冬天的东西。）
  - 海外旅行に行きたいとか、新しい車が欲しいとか、いつも夢を語っています。（总是说想去海外旅行啦、想要新车啦什么的。）
- **Bunpro 校验**：待验证

## 课文

> Arc 10「北京游」第1话 — 万里の長城

**登場人物**：陳明、田中 — 長城の入口

> 使用语法：≤L37（+ば形、ば条件、なら建議、でも提議、とか列挙）

---

陳：天安門へ 行くなら、地下鉄が 便利ですよ。

田中：長城は どうやって 行けば いいですか。

陳：バスで 行けば、2時間ぐらいです。

田中：お土産は 何が いいですか。

陳：お茶でも 買いませんか。

田中：いいですね。Tシャツとか キーホルダーとか、いろいろ ありますね。

陳：ええ。安ければ 買いたいですが、ちょっと 高いですね。

田中：天気が よければ、上から の 景色が きれいでしょうね。

陳：そうですね。長城は 世界遺産ですから、一度は 見た ほうが いいですよ。

---

**语法覆盖**：✅ ば形/ば条件（行けば/安ければ/よければ） ✅ なら建議（行くなら地下鉄が） ✅ でも提議（お茶でも買いませんか） ✅ とか列挙（Tシャツとかキーホルダーとか）

## 表达要点

| 要点 | 说明 |
|------|------|
| 条件表达方式小结 | たら（最自由）/ば（后句有限制）/と（必然结果）/なら（针对对方提建议）四种条件形的对比 |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。确认语法规则是否记住。

### Level 2：组装（30%）

给词造句，3-5 题。主动运用语法。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：带朋友游览名胜古迹**

- **场景**：你带一位第一次来北京的朋友去万里长城。在出租车上，朋友问怎么去最方便，你根据她的情况给出建议。到了之后，你们买纪念品，你举了几种选择。朋友想买特产，你提议买个纪念章之类的。你还介绍了一些关于长城的知识（如果换算成中国单位就超过一万里）。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
