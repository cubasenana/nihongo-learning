## 语法点

### 1. 名 + です

- **含义**：……是……（礼貌体肯定判断）
- **接续**：名词 + です
- **要点**：
  - です 是 だ 的礼貌形式
  - 助词 は 提示主题，读作 わ
  - 结构：「A は B です」= A 是 B
- **例句**：
  - 私は学生です。（我是学生。）
  - これはペンです。（这是笔。）
- **Bunpro 校验**：一致 ✓

### 2. 名 + では ありません / じゃないです

- **含义**：……不是……（礼貌体否定判断）
- **接续**：名词 + では ありません（书面） / じゃないです（口语）
- **要点**：
  - では 口语中常缩为 じゃ
  - 不能用于い形容词
- **例句**：
  - 先生じゃないです。（不是老师。）
  - 犬では ありません。（不是狗。）
- **Bunpro 校验**：一致 ✓

### 3. 名 + ですか（疑问句及应答）

- **含义**：……是……吗？（礼貌体疑问）
- **接续**：名词 + ですか
- **要点**：
  - か 放句尾表示疑问
  - 肯定回答：はい、そうです
  - 否定回答：いいえ、ちがいます / いいえ、～では ありません
  - 不知道时：分かりません
- **例句**：
  - これは本ですか。（这是书吗？）
  - あなたは学生ですか。——はい、そうです。（你是学生吗？——是的。）
- **Bunpro 校验**：一致 ✓

### 4. 名 + の + 名

- **含义**：……的……（所属/从属关系）
- **接续**：名词A + の + 名词B
- **要点**：
  - 表示 B 属于 A、B 是 A 的一部分、B 的属性是 A
  - 日语中名词之间的修饰关系一般都加 の，不能省略
  - 例：私の父（我的父亲），不能说 ×私 父
- **例句**：
  - 先生の傘です。（是老师的伞。）
  - 私の猫です。（是我的猫。）
- **Bunpro 校验**：一致 ✓

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。确认语法规则是否记住。

- 题型：给出句子，判断语法是否正确
- 错误类型：は/が 误用、です/ではありません 混淆、の 遗漏等
- 快速过，不恋战

### Level 2：组装（30%）

给词造句，3-5 题。主动运用语法。

- 给出 2-3 个词（如：私、学生），要求用指定语法点造句
- 每题指定使用哪个语法点
- 逐步增加复杂度：单语法点 → 组合语法点

### Level 3：产出（60%）— 本课核心

场景写作任务 1 个。综合运用全部语法点。

**📋 产出任务：初次见面**

- **场景**：你第一天到日本公司报到，在大厅遇到一个人。写一段你们之间的对话。
- **要求**：
  - 6-8 句对话
  - 必须覆盖以下全部语法点，每个至少使用 1 次：
    - ✅ A は B です
    - ✅ A は B では ありません / じゃないです
    - ✅ ～ですか + はい/いいえ 应答
    - ✅ A の B
- **词汇限制**：仅使用本课词汇范围表中的词
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
