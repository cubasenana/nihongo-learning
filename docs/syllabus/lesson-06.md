## 语法点

### 1. 名[場所]へ + 动（目的地）

- **含义**：表示移动行为的目的地
- **接续**：場所 + へ + 行きます / 来ます / 帰ります
- **要点**：
  - へ 在这里读作 え
  - 仅用于移动动词（行きます、来ます、帰ります 等）
  - 提问用「どこへ 行きますか」
- **例句**：
  - 日本へ 行きます。（去日本。）
  - 学校へ 来ました。（来了学校。）
  - うちへ 帰ります。（回家。）
- **Bunpro 校验**：待验证

### 2. 名[場所]から + 动（起点）

- **含义**：表示移动的出发点
- **接续**：場所 + から + 移动动词
- **要点**：
  - 与第5课的时间起点 から 是同一助词
  - 提问用「どこから 来ましたか」
- **例句**：
  - 北京から 来ました。（从北京来的。）
  - 大阪から 帰ります。（从大阪回去。）
  - どこから 来ましたか。（从哪里来的？）
- **Bunpro 校验**：待验证

### 3. 名[人]と + 动（共同动作）

- **含义**：表示共同做某事的对象
- **接续**：人 + と + 动词
- **要点**：
  - 相当于"和...一起"
  - 提问用「だれと〜ますか」
  - 一个人时说「一人で」
- **例句**：
  - 友達と 行きます。（和朋友一起去。）
  - だれと 来ましたか。（和谁一起来的？）
  - 家族と 帰りました。（和家人一起回去了。）
- **Bunpro 校验**：待验证

### 4. 名[交通工具]で + 动（交通手段）

- **含义**：表示使用的交通手段
- **接续**：交通工具 + で + 移动动词
- **要点**：
  - 步行时说「歩いて」，不说「足で」
  - 提问用「何で（なにで）行きますか」；注意口语中「なんで」容易被误解为"为什么"
- **例句**：
  - 電車で 行きます。（坐电车去。）
  - バスで 帰りました。（坐公交回去了。）
  - 自転車で 来ます。（骑自行车来。）
- **Bunpro 校验**：待验证

### 5. 名[場所]から 名[場所]まで + 动（移动范围）

- **含义**：表示移动的起点和终点
- **接续**：場所 + から + 場所 + まで + 动词
- **要点**：
  - 起点 から + 终点 まで
  - 可以和交通手段 で 组合使用
  - 与第5课的时间 から〜まで 是同一结构的空间版
- **例句**：
  - 東京から 大阪まで 新幹線で 行きます。（从东京坐新干线去大阪。）
  - 駅から 家まで 歩いて 帰りました。（从车站走回了家。）
  - ここから 学校まで バスで 行きます。（从这里坐公交去学校。）
- **Bunpro 校验**：待验证

### 6. 复合助词（に/で/へ/から/まで/と + は）

- **含义**：は 可以加在其他助词后面构成复合形式，表示对比
- **接续**：に/で/へ/から/まで/と + は
- **要点**：
  - 表示对比或强调
  - 「韓国へは 行きました。中国へは 行きませんでした」——强调去了韩国但没去中国
  - 否定句中尤其常见
- **例句**：
  - わたしの 部屋には 電話が ありません。（我的房间里没有电话。）
  - 韓国へは 行きました。中国へは 行きませんでした。（去了韩国，没去中国。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| たしか | 表示"凭记忆大概是..."，说话人并非百分百确定 |
| 家 和 うち | 家 侧重建筑物；うち 侧重"家人/我家"，うち 本身可表示"我家"不需要加 わたしの |
| まっすぐ 帰りました | 表示"直接回家，不去别处" |
| お疲れ様でした | 一天工作结束时向同事致意的寒暄语 |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：周末旅行计划**

- **场景**：你和朋友正在讨论这个周末一起出去玩的计划。你需要说明从哪里出发、去哪里、坐什么交通工具、和谁一起去，还要聊聊上次出游的情况。
- **要求**：
  - 6-8 句
  - 必须覆盖以下全部语法点：
    - ✅ へ（目的地）
    - ✅ から（起点）
    - ✅ と（共同）
    - ✅ で（交通）
    - ✅ から〜まで（范围）
    - ✅ 复合助词
- **词汇限制**：仅使用 lesson ≤ 6 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
