## 语法点

### 1. 小句1 て / 小句1 ないで + 小句2（伴随状态）

- **含义**：保持某种状态做某事 / 不做某事就做另一事
- **接续**：
  - 动词て形 + 小句2
  - 动词ない形 + で + 小句2
- **要点**：
  - 与之前学过的「て」表示顺序/原因不同，这里前句描述后句动作主体的状态
  - 前后主语必须一致（与表示顺序/原因的「て」不同）
  - 「ないで」也可写成书面语形式「ずに」（古语残留）；特殊：する→せずに
  - 例：眼鏡をかけて読む＝戴着眼镜读，傘を持たないで出かける＝没带伞就出门
- **例句**：
  - コートを着て出かけましょう。寒いですから。（穿上外套出门吧，很冷。）
  - 今朝は朝ご飯を食べないで来てしまいました。（今天早上没吃早饭就来了。）
  - 手を挙げて横断歩道を渡ってください。（请举着手过人行横道。）
- **Bunpro 校验**：待验证

### 2. 小句1（简体形）ために、小句2（非意志性原因）

- **含义**：由于……（不如意的事情的原因，偏正式/书面）
- **接续**：
  - 动词/形容词简体形 + ために
  - 二类形容词 + な + ために
  - 名词 + の + ために
- **要点**：
  - 与第34课的「ために」（目的）区分：目的用意志动词基本形，原因用非意志性事态
  - 多用于不如意的事情的原因，但本身不严格限定结果好坏
  - 比「から」「ので」更正式，多用于书面语或郑重场合
  - 不能用裸的「ために」结句，但可用「ためです」：事故があったためです
- **例句**：
  - 台風のために、飛行機が飛びませんでした。（因为台风，飞机停飞了。）
  - 事故があったために、電車が1時間も遅れました。（因为发生了事故，电车晚了整整一个小时。）
  - この村は交通が不便なために、若い人がだんだん減っています。（这个村子因为交通不便，年轻人越来越少了。）
- **Bunpro 校验**：待验证

### 3. 動て形＋いきます/きます

- **含义**：表示移动方向（远离/靠近）、相继动作、伴随状态移动、往返动作
- **接续**：动词て形 + いきます/きます
- **要点**：
  - 用法一（方向）：移动动词＋ていきます（由近及远）/ てきます（由远及近）
  - 用法二（相继动作）：做完某事再去/来（見てきました＝去看了回来了）
  - 用法三（伴随状态移动）：保持某状态去/来（かばんを持っていきます＝带着包去）
  - 用法四（去做某事后回来）：ちょっと手紙を出してきます＝我去寄封信就回来
  - 用法五（时间上的变化延续）：増えていきます（将来越来越多）、寒くなってきました（变冷了）
- **例句**：
  - 森さんは急いで走っていきました。（森先生急匆匆地跑走了。）
  - ちょっとコンビニで飲み物を買ってきます。（我去便利店买点饮料就回来。）
  - 明日はお弁当を持ってきてください。（明天请带便当来。）
- **Bunpro 校验**：待验证

## 课文

> Arc 10「北京游」第3话 — 故宫一日游

**登場人物**：陳明、田中 — 故宮

> 使用语法：≤L39（+て/ないで伴隨、ために原因、ていきます/てきます）

---

陳：かばんを 持って 入って ください。傘は 持たないで ください。

田中：分かりました。チケットを 買って きます。

陳：じゃあ、ここで 待って います。

（中に 入って）

田中：霧の ために、遠くが よく 見えませんね。

陳：そうですね。でも、建物は すごいですね。

田中：この 建物は 600年前に 建てられたんですよ。火事の ために 2回 焼けて しまいましたが。

陳：そうですか。暑く なって きましたね。飲み物を 買って きましょうか。

田中：お願いします。私は ここで 写真を 撮って います。

陳：分かりました。すぐ 戻って きます。

---

**语法覆盖**：✅ て伴隨（持って入って/持たないで） ✅ ために原因（霧のために/火事のために） ✅ ていきます/てきます（買ってきます/戻ってきます/暑くなってきた）

## 表达要点

| 要点 | 说明 |
|------|------|
| も＋数量词（超出预想） | 助词「も」接数量词表示比预想的多：「2度も焼けた」＝竟然烧了两次 |
| 遠く/近く | 一类形容词「遠い/近い」的く形用作名词（远处/近处），不做副词用 |
| せっかく | "好不容易、特意"：せっかく来たのに＝好不容易来了却…… |
| 絶対 | "绝对、一定"：絶対に行きたい（可加「に」，意思不变） |
