# L33-L36 课文与生词表审核报告

审核对象：`docs/syllabus/lesson-33.md` 到 `docs/syllabus/lesson-36.md` 的新增 `## 课文` 与 `## 生词表`。

审核基准：

- 语法解锁：`docs/syllabus/grammar-unlock.md`
- 只检查 `lesson <= X` 的语法是否越界
- 对照每课 `## 语法点` 下的 `###` 标题检查覆盖
- 同时检查日语自然度、叙事连续性、生词表 `📖` 标记、表格列数、词源一致性

## 总结

本批没有大面积错误，但有几个需要优先修：

- L33 `自他動詞②` 覆盖不严：课文几乎只展示自动词状态，没有真正展示对应他动词。
- L34 `ために` 覆盖不完整：只出现 `Nのために`，没有出现 `動基本形 + ために`。
- L35 `作れませんでした / 作れます` 使用可能形，按 `grammar-unlock.md` 属于 L38 越界。
- L36 `動て形 + ばかりいます` 没有覆盖；现在只有 `Nばかり`。
- L36 `小句 + のが聞こえます` 没有覆盖；现在只有 `声が聞こえます`。

Markdown 表格列数检查通过，L33-L36 没有发现表格列数不一致。

## L33

### ⚠ 语法覆盖不严：`自他動詞②` 没有真正成对展示

位置：`lesson-33.md:83`, `lesson-33.md:85`, `lesson-33.md:91`, `lesson-33.md:97`

原文：

```text
出口の ドアが 閉まって いますね。
隣の ドアは 開いて いますよ。
あの ビルの 電気が 全部 消えて いますね。
```

覆盖清单写：

```text
自他對應（閉まっている/開いている/消えている）
```

问题：

这些都是自动词结果状态，主要覆盖的是 `自動詞 + ています`。但本课第 1 个语法点是“自动词和他动词②（词形对应关系）”，如果课文中没有他动词侧，学生只能看到自动词，无法从课文中体会 `閉まる/閉める`、`開く/開ける`、`消える/消す` 的对应。

建议补一句他动词侧，例如：

```text
陳：隣の ドアを 開けましょうか。
田中：はい、お願いします。
```

或：

```text
スタッフが 電気を 消しました。
```

### ✅ 语法越界检查

未发现明确硬越界。`お元気そうで`、`重そう` 都属于 L33 样态 `そうです` 范围；`遅れてしまいました`、`入れてしまいました` 也符合本课 `てしまいます`。

### ✅ 叙事连续性

L33 田中从日本到上海，陈明在上海机场接人，承接 L32 “田中さんが来ることになりました”，叙事连贯。

位置：`lesson-33.md:114-121`

```

建议改成：

```

如果也想标记他动词侧是否出现，可再加右侧 `📖` 列。

## L34

### ❌ 语法覆盖缺失：`動基本形 + ために`

位置：`lesson-34.md:81`, `lesson-34.md:87`, `lesson-34.md:95`

原文：

```text
田中さんの ために、中華料理を 作りました。
明日の ために、レストランも 予約して おきました。
```

问题：

这两句都是 `名詞 + の + ために`。本课第 4 个语法点标题是：

```text
小句（基本形）ために / 名詞＋の＋ために
```

课文没有出现 `動基本形 + ために`，所以覆盖不完整。

建议补一句：

```text
明日 北京を 案内するために、レストランを 予約して おきました。
```

或：

```text
田中さんに 中華料理を 食べてもらうために、材料を 買って おきました。
```

### ⚠ 自然度/语义：`明日のために` 目的不够具体

位置：`lesson-34.md:87`

原文：

```text
明日の ために、レストランも 予約して おきました。
```

问题：

`明日のために` 可以理解，但作为目的表达比较空，学生不容易理解“为了明天的什么”。更自然的是把目的名词具体化。

建议：

```text
明日の 観光の ために、レストランも 予約して おきました。
```

或用 `動ために`：

```text
明日 食事するために、レストランを 予約して おきました。
```

### ✅ 语法越界检查

未发现明确硬越界。`てあります`、`ておきます`、`てみます`、`ために` 都是 L34 新增语法；`かもしれません`、`てもいいですか` 已在前课解锁。

### ✅ 叙事连续性

L34 田中到陈明家，陈明招待田中，承接 L33 机场接人后去酒店/住处的接待线，没有发现硬矛盾。

### 💡 生词表/词源一致性

若严格执行 CSV 词源，以下词条需要确认：

- `中華料理`：CSV 中有 `料理`，并在例句/语境中出现中餐相关表达，但未必有精确词条 `中華料理`。
- `ビール`：课文使用 `ビール`，但 CSV 中未能精确匹配到词条。

这两个不一定是语法问题，但如果词表要求“仅从 CSV 取词”，需要统一处理。

## L35

### ❌ 语法越界：可能形 `作れませんでした / 作れます`

位置：`lesson-35.md:98`, `lesson-35.md:104`, `lesson-35.md:122`

原文：

```text
田中：私は 5つしか 作れませんでした。難しいですね。
陳：はい、子供でも 上手に 作れますよ。
```

问题：

`作れませんでした`、`作れます` 是 `作ります` 的可能形。根据 `grammar-unlock.md`，可能形式是 L38。L35 尚未解锁。

建议用 L20 已学的 `動ことができます` 替换：

```text
田中：私は 5つしか 作ることが できませんでした。難しいですね。
陳：はい、子供でも 上手に 作ることが できますよ。
```

这样仍然能覆盖：

- `しか + 否定`
- `でも` 极端例示

### ❌ 生词表错误：`作れます` 标为二类动词

位置：`lesson-35.md:122`

原文：

```markdown
| 作れます | つくれます | 能做(可能) | 二类 | 📖 |
```

问题：

`作れます` 是 `作ります` 的可能形，不应作为普通“二类动词”提前收进 L35。对 0 基础学生来说，这会误导他们以为 `作れます` 是一个独立二类动词。

建议：

- 如果修正文为 `作ることができます`，生词表删除 `作れます`。
- 或把它改成表达说明，不放入“新动词”表，等 L38 可能形再系统讲。

### ✅ 语法覆盖

除可能形越界外，L35 语法覆盖本身完整：

- `たら`：`分からなかったら / 足りなかったら / あったら`
- `ても`：`何回失敗しても`
- `だけ`：`30枚だけ`
- `しか + 否定`：当前句可保留结构，但需换成 `作ることができませんでした`
- `でも`：`子供でも`

### ✅ 叙事连续性

L35 田中在陈明家学包饺子，承接 L34 招待准备和中餐场景，叙事连贯。

### 💡 生词表/词源一致性

若严格执行 CSV 词源，`作れます` 不应独立作为词条。`作る` 在 CSV 中存在，建议用基础形 `作ります/作る` 管理。

## L36

### ❌ 语法覆盖缺失：`動て形 + ばかりいます`

位置：`lesson-36.md:108`, `lesson-36.md:110`, `lesson-36.md:118`

原文：

```text
最近、肉ばかり 食べて いるので、今日は 野菜を 買いましょう。
田中さんは 肉ばかり ですか。
```

问题：

这两句覆盖的是 `名詞 + ばかり`，没有覆盖本课第 4 个语法点中的 `動て形 + ばかりいます`。

建议补一句：

```text
田中：最近、肉を 食べて ばかり います。
```

或：

```text
陳：肉を 食べて ばかり いると、体に 悪いですよ。
```

### ❌ 语法覆盖缺失：`小句 + のが聞こえます`

位置：`lesson-36.md:104`, `lesson-36.md:106`, `lesson-36.md:118`

原文：

```text
遠くで 子供が 遊んで いるのが 見えますね。
市場は にぎやかですね。いろいろな 声が 聞こえます。
```

问题：

`子供が遊んでいるのが見えます` 正确覆盖了 `小句 + のが見えます`。但 `声が聞こえます` 只是名词主语 + `聞こえます`，没有覆盖 `小句 + のが聞こえます`。

建议补一句：

```text
子供が 話して いるのが 聞こえます。
```

或：

```text
店の 人が 大きな 声で 話して いるのが 聞こえます。
```

### ⚠ 自然度：`肉ばかりですか`

位置：`lesson-36.md:110`

原文：

```text
田中さんは 肉ばかり ですか。体に 悪いですよ。
```

语法上能理解，但省略较重。对 0 基础学生更建议写完整：

```text
田中さんは 肉ばかり 食べて いるんですか。
```

但 `んですか` 已学 L24，可以使用；如果想更简单：

```text
田中さんは 肉ばかり 食べて いますか。
```

### ✅ 语法越界检查

未发现明确硬越界。`ので` 已在 L30 解锁；`て/で` 原因、`Nに` 用途、`動のに` 用途、`ばかり`、`のが見える/聞こえる` 均属于 L36 或之前。

### ✅ 叙事连续性

L36 带田中逛上海市场，承接 L33-L35 接待朋友游上海/吃中餐的叙事线，没有发现硬矛盾。

### 💡 生词表/词源一致性

若严格执行 CSV 词源，以下 L36 表内词条未能在 CSV 中精确匹配到词条：

- `市場`
- `新鮮`
- `包丁`
- `通じます`

`はさみ` 在 CSV 例句中出现，但未能精确匹配为词条；是否可收录需要看词表规则是否允许从例句取词。

## 建议修复顺序

1. 先修 L35 可能形越界：`作れませんでした / 作れます` 改成 `作ることができませんでした / 作ることができます`。
2. 再补 L36 两个覆盖缺口：`食べてばかりいます` 和 `小句 + のが聞こえます`。
3. 再补 L34 `動基本形 + ために`，并把 `明日のために` 改得更具体。
4. 再调整 L33 自他动词覆盖，让课文真正展示他动词侧。
5. 最后按是否严格执行 CSV 词源，整理 L33-L36 生词表中未精确匹配 CSV 的词条。
