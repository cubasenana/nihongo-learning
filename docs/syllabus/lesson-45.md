## 语法点

### 1. ～ていきます / ～てきました（持续/变化）

- **含义**：一直持续做某事 / 性质或状态逐渐发生变化
- **接续**：动词て形 + いきます/きました
- **要点**：
  - 「～てきました」：从过去到基准时间的变化或持续
  - 「～ていきます」：从基准时间向未来的变化或持续
  - 与第39课的空间移动用法不同，这里表示时间上的变化趋势
  - 常与「だんだん」「ますます」「どんどん」等副词搭配
- **例句**：
  - 少子化が進んで、日本の人口はだんだん減っていくでしょう。（少子化加剧，日本人口将逐渐减少吧。）
  - 最近、中国へ旅行に行く日本人が増えてきましたね。（最近去中国旅行的日本人多起来了。）
  - 20年間ずっと日本語を勉強してきました。（这20年我一直在学习日语。）
- **Bunpro 校验**：待验证

### 2. ～てきました（出现）

- **含义**：开始……了（某种状态的开始或出现）
- **接续**：非意志动词て形 + きました
- **要点**：
  - 只能前接非意志动词（疲れる、降る、すく等）
  - 表示某种状态刚刚开始出现
  - 与「～ていきます/きました」的持续变化用法不同，这里是某状态开始显现/逐渐变得可感知
- **例句**：
  - ずっと本を読んでいたので、目が疲れてきました。（一直在看书，眼睛累了。）
  - お母さん、雨が降ってきたよ。（妈妈，下起雨来了。）
  - おなかがすいてきたんですが、そろそろ食事に行きませんか。（肚子有点饿了，去吃饭好吗？）
- **Bunpro 校验**：待验证

### 3. 小句1し、小句2し、小句3

- **含义**：又……又……（并列陈述多个原因或事项）
- **接续**：
  - 小句简体形/敬体形 + し
  - 二类形容词：～だし
  - 名词：～だし
- **要点**：
  - 并列多个原因来支持结论，也可单纯并列列举特征/事项（结论可省略）
  - 「し」前面可以用简体也可以用敬体
  - 主语的「が」常被「も」替换：天気もいいし
  - 与「て形」的区别：「～し」常用于陈述原因，可连接动词
- **例句**：
  - おいしいし、手軽だし、わたしは冷凍食品をよく食べます。（又好吃又简便，我经常吃冷冻食品。）
  - 休みだし、天気もいいし、どこかに出かけませんか。（今天休息，天气也不错，去哪儿玩玩吧？）
  - 電気も消えているし、かぎもかかっているし、きっと留守ですよ。（灯关着，门也锁着，肯定不在家。）
- **Bunpro 校验**：待验证

### 4. ～ば～ほど（越……越……）

- **含义**：越做某事，越怎样（正比例关系）
- **接续**：
  - 动词：ば形 + 基本形 + ほど
  - 一类形容词：ば形 + 原形 + ほど
  - 名词：なら＋名词＋ほど / であれば＋名词＋であるほど
- **要点**：
  - 表示程度与相关性质/动作的正比例关系
  - 动词：読めば読むほど / 一类形：安ければ安いほど
  - 二类形容词口语用「～なら～なほど」，书面用「～であれば～なほど」
- **例句**：
  - この本は読めば読むほどおもしろいです。（这本书越读越有意思。）
  - 商品は安ければ安いほど売れるんでしょう？（商品越便宜越好卖吧？）
  - 住めば住むほど上海のよさが分かりますよ。（越住越能体会到上海的好。）
- **Bunpro 校验**：待验证

### 5. 助词（で/へ/から/まで/と）＋の

- **含义**：将助词短语转化为修饰名词的结构
- **接续**：で/へ/から/まで/と + の + 名词
- **要点**：
  - 「に」没有后续「の」的用法（× 母にのプレゼント）
  - 常用：「から～までの」「への」
  - 使得整个短语可以做定语修饰名词
- **例句**：
  - 空港から市内までのアクセスがよくなりました。（从机场到市内的交通方便了。）
  - 北京から東京への便は1日に10便以上あります。（从北京到东京的航班一天有10班以上。）
  - これは母へのプレゼントです。（这是给妈妈的礼物。）
- **Bunpro 校验**：待验证

## 课文

> Arc 12「開業」第1话 — 上海出差见闻报告

**登場人物**：陳明 — 社内報告

> 使用语法：≤L45（+ていきます/てきました変化、てきました出現、し並列、ば～ほど、助詞+の）

---

陳：上海事務所の 準備に ついて 報告します。上海は この 数年で 大きく 変わって きました。高いビルが どんどん 増えて きました。

交通も 便利に なりましたし、外国の 会社も 増えましたし、ビジネスの チャンスは 多いと 思います。

先月から 事務所の 準備を 進めて きましたが、まだ 完了して いません。来月からは 本格的に スタッフを 集めて いきます。

上海に いる 間、おなかが すいて きたり、疲れて きたり しましたが、住めば 住むほど 上海の よさが 分かります。

東京から 上海への 出張は 3時間ぐらいです。東京から 上海までの 距離は 近いです。これからも よろしく お願いします。

---

**语法覆盖**：✅ てきました変化（変わってきた/増えてきた/進めてきた） ✅ てきました出現（すいてきた/疲れてきた） ✅ し並列（便利になったし増えたし） ✅ ば～ほど（住めば住むほど） ✅ 助詞+の（上海への出張/上海までの距離）

| 要点 | 说明 |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。区分「てきました」的持续变化和状态出现用法，确认「し」「ば～ほど」的接续。

### Level 2：组装（30%）

给词造句，3-5 题。用给定情境造「ていきます/きました」「し」「ば～ほど」句。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：上海出差见闻报告**

- **场景**：你第一次去上海出差。回来后向同事讲述上海的变化（近年来发展很快、高楼越来越多），你对上海的印象（又繁华又方便），以及你在上海的一些小插曲（走着走着下起雨来了、肚子饿了去找东西吃）。最后感叹越了解上海越觉得有魅力。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ ～ていきます / ～てきました（持续变化）
    - ✅ ～てきました（状态出现）
    - ✅ ～し、～し（并列原因）
    - ✅ ～ば～ほど
- **词汇限制**：仅使用 lesson ≤ 45 的词汇
- **AI 验收标准**：
  1. 语法检查：「てきました」的两种用法是否区分正确
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：出差见闻的叙述是否自然流畅
