## 总结

本批整体比 L37-L40 稳，但仍有几个需要修：

- L41 `基本被动句` 覆盖误判：`足を踏まれました` 不是基本被动，更接近物主/受害被动。
- L43 缺 `自动词使役句（～は～を～させます）`，课文中基本都是他动词使役。
- L43 `ご紹介します` 严格按解锁表看，提前用了 L48 `お/ご～します` 自谦结构。
- L43 商务场景里 `陳に説明させます` 很像“让下属说明”，对客户说略生硬。

Markdown 表格列数检查通过，L41-L44 没有发现表格列数不一致。

## L41

### ❌ 语法覆盖误判：`基本被动句` 未真正覆盖

位置：`lesson-41.md:110`, `lesson-41.md:130`

原文：

```text
陳：まず、電車の 中で 足を 踏まれました。
```

覆盖清单写：

```text
基本被動（足を踏まれました） 物主被動（足を踏まれた）
```

问题：

`足を踏まれました` 是“我的脚被踩了”，所属物 `足` 仍用 `を` 标记，更符合物主被动/受害被动。它不能同时拿来覆盖“基本被动句”。

本课第 2 个语法点 `～は（～に）～（ら）れます` 需要类似：

```text
私は 部長に ほめられました。
王さんは 課長に 質問されました。
```

建议补一句基本被动：

```text
陳：展示会で、私は 係の人に いろいろ 質問されました。
```

或：

```text
陳：でも、会場で 係の人に 助けられました。
```

### ⚠ 自然度：`どこの会社によって`

位置：`lesson-41.md:124`

原文：

```text
王：その 商品は どこの 会社に よって 作られた ものですか。
```

问题：

意思能懂，但 `どこの会社によって` 稍别扭。正式被动句中更自然的是 `どの会社によって`。

建议：

```text
王：その 商品は どの 会社に よって 作られた ものですか。
```

未发现明确后课语法越界。被动、`によって` 均属于 L41；`眠れませんでした` 是 L38 已学可能形，允许使用。

## L43

### ❌ 语法覆盖缺失：自动词使役句 `～は～を～させます`

位置：`lesson-43.md:93`, `lesson-43.md:103`, `lesson-43.md:109`, `lesson-43.md:113`

原文中的使役：

```text
陳に 説明させます。
デザイナーに 修正させます。
部下に 使わせて みます。
```

问题：

这些基本都是他动词使役或省略宾语的他动词使役，动作主体用 `に`。本课第 2 个语法点要求的是自动词使役：

```text
～は～を（自动词）（さ）せます
```

课文目前没有真正的 `人を + 自动词使役`。

建议补一句：

```text
課長：会議の 後で、部下を 先に 帰らせます。
```

或：

```text
課長：今日は 陳を 休ませます。
```

### ❌ 语法越界：`ご紹介します`

位置：`lesson-43.md:95`

原文：

```text
陳：はい。私たちの 新商品を ご紹介します。
```

问题：

`ご紹介します` 是 `お/ご～します` 型自谦语，根据 `grammar-unlock.md`，该结构在 L48。L43 尚未解锁。

建议改成普通礼貌形：

```text
陳：はい。私たちの 新商品を 紹介します。
```

如果想保留商务自然度，可以等 L48 再使用 `ご紹介します`。

### ⚠ 自然度：`陳に説明させます` 对客户偏生硬

位置：`lesson-43.md:93`

原文：

```text
課長：本日は お忙しい 中、ありがとうございます。陳に 説明させます。
```

问题：

语法上可以，但在客户面前说 `陳に説明させます` 有“让下属去说明”的控制感，商务自然度偏硬。实际商务场景更常说：

```text
陳から 説明します。
陳が 説明します。
```

但如果为了教学使役，可以保留一处，并在语境上改成内部指示。

### ⚠ 严格模式下的敬语提前出现

位置：`lesson-43.md:93`, `lesson-43.md:99`

表达：


这些更像固定表达/词汇，不一定要当作语法越界；但若严格遵守“敬语集中到 L47-L48”，需要确认是否允许作为商务固定语提前出现。

## L44

### ✅ 语法越界检查

未发现明确硬越界。`ようです`、`みたいです`、`らしいです`、`過ぎます`、`形容词 + さ` 均属于 L44；`と言われている` 的被动已在 L41 解锁；`によると` 已在 L32 出现。

### ✅ 语法覆盖

L44 五个语法点覆盖完整：

- `ようです`：`売れているようです / 言われているようです`
- `みたいです`：`いいみたいです / 理由みたいです`
- `らしいです`：`言っているらしいです / 高過ぎるらしいです`
- `過ぎます`：`甘過ぎる / 高過ぎる`
- `形容词 + さ`：`甘さ / 新しさ / 安さ`

### ⚠ 自然度/用法微调：`ようです` 与信息来源

位置：`lesson-44.md:102`

原文：

```text
データに よると、よく 売れて いる ようです。
```

这句可以成立，因为根据数据做观察推测。但如果想更清楚地区分 L32 传闻 `そうです` 和 L44 观察推测 `ようです`，可以写成更“从数据看出来”的形式：

```text
データを 見ると、よく 売れて いる ようです。
```

这不是硬错误，只是教学清晰度优化。

## 建议修复顺序

1. 先补 L41 基本被动句，别再用 `足を踏まれました` 同时覆盖基本被动和物主被动。
2. 再补 L43 自动词使役句 `人を帰らせる/休ませる/遊ばせる`。
3. 把 L43 `ご紹介します` 改成 `紹介します`，避免提前引入 L48 自谦语。
4. 调整 L43 `陳に説明させます` 的商务自然度，或明确把它放在内部指示语境。
5. L44 可保留，最多把 `データによると` 改成 `データを見ると`，让 `ようです` 的“观察推测”更清楚。
6. 最后按是否严格执行 CSV 词源，统一处理 L41-L44 未精确匹配 CSV 的词条。
