## 语法点

### 1. 动词的命令形

- **含义**：命令对方做某事（语气强硬）
- **接续**：一类动词え段音 / 二类动词る→ろ / 三类动词：来る→来い、する→しろ
- **要点**：
  - 一类：書く→書け、読む→読め、話す→話せ
  - 二类：食べる→食べろ、見る→見ろ
  - 三类：来る→来い、する→しろ
  - 使用场合有限：上级对下级、紧急情况、观看体育比赛、直接引用
  - 语气粗硬，日常会话中一般用「て形」代替
- **例句**：
  - 早く起きろ、もう8時だぞ。（快起来，都8点了！）
  - 危ない！止まれ！（危险！站住！）
  - 課長は「書類を出せ」と言いました。（科长说了"交文件"。）
- **Bunpro 校验**：待验证

### 2. ～なさい

- **含义**：做……！（上对下的直接指示，比命令形稍缓和）
- **接续**：动词ます形去掉「ます」+ なさい
- **要点**：
  - 多用于老师对学生、父母对孩子
  - 書きます→書きなさい、見ます→見なさい、来ます→来なさい、します→しなさい
  - 比命令形礼貌，但仍属于上对下的表达
- **例句**：
  - 宿題を早くしなさい。（快做作业！）
  - もう遅いから、早く寝なさい。（已经很晚了，快去睡觉！）
  - この問題に答えなさい。（回答这道题！）
- **Bunpro 校验**：待验证

### 3. 动词基本形 + な（禁止）

- **含义**：不许做……！（禁止对方做某事）
- **接续**：动词基本形 + な
- **要点**：
  - 书く→書くな、見る→見るな、来る→来るな、する→するな
  - 与命令形一样，使用场合有限
  - 常见于标识和引用：「タバコを吸うな」
- **例句**：
  - ここでサッカーをするな。（不许在这里踢足球！）
  - 人の日記を読むな。（不许看别人的日记！）
  - 授業をサボるな。（不许逃课！）
- **Bunpro 校验**：待验证

### 4. ～て / ～ないで（祈使）

- **含义**：请做…… / 请不要做……（日常中关系亲密的人之间的祈使）
- **接续**：动词て形 / 动词ない形 + で
- **要点**：
  - 男女都常用，适用于关系亲密的人之间
  - 是「～てください」「～ないでください」的省略形式
  - 紧急时常用省略形式：助けて！（也可说 助けてください！）
  - 遠慮しないで！（别客气！）
- **例句**：
  - ちょっと手伝って！（帮我一下！）
  - 心配しないで、大丈夫だから。（别担心，没问题的。）
  - もう少しゆっくり話して。（再说慢一点。）
- **Bunpro 校验**：待验证

### 5. 名1 + という + 名2

- **含义**：叫做……的（提供对方不知道的新信息）
- **接续**：名词1 + という + 名词2
- **要点**：
  - 用于向对方介绍不认识的花名、地名、人名等
  - フジという花（叫做"紫藤"的花）
  - 世田谷という所（叫做"世田谷"的地方）
- **例句**：
  - これはラベンダーという花です。（这是一种叫"薰衣草"的花。）
  - 田中というお客さんから電話がありました。（有一位叫田中的客人打来了电话。）
  - わたしは青島という町で生まれました。（我在一个叫青岛的城市出生的。）
- **Bunpro 校验**：待验证

### 6. 名1は ～ / 小句 + という + 名2です

- **含义**：……是……的意思（给名词下定义或做解释）
- **接续**：名1は + 小句（命令形/简体形/禁止形〈基本形+な〉）+ という + 意味です
- **要点**：
  - 常与「意味」搭配使用
  - 小句部分可以是命令形或禁止形：タバコを吸うなという意味
  - 「どういう意味ですか」是固定问法
- **例句**：
  - 「禁煙」はタバコを吸うなという意味です。（"禁烟"是"不许吸烟"的意思。）
  - 「徐行」はゆっくり走れという意味です。（"徐行"是"慢行"的意思。）
  - 「立入禁止」はここに入るなという意味です。（"立入禁止"是"禁止入内"的意思。）
- **Bunpro 校验**：待验证

## 课文

> Arc 8「工作推进」第1话 — 日本标识解读

**登場人物**：陳明、王 — 上海の日本料理店街を歩いている

> 使用语法：≤L29（+命令形、なさい、動基本形+な禁止、て/ないで祈使、という）

---

陳：あの 看板の 「止まれ」は どういう 意味ですか。

王：「止まれ」は 「止まりなさい」という 意味です。命令形ですよ。

陳：あの 赤い 看板は？

王：「立入禁止」は 「入るな」という 意味です。ここに 入っては いけません。

陳：日本語の 命令形は 難しいですね。

王：普段は 使わないで ください。友達には 「心配しないで」と 言いますよ。

陳：そうですね。じゃあ、「静かに して」は？

王：それは 「静かに しなさい」と 同じです。先生が 学生に 言います。

陳：なるほど。「一番」という ラーメン屋が ありますね。入りましょう。

---

**语法覆盖**：✅ 命令形（止まれ） ✅ なさい（止まりなさい/しなさい） ✅ な禁止（入るな） ✅ て/ないで祈使（静かにして/心配しないで） ✅ という（「一番」というラーメン屋/という意味）

## 表达要点

| 要点 | 说明 |
|------|------|
| 命令形/禁止形的使用场合 | 上下级、紧急情况、体育赛事、直接引用/解释词义；语气粗硬，日常会话一般用「て形」 |
| ～ないといけません的省略 | 口语中常省略后半：急がないと…（不快点就……）、見なければ（得看） |
