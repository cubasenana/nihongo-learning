## 语法点

### 1. 被动形式规则

- **含义**：表示主语承受某种动作或影响的表达方式
- **接续**：
  - 一类动词：ない形去「ない」+ れる
  - 二类动词：ない形去「ない」+ られる
  - 三类动词：来る → 来られる、する → される
- **要点**：
  - 被动形式的活用方式与二类动词相同
  - 一类动词先变ない形，再把ない替换为れる：書く → かかれる
  - 二类动词先变ない形，再把ない替换为られる：食べる → たべられる
- **例句**：
  - 「読む」の被動形式は「読まれる」です。（"读"的被动形式是"被读"。）
  - 「食べる」は二類動詞ですから、「食べられる」になります。（"吃"是二类动词，所以变成"食べられる"。）
- **Bunpro 校验**：待验证

### 2. ～は（～に）～（ら）れます（基本被动句）

- **含义**：被……做了某事（动作对象做主语，动作主体用に表示）
- **接续**：主语は + 动作主体に + 动词被动形式
- **要点**：
  - 主动句的宾语在被动句中做主语
  - 动作的实施者用助词「に」标记
  - 受益语境也常用「～てもらう」表达，但被动不一定都是受害（ほめられる、誘われる等可为中性/正向）
- **例句**：
  - 李さんは部長にほめられました。（小李受到了部长的表扬。）
  - 森さんは小野さんに食事に誘われました。（森先生被小野女士邀请去吃饭了。）
  - 新入社員は先輩に仕事のやり方を教えられました。（新员工被前辈教了工作方法。）
- **Bunpro 校验**：待验证

### 3. ～は～に～を～（ら）れます（物主被动句）

- **含义**：某人被别人对自己所拥有的事物做了某事（含受害语气）
- **接续**：物主は + 动作主体に + 所属物を + 动词被动形式
- **要点**：
  - 所属物的拥有者做主语，所属物仍做宾语
  - 一般含有遭受麻烦或损失的语气
  - 不用「の」连接所属关系，直接用「を」标记所属物
- **例句**：
  - 馬さんは森さんにカメラを壊されました。（小马让森先生把照相机给弄坏了。）
  - 李さんは昨日犬に手をかまれました。（小李昨天被狗咬了手。）
  - 電車の中で知らない人に足を踏まれました。（在电车里被不认识的人踩了脚。）
- **Bunpro 校验**：待验证

### 4. ～は～に～（ら）れます（间接受害被动）

- **含义**：因为别人的行为而间接受到不利影响
- **接续**：受害者は + 动作主体に + 自动词/他动词被动形式
- **要点**：
  - 受害者不是动作的直接对象，而是间接受到影响
  - 自动词也可以构成这种被动句
  - 本课主要学习肯定叙述；否定形式语法上存在，但实际使用要看语境
  - 「に」前面一般是人或动物，但「雨」「雪」+「降ります」例外
- **例句**：
  - 陳さんは飼っていた小鳥に逃げられました。（老陈养的小鸟飞走了。）
  - 張さんは隣の人に夜遅くまで騒がれました。（邻居闹到很晚，小张休息受到了影响。）
  - キャンプの時、雨に降られて大変でした。（露营时被雨淋了，很狼狈。）
- **Bunpro 校验**：待验证

### 5. ～が/は～（ら）れます（事物主语被动）

- **含义**：用事物做主语的被动句（动作主体一般不出现）
- **接续**：事物が/は + 动词被动形式
- **要点**：
  - 动作主体一般是不确定的人群，句中不出现
  - 常用于客观描述事件、制度、建筑等
  - 书面语中比口语更常见
- **例句**：
  - 2010年に上海で万博が開かれました。（2010年在上海举办了世博会。）
  - 駅前に高いビルが建てられました。（车站附近盖起了高楼。）
  - この小説は多くの国で翻訳されています。（这部小说被翻译成了许多国家的语言。）
- **Bunpro 校验**：待验证

### 6. ～は～によって～（ら）れます

- **含义**：由特定的人/组织做的（事物主语被动句中标记特定动作主体）
- **接续**：事物は + 特定主体によって + 动词被动形式
- **要点**：
  - 正式/客观叙述中，标记特定的动作主体（发明、发现、创作等）
  - 常见于事物做主语的被动句中
  - 与「に」不能自由互换，但并非所有场景都绝对禁止
  - 常见于学术、历史、作品等描述
- **例句**：
  - この車は日本の有名なデザイナーによって設計されました。（这款车是由日本著名设计师设计的。）
  - 「万有引力の法則」はニュートンによって発見されました。（"万有引力定律"是牛顿发现的。）
  - この報告書は調査チームによってまとめられました。（这份报告是由调查团队汇总的。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| ～とも | 接数量词后表示全部：「2日とも雨でした」（两天都下了雨） |
| 参りました | 遇到麻烦时的口语表达，表示苦恼、狼狈："真糟糕" |
| 巻き込まれて | 被卷入某种不好的事情中受到影响：「渋滞に巻き込まれて」 |
| 书面语 vs 口语 | 报告等正式书面语用简体形结尾、汉字词汇多、被动句频繁出现 |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。确认被动形式变形规则及各类被动句的区别。

### Level 2：组装（30%）

给词造句，3-5 题。主动运用六种被动句型。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：出差遭遇连环不顺**

- **场景**：你出差去外地参加行业展会。从出发到结束，遭遇了一系列不顺：被堵在路上、被同事弄丢了资料、被淋了雨、被陌生人踩了脚。回到公司后，你向同事讲述这次出差的经历，同时提到展会上看到的一些被知名企业开发的新产品。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ 基本被动句（～は～に～（ら）れます）
    - ✅ 物主被动句（～は～に～を～（ら）れます）
    - ✅ 间接受害被动（自动词被动）
    - ✅ 事物主语被动（～が～（ら）れます）
    - ✅ によって（～は～によって～（ら）れます）
- **词汇限制**：仅使用 lesson ≤ 41 的词汇
- **AI 验收标准**：
  1. 语法检查：被动形式变形是否正确
  2. 覆盖检查：是否全部五种被动句型都用到了
  3. 自然度评分：被动句的使用是否自然、语境是否合理
