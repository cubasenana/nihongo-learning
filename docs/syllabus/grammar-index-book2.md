## 第25課：これは明日会議で使う資料です

1. 动（简体形）+ 名 — 动词简体形修饰名词（连体修饰）
2. 小句（动词简体形）+ 名 は ～です — 动词小句修饰的名词做主语
3. 小句（动词简体形）+ 名 を/に/から ～ます — 动词小句修饰的名词做宾语等
4. 小句（一类形/二类形/名）+ 名 — 形容词/名词小句修饰名词

## 第26課：自転車に2人で乗るのは危ないです

1. 小句（动词简体形）+ のは ～です — 动词小句名词化做主语
2. 小句（动词简体形）+ のを ～ — 动词小句名词化做宾语
3. 小句（简体形）でしょう — 推测（大概……）
4. 小句（简体形）かもしれません — 也许……

## 第27課：子供の時、大きな地震がありました

1. 小句（简体形）+ 時 — ……的时候
2. 动 ながら — 一边……一边……（同时进行）
3. 小句（简体形）でしょう？ — 确认（……对吧？）
4. 动 ています③ — 反复/习惯性动作
5. 名 で — 名词表示原因、理由
6. ～と言います — 引用（说……）

## 第28課：馬さんはわたしに地図をくれました

1. 名1は 名2に 名3を くれます — 别人给我/我方（接受者视角的给予）
2. 动てあげます — 为别人做某事
3. 动てもらいます — 请别人为自己做某事
4. 动てくれます — 别人为自己做某事

## 第29課：電気を消せ

1. 动词的命令形 — 命令形变形规则
2. 动なさい — 稍柔和的命令（请做……）
3. 动（基本形）な — 禁止（不许……）
4. ～という + 名 — 叫做……的（引用/命名）

## 第30課：もう11時だから寝よう

1. 动词的意志形 — 意志形变形规则
2. 动（意志形）と思います — 当场表示意志（我想……）
3. 动（意志形）と思っています — 持续的意志/打算
4. 小句1 ので、小句2 — 因为……（比から更客观）

## 第31課：このボタンを押すと、電源が入ります

1. 小句1（动基本形/ない形）と、小句2 — 条件（一……就……，用于恒常/真理/习惯）
2. 动（基本形/ない形）ことがあります — 有时会……
3. 小句（简体形）でしょうか — 委婉疑问

## 第32課：今度の日曜日に遊園地へ行くつもりです

1. 动（基本形/ない形）つもりです — 打算……
2. 动（基本形/ない形）ことにします — 自己决定……
3. 动（基本形/ない形）ことになりました — （客观/集体）决定……
4. 小句（简体形）そうです① — 传闻（听说……）

## 第33課：電車が急に止まりました

1. 自动词和他动词② — 自他动词的词形对应关系
2. 自动 ています④ — 自动词的结果状态（第二种）
3. 动 てしまいます — 完了/遗憾（……了/不小心……）
4. 动/形 そうです② — 样态推测（看起来要……）

## 第34課：壁にカレンダーが掛けてあります

1. 他动 てあります — 有意动作结果的存续状态
2. 动 ておきます — 事先准备（预先做好……）
3. 动 てみます — 尝试做……
4. 小句（基本形）ために、小句① — 目的（为了……）

## 第35課：明日雨が降ったら、マラソン大会は中止です

1. 小句1 たら、小句2 — 假定条件（如果……就……）
2. 小句1 ても、小句2 — 让步条件（即使……也……）
3. 名 しか + 否定 — 限定（只有……）
4. 名 でも — 提示极端例子（就连……也……）

## 第36課：遅くなって、すみません

1. 小句 て、小句 — て形表示原因、理由
2. 名 に — 名词表示用途/基准
3. 动（基本形）のに — 动词表示用途/基准（用于……）
4. 名 ばかり + 动 — 尽是……/光是……
5. 小句（动词简体形）のが 見えます/聞こえます — 能看到/听到……

## 第37課：優勝すれば、オリンピックに出場することができます

1. 动词的ば形 — ば形变形规则
2. 小句1 ば、小句2 — 假定条件（如果……的话）
3. 小句（简体形）なら、小句 — 话题承接条件（如果说……的话）
4. 名 でも — 示例（……之类的）
5. 名1 とか 名2 とか — 列举（……啦……啦）

## 第38課：戴さんは英語が話せます

1. 可能形式 — 可能形变形规则（能够……）
2. 小句（基本形/ない形）ように、小句 — 以便……/为了……
3. 小句（基本形/ない形）ようになります — 变得能够……
4. 小句（基本形/ない形）ようにします — 努力做到……

## 第39課：眼鏡をかけて本を読みます

1. 小句1 て/ないで + 小句2 — 附带状态（……着做……/不……地做……）
2. 小句1（简体形）ために、小句2② — 原因、理由（因为……）
3. 动 ていきます/きます — 空间移动方向

## 第40課：これから友達と食事に行くところです

1. 动 ところです — 动作的某个阶段（正要/正在/刚……）
2. 动 たばかりです — 刚刚做完……
3. 动 はじめます/出します/続けます/終わります — 复合动词（开始/持续/结束）

## 第41課：李さんは部長にほめられました

1. 被动形式 — 被动形变形规则
2. 名は（名に）动（ら）れます — 直接被动句
3. 名は 名に 名を 动（ら）れます — 间接被动（所有物被动）
4. 名は 名に 动（ら）れます — 间接被动（受害被动）
5. 名が/は 动（ら）れます — 非情物主语被动
6. 名は 名によって 动（ら）れます — 创造/发明的被动

## 第42課：テレビをつけたまま、出かけてしまいました

1. 小句（动た形/ない形）まま、小句 — 保持……状态
2. 小句1（简体形）のに、小句2 — 明明……却……（转折/不满）
3. 小句（简体形）はずです — 应该……（合理推断）
4. 小句（简体形）はずがありません — 不可能……

## 第43課：陳さんは、息子をアメリカに留学させます

1. 使役形式 — 使役形变形规则
2. 名は 名を 自动（さ）せます — 使役句（自动词）
3. 名は 名に 名を 他动（さ）せます — 使役句（他动词）
4. 动（さ）せてください — 请让我做……
5. 动 やすいです/にくいです — 容易……/难……

## 第44課：玄関のところにだれかいるようです

1. 小句（简体形）ようです① — 推测（好像……）
2. 小句（简体形）みたいです① — 推测口语版（好像……）
3. 小句（简体形）らしいです — 推测/传闻（似乎……）
4. 动/一类形/二类形 + 過ぎます — 过于……/太……了
5. 一类形 + さ — 形容词名词化（……度/……性）

## 第45課：少子化が進んで、日本の人口はだんだん減っていくでしょう

1. 动 ていきます/きました — 变化趋势（持续/变化）
2. 动 てきました — 开始出现
3. 小句1 し、小句2 し、小句3 — 并列列举理由
4. 动/一类形 ば ～ ほど、小句 — 越……越……
5. で/へ/から/まで/と + の — 复合助词 + の 修饰名词

## 第46課：これは柔らかくて、まるで本物の毛皮のようです

1. ようです② — 比喻（像……一样）
2. 名1 + らしい + 名2 — 典型性（像样的……）
3. 名（时间）までに — 在……之前（期限）
4. 名/小句（简体形）+ 間/間に — 在……期间/趁……
5. 名のような + 名/においがします — 像……的/有……气味

## 第47課：周先生は明日日本へ行かれます

1. 敬语概述 — 尊他语/自谦语/礼貌语三分类
2. 尊他语概述 — 抬高对方的表达方式
3. 动（ら）れます — 尊敬语（被动形式表尊敬）
4. お + 一类动/二类动 になります — 尊敬语（お……になる）
5. お + 一类动/二类动 ください — 尊敬语（お……ください）
6. 尊他语的特殊形式 — いらっしゃる/おっしゃる 等

## 第48課：お荷物は私がお持ちします

1. 自谦语概述 — 压低自己的表达方式
2. お + 一类动/二类动 します — 自谦语（お……する）
3. 自谦语的特殊形式 — まいる/いただく 等
4. 动 ていただけますか — 请求（能否请您……）
