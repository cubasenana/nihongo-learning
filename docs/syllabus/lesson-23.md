## 语法点

### 1. 動たり～たりします

- **含义**：有时做……有时做……（列举代表性动作）
- **接续**：动词た形 + り + 动词た形 + り + します（読んだ→読んだり、食べた→食べたり）
- **要点**：
  - 列举若干有代表性的动作，暗示还有其他
  - 句尾用「します」（可变时态：しました、したいです等）
  - 不表示动作的先后顺序
- **例句**：
  - 休みの日は本を読んだり、音楽を聞いたりします。（休息日看看书、听听音乐。）
  - 週末、友達と映画を見たり、買い物に行ったりしました。（周末和朋友看了电影、去了购物。）
- **Bunpro 校验**：待验证

### 2. 一类形かったり / 二类形だったり

- **含义**：有时……有时……（状态的多种可能）
- **接续**：
  - 一类形容词：～かったり ～かったりです
  - 二类形容词：～だったり ～だったりです
  - 名词：～だったり ～だったりです
- **要点**：
  - 列举代表性的多种可能状态，不一定是反义对
  - 后面可接「です」结尾，也可接其他表达
  - 常与「～によって」搭配使用
- **例句**：
  - 天気は暖かかったり涼しかったりです。（天气有时暖和有时凉快。）
  - 天気は暑かったり寒かったりで、服を選ぶのが大変です。（天气忽冷忽热，选衣服很麻烦。）
- **Bunpro 校验**：待验证

### 3. 小句＋か（不确定的内容）

- **含义**：是否……/……什么（将疑问内容嵌入长句）
- **接续**：
  - 无疑问词时：简体形 + か どうか
  - 有疑问词时：疑问词小句简体形 + か
- **要点**：
  - 「か どうか」用于没有疑问词的是非问
  - 疑问词小句直接加「か」
  - 名词/二类形容词现在肯定形前不加「だ」：中国人かどうか（非×中国人だかどうか）
- **例句**：
  - 明日、雨が降るかどうか分かりません。（不知道明天会不会下雨。）
  - 駅までどうやって行くか教えてください。（请告诉我怎么去车站。）
  - この料理が辛いかどうか知りません。（不知道这道菜辣不辣。）
- **Bunpro 校验**：待验证
