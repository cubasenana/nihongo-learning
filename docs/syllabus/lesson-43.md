## 语法点

### 1. 使役形式规则

- **含义**：使役主体强制、指示或允许动作主体进行动作
- **接续**：
  - 一类动词：ない形去「ない」+ せる
  - 二类动词：ない形去「ない」+ させる
  - 三类动词：来る → 来させる、する → させる
- **要点**：
  - 使役形式的活用方式与二类动词相同
  - 一类动词先变ない形，再把ない替换为せる：書く → かかせる
  - 二类动词先变ない形，再把ない替换为させる：食べる → たべさせる
  - 注意不要把「話させる」（使役）误写成「話せる」（可能形）
- **例句**：
  - 「行く」の使役形式は「行かせる」です。（"去"的使役形式是"让去"。）
  - 「食べる」は二類動詞ですから、「食べさせる」になります。（"吃"是二类动词，所以变成"食べさせる"。）
- **Bunpro 校验**：待验证

### 2. ～は～を（自动词）（さ）せます

- **含义**：让某人做某事（使役主体做主语，自动词时动作主体用を标记）
- **接续**：使役主体は + 动作主体を + 自动词使役形式
- **要点**：
  - 使役句中使役主体做主语
  - 自动词时，被使役的人多用「を」标记（「に」也存在，语气上有许可/强制的细微区别）
  - 可表示强制、命令，也可表示许可、放任
- **例句**：
  - 陳さんは息子をアメリカに留学させます。（老陈让儿子去美国留学。）
  - 田中さんは公園で子供を遊ばせています。（田中先生让孩子在公园玩耍。）
  - あの課長は部下を休ませません。（那个科长不让部下休息。）
- **Bunpro 校验**：待验证

### 3. ～は～に～を（他动词）（さ）せます

- **含义**：让某人做某事（他动词时动作主体用に标记，因为を已被宾语占用）
- **接续**：使役主体は + 动作主体に + 宾语を + 他动词使役形式
- **要点**：
  - 他动词使役句中，被使役的人用「に」标记
  - 原因：日语中尽量避免一句话出现两个「を」
  - 切勿写成：× 森さんを歌を歌わせます
- **例句**：
  - 部長は太田さんにレポートを書かせました。（部长让太田先生写报告。）
  - 先生は生徒にこの本を読ませます。（老师让学生读这本书。）
  - 母は子供に部屋を掃除させました。（妈妈让孩子打扫了房间。）
- **Bunpro 校验**：待验证

### 4. ～（さ）せてください

- **含义**：请让我做……（请求对方允许自己做某事）
- **接续**：动词使役て形 + ください
- **要点**：
  - 说话人请求许可做某事，语气礼貌
  - 一般省略「わたしを/わたしに」
  - 需要强调时可以加上：「わたしにやらせてください」
  - 商务场景中很常用
- **例句**：
  - 疲れました。少し休ませてください。（累了，请让我稍微休息一会儿。）
  - この仕事はわたしにやらせてください。（这份工作请让我来做。）
  - 部長、気分が悪いので、早く帰らせてください。（部长，我不舒服，请让我早点回去。）
- **Bunpro 校验**：待验证

### 5. ～やすいです / ～にくいです

- **含义**：容易做……/难以做……（表示事物具有易于或难于做某事的性质）
- **接续**：动词ます形去「ます」+ やすい/にくい
- **要点**：
  - 描述事物的固有性质，不是某次具体体验
  - 活用与一类形容词相同：やすかった、にくくない
  - 「やすい」既可表示"好做/方便做"，也可表示"容易发生/倾向于"（壊れやすい、太りやすい）
  - 「にくい」= 不好做、不容易发生
- **例句**：
  - このボールペンはとても書きやすいです。（这支圆珠笔很好用。）
  - この駅は乗り換えが分かりにくいですね。（这个车站换乘很难找。）
  - 運動をしている人は太りにくいです。（经常做运动的人不容易发胖。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| 「行ってもらいます」vs「行かせます」 | 前者委托语气柔和，后者命令语气强硬 |
| ～のではないでしょうか | 委婉陈述意见："不是～吗？" 比「～と思います」更柔和 |
| ぐっと | 表示变化幅度大或动作迅猛："商品のイメージがぐっとよくなる" |
