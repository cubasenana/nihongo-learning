## 语法点

### 1. 小句（简体形）+ 時（时间从句）

- **含义**：……的时候（表示动作或状态发生的时间）
- **接续**：动词基本形/た形 + 時；一类形基本形 + 時；二类形 + な + 時；名词 + の + 時
- **要点**：
  - 动词基本形 + 時：后项发生时前项动作尚未完成（出发前或途中）
  - 动词た形 + 時：后项发生时前项已完成（行った時 = 到了之后）
  - 只有名词才加「の」：子供の時 ○ / 忙しいの時 ×
  - 加「に」强调具体的时间点/场合：病気の時に病院に行きました
- **例句**：
  - 日本に行く時、パスポートを準備しました。（去日本的时候，准备好了护照。）
  - 疲れた時、甘いものが食べたくなります。（累的时候就想吃甜食。）
  - 暇な時、よく映画を見に行きます。（有空的时候，常去看电影。）
- **Bunpro 校验**：待验证

### 2. 动词ます形（去掉ます）+ ながら（同时动作）

- **含义**：一边……一边……（同一主体同时进行两个动作）
- **接续**：动词ます形去掉「ます」+ ながら
- **要点**：
  - 后面的动作是主要动作
  - 主体必须是同一个人
  - 音楽を聞きながら勉強する（一边听音乐一边学习）→ 学习是主要动作
- **例句**：
  - 音楽を聞きながら料理を作ります。（一边听音乐一边做饭。）
  - コーヒーを飲みながら新聞を読むのが好きです。（喜欢一边喝咖啡一边看报纸。）
  - 歩きながらスマホを見るのは危ないです。（边走边看手机很危险。）
- **Bunpro 校验**：待验证

### 3. 小句（简体形）+ でしょう?（确认）

- **含义**：……吧？（向对方确认某件事）
- **接续**：小句简体形 + でしょう?
- **要点**：
  - 升调（↗）：请对方告诉自己信息——李さん、明日行くでしょう?
  - 降调（↘）：叮问或意见不同时——おいしいでしょう↘
  - 与「～ね」不同：「ね」是求确认/共鸣（预设对方会同意）；「でしょう」带有说话人自己的判断，也可用于对方未必同意时
- **例句**：
  - 昨日、駅前の本屋に行ったでしょう？（你昨天去了车站前的书店吧？）
  - この料理、おいしいでしょう？（这道菜好吃吧？）
  - 来週の会議、月曜日でしょう？（下周的会议是周一吧？）
- **Bunpro 校验**：待验证

### 4. 动词て形 + います③（反复/习惯）

- **含义**：（一直）做……（表示反复性或习惯性的动作）
- **接续**：动词て形 + います
- **要点**：
  - 第15/16课学了「ています」表正在进行和结果状态，本课是第三种用法
  - 表示习惯时接近「ます形」，但「ています」更强调当前阶段持续的习惯：毎日散歩しています
  - 有「先月から」等时间起点时，一般只用「ています」不用「ます」
  - 也可表示职业：高校の教師をしています
- **例句**：
  - 毎朝ジョギングをしています。（每天早上跑步。）
  - この電車は10分に1本走っています。（这趟电车每10分钟一班。）
  - 去年から日本語を勉強しています。（从去年开始学日语。）
- **Bunpro 校验**：待验证

### 5. 名词 + で（原因/理由）

- **含义**：因为……（助词「で」表示原因、理由）
- **接续**：名词 + で
- **要点**：
  - 只接名词，不接动词/形容词
  - 仕事で遅れました（因工作迟到了）
  - 台風で電車が止まりました（因台风电车停了）
- **例句**：
  - 風邪で会社を休みました。（因为感冒没去上班。）
  - 事故で電車が遅れています。（因事故电车晚点了。）
  - 仕事で東京に行くことになりました。（因工作要去东京了。）
- **Bunpro 校验**：待验证

### 6. 名词 + と + 动词（共同/互动）

- **含义**：和……一起做……/与……互动（表示共同行为或互动对方）
- **接续**：名词（人）+ と + 动词
- **要点**：
  - 互动动词只能用「と」：結婚します、けんかします
  - 「会います」可用「に」也可用「と」，用「と」时含互动语感
  - 「甲は乙と～」甲是主语；「甲と乙（と）が～」双方都是主语
- **例句**：
  - 週末、友達と映画を見に行きました。（周末和朋友去看了电影。）
  - 田中さんは佐藤さんと結婚しました。（田中先生和佐藤女士结婚了。）
  - 昨日、弟とけんかしてしまいました。（昨天和弟弟吵架了。）
- **Bunpro 校验**：待验证

## 课文

> Arc 7「新城市」第3话 — 公园散步

**登場人物**：陳明、王 — 日曜日の公園

> 使用语法：≤L27（+小句+時、動ながら、でしょう?確認、動ています③習慣、Nで原因、Nと動）

---

陳：子供の 時、よく 公園で 遊びましたか。

王：はい、毎日 遊びました。陳さんは 毎朝 何を して いますか。

陳：音楽を 聞きながら 散歩して います。

王：雨の 日も 散歩しますか。

陳：いいえ、雨の 時は 散歩しません。家で 本を 読みます。

王：陳さんは 日本に いた 時、よく 散歩しましたか。

陳：はい。田中さんと 公園で 散歩しました。

王：田中さん、先週 上海に 来たでしょう？

陳：はい、仕事で 来ました。忙しくて、会えませんでした。

---

**语法覆盖**：✅ 小句+時（子供の時/雨の時/日本にいた時） ✅ 動ながら（聞きながら散歩） ✅ でしょう?確認（来たでしょう？） ✅ ています③習慣（散歩しています） ✅ Nで原因（仕事で来ました） ✅ Nと動（田中さんと散歩しました）

## 表达要点

| 要点 | 说明 |
|------|------|
| 大勢 | 表示人多，只能用于人（「たくさん」可用于人和物）；修饰名词或动词，不做谓语 |
| そう言えば | 由正在进行的对话联想到另一件事时使用："说起来……" |
| 学校に通っています | 「通います」表示定期来往同一场所（含往返），「行きます」只是单向移动 |
