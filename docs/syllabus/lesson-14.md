## 语法点

### 1. 动词的て形规则

- **含义**：动词的一种变形，用于连接、请求等多种表达
- **接续**：根据动词类别，变形规则不同
- **要点**：
  - 一类动词：き→いて、ぎ→いで、び/み/に→んで、ち/り/い→って、し→して
  - 二类动词：ます形去ます + て（食べます→食べて）
  - 三类动词：来ます→来て、します→して
  - 特殊：行きます→行って（例外）
- **例句**：
  - 書きます→書いて、読みます→読んで、待ちます→待って
  - 食べます→食べて、見ます→見て、来ます→来て
- **Bunpro 校验**：待验证

### 2. 动て 动（相继发生）

- **含义**：表达两个以上动作按时间顺序相继发生
- **接续**：动词て形 + 动词（可连续使用）
- **要点**：
  - 按时间先后顺序排列
  - 可以连接两个或三个动作
  - 连接太多时使用「それから」分句
- **例句**：
  - 朝起きて、顔を洗って、朝ごはんを食べます。（早上起床，洗脸，吃早饭。）
  - 図書館へ行って、本を借ります。（去图书馆借书。）
- **Bunpro 校验**：待验证

### 3. 动てから 动（先后顺序）

- **含义**：强调先做完前一个动作，再做后一个动作
- **接续**：动词て形 + から + 动词
- **要点**：
  - 与「て」基本相同，但更强调先后关系
  - 初级阶段通常一个句子只用一次「てから」；连续堆叠会不自然
  - 强调"做完A之后才做B"的时间顺序
- **例句**：
  - 宿題をしてから、テレビを見ます。（做完作业后看电视。）
  - シャワーを浴びてから、寝ます。（冲完澡后睡觉。）
- **Bunpro 校验**：待验证

### 4. 动て ください

- **含义**：请求对方做某事，"请……"
- **接续**：动词て形 + ください
- **要点**：
  - 基本的请求表达
  - 日常礼貌请求，对上级在一般情景下也可使用
  - 正式场合可用更委婉的形式：〜ていただけませんか
- **例句**：
  - すみません、もう一度言ってください。（不好意思，请再说一次。）
  - この書類にサインしてください。（请在这份文件上签名。）
- **Bunpro 校验**：待验证

### 5. 名【场所】を 动（经过/离开）

- **含义**：表示经过某场所或离开某场所
- **接续**：场所 + を + 移动动词
- **要点**：
  - 经过：を + 通ります/渡ります/過ぎます
  - 离开：を + 出ます/卒業します
  - 此处「を」不表示宾语，而表示经过/离开的场所
- **例句**：
  - 毎朝この橋を渡ります。（每天早上过这座桥。）
  - 8時に家を出ます。（8点离开家。）
- **Bunpro 校验**：待验证

## 课文

> Arc 4「生活技能」第2话 — 忙碌的周末

**登場人物**：陳明、田中 — 月曜日の朝

> 使用语法：≤L14（+て形、動て動、動てから動、動てください、Nを動(経過/離開)）

---

田中：陳さん、週末 何を しましたか。

陳：土曜日に スーパーへ 行って、野菜や 肉を 買いました。

田中：それから？

陳：うちへ 帰ってから、料理を 作りました。

田中：何を 作りましたか。

陳：餃子を 作りました。まず 野菜を 切って、肉と 混ぜます。それから 皮で 包んで、焼きます。

田中：そうですか。日曜日は？

陳：日曜日は 朝 公園を 散歩しました。それから 図書館へ 行って、本を 読みました。

田中：今度、教えて ください。

陳：はい。

---

**语法覆盖**：✅ て形（行って/切って/包んで/読んで） ✅ 動て動（行って買いました/切って混ぜます） ✅ 動てから（帰ってから作りました） ✅ 動てください（教えてください） ✅ Nを動(経過)（公園を散歩しました）

## 表达要点

| 要点 | 说明 |
|------|------|
| 句子连接用それから | 「〜て」连接太多会不自然，超过2-3项时用「それから」分句 |
| 駅前 | 指车站周围繁华区域，不一定是"车站正前方" |
| なかなか + 肯定 | 表示程度比预想高，"相当……"，不用于对长辈评价 |
| すみませんが、〜 | 请求别人帮忙前的铺垫语，表示歉意 |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：忙碌的周末计划**

- **场景**：你向朋友描述你这个周末的详细计划——先去银行取钱，然后去邮局寄包裹，再去超市买东西。朋友也描述自己的计划，并请你帮忙做一件事。
- **要求**：
  - 8-12 句
  - 必须覆盖以下全部语法点：
    - ✅ て形（至少正确变形3个不同类型的动词）
    - ✅ 动て动（相继动作）
    - ✅ てから（先后顺序）
    - ✅ てください（请求）
    - ✅ 場所を動（经过或离开）
- **词汇限制**：仅使用 lesson ≤ 14 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
