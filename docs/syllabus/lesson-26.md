## 语法点

### 1. 小句（动词简体形）+ のは + 一类形/二类形 + です（名词化做主语）

- **含义**：做某事是……的（将动作名词化后充当主语）
- **接续**：动词简体形 + のは + 形容词です；好恶类形容词对象用「のが」
- **要点**：
  - 一般主语用「のは」：自転車に乗るのは楽しいです
  - 好恶/擅长不擅长的对象用「のが」：絵をかくのが好きです
  - 好恶对象作为话题提出时又变回「のは」：絵をかくのは好きですか
- **例句**：
  - 外国語を勉強するのは大変ですが、おもしろいです。（学外语虽然辛苦，但很有趣。）
  - 朝早く起きるのが苦手です。（不擅长早起。）
  - 一人で旅行するのは不安ですか。（一个人旅行会不安吗？）
- **Bunpro 校验**：待验证

### 2. 小句（动词简体形）+ のを + 动词（名词化做宾语）

- **含义**：做某事……（将动作名词化后充当宾语）
- **接续**：动词简体形 + のを + 动词
- **要点**：
  - 「の」和「こと」有重叠，但「の」偏具体/现场感知，「こと」偏抽象/内容描述
  - 感知动词（見ます/聞きます）、直接作用动词（手伝います/やめます）→ 只能用「の」
  - 语言行为动词（知らせます/伝えます/話します）→ 只能用「こと」
- **例句**：
  - 傘を持っていくのを忘れました。（忘了带伞。）
  - 子供たちが公園で遊んでいるのを見ました。（看到孩子们在公园里玩。）
  - 友達が引っ越しするのを手伝いました。（帮朋友搬家了。）
- **Bunpro 校验**：待验证

### 3. 小句（简体形）+ でしょう（推测）

- **含义**：大概……吧（对不确定的事情进行推测）
- **接续**：动词/一类形简体形 + でしょう；二类形/名词简体形去「だ」+ でしょう
- **要点**：
  - 常与副词「たぶん」呼应使用
  - 简体形为「～だろう」
  - 二类形容词和名词去掉「だ」再接：静かでしょう、雨でしょう
- **例句**：
  - 週末はたぶん暖かいでしょう。（周末大概会暖和吧。）
  - この映画はたぶん人気があるでしょう。（这部电影大概会受欢迎吧。）
  - 会議はもう終わっただろう。（会议已经结束了吧。）
- **Bunpro 校验**：待验证

### 4. 小句（简体形）+ かもしれません（可能性）

- **含义**：也许……（表示不确定的可能性）
- **接续**：动词/一类形简体形 + かもしれません；二类形/名词简体形去「だ」+ かもしれません
- **要点**：
  - 表示不确定的可能性，通常比「でしょう」把握度低
  - 可以表达两种可能都存在：行くかもしれませんし、行かないかもしれません
  - 常与「もしかしたら」呼应
  - 二类形容词和名词去掉「だ」再接：暇かもしれません
- **例句**：
  - 明日の試験は難しいかもしれません。（明天的考试也许很难。）
  - もしかしたら、電車が遅れるかもしれません。（也许电车会晚点。）
  - 財布を家に忘れたかもしれません。（钱包也许忘在家里了。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| もしかしたら | 口语形式，用于可能性较低的场合，常与「かもしれません」呼应 |
| それで | 以前句事态为后句的原因/理由；后面不能接祈使/推测句（那种情况用「だから」） |
| つい | 表示并非有意所为却形成了某种后果，含有不快/后悔的语感 |

### Level 2：组装（30%）

### Level 3：产出（60%）— 本课核心

**📋 产出任务：周末计划与天气讨论**

- **场景**：周五下午，你和同事讨论周末的计划。你谈到自己喜欢或不擅长的活动，推测周末天气如何，并讨论某位朋友是否会参加活动。过程中有人提到忘了做某件事。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ 动词のは＋形容词です（名词化主语）
    - ✅ 动词のを＋动词（名词化宾语）
    - ✅ でしょう（推测）
    - ✅ かもしれません（可能性）
- **词汇限制**：仅使用 lesson ≤ 26 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
