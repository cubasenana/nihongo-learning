## 语法点

### 1. 小句1 たら、小句2（假定条件）

- **含义**：如果……就……（假定某种条件成立时的结果）
- **接续**：动词/形容词/名词的过去形「た」→「たら」
- **要点**：
  - 构成方式：把过去形的「た」换成「たら」
  - 名词/二类形容词现在肯定：「だ」→「だったら」（雨だったら、静かだったら）
  - 名词/二类形容词否定：「～ではなかった」→「～ではなかったら」/「～じゃなかったら」
  - 后句可以是命令/请求/意志等任何形式（比「ば」自由）
  - 也可表示"做了之后发现"：窓を開けたら、富士山が見えました
- **例句**：
  - 明日天気がよかったら、公園へ散歩に行きましょう。（明天天气好的话就去公园散步吧。）
  - 分からないことがあったら、いつでも聞いてください。（有不懂的事随时问我。）
  - 駅に着いたら、電話してください。（到了车站请给我打电话。）
- **Bunpro 校验**：待验证

### 2. 小句1 ても、小句2（即使/让步）

- **含义**：即使……也……（条件成立但结果不变）
- **接续**：动词/形容词て形的「て」→「ても」、「で」→「でも」
- **要点**：
  - 假定性让步：薬を飲んでも治りません（即使吃药也不会好）
  - 既成事实让步：辞書で調べても分かりませんでした（查了词典也没弄懂）
  - 形容词：忙しくても／静かでも
  - 名词：休日でも
- **例句**：
  - いくら練習しても、なかなか上手になりません。（无论怎么练也不太能变好。）
  - 雨が降っても、サッカーの試合はあります。（即使下雨，足球比赛也照常进行。）
  - 安くても、必要ないものは買いません。（即使便宜，不需要的东西也不买。）
- **Bunpro 校验**：待验证

### 3. 名詞＋だけ（限定）

- **含义**：只有……、仅仅……（客观地限定范围）
- **接续**：名词 + だけ
- **要点**：
  - 表示"这就是全部，不再有其他"
  - 后续「が」「を」可省略：森さんだけ（が）来ました
  - 与「しか」的区别：だけ是客观限定，不带主观感情色彩
- **例句**：
  - 今日は午前中だけ仕事があります。（今天只有上午有工作。）
  - このクラスで日本に行ったことがあるのは田中さんだけです。（这个班去过日本的只有田中。）
  - あと10分だけ待ってください。（请再等我10分钟就好。）
- **Bunpro 校验**：待验证

### 4. 名詞＋しか＋否定形式

- **含义**：只有……（带有"太少了/不够"的主观感情色彩）
- **接续**：名词 + しか + 否定形式
- **要点**：
  - 后面必须接否定形式
  - 与「～だけ＋肯定」意思基本相同，但「しか」带主观色彩
  - 与数量词搭配时差异明显：「3日だけ」客观 vs「3日しか」嫌少
- **例句**：
  - 財布の中に500円しかありません。（钱包里只有500日元。）
  - 日本語の授業は週に1回しかありません。（日语课一周只有一次。）
  - 今回の旅行では京都にしか行きませんでした。（这次旅行只去了京都。）
- **Bunpro 校验**：待验证

### 5. 名詞＋でも（极端例示）

- **含义**：连……都……（举出极端的例子）
- **接续**：名词 + でも
- **要点**：
  - 提示极端的一个例子，暗示"更不用说其他了"
  - 肯定句：子供でもできます（连小孩都会做）
  - 否定句：大人でもできません（连大人都不会做）
  - 被形容词/小句修饰的名词也可接でも：よく効く薬でも、飲みすぎはよくない
- **例句**：
  - このアプリは子供でも簡単に使えます。（这个APP连小孩都能轻松使用。）
  - プロの選手でも、毎日練習が必要です。（即使是专业选手，每天也需要练习。）
  - こんなに寒いと、元気な人でも風邪を引きます。（这么冷的话，连身体好的人也会感冒。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| 何かあった | 「あった」不表示"存在"，而表示"发生了什么事" |
| あと～ | 后接数量词表示"只差……就完成"：あと10分です |
| なかなか＋否定 | 表示事态不如期待的那样简单："怎么也不……" |
| だんだん | 逐渐、渐渐地（变化不显著），常与「～なります」搭配 |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：在朋友家包饺子**

- **场景**：你受邀去一位中国朋友家包饺子。朋友问你会不会包，你说照着书做但怎么也做不好。朋友说多做几次就会好的，如果有时间的话下次一起再做。吃饺子时你发现自己只能吃20个，而朋友连孩子都能包得很好。你感叹即使看了教程也很难学会。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ ～たら（假定条件）
    - ✅ ～ても（即使/让步）
    - ✅ ～だけ
    - ✅ ～しか＋否定
    - ✅ ～でも（极端例示）
- **词汇限制**：仅使用 lesson ≤ 35 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
