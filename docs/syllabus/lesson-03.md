## 语法点

### 1. ここ / そこ / あそこ は 名 です

- **含义**：指示场所的代词，用于说明某地点是什么
- **接续**：ここ / そこ / あそこ + は + 名词 + です
- **要点**：
  - 位置关系与 これ/それ/あれ 相同
  - ここ：距说话人近的场所
  - そこ：距听话人近的场所
  - あそこ：距双方都远的场所
- **例句**：
  - ここは 教室です。（这里是教室。）
  - そこは 受付です。（那里是前台。）
  - あそこは 公園です。（那儿是公园。）
- **Bunpro 校验**：待验证

### 2. 名 は 名［場所］です

- **含义**：表示某人或某物存在于某个场所
- **接续**：名词 + は + 場所名词 + です
- **要点**：
  - 中文可译为"在"，但日语结构仍是名词谓语句：A は 場所名詞 です
  - 「トイレは ここです」（厕所在这儿）与「ここは トイレです」（这里是厕所）句型相同但重点不同
  - 也可以用于表示人的所在地
- **例句**：
  - 食堂は 3階です。（食堂在3楼。）
  - 田中さんは 事務所です。（田中先生在办公室。）
  - 郵便局は 駅の 隣です。（邮局在车站旁边。）
- **Bunpro 校验**：待验证

### 3. 名 は どこですか

- **含义**：用于询问人或物所在的场所
- **接续**：名词 + は + どこ + ですか
- **要点**：
  - どこ 的礼貌说法为 どちら
  - 回答时可用 ここ/そこ/あそこ/あちら 等
- **例句**：
  - お手洗いは どこですか。（洗手间在哪里？）
  - あなたの 学校は どこですか。（你的学校在哪里？）
- **Bunpro 校验**：待验证

### 4. 名 も 名 です

- **含义**：助词 も 表示"也"，用于追加同类信息
- **接续**：名词 + も + 名词 + です
- **要点**：
  - も 替换 は，表示与前文相同的情况
  - 基本相当于汉语的"也"
- **例句**：
  - ここは 銀行です。あそこも 銀行です。（这里是银行。那里也是银行。）
  - 李さんは 中国人です。張さんも 中国人です。（小李是中国人。小张也是中国人。）
- **Bunpro 校验**：待验证

### 5. 名 は 名 ですか、名 ですか（选择疑问句）

- **含义**：在多种可能中询问具体是哪一种
- **接续**：名词 + は + 名词A + ですか、名词B + ですか
- **要点**：
  - 重复谓语 〜ですか 来列出选项
  - 不能用 はい/いいえ 来回答，需直接回答具体选项
  - 选项可以是两个以上
- **例句**：
  - これは ペンですか、鉛筆ですか。（这是钢笔还是铅笔？）
  - 今日は 土曜日ですか、日曜日ですか。（今天是星期六还是星期天？）
- **Bunpro 校验**：待验证

## 课文

> Arc 1「到达·安顿」第3话 — 公司周边

**登場人物**：陳明（ちん めい）、田中（たなか）

> 使用语法：≤L03（+ここ/そこ/あそこ、名は場所です、どこですか、も、选择疑问句）

---

陳：田中さん、食堂はどこですか。

田中：食堂は3階です。

陳：コンビニはどこですか。

田中：コンビニは1階です。あそこです。

陳：ああ、あそこですか。郵便局はこの近くですか。

田中：はい。郵便局は駅の隣です。

陳：銀行もこの近くですか。

田中：はい。銀行も駅の近くです。

陳：あのう、ここはトイレですか、会議室ですか。

田中：そこはトイレです。会議室は2階です。

陳：そうですか。ありがとうございます。

田中：あ、陳さん、あそこはデパートです。

---

**语法覆盖**：
- ✅ ここ/そこ/あそこ は 名 です（あそこです / そこはトイレです / あそこはデパートです）
- ✅ 名は場所です（食堂は3階です / 郵便局は駅の隣です）
- ✅ どこですか（食堂はどこですか / コンビニはどこですか）
- ✅ も（銀行もこの近くですか / 銀行も駅の近くです）
- ✅ 选择疑问句（ここはトイレですか、会議室ですか）

## 表达要点

| 要点 | 说明 |
|------|------|
| こちら/そちら/あちら/どちら | ここ/そこ/あそこ/どこ 的礼貌说法；原本是表示方向的词 |
| いくら | 询问价格时使用，如「これは いくらですか」 |
| 省略（谓语） | 进一步询问时可省略已说过的谓语部分，句尾读升调，如「あれは（↗）」 |
| あのう | 向对方搭话、引起注意时使用；也用于难以启齿的事情或新话题 |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。确认语法规则是否记住。

### Level 2：组装（30%）

给词造句，3-5 题。主动运用语法。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：逛商场问路**

- **场景**：你第一次来到一家大型商场，需要找到食堂、洗手间和朋友约好见面的书店。你向商场的工作人员询问这些地方的位置，还顺便问了一下商品的价格。
- **要求**：
  - 5-7 句
  - 必须覆盖以下全部语法点：
    - ✅ ここ/そこ/あそこ
    - ✅ 名は名［場所］です
    - ✅ どこですか
    - ✅ も
    - ✅ 选择疑问句
- **词汇限制**：仅使用 lesson ≤ 3 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
