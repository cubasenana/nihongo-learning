## 语法点

### 1. ここ / そこ / あそこ は 名 です

- **含义**：指示场所的代词，用于说明某地点是什么
- **接续**：ここ / そこ / あそこ + は + 名词 + です
- **要点**：
  - 位置关系与 これ/それ/あれ 相同
  - ここ：距说话人近的场所
  - そこ：距听话人近的场所
  - あそこ：距双方都远的场所
- **例句**：
  - ここは 教室です。（这里是教室。）
  - そこは 受付です。（那里是前台。）
  - あそこは 公園です。（那儿是公园。）
- **Bunpro 校验**：待验证

### 2. 名 は 名［場所］です

- **含义**：表示某人或某物存在于某个场所
- **接续**：名词 + は + 場所名词 + です
- **要点**：
  - 中文可译为"在"，但日语结构仍是名词谓语句：A は 場所名詞 です
  - 「トイレは ここです」（厕所在这儿）与「ここは トイレです」（这里是厕所）句型相同但重点不同
  - 也可以用于表示人的所在地
- **例句**：
  - 食堂は 3階です。（食堂在3楼。）
  - 田中さんは 事務所です。（田中先生在办公室。）
  - 郵便局は 駅の 隣です。（邮局在车站旁边。）
- **Bunpro 校验**：待验证

### 3. 名 は どこですか

- **含义**：用于询问人或物所在的场所
- **接续**：名词 + は + どこ + ですか
- **要点**：
  - どこ 的礼貌说法为 どちら
  - 回答时可用 ここ/そこ/あそこ/あちら 等
- **例句**：
  - お手洗いは どこですか。（洗手间在哪里？）
  - あなたの 学校は どこですか。（你的学校在哪里？）
- **Bunpro 校验**：待验证

### 4. 名 も 名 です

- **含义**：助词 も 表示"也"，用于追加同类信息
- **接续**：名词 + も + 名词 + です
- **要点**：
  - も 替换 は，表示与前文相同的情况
  - 基本相当于汉语的"也"
- **例句**：
  - ここは 銀行です。あそこも 銀行です。（这里是银行。那里也是银行。）
  - 李さんは 中国人です。張さんも 中国人です。（小李是中国人。小张也是中国人。）
- **Bunpro 校验**：待验证

### 5. 名 は 名 ですか、名 ですか（选择疑问句）

- **含义**：在多种可能中询问具体是哪一种
- **接续**：名词 + は + 名词A + ですか、名词B + ですか
- **要点**：
  - 重复谓语 〜ですか 来列出选项
  - 不能用 はい/いいえ 来回答，需直接回答具体选项
  - 选项可以是两个以上
- **例句**：
  - これは ペンですか、鉛筆ですか。（这是钢笔还是铅笔？）
  - 今日は 土曜日ですか、日曜日ですか。（今天是星期六还是星期天？）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| こちら/そちら/あちら/どちら | ここ/そこ/あそこ/どこ 的礼貌说法；原本是表示方向的词 |
| いくら | 询问价格时使用，如「これは いくらですか」 |
| 省略（谓语） | 进一步询问时可省略已说过的谓语部分，句尾读升调，如「あれは（↗）」 |
| あのう | 向对方搭话、引起注意时使用；也用于难以启齿的事情或新话题 |

- **要求**：
  - 5-7 句
  - 必须覆盖以下全部语法点：
    - ✅ ここ/そこ/あそこ
    - ✅ 名は名［場所］です
    - ✅ どこですか
    - ✅ も
    - ✅ 选择疑问句
