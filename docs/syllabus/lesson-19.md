## 语法点

### 1. 动词的「ない形」规则

- **含义**：动词否定的基础变形
- **接续**：
  - 一类动词：ます形去ます → 最后一音变「あ段」+ ない（い→わ+ない）
  - 二类动词：ます形去ます + ない
  - 三类动词：来ます（きます）→ 来ない（こない）、します→しない
- **要点**：
  - 一类动词「い」结尾时变「わ」而非「あ」：買います→買わない
  - ない形是后续多种句型的基础
  - ある的ない形是「ない」（特殊）
- **例句**：
  - 書きます→書かない、食べます→食べない、来ます→来ない
  - 今日は会社へ行かない。（今天不去公司。）
- **Bunpro 校验**：待验证

### 2. 動ないでください

- **含义**：请不要做……（否定请求）
- **接续**：动词ない形 + で + ください
- **要点**：
  - 是「～てください」的否定对应
  - 本质是"否定请求/请不要"，与规则性禁止的「てはいけません」语气不同
  - 语气比命令形柔和
- **例句**：
  - ここでは写真を撮らないでください。（请不要在这里拍照。）
  - 心配しないでください。（请别担心。）
- **Bunpro 校验**：待验证

### 3. 動なければなりません

- **含义**：必须做……（义务/必要性）
- **接续**：动词ない形去「ない」+ なければなりません
- **要点**：
  - 也可用口语形式「～ないといけません」
  - 口语中常缩成「～なきゃ」「～ないと」
  - 表示客观义务或必要性
- **例句**：
  - 明日までにレポートを書かなければなりません。（必须在明天之前写完报告。）
  - 毎日8時間寝なければなりません。（每天必须睡8个小时。）
- **Bunpro 校验**：待验证

### 4. 動なくてもいいです

- **含义**：不做……也可以（免除义务）
- **接续**：动词ない形去「ない」+ なくてもいいです
- **要点**：
  - 是「なければなりません」的语义否定
  - 用于告知对方某事不是必须的
  - 回答「～なければなりませんか」时常用
- **例句**：
  - 明日は早く起きなくてもいいです。（明天不用早起。）
  - この薬はもう飲まなくてもいいですか。（这个药不用再吃了吗？）
- **Bunpro 校验**：待验证

### 5. 名1が 名2です（强调主语）

- **含义**：（是）……才是……（用「が」标记被聚焦/被识别的项）
- **接续**：名词1 + が + 名词2 + です
- **要点**：
  - 「が」标记被聚焦的项，常用于回答「だれが」「どれが」等疑问
  - 例：だれが田中さんですか → わたしが田中です
  - 疑问词（何・だれ・どこ等）做主语时只能用「が」
  - 与「は」的区别不是简单的"新旧信息"公式，要结合语境理解
- **例句**：
  - あの方が山田先生です。（那位就是山田老师。）
  - だれが田中さんですか。——わたしが田中です。（谁是田中？——我就是田中。）
- **Bunpro 校验**：待验证

## 课文

> Arc 5「年末年始」第3话 — 滑雪场规则

**登場人物**：陳明、スタッフ — スキー場

> 使用语法：≤L19（+ない形、動ないでください、動なければなりません、動なくてもいいです、N1がN2です）

---

スタッフ：お客様、スキー場の ルールを 説明します。

陳：はい。

スタッフ：まず、ヘルメットを かぶらなければ なりません。

陳：分かりました。手袋も しなければ なりませんか。

スタッフ：はい、手袋も しなければ なりません。でも、ゴーグルは しなくても いいです。

陳：ここで 写真を 撮っても いいですか。

スタッフ：写真は いいですが、コースの 中で 止まらないで ください。

陳：分かりました。初心者コースは どれですか。

スタッフ：あの 赤い 旗が 初心者コースです。

陳：ありがとうございます。

---

**语法覆盖**：✅ ない形 ✅ ないでください（止まらないで） ✅ なければなりません（かぶらなければ/しなければ） ✅ なくてもいいです（しなくてもいいです） ✅ N1がN2です（赤い旗が初心者コース）

## 表达要点

| 要点 | 说明 |
|------|------|
| ～なければ vs ～ないと | 前者正式/书面，后者口语；口语中常省略为「～ないと」 |
| だいぶ | 表示程度相当高，强调变化的幅度："だいぶよくなりました" |
| 大丈夫です | 不是"大丈夫"，而是"没问题/可以"的意思 |
| 何度も | 「何+量词+も+肯定」表示数量多 |
