## 语法点

### 1. 一类形く なります

- **含义**：变得……（关注状态变化本身）
- **接续**：一类形容词词尾「い」→「く」+ なります
- **要点**：
  - 关注状态变化本身，不直接表达施事者
  - 可以是自然变化（暖かくなる）也可以是有意志的变化（医者になる）
  - 常与「だんだん」「もうすぐ」搭配
- **例句**：
  - 春になると、だんだん暖かくなります。（到了春天会渐渐暖和起来。）
  - 最近、日本語の勉強が楽しくなりました。（最近日语学习变得有趣了。）
- **Bunpro 校验**：待验证

### 2. 一类形く します

- **含义**：使……变得……（主观改变）
- **接续**：一类形容词词尾「い」→「く」+ します
- **要点**：
  - 表示人为地使事物发生变化
  - 主语有意志性
  - 他动词性质：宾语用「を」标记
- **例句**：
  - エアコンの温度を低くしてください。（请把空调温度调低。）
  - 旅行の荷物をできるだけ軽くします。（尽量减轻旅行的行李。）
- **Bunpro 校验**：待验证

### 3. 二类形/名に なります

- **含义**：变成……（性质/身份的变化）
- **接续**：二类形容词/名词 + に なります
- **要点**：
  - 二类形容词和名词接续相同
  - 可表示身份变化：医者になります
  - 可表示状态变化：元気になります
- **例句**：
  - 娘は来年、大学生になります。（女儿明年就是大学生了。）
  - 薬を飲んだら、元気になりました。（吃了药之后精神好了。）
- **Bunpro 校验**：待验证

### 4. 二类形/名に します

- **含义**：使……变成……/ 决定选……（两种用法）
- **接续**：二类形容词/名词 + に します
- **要点**：
  - 用法①「使变」：部屋をきれいにする = 使房间变干净（人为改变状态）
  - 用法②「选择」：コーヒーにする = 决定选咖啡（做出选择/决定）
  - 两种用法接续相同但语义不同，注意区分
- **例句**：
  - この部屋をきれいにしてください。（请把这个房间打扫干净。）
  - 飲み物はコーヒーにします。（饮料我要咖啡。）
- **Bunpro 校验**：待验证

### 5. 一类形/二类形 ほうが いいです

- **含义**：还是……比较好（建议/选择）
- **接续**：一类形容词 + ほうがいいです / 二类形容词 + な + ほうがいいです
- **要点**：
  - 用于比较后表示偏好或建议
  - 一类形直接加「ほうがいいです」
  - 二类形需加「な」再接「ほうがいいです」
- **例句**：
  - 部屋は広いほうがいいです。（房间还是宽敞点好。）
  - 公園は静かなほうがいいです。（公园还是安静点好。）
- **Bunpro 校验**：待验证

### 6. 自動詞と他動詞①

- **含义**：自动词与他动词的基本区别
- **接续**：自动词：名が＋自動詞 / 他动词：名は/が＋名を＋他動詞
- **要点**：
  - 自动词：不带直接宾语，关注主体自身发生的变化或状态
  - 他动词：带宾语（を），表示施事者作用于对象使其变化
  - 部分自他动词在词形上有对应：開く/開ける、始まる/始める
- **例句**：
  - 窓が開きました。（窗户开了。）——自动词
  - 森さんが窓を開けました。（森先生打开了窗户。）——他动词
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| 声と音 | 人/动物发出的声音用「声」，物体发出的声音用「音」 |
| さらに vs もっと | 都表示"更加"，前者郑重/书面，后者随便/口语 |
| 似合います | 「人に似合います」表示适合/相称，是自动词 |
| ～割引 | 日语"3割引"＝中文"七折"，用减掉的部分表示 |
