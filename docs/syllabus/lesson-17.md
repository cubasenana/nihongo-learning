## 语法点

### 1. 名が 欲しいです

- **含义**：想要某物（表示愿望）
- **接续**：名词1は 名词2が 欲しいです
- **要点**：
  - 愿望对象用「が」标记，不能用「を」
  - 陈述句主语默认为「わたし」，疑问句主语默认为「あなた」
  - 不能直接用于长辈或上级（失礼）
- **例句**：
  - 新しいパソコンが欲しいです。（想要一台新电脑。）
  - 今、何がいちばん欲しいですか。（你现在最想要什么？）
- **Bunpro 校验**：待验证

### 2. 名を/が 動たいです

- **含义**：想做某事（表示行为愿望）
- **接续**：动词ます形去掉「ます」+ たいです
- **要点**：
  - 对象助词可用「を」也可用「が」
  - 否定形式：～たくないです / ～たくありません
  - 与「欲しい」相同，直接询问对方愿望（特别是对上级/长辈）显得唐突，可用「どうですか」等委婉表达替代
- **例句**：
  - 週末、映画を見たいです。（周末想看电影。）
  - 今日はどこへも行きたくないです。（今天哪儿也不想去。）
- **Bunpro 校验**：待验证

### 3. 動ませんか

- **含义**：做……好吗？（礼貌邀请/提议）
- **接续**：动词ます形去掉「ます」+ ませんか
- **要点**：
  - 形式上是否定疑问，功能上是邀请
  - 比「～ましょう」更加礼貌
  - 对方可用「いいですね」等接受
- **例句**：
  - いっしょに昼ご飯を食べませんか。（一起吃午饭好吗？）
  - 今度の日曜日、公園へ行きませんか。（这周日去公园怎么样？）
- **Bunpro 校验**：待验证

### 4. 動ましょう

- **含义**：……吧（提议/表示意志）
- **接续**：动词ます形去掉「ます」+ ましょう
- **要点**：
  - 可表示提议共同做某事（いっしょに食べましょう）
  - 也可表示说话人自己的决意或引导（始めましょう = 开始吧）
  - 礼貌程度低于「～ませんか」
- **例句**：
  - そろそろ出かけましょう。（差不多该出发了吧。）
  - いいですね。行きましょう。（好啊，走吧。）
- **Bunpro 校验**：待验证

### 5. 疑問詞 + でも

- **含义**：无论……都……（全面肯定）
- **接续**：疑问词 + でも + 肯定句
- **要点**：
  - 肯定用「でも」：だれでも、何でも（なんでも）、いつでも、どこでも
  - 否定用「も」：だれも～ない、何も～ない
  - 注意「いつも」（经常）与「いつでも」（任何时候）的区别
- **例句**：
  - 何でもいいですよ。遠慮しないでください。（什么都行，别客气。）
  - このアプリはだれでも使うことができます。（这个应用任何人都能用。）
- **Bunpro 校验**：待验证

### 6. ね②（缓和语气）

- **含义**：舒缓语气，传达内心感受
- **接续**：句末 + ね
- **要点**：
  - 与第4课确认用的「ね」不同，此处是缓和语气并向听者求共鸣
  - 使语气柔和，预设对方会理解或同意
  - 常用于表达自己的希望、感想（真正的自言自语用「なあ/なぁ」）
- **例句**：
  - やっぱりお寿司が好きですね。（果然还是喜欢寿司啊。）
  - 来年、旅行に行きたいですね。（明年想去旅行啊。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| ぜひ | 表示强烈愿望，后接「～たい」「～てください」等 |
| ～中に（じゅうに/ちゅうに） | 表示在该期间结束之前完成，如「今年中に」 |
| どうですか | 对长辈/不太熟的人用「どうですか」代替直接问愿望 |

- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：

    - ✅ 動ませんか
    - ✅ 動ましょう
    - ✅ 疑問詞+でも
