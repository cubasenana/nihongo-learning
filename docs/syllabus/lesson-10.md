## 语法点

### 1. 名は 二类形 です / でした

- **含义**：用二类形容词描述事物的性质或状态（现在/过去）
- **接续**：名词 + は + 二类形容词 + です（现在）/ でした（过去）
- **要点**：
  - 多数二类形容词不以「い」结尾，但「きれい」「嫌い」虽以い结尾，仍属二类形容词
  - 现在肯定：二类形 + です
  - 过去肯定：二类形 + でした
- **例句**：
  - この町はとても静かです。（这个城市很安静。）
  - 昨日の公園はにぎやかでした。（昨天的公园很热闹。）
- **Bunpro 校验**：待验证

### 2. 二类形 では ありません / では ありませんでした

- **含义**：二类形容词的否定及过去否定形式
- **接续**：二类形容词 + では ありません（否定）/ では ありませんでした（过去否定）
- **要点**：
  - 口语中「では」常缩为「じゃ」
  - 否定：二类形 + では ありません / じゃ ありません
  - 过去否定：二类形 + では ありませんでした / じゃ ありませんでした
- **例句**：
  - あの店は便利ではありません。（那家店不方便。）
  - 先週は暇じゃありませんでした。（上周不空闲。）
- **Bunpro 校验**：待验证

### 3. 二类形な + 名

- **含义**：用二类形容词修饰名词
- **接续**：二类形容词 + な + 名词
- **要点**：
  - 不同于一类形容词直接修饰，二类形必须加「な」
  - 本课先掌握肯定修饰：二类形 + な + 名。否定修饰（「静かではない町」）以后再学
  - 注意：不能省略「な」
- **例句**：
  - 有名な大学に入りたいです。（想进有名的大学。）
  - きれいな花を買いました。（买了漂亮的花。）
- **Bunpro 校验**：待验证

### 4. 名词谓语の过去 / 否定 / 过去否定

- **含义**：名词做谓语时的过去、否定、过去否定形式
- **接续**：名词 + でした / ではありません / ではありませんでした
- **要点**：
  - 过去：名词 + でした
  - 否定：名词 + ではありません
  - 过去否定：名词 + ではありませんでした
  - 形式与二类形容词相同
- **例句**：
  - 昨日は休みでした。（昨天是休息日。）
  - おとといは晴れではありませんでした。（前天不是晴天。）
- **Bunpro 校验**：待验证

### 5. どんな + 名

- **含义**：询问人或事物的性质，"什么样的……"
- **接续**：どんな + 名词 + ですか
- **要点**：
  - 主要问性质、类型：どんな映画（什么样的/什么类型的电影）
  - 区分：どんな（性质/类型）vs 何の（内容/所属/材料）
  - 回答用形容词 + 名词的形式
- **例句**：
  - どんな映画が好きですか。（喜欢什么样的电影？）
  - どんな町ですか。——静かできれいな町です。（是什么样的城市？——安静又漂亮的城市。）
- **Bunpro 校验**：待验证

### 6. どうですか / どうでしたか

- **含义**：询问对方对某状态的意见或感想
- **接续**：名词 + は + どうですか（现在/未来）/ どうでしたか（过去）
- **要点**：
  - 询问现在或未来：どうですか
  - 询问过去：どうでしたか
  - 还可用于劝请对方做某事（如：お茶はどうですか）
- **例句**：
  - 日本の生活はどうですか。（日本的生活怎么样？）
  - 週末の旅行はどうでしたか。（周末的旅行怎么样？）
- **Bunpro 校验**：待验证

### 7. でも / そして

- **含义**：连接两个句子的连词——转折（でも）和并列（そして）
- **接续**：小句1。でも / そして、小句2。
- **要点**：
  - 「でも」表示转折，多用于口语，相当于"但是"
  - 「そして」表示并列/递进，相当于"而且、然后"
  - 两者都是连词，放在第二个句子开头
- **例句**：
  - あの店は安いです。でも、あまりおいしくないです。（那家店便宜。但是不太好吃。）
  - この部屋は広いです。そして、明るいです。（这个房间宽敞。而且很明亮。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| きれい是二类形容词 | 虽然以「い」结尾，但属于二类形；修饰名词要加「な」 |
| 天气用名词句表达 | 「雨です」「晴れです」「曇りです」而非动词句 |
| ところで | 转换话题时使用，相当于"对了""另外" |
| 形容词 + の | 「おもしろいの」中的「の」代替前文已提到的名词 |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：城市推荐指南**

- **场景**：一个外国朋友即将来日本旅游，问你推荐哪个城市。你用对比的方式介绍两个城市（如东京和京都），描述各自的特点。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ 二类形容词谓语（肯定/过去）
    - ✅ 二类形容词否定或过去否定
    - ✅ 二类形 + な + 名词
    - ✅ 名词谓语过去或否定
    - ✅ どんな + 名
    - ✅ どうですか / どうでしたか
    - ✅ でも 或 そして
- **词汇限制**：仅使用 lesson ≤ 10 的词汇

  2. 覆盖检查：是否全部语法点都用到了
