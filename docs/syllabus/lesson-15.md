## 语法点

### 1. 动ています①（正在进行）

- **含义**：表示动作正在进行中
- **接续**：动词て形 + います
- **要点**：
  - 表示说话时刻正在发生的动作
  - 常与「今」搭配使用
  - 否定：〜ていません
- **例句**：
  - 母は台所で料理を作っています。（妈妈正在厨房做饭。）
  - 今、何をしていますか。——本を読んでいます。（现在在做什么？——在看书。）
- **Bunpro 校验**：待验证

### 2. 动てもいいです（许可）

- **含义**：表示允许或许可，"可以做……"
- **接续**：动词て形 + も + いいです
- **要点**：
  - 疑问形式用于请求许可：〜てもいいですか
  - 肯定回答：ええ、いいですよ / どうぞ
  - 否定回答：いえ、ちょっと…… / すみません、〜から
  - 一般不用于对长辈表示许可（显得傲慢）
- **例句**：
  - ここで写真を撮ってもいいですか。——ええ、どうぞ。（这里可以拍照吗？——可以，请。）
  - 明日休んでもいいですか。（明天可以休息吗？）
- **Bunpro 校验**：待验证

### 3. 动てはいけません（禁止）

- **含义**：表示禁止，"不可以做……"
- **接续**：动词て形 + は + いけません
- **要点**：
  - 表示规则、法律或说话人的禁止
  - 对长辈一般不用，改用「〜ないでください」
  - 用于规则、告示、法律时不看对象身份：「ここで泳いではいけません」
  - 肯定回答（确认禁止）：はい、〜てはいけません
  - 否定回答（解除禁止）：いいえ、〜てもいいです
- **例句**：
  - 図書館で大きい声で話してはいけません。（图书馆里不能大声说话。）
  - ここでタバコを吸ってもいいですか。——はい、いいですよ。（这里可以吸烟吗？——可以。）
- **Bunpro 校验**：待验证

### 4. 名【附着点】に 动

- **含义**：表示人或物体停留、附着的位置
- **接续**：附着点名词 + に + 动词（乗ります/座ります/入れます等）
- **要点**：
  - 乗り物に乗ります（上交通工具）
  - いすに座ります（坐在椅子上）
  - かばんに入れます（放进包里）
  - 「乗る」的交通工具对象用「に」：× 電車を乗ります → ○ 電車に乗ります
  - 注意：移动动词的「を」另有用法（经过/离开，见第14课），不要混淆
- **例句**：
  - 毎朝バスに乗って会社へ行きます。（每天早上坐公交去公司。）
  - そのかばんに本を入れてください。（请把书放进那个包里。）
- **Bunpro 校验**：待验证

### 5. 名【目的地】に 动

- **含义**：表示移动行为的目的地（与「へ」通用）
- **接续**：目的地名词 + に + 行きます/来ます/帰ります
- **要点**：
  - 只限于行きます/来ます/帰ります等典型移动动词
  - 「に」和「へ」在表示目的地时可互换
  - 「に」的其他用法（存在、时间、对象、附着）不能换成「へ」
- **例句**：
  - 来週京都に行きます。（下周去京都。）
  - 昨日何時に家に帰りましたか。（昨天几点回的家？）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| 请求许可的回答 | 肯定用「どうぞ/いいですよ」；否定用「ちょっと……」婉拒 |
| 無理をしてはいけませんよ | 固定表达，劝告对方不要勉强 |
| ゆっくり | 除了"慢慢地"还有"好好地"的意思（ゆっくり休んでください） |
| お大事に | 对生病的人表示关心的寒暄语，"请保重" |

### Level 2：组装（30%）

### Level 3：产出（60%）— 本课核心

**📋 产出任务：新员工入职须知**

- **场景**：你是公司前辈，一位新员工第一天上班。你带他参观办公室，告诉他哪些事情可以做（茶水间喝咖啡、用会议室等），哪些事情不可以做（办公区吃东西、打私人电话等），并回答他的提问。
- **要求**：
  - 8-12 句
  - 必须覆盖以下全部语法点：
    - ✅ ています（正在进行）
    - ✅ てもいいです（许可）
    - ✅ てはいけません（禁止）
    - ✅ に（附着点）
    - ✅ に（目的地）
- **词汇限制**：仅使用 lesson ≤ 15 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
