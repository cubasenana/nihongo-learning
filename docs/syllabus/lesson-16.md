## 语法点

### 1. 一类形くて + 后续形容词/小句（并列）

- **含义**：用くて连接形容词，并列描述同一事物
- **接续**：一类形容词词尾「い」→「くて」+ 后续形容词/小句（可跨类型连接）
- **要点**：
  - 变形规则：い → くて（広い→広くて、新しい→新しくて）
  - 特殊：いい → よくて
  - 可连续使用多个：大きくて新しくて便利です
  - 也可修饰名词：軽くて暖かいコート
- **例句**：
  - この部屋は明るくて静かです。（这个房间又明亮又安静。）
  - 安くておいしいレストランを知っていますか。（你知道又便宜又好吃的餐厅吗？）
- **Bunpro 校验**：待验证

### 2. 二类形で 二类形（并列）

- **含义**：两个以上二类形容词并列描述同一事物
- **接续**：二类形容词 + で + 后续形容词
- **要点**：
  - 变形规则：二类形 + で（静か→静かで、きれい→きれいで）
  - 可以连接一类形和二类形混合使用
  - 修饰名词时最后一个形容词按正常规则接续
- **例句**：
  - この公園は静かできれいです。（这个公园安静又漂亮。）
  - 親切で元気な先生です。（是一位热情又有活力的老师。）
- **Bunpro 校验**：待验证

### 3. 名で 名（并列）

- **含义**：两个以上名词谓语并列说明同一主语
- **接续**：名词1 + で、+ 名词2 + です
- **要点**：
  - 用于列举同一主语的多个身份或属性
  - 形式上与二类形相同（名词 + で）
  - 不用于列举不同主语的事物
- **例句**：
  - 田中さんは先生で、サッカーのコーチです。（田中是老师，也是足球教练。）
  - あの建物は図書館で、市の施設です。（那栋建筑是图书馆，是市里的设施。）
- **Bunpro 校验**：待验证

### 4. 动ています②（结果状态）

- **含义**：表示动作结束后留下的结果状态
- **接续**：动词て形 + います
- **要点**：
  - 区别于「ています①」（正在进行），此处表示持续的结果状态
  - 常用动词：住んでいます、持っています、知っています、結婚しています
  - 「知っていますか」的否定回答是「知りません」（会话中通常不用「知っていません」）
- **例句**：
  - 東京に住んでいます。（住在东京。）
  - 車を持っていますか。——いいえ、持っていません。（有车吗？——没有。）
- **Bunpro 校验**：待验证

### 5. 小句が、小句（转折）

- **含义**：连接两个有转折关系的小句，"但是、可是"
- **接续**：小句1 + が、+ 小句2
- **要点**：
  - 「が」放在第一个小句末尾，表示转折
  - 「が」是连接小句的助词，「でも」是独立的接续词（放句首开始新句）
  - 「が」更正式/书面，口语中也常用
- **例句**：
  - この料理はおいしいですが、少し高いです。（这道菜好吃，但有点贵。）
  - 日本語を1年勉強しましたが、まだあまり上手ではありません。（学了一年日语，但还不太好。）
- **Bunpro 校验**：待验证

### 6. まだ + 动词否定

- **含义**：表示还未达到某种状态，"还没……"
- **接续**：まだ + 动词否定 / まだ + ～ていません
- **要点**：
  - 对「もう〜ましたか」的否定回答用「まだ〜ていません」
  - 也可与能力/可能否定搭配：まだ日本語があまりできません
  - 简略回答可只说「まだです」
- **例句**：
  - もう昼ごはんを食べましたか。——いいえ、まだ食べていません。（已经吃午饭了吗？——不，还没吃。）
  - まだ日本語があまりできません。（日语还不太会。）
- **Bunpro 校验**：待验证

## 课文

> Arc 4「生活技能」第4话 — 找房子

**登場人物**：陳明、不動産屋のスタッフ — 不動産屋にて

> 使用语法：≤L16（+一类形くて並列、二类形で並列、Nで並列、動ています②結果状態、が転折、まだ+否定）

---

陳：すみません、アパートを 探して います。

不動産屋：はい。どんな 部屋が いいですか。

陳：広くて 明るい 部屋が いいです。

不動産屋：駅から 近くて、静かで 便利な ところが ありますよ。

陳：家賃は いくらですか。

不動産屋：8万円です。部屋は 2階で、南向きです。

陳：エアコンは ついて いますか。

不動産屋：はい、エアコンは ついて います。冷蔵庫も ついて います。

陳：ペットは まだ 確認して いませんか。

不動産屋：すみません、まだ 確認して いません。

陳：分かりました。部屋は きれいですが、少し 狭いですね。

不動産屋：そうですね。でも、駅から 5分ですよ。

---

**语法覆盖**：
- ✅ 一类形くて並列（広くて明るい / 近くて）
- ✅ 二类形で並列（静かで便利な）
- ✅ Nで並列（2階で南向き）
- ✅ 動ています②結果状態（ついています）
- ✅ が転折（きれいですが狭い）
- ✅ まだ+～ていません（まだ確認していません）

## 表达要点

| 要点 | 说明 |
|------|------|
| 背が高くて髪が長くて… | 描述人物外貌时用形容词て形连接多个特征 |
| ～料/～費/～代 | 费用的三种后缀：入場料、交通費、電気代，用法不同 |
| ちゃんと | 副词，"好好地、正经地"，只用于口语 |
| 嫌いじゃありません | 双重否定表示"并非讨厌"＝含蓄地表示还可以 |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：找房子**

- **场景**：你正在日本找房子。向房产中介描述你的理想住所条件（安静、离车站近、房间明亮等），中介向你介绍一套房子的优缺点，你们讨论这套房子是否合适。
- **要求**：
  - 8-12 句
  - 必须覆盖以下全部语法点：
    - ✅ 一类形くて（并列）
    - ✅ 二类形で（并列）
    - ✅ 名で（并列）
    - ✅ ています②（结果状态）
    - ✅ が（转折）
    - ✅ まだ + 否定
- **词汇限制**：仅使用 lesson ≤ 16 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
