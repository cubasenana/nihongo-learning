## 语法点

### 1. 动词的「基本形」规则

- **含义**：动词的词典形/原形，是各种句型的基础
- **接续**：
  - 一类动词：ます形去ます → 最后一音变「う段」
  - 二类动词：ます形去ます + る
  - 三类动词：来ます→来る、します→する
- **要点**：
  - 基本形即词典中收录的形式
  - 一类动词的变化规律：い段→う段（書き→書く、飲み→飲む）
  - 是「ことができます」「ことです」「前に」等句型的前接形式
- **例句**：
  - 書きます→書く、食べます→食べる、来ます→来る
  - 明日、友達に会う。（明天见朋友。）
- **Bunpro 校验**：待验证

### 2. 動（基本形）ことができます

- **含义**：能够做……/会做……（能力/可能/条件允许）
- **接续**：动词基本形 + ことができます
- **要点**：
  - 表示能力：スミスさんはピアノを弾くことができます
  - 也可表示条件上可行/规则允许
  - 肯定简答「できます」、否定简答「できません」
- **例句**：
  - 日本語で手紙を書くことができます。（能用日语写信。）
  - この図書館では飲食することができません。（这个图书馆不能饮食。）
- **Bunpro 校验**：待验证

### 3. 名は 動（基本形）ことです

- **含义**：……是做……（用动作描述事物）
- **接续**：名词は + 动词基本形 + ことです
- **要点**：
  - 「動ること」整体作为名词使用
  - 常用于描述趣味、梦想、特技等
  - 也可用名词短语替代：車を運転すること＝車の運転
- **例句**：
  - わたしの趣味は音楽を聞くことです。（我的爱好是听音乐。）
  - 彼の夢は外国で働くことです。（他的梦想是在国外工作。）
- **Bunpro 校验**：待验证

### 4. 動（基本形）前に、～

- **含义**：在做……之前（时间先后）
- **接续**：动词基本形 + 前に / 名词 + の + 前に
- **要点**：
  - 「前に」前面的动词必须是基本形，即使后句是过去时也不变：日本へ来る前に勉強しました（× 来た前に）
  - 名词接续时加「の」：会議の前に
  - 对比「～た後で」：前に用基本形，後で用た形
- **例句**：
  - 寝る前に、歯を磨きます。（睡觉前刷牙。）
  - 日本へ来る前に、少し日本語を勉強しました。（来日本之前学了一点日语。）
- **Bunpro 校验**：待验证

### 5. 疑問詞 + か（不确定）

- **含义**：某……（表示不确定的事物）
- **接续**：疑问词 + か
- **要点**：
  - 何か（なにか/某物）、どこか（某处）、だれか（某人）、いつか（某时）
  - 不表示疑问，可用于陈述句
  - 助词顺序：どこかへ、だれかに（か 在格助词前）
  - 与第17课「疑問詞+でも」的区别：でも＝全部，か＝某一个不确定的
- **例句**：
  - 夏休み、どこかへ旅行に行きたいです。（暑假想去什么地方旅行。）
  - おなかがすきました。何か食べましょう。（肚子饿了，吃点什么吧。）
- **Bunpro 校验**：待验证

### 6. ～よね

- **含义**：……吧？（自己判断 + 求对方确认/认同）
- **接续**：句末 + よね
- **要点**：
  - 「よ」（我这样认为）+「ね」（你也同意吧？）的组合
  - 可用于求确认：10時からですよね（是10点开始吧？）
  - 也可用于求共鸣：おいしいですよね（很好吃对吧？）
- **例句**：
  - この店の料理、おいしいですよね。（这家店的菜很好吃吧？）
  - 明日の会議は10時からですよね。（明天的会议是10点开始吧？）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| 趣味 vs 興味 | 「趣味」＝爱好，「興味」＝兴趣；日语「趣味」不等于中文"趣味" |
| みんなで | 「で」表示动作主体样态：1人で、自分で、全員で |
| ごちそうします | 请别人吃饭的表达，不是单纯的"做菜" |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：春节聚会上的趣味分享**

- **场景**：春节期间，你和几个朋友在中华街聚餐。大家聊各自的爱好和特长，讨论聚会前做了什么准备，还商量下次聚会去哪儿。
- **要求**：
  - 10-12 句
  - 必须覆盖以下全部语法点：
    - ✅ 基本形（正确变形）
    - ✅ ～ことができます
    - ✅ ～ことです
    - ✅ ～前に
    - ✅ 疑問詞＋か
    - ✅ ～よね
- **词汇限制**：仅使用 lesson ≤ 20 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
