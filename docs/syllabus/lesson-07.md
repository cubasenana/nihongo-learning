## 语法点

### 1. 名を + 动（对象）

- **含义**：助词 を 表示动作的直接对象
- **接续**：名词 + を + 他动词
- **要点**：
  - を 读作 お
  - 用于各种他动词：飲みます、食べます、読みます、買います 等
  - 提问用「何を〜ますか」
- **例句**：
  - 毎朝 コーヒーを 飲みます。（每天早上喝咖啡。）
  - 新聞を 読みます。（看报纸。）
  - 何を 食べますか。（吃什么？）
- **Bunpro 校验**：待验证

### 2. 名[場所]で + 动（动作场所）

- **含义**：助词 で 表示动作发生的场所
- **接续**：場所 + で + 动词
- **要点**：
  - 注意区分：存在场所用 に，动作场所用 で
  - 提问用「どこで〜ますか」
  - 这里的 で 与第6课的交通手段 で 是同一助词的不同用法
- **例句**：
  - 図書館で 勉強します。（在图书馆学习。）
  - コンビニで お弁当を 買います。（在便利店买盒饭。）
  - どこで 昼ご飯を 食べますか。（在哪里吃午饭？）
- **Bunpro 校验**：待验证

### 3. 名か 名（选择 or）

- **含义**：在两个以上的名词间表示"或者"
- **接续**：名词A + か + 名词B
- **要点**：
  - 相当于汉语的"或者"
  - 可以用于两个以上的选项
  - 与第3课的选择疑问句不同：这里是名词层面的选择
- **例句**：
  - パンか ご飯を 食べます。（吃面包或者米饭。）
  - 休みは 土曜日か 日曜日です。（休息日是周六或者周日。）
  - テニスか ジョギングを します。（打网球或者慢跑。）
- **Bunpro 校验**：待验证

### 4. 名を ください

- **含义**：买东西、点菜或向人索取某物时使用
- **接续**：名词 + を + ください
- **要点**：
  - 可以用 と 连接多个物品
  - 语气直接，适用于商店、餐厅等场合
  - 也可用于索取免费物品（申请表等）
- **例句**：
  - コーヒーを ください。（请给我咖啡。）
  - この 本と あの 雑誌を ください。（请给我这本书和那本杂志。）
  - 水を ください。（请给我水。）
- **Bunpro 校验**：待验证

## 课文

> Arc 2「日常节奏」第3话 — 在餐厅点餐

**登場人物**：陳明、田中、店員 — 昼休み

> 使用语法：≤L07（+Nを+動、N[場所]で+動、NかN、Nをください）

---

陳：田中さん、昼ごはんは どこで 食べますか。

田中：あの レストランで 食べます。

陳：何を 食べますか。

田中：今日は 魚を 食べます。陳さんは？

陳：私は 肉か 魚を 食べます。

（レストランで）

店員：肉ですか、魚ですか。

田中：魚定食を ください。

陳：肉定食を ください。コーヒーも ください。

田中：お茶を ください。

陳：田中さんは 毎日 ここで 食べますか。

田中：いいえ。水曜日と 金曜日に ここで 食べます。

陳：そうですか。私は 毎日 コンビニで お弁当を 買います。

田中：私も コンビニで 買います。

---

**语法覆盖**：
- ✅ Nを+動（魚を食べます / お弁当を買います）
- ✅ N[場所]で+動（レストランで食べます / ここで食べます / コンビニで買います）
- ✅ NかN（肉か魚）
- ✅ Nをください（魚定食をください / お茶をください / コーヒーもください）

## 表达要点

| 要点 | 说明 |
|------|------|
| 何 的读法 | なに：后续 を/が/と 时；なん：后续 の/量词/时间词时；で 前两种都可 |
| そうですか（↘） | 降调表示理解了新信息（"是吗"）；升调表示对内容持疑义 |
| じゃあ / では | 接过话题继续往下说，"那么"的意思；正式场合用 では |
| すみません | 除了"对不起"，还可用于搭话、请人帮忙，相当于"劳驾、请问" |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：在餐厅点餐**

- **场景**：周日中午你和朋友一起去附近的一家餐厅吃午饭。你们看菜单、讨论吃什么，然后向店员点餐。之后你还聊起了自己平时的饮食习惯。
- **要求**：
  - 6-8 句
  - 必须覆盖以下全部语法点：
    - ✅ を（对象）
    - ✅ で（场所）
    - ✅ か（or）
    - ✅ をください
- **词汇限制**：仅使用 lesson ≤ 7 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
