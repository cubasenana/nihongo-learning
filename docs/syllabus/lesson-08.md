## 语法点

### 1. 名[工具/手段]で + 动

- **含义**：助词 で 表示动作使用的工具、手段或原材料
- **接续**：工具/手段/材料 + で + 动词
- **要点**：
  - 是第6课交通手段 で 的扩展用法
  - 可表示语言（日本語で）、工具（ペンで）、材料（紙で）
  - 提问用「何で（なにで）〜ますか」；注意口语中「なんで」容易被误解为"为什么"
- **例句**：
  - 日本語で 手紙を 書きます。（用日语写信。）
  - はさみで 紙を 切ります。（用剪刀剪纸。）
  - 何で 送りますか。（用什么方式寄？）
- **Bunpro 校验**：待验证

### 2. 名1[人]は 名2[人]に 名3[物]を あげます

- **含义**：表示"给予"，物品从己方向对方/第三者移动
- **接续**：给予者は + 接受者に + 物を + あげます
- **要点**：
  - 物品移动方向：第一人称→第二/三人称，或第三→第三
  - 接受者用助词 に 标记
  - **あげます 的接受者不能是说话人自己**：× 先生は私にプレゼントをあげました → ○ 先生が私にプレゼントをくれました
  - 说话人的亲属按说话人立场处理
  - 不宜对人直接说「あげます」，改用「どうぞ」更礼貌
- **例句**：
  - わたしは 友達に プレゼントを あげます。（我送给朋友礼物。）
  - 母は 先生に お菓子を あげました。（妈妈送给老师了点心。）
  - だれに あげますか。（给谁？）
- **Bunpro 校验**：待验证

### 3. 名1[人]は 名2[人]に/から 名3[物]を もらいます

- **含义**：表示"得到/收到"，物品从对方向己方移动
- **接续**：接受者は + 给予者に/から + 物を + もらいます
- **要点**：
  - 物品移动方向：第三/二人称→第一人称，或第三→第三
  - 给予者用 に 或 から；若给予者是组织/团体则多用 から
  - 与 あげます 方向相反
- **例句**：
  - わたしは 先生に 辞書を もらいました。（我从老师那里得到了词典。）
  - 父は 会社から 記念品を もらいました。（父亲从公司得到了纪念品。）
  - だれに もらいましたか。（从谁那里得到的？）
- **Bunpro 校验**：待验证

### 4. 名[人]に 会います

- **含义**：表示见面，所见对象用 に 标记
- **接续**：人 + に + 会います
- **要点**：
  - 会います 的对象助词是 に（不是 を）
  - 可以和场所 で 组合使用
- **例句**：
  - 明日 田中さんに 会います。（明天见田中先生。）
  - 駅で 友達に 会いました。（在车站碰到了朋友。）
  - だれに 会いましたか。（见了谁？）
- **Bunpro 校验**：待验证

### 5. よ（提醒）

- **含义**：句尾助词，提醒对方注意其不知道的事情
- **接续**：句子 + よ
- **要点**：
  - 用于告知、提醒、强调对方不知道的新信息
  - 语调随语气变化，不固定为升调
  - 根据上下文可表示提醒、警告或柔和的告知
- **例句**：
  - もう 帰りましたよ。（已经回去了哦。）
  - 明日は 休みですよ。（明天休息哦。）
  - 遅刻しますよ。（要迟到了哦。）
- **Bunpro 校验**：待验证

### 6. もう（已经）

- **含义**：表示动作已经完了，或不再做某事
- **接续**：もう + 过去肯定（已经…） / もう + 否定（不再…）
- **要点**：
  - もう + ました：已经做了（本课重点）
  - もう + ません：不再做了（了解即可）
  - 回答时「ええ、もう〜ました」表示已完成
- **例句**：
  - もう 食べましたか。（已经吃了吗？）
  - ええ、もう 食べました。（是的，已经吃了。）
  - もう 送りましたよ。（已经寄了哦。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| さっき / たった今 | 都表示刚才；たった今＝刚刚（离现在非常近），さっき＝稍早前；后续必须用过去形 |
| 〜の 件 | 比较郑重地提到某件事情时使用，多用于正式场合、传真或邮件中 |
| お願いします | 请求对方做某事时用「名词＋（を）お願いします」 |
| 分かりました | 除了表示理解，还可表示对对方的话表示承诺或应答（"好的"） |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：生日礼物**

- **场景**：下周是朋友的生日。你和另一位朋友商量送什么礼物，聊到最近谁送了你什么、你又打算送别人什么。你还提到昨天偶然在书店遇到了那位过生日的朋友。
- **要求**：
  - 6-8 句
  - 必须覆盖以下全部语法点：
    - ✅ で（手段）
    - ✅ あげます
    - ✅ もらいます
    - ✅ に会います
    - ✅ よ
    - ✅ もう
- **词汇限制**：仅使用 lesson ≤ 8 的词汇

  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
