## 语法点

### 1. あります 和 います

- **含义**：表示事物的存在
- **接续**：名词 + が + あります / います
- **要点**：
  - あります：无生命物、植物、抽象事物
  - います：人、动物等有生命/会动的存在
  - 两者不可互换
- **例句**：
  - 机の 上に 本が あります。（桌上有书。）
  - 庭に 猫が います。（院子里有猫。）
- **Bunpro 校验**：待验证

### 2. 場所に 名が あります / います（存在句）

- **含义**：表示"某地方有某物/某人"
- **接续**：場所 + に + 名词 + が + あります / います
- **要点**：
  - 助词 に 表示存在的场所
  - 助词 が 标记存在的主体
  - 提问时用「何が ありますか」或「だれが いますか」
- **例句**：
  - 教室に 学生が います。（教室里有学生。）
  - 冷蔵庫に 牛乳が あります。（冰箱里有牛奶。）
  - 公園に 何が ありますか。（公园里有什么？）
- **Bunpro 校验**：待验证

### 3. 名は 場所に あります / います（位置句）

- **含义**：表示"某物/某人在某地方"
- **接续**：名词 + は + 場所 + に + あります / います
- **要点**：
  - は 标记话题（已知信息），に 标记所在地
  - 与存在句的区别：话题不同，重点不同
  - 提问时用「どこに ありますか / いますか」
- **例句**：
  - 銀行は 駅の 前に あります。（银行在车站前面。）
  - 田中さんは 図書館に います。（田中先生在图书馆。）
  - 猫は どこに いますか。（猫在哪里？）
- **Bunpro 校验**：待验证

### 4. 名1 と 名2

- **含义**：助词 と 连接两个名词，表示并列
- **接续**：名词1 + と + 名词2
- **要点**：
  - 相当于汉语的"和"
  - 只连接名词，不用于连接句子
- **例句**：
  - 机と 椅子が あります。（有桌子和椅子。）
  - ペンと ノートを ください。（请给我笔和笔记本。）
- **Bunpro 校验**：待验证

### 5. 方位词（上/下/前/後ろ/隣/中/外）

- **含义**：表示事物的具体位置关系
- **接续**：名词 + の + 上/下/前/後ろ/隣/中/外
- **要点**：
  - の 不可省略（汉语说"桌子上"，日语必须说「机の 上」）
  - 日语的 上 只表示垂直上方，范围比汉语的"上"窄
  - 墙壁表面不说「壁の 上」而直接说「壁に」
- **例句**：
  - テレビの 下に 猫が います。（电视下面有猫。）
  - 学校の 隣に コンビニが あります。（学校旁边有便利店。）
  - 箱の 中に 何が ありますか。（箱子里面有什么？）
- **Bunpro 校验**：待验证

### 6. ね（确认）

- **含义**：句尾助词，用于向听话人征求同意或确认
- **接续**：句子 + ね
- **要点**：
  - 确认时常读升调；表示共鸣、感叹时语调可不同
  - 说话人就某事征求听话人同意时使用
- **例句**：
  - あそこに 犬が いますね。（那儿有只狗吧。）
  - この 本は 森さんのですね。（这本书是森先生的吧。）
- **Bunpro 校验**：待验证

### 7. 疑问词 + も + 否定

- **含义**：表示全面否定
- **接续**：だれ/何 + も + 否定形；どこ + にも + 否定形
- **要点**：
  - 何も ありません：什么也没有
  - だれも いません：谁也没有
  - どこにも ありません / いません：哪里都没有
  - 其他助词也保留后再加 も，如 だれにも、どこへも
  - 全面否定时，助词 が 变为 も
- **例句**：
  - 部屋に だれも いません。（房间里谁也没有。）
  - 冷蔵庫に 何も ありません。（冰箱里什么也没有。）
- **Bunpro 校验**：待验证

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。确认语法规则是否记住。

### Level 2：组装（30%）

给词造句，3-5 题。主动运用语法。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：描述你的房间**

- **场景**：你刚搬进新宿舍，朋友打电话来问你房间里有什么。你向朋友描述房间里各种物品的位置，也提到了自己养的宠物在哪里。
- **要求**：
  - 6-8 句
  - 必须覆盖以下全部语法点：
    - ✅ あります/います
    - ✅ 場所に名がある/いる
    - ✅ 名は場所にある/いる
    - ✅ と（并列）
    - ✅ 方位词
    - ✅ ね
    - ✅ 疑问词+も+否定
- **词汇限制**：仅使用 lesson ≤ 4 的词汇
- **AI 验收标准**：
  1. 语法检查：语法点是否正确使用
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：对话是否自然，指出不自然的地方
