## 语法点

### 1. 今 〜時 〜分です（时间表达）

- **含义**：表示现在的时间
- **接续**：今 + 数字 + 時 +（数字 + 分）+ です
- **要点**：
  - 问时间用「今 何時ですか」
  - 30分 也可以说成 半
  - 可以加 午前/午後 来区分上下午
- **例句**：
  - 今 9時です。（现在9点。）
  - 今 午後 3時半です。（现在下午3点半。）
  - 今 何時ですか。（现在几点？）
- **Bunpro 校验**：待验证

### 2. 〜ます / 〜ません / 〜ました / 〜ませんでした

- **含义**：动词 ます 形的四种时态变化
- **接续**：动词词干 + ます / ません / ました / ませんでした
- **要点**：
  - 〜ます：现在/将来的肯定
  - 〜ません：现在/将来的否定
  - 〜ました：过去的肯定
  - 〜ませんでした：过去的否定
  - 与中文不同，日语句末谓语表达过去事件时必须用过去形（不能用非过去形代替）
- **例句**：
  - 毎日 働きます。（每天工作。）
  - 昨日 休みました。（昨天休息了。）
  - 明日 働きません。（明天不工作。）
  - 先週 休みませんでした。（上周没有休息。）
- **Bunpro 校验**：待验证

### 3. 名[时间]に + 动

- **含义**：表示动作发生的具体时间点
- **接续**：时间词 + に + 动词
- **要点**：
  - 含数字的时间加 に：3時に、9月に、2日に
  - 今日/昨日/明日/毎日/去年/来年 等不加 に
  - 曜日（月曜日、火曜日等）后一般加 に，但也可以不加
- **例句**：
  - 7時に 起きます。（7点起床。）
  - 11時に 寝ます。（11点睡觉。）
  - 明日 休みます。（明天休息。）
- **Bunpro 校验**：待验证

### 4. 名[时间]から 名[时间]まで + 动

- **含义**：表示动作发生的时间范围（从...到...）
- **接续**：时间 + から + 时间 + まで + 动词
- **要点**：
  - から 表示起点，まで 表示终点
  - から 和 まで 可以单独使用
  - 也可用于场所（第6课进一步学习）
- **例句**：
  - 9時から 5時まで 働きます。（从9点工作到5点。）
  - 月曜日から 金曜日まで 勉強します。（从周一学到周五。）
  - 8時から 始まります。（从8点开始。）
- **Bunpro 校验**：待验证

### 5. いつ 〜ますか

- **含义**：询问动作或事情在什么时候发生
- **接续**：いつ + 动词ますか / 名词はいつですか / いつから〜ですか / いつまで〜ですか
- **要点**：
  - いつ：什么时候
  - 可以与 から/まで 组合使用
  - 回答可简答「〜です」，也可完整回答动词句
- **例句**：
  - 試験は いつですか。（考试什么时候？）
  - 仕事は 何時に 終わりますか。（工作几点结束？）
  - 休みは いつからですか。（假期从什么时候开始？）
- **Bunpro 校验**：待验证

### 6. は（对比用法）

- **含义**：は 除了表示主题外，还可以表示对比
- **接续**：名词 + は（暗含"与其他不同"的语感）
- **要点**：
  - 在对比语境中，「今日は 休みます」可暗示"今天休息，其他时候不同"
  - 「いつもは 7時ごろです」可暗示"今天和平时不一样"
  - 同一句中可出现两个 は（主题は + 对比は）
- **例句**：
  - 今日は 休みます。（今天休息。——暗示平时不休息）
  - いつもは 7時に 起きます。（平时7点起床。——暗示今天不同）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| 〜ごろ | 接在时间词后表示"大约"，如「7時ごろ」；后面一般不加 に |
| 〜です 的简答 | 已明确的内容可简洁回答，如问几点起床答「7時ごろです」 |
| 毎＋时间 | 前面的名词带「毎」时中间不加 の：「毎日 7時に」而非「毎日の 7時に」 |
| 称呼上司 | 在公司称呼上司可直接用职务名如「課長」，或姓＋职务如「吉田課長」 |

### Level 3：产出（60%）— 本课核心

**📋 产出任务：我的一天**

- **场景**：语言交换伙伴问你平时和周末的日程安排有什么不同。你描述自己工作日和休息日典型的一天，包括起床、上课/工作、休息的时间。
- **要求**：
  - 6-8 句
  - 必须覆盖以下全部语法点：
    - ✅ 时间表达
    - ✅ ます形四种时态
    - ✅ 时间+に
    - ✅ から〜まで
    - ✅ いつ
    - ✅ は（对比）
- **词汇限制**：仅使用 lesson ≤ 5 的词汇

  2. 覆盖检查：是否全部语法点都用到了
