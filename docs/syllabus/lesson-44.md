## 语法点

### 1. 小句（简体形）ようです①（推测）

- **含义**：好像……（根据说话人感知的情况进行推测）
- **接续**：
  - 动词/一类形容词简体形 + ようです
  - 二类形容词：～な + ようです
  - 名词：～の + ようです
- **要点**：
  - 基于说话人自身的观察、感知做出的推测
  - 可以对被观察者本人直接说（「風邪を引いたようですね」）
  - 书面语和口语都可使用
  - 回应对方的推测时可以用「そのようですね」
- **例句**：
  - 玄関のところにだれかいるようです。（门厅那儿好像有人。）
  - 店の前に行列ができています。あの店の料理はおいしいようです。（那家饭店前排着长队，看来菜不错。）
  - 遠くから何か飛んできます。飛行機のようですね。（远处飞来什么东西，好像是飞机。）
- **Bunpro 校验**：待验证

### 2. 小句（简体形）みたいです①（口语推测）

- **含义**：好像……（与ようです基本相同，但更口语化）
- **接续**：
  - 动词/一类形容词简体形 + みたいです
  - 二类形容词/名词：去「だ」+ みたいです
- **要点**：
  - 比「ようです」更随便，不用于正式书面语；聊天、口语化文章中可用
  - 二类形容词和名词直接去掉「だ」接续：雨みたいです
  - 使用场景比「ようです」更日常
- **例句**：
  - 小野さんは森さんが好きみたいです。（小野女士好像喜欢森先生。）
  - 雨の音が聞こえません。雨がやんだみたいです。（听不见雨声了，好像雨停了。）
  - 雲が広がっています。明日は雨みたいですよ。（云层在蔓延，明天好像要下雨。）
- **Bunpro 校验**：待验证

### 3. 小句（简体形）らしいです（观察推测/传闻）

- **含义**：好像……/据说……（基于观察或传闻信息做出推测）
- **接续**：
  - 动词/一类形容词简体形 + らしいです
  - 二类形容词/名词：去「だ」+ らしいです
- **要点**：
  - 既可表示根据观察推测，也可用于婉转转述听来的信息
  - 通常不用来直接判断眼前对方的状态（会显得像听别人说来的，不自然）
  - 站在第三者角度客观叙述时，「ようです」和「らしいです」都可以用
  - 二类形容词/名词去「だ」直接接续
- **例句**：
  - 今度の社員旅行は韓国へ行くらしいです。（下次全社旅行好像是去韩国。）
  - 電気が消えています。李さんは出かけたらしいです。（灯关着，小李大概是出去了。）
  - パトカーのサイレンが聞こえます。近所で事故があったらしいです。（听见了警车的声音，附近好像出了事故。）
- **Bunpro 校验**：待验证

### 4. ～過ぎます

- **含义**：过于……/太……（动作或状态超过正常的量或程度）
- **接续**：
  - 动词：ます形去「ます」+ 過ぎます
  - 一类形容词：去「い」+ 過ぎます
  - 二类形容词：词干 + 過ぎます
- **要点**：
  - 「過ぎます」本身的活用与二类动词相同
  - 隐含"超过合适程度"的负面评价；口语中也可正向夸张（楽しすぎる、かわいすぎる）
  - 可用て形连接后续句：食べ過ぎて、太りました
- **例句**：
  - 昼ご飯を食べ過ぎました。（午饭吃多了。）
  - このスープは熱過ぎませんか。（你不觉得这个汤太烫了吗？）
  - この説明は複雑過ぎます。（这说明太复杂了。）
- **Bunpro 校验**：待验证

### 5. 一类形容词＋さ

- **含义**：将形容词转化为表示某种状态或程度的名词
- **接续**：一类形容词去「い」+ さ
- **要点**：
  - 构成的名词表示该性质的程度或状态
  - 部分构成表示量化概念的名词：高さ（高度）、長さ（长度）、重さ（重量）
  - 二类形容词也可以用「さ」（きれいさ、静かさ等），但不如一类形容词普遍
- **例句**：
  - この暑さはいつまで続くんでしょう。（这种热什么时候才到头呢？）
  - 富士山の美しさが心に残っています。（富士山的美留在了心底。）
  - 仕事のおもしろさがやっと分かりました。（终于体会到了工作的趣味。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| おかげさまで | 对对方的关心表示感谢："托您的福" |
| けっこう | 程度超出预期，可用于量化程度（比「なかなか」范围更广） |
| 何と言っても | 从多种因素中着重选择一种："首先是……/不管怎么说……" |
| 控えめ | 分量少、节制；也形容态度拘谨谦虚 |

## 练习框架

### Level 1：热身（10%）

判断正误 / 选择题，3-5 题。区分ようです/みたいです/らしいです的用法差异，确认過ぎます和～さ的接续。

### Level 2：组装（30%）

给词造句，3-5 题。根据给定线索选择合适的推测表达。

### Level 3：产出（60%）— 本课核心

**📋 产出任务：新产品上市后的市场反馈分析**

- **场景**：你们公司最近推出了一款新饮料。你和同事在办公室讨论市场反馈：根据销售数据推测销路（用ようです/らしいです），谈论消费者的口碑（用みたいです），分析产品的优缺点（用過ぎます指出不足），讨论产品名称的重要性（用～さ）。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ ～ようです（推测）
    - ✅ ～みたいです（口语推测）
    - ✅ ～らしいです（观察推测/传闻）
    - ✅ ～過ぎます
    - ✅ 一类形 + さ
- **词汇限制**：仅使用 lesson ≤ 44 的词汇
- **AI 验收标准**：
  1. 语法检查：三种推测表达的接续和使用场景是否正确
  2. 覆盖检查：是否全部语法点都用到了
  3. 自然度评分：市场讨论的语境是否自然，推测表达的选择是否得当
