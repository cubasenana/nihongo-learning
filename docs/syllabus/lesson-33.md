## 语法点

### 1. 自动词和他动词②（词形对应关系）

- **含义**：日语中大量自他动词在词形上存在系统对应，掌握规律可批量记忆
- **接续**：自动词（～aru/～eru/～ru）⇔ 他动词（～eru/～su）
- **要点**：
  - 主要对应类型一：自动词词尾「aru」↔ 他动词词尾「eru」（如 閉まる↔閉める、止まる↔止める）
  - 主要对应类型二：自动词词尾「eru/ru」↔ 他动词词尾「su」（如 消える↔消す、壊れる↔壊す）
  - 不规则对应也存在：割れる↔割る、付く↔付ける
  - 自动词的发生主体常用「が」标记，他动词的直接宾语用「を」标记
- **例句**：
  - 風でドアが開きました。（门被风吹开了。）
  - 田中さんがドアを開けました。（田中先生打开了门。）
  - 電気が消えました。（灯灭了。）
- **Bunpro 校验**：待验证

### 2. 自动词＋ています④（结果状态）

- **含义**：有对应他动词的自动词接「ています」时，表示变化完成后的结果状态
- **接续**：自动词て形 + います
- **要点**：
  - 与「自动词た形」的区别：た形偏重瞬间变化的完成，ています偏重变化后持续的状态
  - 例：電気が消えました（灯灭了＝变化）vs 電気が消えています（灯灭着＝状态）
  - 本课聚焦有对应他动词的自动词（壊れる、閉まる、開く等）；其他变化/瞬间性自动词（座る、死ぬ等）也可表结果状态
- **例句**：
  - 壁に大きなポスターが掛かっています。（墙上挂着一张大海报。）
  - 窓が閉まっています。外は雨ですから。（窗户关着呢，外面在下雨。）
  - この椅子は壊れていますから、座らないでください。（这把椅子坏了，请不要坐。）
- **Bunpro 校验**：待验证

### 3. 動て形＋しまいます

- **含义**：表示动作结果令人遗憾/不愉快，或单纯表示动作的彻底完成
- **接续**：动词て形 + しまいます
- **要点**：
  - 核心用法：结果不如意/遗憾（うっかり忘れてしまいました）
  - 次要用法：动作彻底完成，无遗憾含义（もう読んでしまいました）
  - 口语缩略形：「～ちゃった」（～てしまった）「～じゃった」（～でしまった）
- **例句**：
  - 大事な資料をうっかりなくしてしまいました。（不小心把重要资料弄丢了。）
  - 無理をすると体を壊してしまいますよ。（太勉强的话会搞坏身体的。）
  - この映画はもう全部見てしまいました。（这部电影已经全部看完了。）
- **Bunpro 校验**：待验证

### 4. ～そうです②（样态推测）

- **含义**：根据外观或迹象进行主观推测——"看起来……""好像要……"
- **接续**：
  - 动词ます形去「ます」+ そうです
  - 一类形容词去「い」+ そうです
  - 二类形容词 + そうです
- **要点**：
  - 特殊形式：ない→なさそうです、いい→よさそうです
  - 否定（形容词）：～くなさそうです / ～そうではありません；否定（动词）：～そうにありません / ～そうにないです
  - 修饰名词用「～そうな＋名词」，修饰动词用「～そうに＋动词」
  - 不要与第32课的「～そうです」（传闻）混淆：传闻接简体形，样态接词干
- **例句**：
  - この料理はとてもおいしそうですね。（这道菜看起来很好吃啊。）
  - 空が暗くなってきました。雨が降りそうです。（天暗下来了，好像要下雨。）
  - 今日の会議はなさそうです。（今天好像没有会议。）
- **Bunpro 校验**：待验证

## 课文

> Arc 9「接待朋友·上」第1话 — 机场接人

**登場人物**：陳明、田中 — 上海の空港

> 使用语法：≤L33（+自他動詞②対応、自動詞ています④結果状態、てしまいます、そうです②様態）

---

陳：田中さん、お久しぶりです。お元気そうで、何よりです。

田中：ありがとうございます。飛行機が 遅れて しまいました。すみません。

陳：大丈夫ですよ。あ、出口の ドアが 閉まって いますね。

田中：本当ですね。隣の ドアを 開けましょうか。

陳：はい、お願いします。あ、開いて いますよ。

陳：タクシーで ホテルへ 行きましょう。荷物が 重そうですね。

田中：ええ、お土産を たくさん 入れて しまいましたから。

陳：あの ビルの 電気が 全部 消えて いますね。もう 夜 遅いですから。

田中：上海の 夜景は きれいですね。

---

**语法覆盖**：✅ 自他對應（閉まっている↔開けましょうか / 消えている） ✅ ています④（閉まっています/開いています/消えています） ✅ てしまいます（遅れてしまった/入れてしまった） ✅ そうです②（元気そう/重そう）

## 表达要点

| 要点 | 说明 |
|------|------|
| ～かしら | 偏柔和/老派的自问表达，与「～かな」意思相近；现代教学不按性别硬分 |
| それにしても | 虽然承认前提，但仍感觉程度超出预期："话虽如此，但……" |
| お久しぶりです | 久别重逢的寒暄语，回答可用较随意的「お久しぶり」 |
| お元気そうで、何よりです | 对熟人久别后使用的寒暄语："看您精神不错，这比什么都好" |
