# L37-L40 课文与生词表审核报告

审核对象：`docs/syllabus/lesson-37.md` 到 `docs/syllabus/lesson-40.md` 的新增 `## 课文` 与 `## 生词表`。

审核基准：

- 语法解锁：`docs/syllabus/grammar-unlock.md`
- 只检查 `lesson <= X` 的语法是否越界
- 对照每课 `## 语法点` 下的 `###` 标题检查覆盖
- 同时检查日语自然度、叙事连续性、生词表 `📖` 标记、表格列数、词源一致性

## 总结

本批主要问题集中在 L37、L39、L40：

- L37 场景写“长城入口”，但对话还在问“长城怎么去”，叙事场景矛盾。
- L37 `上からの景色` 实际用了 L45 `助詞 + の`。
- L38 人物设定不自然：田中说“日本に来てから”，陈明说每天听“中国語のニュース”，都和人物设定冲突。
- L39 `建てられた` 提前用了 L41 被动。
- L39 `ないで` 伴随状态和 `ていきます` 没有真正覆盖。
- L40 `ているところです` 阶段未覆盖；`笑い出す` 只在舞台说明中出现，覆盖偏弱。

Markdown 表格列数检查通过，L37-L40 没有发现表格列数不一致。

## L37

### ❌ 叙事/场景矛盾：人在长城入口，却问长城怎么去

位置：`lesson-37.md:89`, `lesson-37.md:95`

原文：

```text
**登場人物**：陳明、田中 — 長城の入口

田中：長城は どうやって 行けば いいですか。
```

问题：

课文场景已经是 `長城の入口`，田中再问“长城怎么去”不合逻辑。前一句还提到 `天安門へ行くなら`，也和当前长城入口场景跳脱。

建议二选一：

1. 把场景改成“ホテル / 北京市内 / 天安門附近”，保留问路内容。
2. 保留“長城の入口”，把问题改成入口内的路线，例如：

```text
田中：上まで どうやって 行けば いいですか。
陳：歩いて 行けば、30分ぐらいです。
```

### ❌ 语法越界：`上からの景色`

位置：`lesson-37.md:111`

原文：

```text
田中：天気が よければ、上から の 景色が きれいでしょうね。
```

问题：

`上からの景色` 是 `助詞 + の + 名詞`，根据 `grammar-unlock.md` 属于 L45。L37 尚未解锁。

建议：

```text
田中：天気が よければ、長城から 見る 景色が きれいでしょうね。
```

这里用的是 L25 已学的 `動小句 + N`。

### ✅ 语法覆盖

除 `上からの景色` 越界外，L37 目标语法基本覆盖：

- `ば`：`行けば / 安ければ / よければ`
- `なら`：`天安門へ行くなら`
- `でも` 提议：`お茶でも買いませんか`
- `とか`：`Tシャツとかキーホルダーとか`

### ⚠ 自然度：`ラーメン屋` 式命名问题这批没有，但专名偏多

L37 使用了大量旅游专名：`長城`、`天安門`、`世界遺産`、`キーホルダー`、`Tシャツ`。作为旅游课文可以接受，但若严格按 CSV 词源控制，需确认是否允许专名和场景词例外。

### 💡 生词表/词源一致性

若严格执行“词汇源仅使用 `resources/jlpt-vocab-n5n4.csv`”，以下 L37 表内词条未能在 CSV 中精确匹配到词条：

- `長城`
- `天安門`
- `世界遺産`
- `キーホルダー`
- `Tシャツ`

`地下鉄`、`景色` 能在 CSV 中匹配到。

## L38

### ⚠ 叙事/人物设定不自然：田中说 `日本に来てから`

位置：`lesson-38.md:89`

原文：

```text
田中：日本に 来て から、辛い 物が 食べられる ように なりました。
```

问题：

田中是从日本来中国旅游/出差的日本朋友，说“来到日本之后变得能吃辣”不符合当前北京游叙事线。

建议：

```text
田中：中国に 来て から、辛い 物が 食べられる ように なりました。
```

或：

```text
田中：上海に 来て から、辛い 物が 食べられる ように なりました。
```

### ⚠ 人物设定不自然：陈明每天听 `中国語のニュース`

位置：`lesson-38.md:91`

原文：

```text
陳：すごいですね。私は 毎日 中国語の ニュースを 聞く ように して います。
```

问题：

陈明在 L01 已设定为中国人。作为日语学习/接待田中的主角，说自己每天听中文新闻作为努力目标，不太符合人物设定和学习线。更自然的是日语新闻。

建议：

```text
陳：私は 毎日 日本語の ニュースを 聞く ように して います。
```

### ⚠ 自然度：`上手になれるように`

位置：`lesson-38.md:93`

原文：

```text
田中：日本語が もっと 上手に なれる ように、毎日 練習して いますか。
```

问题：

`上手になれるように` 语法上可解释为“为了能够变得更擅长”，但日语自然度偏低。这里更自然的是非意志变化 `上手になるように`。

建议：

```text
田中：日本語が もっと 上手に なる ように、毎日 練習して いますか。
```

### ✅ 语法越界检查

未发现明确硬越界。可能形、`ように`、`ようになります`、`ようにしています` 都属于 L38 范围。

### ✅ 语法覆盖

L38 四个语法点都有覆盖：

- 可能形：`歩けます / 通り抜けられます / 撮れます / 食べられる`
- `ように` 目的：`忘れないように`
- `ようになります`：`食べられるようになりました`
- `ようにしています`：`聞くようにしています`

建议修自然度时保留这些覆盖。

### 💡 生词表/词源一致性

若严格执行 CSV 词源，以下 L38 表内词条未能在 CSV 中精确匹配到词条：

- `胡同`
- `通り抜けます`

`曲がる`、`メモ` 能在 CSV 中匹配到。

## L39

### ❌ 语法越界：被动 `建てられた`

位置：`lesson-39.md:81`

原文：

```text
田中：この 建物は 600年前に 建てられたんですよ。
```

问题：

`建てられた` 是被动形式，根据 `grammar-unlock.md`，被动在 L41 才解锁。L39 尚未解锁。

建议：

```text
田中：この 建物は 600年前に できたんですよ。
```

或：

```text
田中：これは 600年前の 建物なんですよ。
```

### ❌ 语法覆盖缺失：`ていきます`

位置：`lesson-39.md:91`, `lesson-39.md:116`

覆盖清单写：

```text
ていきます/てきます（買ってきます/戻ってきます/暑くなってきた）
```

问题：

课文实际只有 `てきます / てきました`：

- `買ってきます`
- `戻ってきます`
- `暑くなってきました`

没有真正出现 `ていきます`。生词表中 `～ていきます` 也没有 `📖`，说明正文确实未覆盖。

建议补一句：

```text
陳：人が 中へ 入って いきますね。
```

或：

```text
田中：鳥が 遠くへ 飛んで いきますね。
```

### ⚠ 语法覆盖不严：`ないで` 伴随状态

位置：`lesson-39.md:69`, `lesson-39.md:91`

原文：

```text
陳：かばんを 持って 入って ください。傘は 持たないで ください。
```

问题：

`かばんを持って入ってください` 可以覆盖 `て` 伴随状态。
但 `傘は持たないでください` 是 L19 已学的”请不要……”，不是本课 `ないで + 小句2` 的伴随状态。

建议：

```text
陳：傘を 持たないで 入って ください。
```

或：

```text
田中：傘を 持たないで 中に 入ります。
```

### ⚠ 覆盖建议：原因 `ために` 只有名词接续

位置：`lesson-39.md:77`, `lesson-39.md:81`, `lesson-39.md:91`

原文：

```text
霧の ために、遠くが よく 見えませんね。
火事の ために 2回 焼けて しまいましたが。
```

问题：

这两句都是 `Nのために`。L39 语法点说明中还包括 `小句簡体形 + ために`、`二类形容词 + なために`。如果希望课文覆盖更完整，可补一个小句接续。

建议：

```text
霧が 濃い ために、遠くが よく 見えませんね。
```

### ⚠ 自然度：句末 `が` 悬空

位置：`lesson-39.md:81`

原文：

```text
火事の ために 2回 焼けて しまいましたが。
```

问题：

句末 `が` 可以表示铺垫，但这里后续没有接出转折或补充，略显悬空。初级课文建议写完整。

建议：

```text
火事の ために 2回 焼けて しまいました。
```

### ✅ 叙事连续性

L39 在故宫游览，承接 L37-L38 北京游路线，没有发现人物或时间线硬矛盾。

### 💡 生词表/词源一致性

若严格执行 CSV 词源，以下 L39 表内词条未能在 CSV 中精确匹配到词条：

- `故宮`
- `火事`
- `霧`
- `焼けます`

`戻る` 能在 CSV 中匹配到。

## L40

### ⚠ 语法覆盖缺失：`ているところです`

位置：`lesson-40.md:98`, `lesson-40.md:100`, `lesson-40.md:108`, `lesson-40.md:122`

L40 第 1 个语法点明确列了三种阶段：

```text
基本形 + ところです
ている + ところです
た形 + ところです
```

课文目前有：

- `着いたところです`：刚做完
- `出かけるところです`：即将做
- `出るところで`：即将做的时间点，但不是 `ところです`

缺少 `ているところです`。

建议补一句：

```text
陳：今、チケットを 買って いる ところです。
```

### ⚠ 覆盖偏弱：`笑い出す` 只在舞台说明中出现

位置：`lesson-40.md:112`, `lesson-40.md:122`

原文：

```text
田中：本当ですか。（笑い出す）陳さんの 趣味は 変わり続けますね。
```

问题：

`笑い出す` 的确能展示 `出します`，但它出现在括号舞台说明里，不是对话句。若要求“课文对话覆盖语法点”，这个覆盖偏弱。

建议：

```text
田中：本当ですか。急に 笑い出して すみません。
```

或：

```text
佐藤：田中さんが 笑い出しましたね。
```

### ✅ 语法越界检查

未发现明确硬越界。`ところです`、`たばかりです`、`始めます/出します/続けます/終わります` 都属于 L40 范围；`残っています` 是前面已学 `ています` 结果/状态。

### ✅ 叙事连续性

L40 继续北京游，晚上看京剧，田中与佐藤出现也不矛盾；佐藤此前在 L23-L24 东京线出现过，作为电话/同行人物回归可以接受。

### ⚠ 生词表 `📖` 标记边界

位置：`lesson-40.md:134`

`公演` 只出现在舞台说明 `（公演後）`，不出现在对话正文。如果 `📖` 统计整个课文含舞台说明，可以保留；如果只统计对话句，应去掉 `📖` 或把它写入对话。

### 💡 生词表/词源一致性

若严格执行 CSV 词源，以下 L40 表内词条未能在 CSV 中精确匹配到词条：

- `京劇`
- `劇場`
- `公演`
- `すばらしい`（CSV 中有例句包含 `すばらしい`，但未能精确匹配为词条）

`興味`、`耳` 能在 CSV 中匹配到。

## 建议修复顺序

1. 先修 L39 被动越界：`建てられた` 改成 `できた / 600年前の建物`。
2. 再修 L37 场景矛盾和 `上からの景色` 越界。
3. 再补 L39 `ていきます` 和 `ないで + 小句2` 的真正覆盖。
4. 再修 L38 人物设定不自然：`日本に来てから`、`中国語のニュース`、`上手になれるように`。
5. 再补 L40 `ているところです`，并把 `笑い出す` 从舞台说明改成对话句。
6. 最后按是否严格执行 CSV 词源，统一处理专名和未精确匹配 CSV 的词条。
