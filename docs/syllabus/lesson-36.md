## 语法点

### 1. 小句1 て/で、小句2（原因・理由）

- **含义**：因为……所以……（用て/で连接表示原因理由）
- **接续**：
  - 动词て形 + 小句2
  - 一类形容词「い」→「くて」+ 小句2
  - 二类形容词/名词 + で + 小句2
- **要点**：
  - 与「から」「ので」不同，后句不能是命令/请求句；后句多为自然结果、状态、感情等非意志性表达
  - 动词：遅くなって、すみません
  - 一类形容词：頭が痛くて、寝ていました
  - 二类形容词：説明が上手で、よく分かりました
  - 名词：雪で、電車が止まりました
- **例句**：
  - 言葉が通じなくて、とても困りました。（语言不通，非常为难。）
  - 部屋が暗くて、本が読めません。（房间太暗，看不了书。）
  - 火事で、家が焼けてしまいました。（因为火灾，房子烧毁了。）
- **Bunpro 校验**：待验证

### 2. 名詞＋に（用途/基准）

- **含义**：在某种用途上使用 / 以某种基准来评价
- **接续**：名词 + に + 動詞/形容詞
- **要点**：
  - 用途：「何に使いますか」→「パスポートの申請に使います」
  - 基准：「買い物に便利です」「子供には難しいです」
  - 后面常接「使います」「便利です」「いいです」「役立ちます」等
- **例句**：
  - この写真はビザの申請に使います。（这张照片申请签证用。）
  - この辺はスーパーが近くて、買い物にとても便利です。（这附近有超市，购物非常方便。）
  - この問題は大人には簡単ですが、子供には難しいです。（这道题对大人来说简单，对小孩来说难。）
- **Bunpro 校验**：待验证

### 3. 動詞（基本形）のに（用途/基准）

- **含义**：用于做某事 / 对做某事来说
- **接续**：动词基本形 + のに + 動詞/形容詞
- **要点**：
  - 是「名词＋に」的动词版本
  - 用途：この道具は荷物を運ぶのに使います
  - 基准：このカメラは持ち歩くのに便利です
  - 后面常接「使います」「便利です」「いいです」「かかります」等
- **例句**：
  - このはさみは紙を切るのに使います。（这把剪刀用来剪纸。）
  - この本を書くのに3年かかりました。（写这本书花了三年。）
  - このマンションは通勤するのにとても便利です。（这个公寓对通勤来说非常方便。）
- **Bunpro 校验**：待验证

### 4. 名詞＋ばかり / 動て形＋ばかりいます

- **含义**：尽是……、光……（反复同一事物/动作，含"不太理想"的语感）
- **接续**：
  - 名词 + ばかり + 動詞
  - 动词て形 + ばかりいます
- **要点**：
  - 名词＋ばかり：列举的事物全部相同（野菜ばかり食べる＝光吃蔬菜）
  - 動て形＋ばかりいます：总是做同一件事（飲んでばかりいます＝整天光喝酒）
  - 与「だけ」的关键区别：ばかり含"反复"和"不太理想"的语感，だけ纯粹客观限定
  - 与「しか」不同，ばかり不要求后续否定，但可以出现在否定句中（遊んでばかりいないで）
- **例句**：
  - 弟はゲームばかりしていて、全然勉強しません。（弟弟光打游戏，完全不学习。）
  - 甘いものばかり食べていると、太りますよ。（光吃甜食的话会胖的。）
  - 最近、残業してばかりいます。（最近尽是在加班。）
- **Bunpro 校验**：待验证

### 5. 小句（动词普通形）のが＋見えます/聞こえます

- **含义**：能看到/听到某件事正在发生
- **接续**：小句（动词普通形，含ている/ていた等） + のが + 見えます/聞こえます
- **要点**：
  - 見えます：自然地映入眼帘（客观能力/距离）
  - 聞こえます：自然地传入耳中（客观能力/距离）
  - 前接「の」将小句名词化
  - 一类形容词＋のが、二类形容词＋な＋のが
- **例句**：
  - 窓から子供たちが遊んでいるのが見えます。（从窗户能看到孩子们在玩。）
  - 隣の部屋で誰かがピアノを弾いているのが聞こえます。（听到隔壁房间有人在弹钢琴。）
  - 遠くで花火が上がっているのが見えました。（看到远处在放烟花。）
- **Bunpro 校验**：待验证

## 课文

> Arc 9「接待朋友·上」第4话 — 逛菜市场

**登場人物**：陳明、田中 — 上海の市場

> 使用语法：≤L36（+て/で原因、Nに用途、動のに用途、ばかり、のが見える/聞こえる）

---

陳：言葉が 通じなくて、困りませんか。

田中：大丈夫ですよ。陳さんが いますから。

陳：このはさみは 紙を 切るのに 使います。あの 包丁は 野菜を 切るのに 使います。

田中：なるほど。あ、遠くで 子供が 遊んで いるのが 見えますね。

陳：市場は にぎやかですね。いろいろな 声が 聞こえます。

田中：最近、肉ばかり 食べて いるので、今日は 野菜を 買いましょう。

陳：田中さんは 肉ばかり ですか。体に 悪いですよ。

田中：分かって います。この 市場は 買い物に 便利ですね。

陳：はい、新鮮で、値段も 安いですから。

---

**语法覆盖**：✅ て/で原因（通じなくて困る） ✅ Nに用途（買い物に便利） ✅ 動のに（切るのに使います×2） ✅ ばかり（肉ばかり食べている） ✅ のが見える/聞こえる（遊んでいるのが見える/声が聞こえる）

**📋 产出任务：带外国朋友逛菜市场**

- **场景**：你带一位刚到中国的外国朋友去逛当地的菜市场。朋友因为语言不通而发愁，你安慰她说这里买东西很方便。你们一路走，看到各种摊位，听到各种声音。朋友抱怨自己总是买同样的东西，你介绍某些工具的用途。途中你们站在高处远眺，看到远处有人在放风筝。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：

    - ✅ 名詞＋に（用途/基准）或 動詞のに（用途/基准）

- **词汇限制**：仅使用 lesson ≤ 36 的词汇
