## 语法点

### 1. 小句1（动词基本形/ない形）と、小句2（と条件：恒常/真理）

- **含义**：一……就……（表示恒常性状态、真理、反复性状态或发现新情况）
- **接续**：动词基本形/ない形 + と、小句2
- **要点**：
  - 用于恒常/真理/反复：ボタンを押すと、電源が入ります
  - 也可用于"由于某行为发现新情况"：窓を開けると、山が見えました
  - 小句2不能是意志/愿望/祈使/劝诱等表达
  - × 夜になると、食事に行きましょう
- **例句**：
  - 春になると、桜が咲きます。（一到春天，樱花就开了。）
  - この薬を飲むと、すぐ眠くなります。（一吃这个药，就会马上犯困。）
  - まっすぐ行くと、右に郵便局があります。（一直走的话，右边有邮局。）
- **Bunpro 校验**：待验证

### 2. 动（基本形/ない形）ことがあります（偶尔发生）

- **含义**：有时会……/偶尔会……
- **接续**：动词基本形/ない形 + ことがあります
- **要点**：
  - 常与「たまに」「時々」呼应
  - 注意与「～たことがあります」（经验）完全不同
  - 基本形：時々寝坊することがあります（有时会睡懒觉）
  - ない形：たまに朝ご飯を食べないことがあります（偶尔不吃早饭）
- **例句**：
  - たまに電車で寝過ごすことがあります。（偶尔会在电车上坐过站。）
  - 忙しい時、昼ご飯を食べないことがあります。（忙的时候偶尔不吃午饭。）
  - このパソコンは時々動かなくなることがあります。（这台电脑有时候会不动了。）
- **Bunpro 校验**：待验证

### 3. 形容词的副词用法

- **含义**：用形容词修饰动词（像副词一样使用形容词）
- **接续**：一类形容词「い」→「く」/ 二类形容词 +「に」
- **要点**：
  - 一类：早い→早く、大きい→大きく、美しい→美しく
  - 二类：上手→上手に、静か→静かに、きれい→きれいに、丁寧→丁寧に
  - 早く帰りましょう（早点回去吧）
  - 静かにドアを閉めてください（请轻轻关门）
- **例句**：
  - もう少し大きく書いてください。（请再写大一点。）
  - 田中さんは上手に日本語で説明しました。（田中先生用日语做了出色的说明。）
  - 子供たちは元気に遊んでいます。（孩子们精神十足地在玩。）
- **Bunpro 校验**：待验证

### 4. 小句（简体形）でしょうか（礼貌疑问）

- **含义**：……吗？（比「～ますか」更礼貌的疑问表达）
- **接续**：动词/一类形简体形 + でしょうか；二类形/名词简体形去「だ」+ でしょうか
- **要点**：
  - 带有不确定、委婉探问的语气，比「～ますか」更礼貌
  - 二类形容词/名词去掉「だ」：中国のお菓子でしょうか
  - 常用于向不太熟的人提问
- **例句**：
  - すみません、この近くに薬局はあるでしょうか。（请问，这附近有药店吗？）
  - 明日の会議は何時に始まるでしょうか。（明天的会议几点开始？）
  - 森さんはもう出発したでしょうか。（森先生已经出发了吗？）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| きちんと | 表示正确、无误、恰当：きちんと片づけてください（请彻底打扫干净） |
| ～でできた～ | 表示"用……做成的"：コンクリートでできた卓球台（混凝土做的乒乓球台） |
| ただし | 基本肯定前句，附加条件/限制：だれでも利用できます。ただし、有料ですが |
| ほど vs ぐらい | 都表大致数量，但表序列中的某一点时只能用「ぐらい」：30歳ぐらいから |

- **场景**：你和朋友散步时路过一个体育中心和周边设施。你向朋友介绍这些设施的使用规则和条件（恒常条件），聊到自己偶尔使用的经历，以及向路人礼貌地问路。描述时注意使用形容词的副词用法让表达更生动。
