## 语法点

### 1. 可能形式

- **含义**：能够做……（表示具备做某动作的能力或条件）
- **接续**：
  - 一类动词：基本形末音变为「え」段 + る（書く→書ける）
  - 二类动词：基本形去「る」+ られる（食べる→食べられる）
  - 三类动词：来る→来られる、する→できる
- **要点**：
  - 可能形式的活用方式与二类动词相同
  - 对象助词「を」可变为「が」，两者都可用（日本語が話せます / 日本語を話せます）；「できる」的对象通常用「が」
  - 与「～ことができます」意思相同，可能形式更口语化
  - 口语中二类动词常省略「ら」（ら抜き言葉）：食べれる、見れる（非正式）
- **例句**：
  - 田中さんは中国語が読めます。（田中先生会读中文。）
  - すみません、ここで写真が撮れますか。（请问这里能拍照吗？）
  - この店では予約なしでも入れますよ。（这家店不预约也能进。）
- **Bunpro 校验**：待验证

### 2. 小句（基本形/ない形）ように、小句（目的）

- **含义**：为了使某种状态成立（前句为非意志性变化/可能形式）
- **接续**：
  - 非意志动词基本形/ない形 + ように
  - 意志动词可能形式基本形/ない形 + ように
- **要点**：
  - 与「ために」的区别：ために前接意志动词，ように前接非意志/可能形式
  - ように前后主语可以不一致，ために前后主语必须一致
  - 常见搭配：見えるように、聞こえるように、忘れないように、風邪を引かないように
- **例句**：
  - よく聞こえるように、大きな声で話してください。（为了能听清楚，请大声说。）
  - 忘れないように、メモしておきましょう。（为了不忘记，记个备忘吧。）
  - 子供が触れないように、薬は高い所に置きました。（为了不让孩子碰到，把药放在了高处。）
- **Bunpro 校验**：待验证

### 3. 小句（基本形/ない形）ようになります

- **含义**：变得能够……/变得不再……（能力、状况、习惯的变化）
- **接续**：动词基本形/ない形/可能形式 + ようになります
- **要点**：
  - 表示从"不能"到"能"或从"能"到"不能"的变化过程
  - 常用于描述能力的获得：歩けるようになりました
  - 也用于习惯的变化：野菜を食べるようになりました
  - 反方向的变化：「食べなくなりました」＝不再吃了（习惯变化）；「食べられなくなりました」＝变得不能吃了（能力丧失）——两者含义不同
- **例句**：
  - 毎日練習したので、やっと漢字が書けるようになりました。（因为每天练习，终于能写汉字了。）
  - 赤ちゃんが歩くようになりました。（宝宝会走路了。）
  - 日本に来て、刺し身が食べられるようになりました。（来了日本之后变得能吃生鱼片了。）
- **Bunpro 校验**：待验证

### 4. 小句（基本形/ない形）ようにします

- **含义**：努力做到……/注意（不）……（有意识地使行为成为习惯）
- **接续**：意志动词基本形/ない形 + ようにします/ようにしています
- **要点**：
  - 「～ようにしています」：已经成为习惯，平时一直注意在做
  - 「～ようにします」：从现在开始要努力做到
  - 常用于健康、生活习惯类话题
- **例句**：
  - 毎朝7時に起きるようにしています。（我坚持每天早上7点起床。）
  - 甘いものをあまり食べないようにしています。（我尽量不吃甜食。）
  - 今日からエレベーターを使わないようにします。（从今天起不坐电梯了。）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| 入り組んでいます | 形容道路或事物错综复杂、曲里拐弯 |
| なんだか | "总觉得……"：理由不明确的主观感受（なんだか残念です） |
| 見えます vs 見られます | 見えます＝客观能力/距离（看得到），見られます＝人为条件（可以看到） |
| ら抜き言葉 | 二类动词可能形式口语中常省「ら」：食べれる、見れる（非正式用法） |

**📋 产出任务：胡同散步与健康生活分享**

- **场景**：你和一位外国朋友走在北京的胡同里。朋友问这条小巷能不能走通，你说虽然路弯弯曲曲但能走通。你提醒朋友注意不要拍照、注意别烫伤。你们聊到各自的健康习惯——你坚持每天做什么、注意不做什么。朋友说来了中国之后变得能吃很多以前不敢吃的东西了。
- **要求**：
  - 8-10 句
  - 必须覆盖以下全部语法点：
    - ✅ 可能形式
    - ✅ ～ように（目的）
    - ✅ ～ようになります（变化）

- **词汇限制**：仅使用 lesson ≤ 38 的词汇
