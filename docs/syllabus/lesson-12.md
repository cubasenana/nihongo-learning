## 语法点

### 1. 名1は 名2より 一类形 / 二类形 です

- **含义**：比较两者，名1比名2更具有某种性质
- **接续**：名1 + は + 名2 + より + 形容词 + です
- **要点**：
  - 名1 是被描述的主体，名2 是比较基准
  - 程度差异大时加副词「ずっと」强调
  - 适用于一类形和二类形
- **例句**：
  - 電車はバスより速いです。（电车比公交快。）
  - 夏は冬よりずっと暑いです。（夏天比冬天热得多。）
- **Bunpro 校验**：待验证

### 2. 名1より 名2のほうが 一类形 / 二类形 です

- **含义**：与名1相比，名2更具有某种性质
- **接续**：名1 + より + 名2 + のほうが + 形容词 + です
- **要点**：
  - 「のほうが」强调名2一侧
  - 与句型1的区别在于哪个是被突出的对象
  - 「ほう」写作「方」
- **例句**：
  - バスより電車のほうが便利です。（比起公交，电车更方便。）
  - 紅茶よりコーヒーのほうが好きです。（比起红茶，更喜欢咖啡。）
- **Bunpro 校验**：待验证

### 3. 名1は 名2ほど 一类形くないです / 二类形ではありません

- **含义**：名1在程度上不及名2，"不像……那么……"
- **接续**：名1 + は + 名2 + ほど + 形容词否定
- **要点**：
  - 句尾必须使用否定形式
  - 表示名1的程度低于名2
  - 一类形用「くないです」，二类形用「ではありません」
  - 注：Bunpro 将此语法标为 N3，标日初级已提前引入
- **例句**：
  - 今日は昨日ほど寒くないです。（今天没有昨天那么冷。）
  - この店はあの店ほど有名ではありません。（这家店没有那家店那么有名。）
- **Bunpro 校验**：待验证

### 4. 名1の中で 名2が いちばん 一类形 / 二类形 です

- **含义**：在某范围内，某事物最具有某种性质（最高级）
- **接续**：名1 + の中で + 名2 + が + いちばん + 形容词 + です
- **要点**：
  - 名1 是范围，名2 是最突出的对象
  - 场所/时间范围不加「の中」，直接用「～で」
  - 「いちばん」通常需要有明示或语境可推断的比较范围
- **例句**：
  - 果物の中でりんごがいちばん好きです。（水果当中最喜欢苹果。）
  - クラスの中で田中さんがいちばん背が高いです。（班里田中个子最高。）
- **Bunpro 校验**：待验证

### 5. 名1と 名2と どちらが 一类形 / 二类形 ですか

- **含义**：询问两者中哪一个更具有某种性质
- **接续**：名1 + と + 名2 + と + どちらが + 形容词 + ですか
- **要点**：
  - 回答用「～のほうが」
  - 两者相同时用「どちらも」
  - 口语中「どちら」可说成「どっち」
- **例句**：
  - 犬と猫とどちらが好きですか。——猫のほうが好きです。（狗和猫哪个更喜欢？——更喜欢猫。）
  - 東京と大阪とどちらが大きいですか。——どちらも大きいです。（东京和大阪哪个更大？——都很大。）
- **Bunpro 校验**：待验证

### 6. どの名 / いつ / だれが いちばん 一类形 / 二类形 ですか

- **含义**：在三个以上事物中询问最突出的是哪一个
- **接续**：疑问词（どの/いつ/だれ/何）+ が + いちばん + 形容词 + ですか
- **要点**：
  - 人→だれ、物→どれ/どの/何、时间→いつ
  - 选项已知用「どの/どれ」；范围笼统用「何が」
  - 回答重复范围 +「が いちばん～です」
- **例句**：
  - どの季節がいちばん好きですか。——秋がいちばん好きです。（哪个季节最喜欢？——最喜欢秋天。）
  - 家族の中でだれがいちばん早く起きますか。（家里谁起得最早？）
- **Bunpro 校验**：待验证

## 表达要点

| 要点 | 说明 |
|------|------|
| やっぱり/やはり | "果然、还是"——经过考虑后确认某个想法 |
| ～がいいです | 根据语境可表示"我要某种～"或选择偏好 |
| ～が…… | 用「が」收尾不说完，语气比直说更含蓄委婉 |
| 最近 | 表示从较近的过去到现在的一段时间，不能用于将来 |
